        季雅珊身心一怔，踱步上前，一把从后抱住了十爷。
<br/>
<br/>        她纤瘦的双臂紧紧地环住了他的腰身，感受着他身上那温厚的气息，让她十分安心。
<br/>
<br/>        季雅珊斩钉截铁地回答道：“不！我不走！阿十在哪，我就在哪。”
<br/>
<br/>        “我们之间的契约，已经……”
<br/>
<br/>        “我是心甘情愿的。”季雅珊再次打断了十爷的话，直到十爷沉默，她依旧抱着他不肯放手，“我知道你忘不了婴宁，但是我不介意，我会等，哪怕等一辈子都愿意。”
<br/>
<br/>        “随你。”十爷掰开了季雅珊环在自己腰际上的双手，转过身去，接着说道，“你早点去休息。”
<br/>
<br/>        季雅珊微笑着点了点头。
<br/>
<br/>        十爷连夜带着童筱颖离开了古堡，季雅珊送走他们后，转身遇到了古堡里的制药学徒何明旭。
<br/>
<br/>        这个男人是最近才被八爷的首席制药师带进来的关门弟子，听说是临海大学刚毕业，学的是生化专业，是学校里的尖子生，能力很强。
<br/>
<br/>        听说这个何明旭三年前在学校里沾了命案，是八爷背后的那个大人物保了他，才让他毕业后得意为他们效命。
<br/>
<br/>        其实，季雅珊对这里的事情，只是知道点皮毛，听说他们私底下在制药，各种药都有，“夜惑”里的药，便是从古堡里出去的。
<br/>
<br/>        “雅珊，刚刚那个女人跟十爷是什么关系？”何明旭凝眉问道。
<br/>
<br/>        季雅珊见何明旭仍旧穿着白马褂，淡淡地笑了笑：“何药师不去制药，反倒跑来八卦起爷的私事呢？”
<br/>
<br/>        “我刚下班，准备去休息了。”何明旭答非所问。
<br/>
<br/>        季雅珊咧嘴道：“那何药师好好休息，我也先去休息了。”
<br/>
<br/>        “好。”何明旭点了点头，看着季雅珊上楼去后，不由地皱起了眉头。
<br/>
<br/>        现在是什么情况？
<br/>
<br/>        童筱颖一觉醒来后，发现自己现在居然在一个完全陌生的房间里。
<br/>
<br/>        这里不是那座古堡的阁楼，而是……酒店套房！
<br/>
<br/>        童筱颖猛然坐起身来。
<br/>
<br/>        “终于睡醒了？”耳边忽然响起一个熟悉得不能再熟悉的声音。
<br/>
<br/>        童筱颖惊愕地偏过头去，看到睡在自己身边的男人是傅昼景时，顿时脸色都变了。
<br/>
<br/>        “好久没抱着你睡了，没想到，自己还能有机会抱着你一起睡觉。”傅昼景一边坐起身来，一边苦笑道。
<br/>
<br/>        童筱颖吓得本能地抱着被子往后退。
<br/>
<br/>        傅昼景见她快要退到床边上，即将要摔下去了，连忙伸手捉住她的手腕，将她给搂了回来。
<br/>
<br/>        一头撞入了傅昼景的怀中，童筱颖心里莫名其妙地对他产生了抗拒感。
<br/>
<br/>        她刚要推开他，他却使力拥她入怀：“筱颍，我真的很意外，没想到你在外人面前，竟然仍旧承认我是你丈夫。其实，你不恨我。对不对？”
<br/>
<br/>        外人？！
<br/>
<br/>        难道是十爷将她送回来的？！
<br/>
<br/>        童筱颖心里思忖了片刻，收回思绪时，再次推开了傅昼景。
<br/>
<br/>        他从她的眼神里看到了对自己的淡漠，心口腾地像被绞过一样痛着。
<br/>
<br/>        “我要走了！”童筱颖毫不犹豫地掀开被褥下床，她身上的衣服都还在，其实她也不担心傅昼景会对她趁人之危。
<br/>
<br/>        他虽然出轨在先，但是至少在这方面，他是尊重她的。
<br/>
<br/>        “筱颍，那个送你回来的男人跟我说，让我带着你远离我大哥。所以，筱颍不要回到大哥身边去了，也不要去利用大哥了，很危险。”傅昼景曲起长腿，手肘撑在膝盖上，手背托腮，凝望着童筱颖的背影说道。
<br/>
<br/>        童筱颖忽然间想到了什么，转过身去，坐回床边，问道：“你跟我提起过，大哥以前是特种兵，对吗？大哥以前是哪方面的特种兵？他是不是得罪过什么人？还有，他……”
<br/>
<br/>        “筱颍是爱上我大哥了吗？”傅昼景眸光黯然地问，顿了顿后，不禁感慨道，“筱颍之前对大哥的事情似乎一点都不感兴趣了。”
<br/>
<br/>        “你难道就一点都不担心大哥的安危吗？你们是亲兄弟啊！”童筱颖凝眉质问。
<br/>
<br/>        傅昼景却冷冷一笑：“如果，大哥没有娶你。我跟他就还算是亲兄弟，但是他娶了你。我跟他就已经不是亲兄弟了！”
<br/>
<br/>        “这件事情，是我的错。跟大哥……”
<br/>
<br/>        “筱颍，我们重新开始，好不好？”傅昼景再次打断童筱颖的话，放下手来，轻轻地拉起了童筱颖的手。
<br/>
<br/>        童筱颖毫不犹豫地便甩开了他的手，但是在甩手的那一瞬间，一不小心，指甲划到了他的脸颊上，在他脸颊上留下了一线红印。
<br/>
<br/>        傅昼景生疼地呲了呲牙。
<br/>
<br/>        童筱颖下意识地说了声：“对不起。”
