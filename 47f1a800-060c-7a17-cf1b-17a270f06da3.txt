　　艾萨克盯着大屏幕上快速突进的两人，脸上满是阴沉。
<br/>　　“博士，他们已经到门外了！”助理满头冷汗提醒。
<br/>　　艾萨克非常愤怒，不是因为爱丽丝闯入基地，也不是因为爱丽丝沿途杀了多少守卫。让他愤怒的是，本应该仓皇而逃的爱丽丝，竟敢大摇大摆的攻打基地！
<br/>　　他打开通话频道，盯着大屏幕上的爱丽丝：“爱丽丝，本来我还想和你好好合作，但现在，我更想把你放在试验台上解剖！就像你的那些克隆体！”
<br/>　　“放新型丧尸！”艾萨克对小助理狠狠道。
<br/>　　小助理立刻开始操作手中平板。
<br/>　　门外，走廊两侧光滑墙壁，突然变得透明。一个个透明关押室展现在两人面前。一群穿着病号服的丧尸紧贴屏幕，满是贪婪的看着他们。
<br/>　　和外面的丧尸不同，这些丧尸的身体腐烂程度很低，只有不同程度的萎缩或者畸变，眼中布满青黑血丝，竟有一种灵动感觉。
<br/>　　“看到了吗，这是我的研究成果。相比那些只知道进食的死物，他们的身体更有活性，速度更快，力量更大！”艾萨克很满意两人的表情。
<br/>　　外面的丧尸就是死人，时间长了会腐烂枯朽，即便腐烂速度非常慢，但终究会死。眼前这些，不仅不会继续腐烂，反而在渐渐恢复活性！
<br/>　　“爱丽丝，我再给你一次选择机会。要么和我合作，要么被这些丧尸咬死！”艾萨克好整以暇。
<br/>　　“博士……”爱丽丝脸色难看。只是看着这些丧尸就知道不好对付，真要被围起来，杀都杀不过来。
<br/>　　“博士？他也配称博士！任何一所大学都不会承认他的博士学位！也从未给他过博士学位！你竟然会称呼他博士？这个笑话我可以笑一年。”艾萨克忽然有点儿愤愤不平，你说我提供那么好的条件，你就是不合作。反而整天和这个小年轻混在一起，还博士？什么意思，看不起我保护伞大boss还是咋地。
<br/>　　“我决定了，你们还是躺在试验台上比较好。开门，放丧尸！”艾萨克恶狠狠道。
<br/>　　透明玻璃门刷的一下就打开，满屋子的新型丧尸就像猛虎出笼，一下冲向苏重两人。
<br/>　　艾萨克笑吟吟的看着被包围起来的两人，对身旁小助理得意道：“两个人就敢闯我的基地，这下我看他们怎么……人呐！”
<br/>　　硕大的显示屏上，只有乌泱泱一片丧尸，爱丽丝和苏重两个人竟消失不见！
<br/>　　“人呐！”艾萨克有些蒙，这些丧尸虽然速度力量更强，但也不能吃的这么快啊。
<br/>　　正纳闷呢，一个硕大人脸骤然出现在屏幕上，把艾萨克吓的心脏差点儿跳出来。
<br/>　　人脸慢慢向后移动，视线扩大，然后就看到爱丽丝一脸惬意的横躺在半空。
<br/>　　“傻了吧？老娘我会飞！”
<br/>　　艾萨克：“……”人怎么能会飞？这不科学！
<br/>　　苏重脸皮直抽抽，这是觉醒了逗逼属性了吗？
<br/>　　“你怎么也会飞！”艾萨克觉得天都变了。爱丽丝能飞，他还能理解。毕竟爱丽丝是病毒完美融合体，可这个名不见经传的亚裔是怎么回事。
<br/>　　不对呀，这家伙不是和爱丽丝闹掰了吗，怎么两人还会合作闯基地，而且还他么能飞！
<br/>　　他造的变异丧尸确实厉害，几乎有着常人的力量和速度，而且数量众多，走的是蚁多咬死象的路子。可人家他么飞到天上，你就是蹦起来都捞不着人家，那还怎么咬？！
<br/>　　苏重不理爱丽丝搞怪，决定先办正事，虽然看着艾萨克这老家伙气急败坏的样子挺乐呵的。
<br/>　　飘到门前，苏重把念力化作一个大锤，狠狠砸在大门上。
<br/>　　轰！
<br/>　　一声巨响，超强硬度的合金大门上就出现一个弧形凹陷。
<br/>　　艾萨克当时就吸了一口冷气。
<br/>　　爱丽丝对着监控器笑嘻嘻：“你的丧尸不是厉害吗，等门开了，你自己也试试。”
<br/>　　艾萨克脸立刻就白了。新型丧尸是他发明的，他当然知道丧尸的厉害。不知有多少研究员，被这些暴躁丧尸生吞活剥。想想他们的下场艾萨克就不寒而栗。
<br/>　　阴冷等着爱丽丝片刻，艾萨克声音沙哑：“这里不能呆了。”
<br/>　　“你留下来挡住他们，我去去就来！”艾萨克一脸阴狠的等着小助理。
<br/>　　这里是地下核心，只有眼前一条离开的路，路上还被一大群丧尸堵着，他要舍命一搏。
<br/>　　小助理抱着平板欲哭无泪，让我挡住，那么多丧尸冲过来，我挡得住吗？
<br/>　　注视着艾萨克顺着一个小门走出房间，小助理干脆利落的拿起通话器：“我投降！”
<br/>　　苏重：“……”还有这种好事。
<br/>　　有小助理配合，苏重爱丽丝两人利用念力，把新型丧尸赶回囚禁室。然后打开大门走进监控室。
<br/>　　“艾萨克去哪里？”苏重看着战战兢兢的小助理闻到。
<br/>　　“他去了基地控制核心！”小助理果断出卖艾萨克。实在是这老家伙不拿人命当回事，已经有不少研究员被老家伙拿去喂丧尸了。他这小助理天天当的提心吊胆。一个是自己死一个是老家伙死，他当然不会犹豫。
<br/>　　顺着小门走过一段螺旋楼梯，一片巨大空间出现在苏重眼前。
<br/>　　整个空间一片纯白，苏重一眼就看到站在地下空间深处的艾萨克。
<br/>　　“爱丽丝已经让我惊喜，没想到还有你这个完美样品自动上门。今天真是个好日子。”艾萨克低沉道。
<br/>　　苏重：“……”老家伙这是没睡醒吧，在这自言自语什么呢。我都打到你的老巢了，你还在这里口放狂言，迷之自信。
<br/>　　“放心，我会好好利用你们的身体。”艾萨克砖头过来，一张嘴裂开到耳根，手臂一摆伸出一根两根触须，满脸都是癫狂。
<br/>　　苏重当时就倒吸一口冷气。
<br/>　　这老家伙真他么变态，竟然敢拿自己做实验，把自己变成这么个不人不鬼的怪物！
<br/>　　“去死吧！”老家伙一声大吼，猛然冲向苏重。
<br/>　　速度之快，竟然只能看到残影！
<br/>　　轰！
<br/>　　一声巨响，血浆飞溅，艾萨克来的快，去的更快。
<br/>　　一张无形屏障出现在苏重身前。艾萨克一头装上去，脸都撞扁啦！
<br/>　　这老家伙爬起来，扁了的脸竟然快速恢复，然后满脸狞笑的看着苏重：“没想到吧，我这是不死之……”
<br/>　　轰！
<br/>　　苏重念力化作大手，一把抓起艾萨克狠狠摔在地上。
<br/>　　“没用的，你摔不死……”
<br/>　　轰轰！
<br/>　　“你找死……”
<br/>　　轰轰轰！
<br/>　　艾萨克：“……”
<br/>　　轰轰轰……噗嗤。
<br/>　　一摊肉泥瘫在地上。
<br/>　　“一下摔不死，那就二十下，摔成肉泥我看你死不死！”苏重有念力在，简直就是作弊。
<br/>　　根本不用和艾萨克近身，抡起艾萨克就往地上砸。别看他变身怪物挺吓人，但对苏重是真心没用！
