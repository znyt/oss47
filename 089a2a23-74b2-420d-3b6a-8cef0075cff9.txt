        傅夜沉不以为然地冷哼：“干爹，如果筱颖真是您的女儿，或许，我会替她感到高兴。只可惜了……”
<br/>
<br/>        夏阳枭身心一怔，淡定自若地接着说道：“我说筱颖是我女儿，那么她就是我的女儿。我女儿为你而死，我好不容易派人将她的命给救回来，我希望你放过她。”
<br/>
<br/>        他说这话就像一个老父亲在替自己的女儿说情一样。
<br/>
<br/>        傅夜沉冷冷一笑：“干爹是在转移话题吗？不过，到底是我不放过她，还是您不放过她，干爹心里应该比我更清楚。”
<br/>
<br/>        夏阳枭此刻沉默了。
<br/>
<br/>        傅夜沉竟然能做到在神不知鬼不觉地冒充他的司机，那就证明，他傅夜沉一定还查到了些其他东西，不然，他也不会如此胸有成竹地主动来找他。
<br/>
<br/>        片刻后，夏阳枭终于放低了自己的姿态，和傅夜沉谈起了条件：“筱颖拿走了实验室里所有的解药，只要她把解药全部交出来，我撤销你的通缉令。”
<br/>
<br/>        “干爹觉得，我来是跟您谈这个的？”傅夜沉剑眉微蹙。
<br/>
<br/>        夏阳枭收起手枪，不再跟傅夜沉拐弯抹角：“我只要解药，以后我和你熄战，撤销你的通缉令。你放了我父亲，我放了你母亲。你解散你的幕后势力，我不再派人抓筱颖回实验室。要知道，你现在有了弱点，是斗不过我的。”
<br/>
<br/>        “成交！希望干爹，言而有信。”傅夜沉淡定地应道，顿了顿后，若有所思地问，“我想知道，干爹是什么时候和冒险者集团联手的？”
<br/>
<br/>        夏阳枭笑了笑：“你都已经不是我的手下了，还关心这个做着什么？”
<br/>
<br/>        “干爹，您还真是阴险狡诈。一边与冒险者集团的冒先生早已暗中结盟，一边又派我们去查他们的底细。”傅夜沉嗤之以鼻。
<br/>
<br/>        查这个集团，他们查了整整三年多的时间，却没有查出个什么所以然来。
<br/>
<br/>        那就只能说明一点，安防局里有冒险者集团的内线。
<br/>
<br/>        这件事情，他和连华生早就心知肚明，只是没料到这内线竟然就他们的老大。
<br/>
<br/>        上次他诈死，是夏阳枭特意安排的。
<br/>
<br/>        夏阳枭很想知道冒险者集团的内部结构，并企图控制这个集团。
<br/>
<br/>        正因为冒险者集团的人知道他傅夜沉是他夏阳枭的干儿子，傅夜沉若是死了，他夏阳枭正好有理由“讨伐”那位冒先生。
<br/>
<br/>        夏阳枭这是一箭双雕。
<br/>
<br/>        “小沉，干爹不得不给你提个醒。有些事情，你若是放不下。你将会再次失去你所有爱的人。”夏阳枭双眼微眯，语重心长地劝说。
<br/>
<br/>        夏阳枭所暗指的“有些事情”其实就是指他傅夜沉父亲的死因。
<br/>
<br/>        傅夜沉一直抓着这件事情不放，夏阳枭心里一清二楚。
<br/>
<br/>        “过去、现在、未来，哪个更重要，我想你不需要我这个当干爹的来提醒。”夏阳枭接着说道。
<br/>
<br/>        傅夜沉按下开锁键，打开了驾驶座的车门：“是，干爹说得很对。希望干爹您今后，能好自为之。”
<br/>
<br/>        “乓”地一声，傅夜沉下车后，反手关上了车门。
<br/>
<br/>        夏阳枭这才暗自松了口气。
<br/>
<br/>        这一天，比之前的任何一天，都要令傅夜沉感到疲惫。
<br/>
<br/>        和夏阳枭谈完判，傅夜沉得知自己的母亲安然无恙地回到傅家大庄园里后，才派人将夏爷爷也安然无恙地送回了夏家。
<br/>
<br/>        一并送去的，还有夏阳枭要的解药。
<br/>
<br/>        不过，傅夜沉只给了夏阳枭一盒。其余的，被他藏了起来，算是给自己和筱颖留一条后路。
<br/>
<br/>        当他回到自己的地下城时，刚好凌晨。
<br/>
<br/>        傅夜沉打开家门，直接去了卧室。
<br/>
<br/>        但是卧室里空无一人，让他身心顿时紧绷。
<br/>
<br/>        “筱颖？筱颖？”傅夜沉一边呼唤着，一边从卧室里找了出去，每个角落都不放过。
<br/>
<br/>        当他找去厨房的时候，身后忽然响起了细微的脚步声，令他微微蹙起了剑眉。
<br/>
<br/>        傅夜沉屏气凝神，当身后的人向自己逼近的那一秒，他敏捷地捉住了两只手臂，刚准备来个过肩摔的时候，闻到这人身上的气息，连忙转过身去抱住。
<br/>
<br/>        因重心不稳，他抱着这个偷袭他的小女人，一起往后跌倒，后背硬生生地撞到了地面上，令脊椎一阵发痛。
<br/>
<br/>        童筱颖迷离的双眼，对上了一双黑钻般的眸子。她趴在傅夜沉的胸膛上，口鼻间全是这个男人温热的气息。
<br/>
<br/>        她身上还穿着他的衣服，宽松的衬衣下，被他搂在怀里的是她那软软的身体。
<br/>
<br/>        傅夜沉胸膛微微起伏着，凝了凝神，连忙抱着童筱颖从地面上坐起来。
<br/>
<br/>        “大哥，我好饿……”童筱颖说得气若游丝。
<br/>
<br/>        傅夜沉收敛了刚刚那异样的思绪，纳闷地反问：“你一天都没吃东西吗？”
