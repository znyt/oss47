　　第438章
<br/>　　天下大乱，诸侯林立。
<br/>　　各路诸侯都在拼命的网罗人才，谋士、杀手、盗贼，鸡鸣狗盗之徒，将其美其名曰门客。江湖草莽有了出路，也未必愿意提着脑袋在江湖上干活。除了一些人天生不愿寄人篱下，大多投在诸侯门下。
<br/>　　将军府有将军府的门客，相府也有相府的门客。这万湖心就是相府的门客，据说此人学富五车，有过目不忘之能，思维缜密得很。
<br/>　　“殿下，也未必是大皇子派来的人，大皇子不一定知道这件事。”万湖心。
<br/>　　“有什么区别么，总要走到这一步的。”李行哉依旧笑着，从脸上表情没办法看出他心里想些什么。
<br/>　　总得来说，明帝的几个儿子互相之间还是很和睦的，从前并没有那种闹得不可开交的事情出现。而李行哉本人，很早就表达了自己对皇位不感兴趣的态度。
<br/>　　但在这张赌局上，要么你就干脆别上去，既然上了赌局，就要有打算输掉全部。
<br/>　　万湖心默默退出了营帐，宋游渠抱起一个盒子，要将帅案上的头颅收起来，然后处理掉。
<br/>　　李行哉和宋游渠在蛤蟆城就见过面，关系自然谈不上好。但现在的李行哉孤立无援，有个认识的人总是好事。
<br/>　　他将草十七的头颅装进盒子中，看了一眼，欲言又止。
<br/>　　“怎么了？”李行哉问。
<br/>　　“殿下可想知道这草十七的名字是怎么来的？”宋游渠问。
<br/>　　李行哉点点头，目光看着对方。
<br/>　　“江湖传言，有人在大山中豢养死士，都是从各地找来的孤儿，男孩以草为姓，女人以花为姓。”
<br/>　　“那这草十七就是其中之一了？”
<br/>　　宋游渠点点头，接着又摇摇头：“草十七不止有一人，死士之间，强者生，弱者死，就像苗疆养蛊一般。如果有人能取而代之，那么人人都可以是草十七。”
<br/>　　宋游渠接着道：“所以草十七是谁并不重要，重要的是大山之中养蛊之人。”
<br/>　　“这种江湖传言，帝国没有一千也有八百，大多是以讹传讹，难道还真有什么杀手组织。”
<br/>　　“很多都是假的，但这个是真的。”
<br/>　　“你怎么知道的？”李行哉看着宋游渠，终于发现他此刻有些不寻常。
<br/>　　在军营中，李行哉被视为废物，宋游渠何尝不是被当做纨绔。但废物未必是废物，纨绔也不一定是纨绔。
<br/>　　“有些事想要知道，总是能知道的，不是太困难的事情。”宋游渠道：“而我还知道，草十七很早之前就死了。”
<br/>　　“怎么死的？”
<br/>　　“被大当家杀死的呀。”
<br/>　　李行哉一怔，玩味的看着对方：“那个大当家呀？”
<br/>　　宋游渠自知失言，立刻闭上嘴巴，然后又道：“当初，程大雷阻挠和亲，各路势力派出很多杀手杀他，最后却都被他杀了，其中便有这草十七。”
<br/>　　“这件事我怎么没听程大雷说起过？”
<br/>　　“当时有很多高手，草十七在里面的确不算最强的，怕是程大雷自己，也不知道草十七这个名字。”宋游渠道：“但自从那件事后，真正的草十七就死了，后来又冒出一个草十七，接着成了将军府的门客，又接着，这次过来刺杀殿下。”
<br/>　　李行哉坐在椅子上，思维一时有些短路，他恍然想起什么，道：“你还没有告诉我，谁是养蛊之人？”
<br/>　　“殿下当已经知道了，何必又来问我。”
<br/>　　李行哉点点头，他心中的确已经有了猜测，只是一时有些难以接受。
<br/>　　当初，派去草原杀程大雷的，大部分是崔相的人，有很大可能，幕后养蛊之人便是崔相。如此，事情就变得复杂了。
<br/>　　要杀自己的究竟是将军府，尉迟离……或者，是崔相故意给自己演这处戏。
<br/>　　李行哉坐在那里，久久无语，过了半晌，他拒绝在思考下去。
<br/>　　因为思考下去，也是没有任何意义的，自己孤立无援，知道的再多，却也是什么都做不了。
<br/>　　“你为什么要告诉我这件事？”李行哉看着对方，与这件事情背后的真相，却是宋游渠的态度更值得琢磨。
<br/>　　宋游渠道：“因为殿下需要知道，不是么？”
<br/>　　“可你宋家是相府的人，不是么？”
<br/>　　宋游渠点点头，又跟着摇摇头：“宋家是相府的人，我未必支持相府。”
<br/>　　李行哉一怔，不解其意。
<br/>　　宋游渠将头垂下来：“帝国太老了，新的帝国应该是年轻人的，那些老人的一些看法，已经跟不上时代了。”
<br/>　　“不仅仅是我，还有很多人年轻人，对于老一辈做的事情，其实很是看不上的。”
<br/>　　李行哉盯着宋游渠，从对方身上看到一团火，一团灼目的火。
<br/>　　“你……太莽撞了，崔相耿耿忠心，你们要长大才能明白。”
<br/>　　宋游渠笑了笑，道：“我之所以愿意对殿下说这番话，是因为觉得殿下是懂我的，毕竟，我们是从同一个地方来。”
<br/>　　从同一个地方来……从蛤蟆寨来。李行哉当真觉得有些不可思议，那蛤蟆城究竟有怎样的魔力，林少羽在蛤蟆寨根本算不得起眼人物，但丢出来，却踹翻了大半个帝国。
<br/>　　而眼前这个被当做人质的宋游渠，假若不早夭的话，日后怕也不是池中之物。
<br/>　　至于自己……似乎自己也不该太妄自菲薄。这未来的天下，自己未必不能争一争。
<br/>　　其实李行哉一直很苦闷，傀儡不是那么好当的。大家都想从他身上榨取更多资源，而不像拿出任何东西交换。他想有自己的人马，收买人心，可稍微做得过分些，就有太多双眼睛盯着自己。
<br/>　　算来算去，似乎继续伪装成一个废物，能够活得更轻松些。
<br/>　　而眼前的宋游渠，给自己带来了力量。虽然这力量还很小，火苗也很弱，但日后未必不能熊熊燃烧起来，直到，将那些丑陋的，讨厌的，肮脏的统统烧成灰烬。
<br/>　　“帝国的未来应该是崭新的，年轻的，也同样应该属于年轻人的。”李行哉站起身来，面带微笑道：“或许，我真的该感谢程大雷了。”
<br/>(本章完)
