　　罗昊正在往一支紧凑型HK416自动步枪的P-MAg弹匣中装填着子弹，这次行动，所有战术中队队员弹匣中所装填的都是钨芯穿甲弹。
<br/>　　在树木叠嶂，稍有开阔地带的丛林区域作战，树木会对子弹的弹道轨迹造成影响，而钨芯穿甲弹能有效的击穿树干，对藏身在树干背后，以及建筑背后的目标造成杀伤。
<br/>　　司徒空用磨刀石将他的飞刀打磨后，开始安装反曲弓弓箭的箭镞。司徒空这次所使用的箭簇是特制的。
<br/>　　参考了毒天使那两件特制的武器构造后，老章头给战术中队打造了一批特殊的反曲弓弓箭箭簇。
<br/>　　这些箭簇的中心是空心结构，中间注入了三毫升的剧毒，箭镞四周被钻了四个小孔，使用特制的封蜡将小孔堵死，箭镞一旦进入人的身体，小孔上的封蜡会瞬间被化开，剧毒会通过箭镞上的四个小孔释放进人身体中，只需要不到两秒钟的时间，剧毒就会人的心脏骤停，然后毙命。
<br/>　　司徒空将一个个箭镞旋在箭杆上，然后装入箭袋，这些反曲弓弓箭会给布克尔特手下那些武装毒贩带去噩梦，不管他们是不是从特种部队退役的老兵，只要被弓箭射中都会让他们瞬间毙命。
<br/>　　咔！
<br/>　　罗昊将两个压满子弹，并且使用胶布一正一反缠在一起的弹匣，塞进手中的自动步枪中，站起身看着屋子中已经在脸上画好伪装迷彩，身上披着伪装网的战术中队队员和“龙牙”佣兵团的四个人，说道：“伙计们，我们出发！”
<br/>　　布谷翻身跳上皮卡车的车斗，在车顶上拍了拍，天蛇驾驶着吉普顺着丛林中那条因为昨天晚上那场大雨而变得泥泞不堪的泥路颠簸着向前行驶。
<br/>　　玄龟降下车窗对着坐在车斗中的布谷大声喊道：“布谷鸟，你是这次行动的关键，如果巴坦伦真的搭乘直升机从他的秘密别墅离开，你必须要用你可爱的狙击步枪把他的直升机打下来。”
<br/>　　“放心，我从来没有失手过。”布谷充满信心的回答道。
<br/>　　狙击手除了要狙击人外，有时需要狙击没有生命特征的战略目标，比如雷达车、通讯车、直升机！不同的直升机结构也不相同，而很多时候，狙击手只有开一枪的机会，所以一个战略狙击手需要熟悉不同的车辆和航空器。
<br/>　　布谷跟着夜鹰学习的时候，最开始的两个月时间，夜鹰就是让布谷记忆不同载具的结构，清楚射击什么位置，可以对载具造成最致命的破坏。
<br/>　　……
<br/>　　木屋中，佟千雪的手腕和脚踝上的都被锁着镣铐，所在屋子四个角落的铁环上，镣铐上的铁链很长，所以并不会影响到到佟千雪在木屋中的活动，但是想要从木屋中离开，却是不可能的。
<br/>　　“吱呀！”
<br/>　　木屋门被人从外面打开，娜塔莉娅手中端着吃的东西和水走进木屋。这间木屋除了娜塔莉娅和布克尔特以外，不允许任何人进入，平时佟千雪的生活起居也是由娜塔莉娅来照顾。
<br/>　　孩子流产后，佟千雪有那么一段时间万念俱灰，甚至拒绝饮食，可是在娜塔莉娅的劝告后，佟千雪心中的希望之火才又重新燃烧起来。
<br/>　　因为是遭受外部暴力导致流产，对佟千雪身体的伤害很大，很有可能伤及到了她的生殖系统，加上本身年龄的关系，如果不注意调养，佟千雪很有可能无法再次怀孕。
<br/>　　“感觉怎么样？”
<br/>　　娜塔莉娅把手中的盘子放在桌子上，看着佟千雪说道：“你的气色看上去好了不少。”
<br/>　　正在房间中走动的佟千雪走到桌子边坐下，因为手脚都戴着镣铐的关系，所以只要佟千雪一有动作铁链就会发出“丁零当啷”的声响来。
<br/>　　佟千雪拿起盘子里那块又干又硬的面包，撕下一块，放在水杯中泡软后，塞进嘴里，对娜塔莉娅说道：“这还得多谢你，如果没有你的那些话刺激，我可能还是跟行尸走肉一样，很有可能连南美洲都没到就死在了海上。”
<br/>　　“我帮助你，也是为了我自己。”娜塔莉娅朝屋子门口的方向看了一眼，确认屋子外的守卫不再偷听后，一把钥匙放在桌子上推到佟千雪面前，说道：“布克尔特刚从种植园那边回来，他一回来就对这里的防御重新进行了布置，似乎是营救你的特种部队已经来了。”
<br/>　　“真的！？”
<br/>　　佟千雪立刻变得激动起来，从椅子上站了起来，她手腕和脚腕上的镣铐又是发出一阵声响来。
<br/>　　娜塔莉娅抓住佟千雪的手，让她保持冷静后，说道：“现在还不确定，我也只是偷听到了几个毒贩的对话而已，但是布克尔特急匆匆的回来布置，可能真的是要发生什么事情了。记住我们之间的承诺，我尽可能的帮助，你脱身的时候，也必须要带上我。”
<br/>　　佟千雪抓起娜塔莉娅推给自己的钥匙，说道：“我不会忘记，只要他们来救我，我就会带你一起走。”
<br/>　　“里面那个女人怎么样？”
<br/>　　布克尔特的声音突然从门外传来，木门的缝隙中，有着人影闪过。
<br/>　　糟糕！
<br/>　　佟千雪手中抓着钥匙，紧张起来。她身上的衣服没有口袋，根本藏不了东西，而且现在也来不及把钥匙藏到床铺下，眼看着布克尔特就要推门进来，情急之下，佟千雪直接把钥匙塞进了自己的内裤中，然后若无其事的跟娜塔莉娅继续交谈着一些无关痛痒的东西。
<br/>　　布克尔特这个人做事的确是有他的原则，这几天时间，佟千雪吃喝拉撒睡都在这间木屋中，而在外面看守的守卫别说是进来意图不轨，就连平时佟千雪方便的时候，连偷看的行为都没有；但是这并不能让佟千雪对布克尔特有什么好感，毒贩就是毒贩，更何况自己这次流产也是拜他所赐！
<br/>　　“那个女医生刚进去送吃的。”门外的守卫回答道。
<br/>　　“把门打开。”布克尔特对着守卫吩咐道。
