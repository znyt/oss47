　　“叮~检测到宿主存在沟通障碍，推荐开启全语言精通功能，此项功能为临时功能，需缴纳1000积分开启，时限30天。”
<br/>　　听到系统的提示，刘哔微微松了口气，有解决办法就好，花费一些积分并没有什么关系。
<br/>　　虽然，一个月需要缴纳1000点积分有些贵了，但为了更好的从这个世界进行任务，他也只能忍痛开启，别无选择。
<br/>　　事实证明，系统虽然黑心一些，但效率还是很高的，刘哔这边刚刚缴纳了积分，这项功能的作用就凸显出来了。
<br/>　　“喂，你有没有听见我说话啊，你今天怎么这么不正常？”看着呆站在原地的刘哔，少女挥了挥手掌，奇怪的问道。
<br/>　　微微笑了一下，刘哔缓过神来，随口敷衍道：“哦，这个啊，我身体不舒服，对了，有什么工作需要我去做吗？”
<br/>　　“诶，现在可是到搜查时间了，刘君该不会已经忘了吧。”脸上的表情更加奇怪，少女眨动了几下眼睛。
<br/>　　仅有的一些信息猜测不出什么，刘哔露出了手腕，看了一眼上面带着的金色手表，装出一副忘了时间的样子。
<br/>　　“额，不好意思，我好像错过了时间，我们这就开始工作吧。”充满歉意的一笑，刘哔用袖口遮住了手腕。
<br/>　　他想要先跟着去看看，没准会从工作中发现什么。
<br/>　　但不曾想，少女听他说完之后，眼睛像是看见了猎物一般泛起了亮光，脸上的表情也逐渐开始兴奋。
<br/>　　“可是，刘君不是我们这组的呢，按照CCG的规矩，我们不可以一起行动呢。”虽然这样说着，但少女脸上的表情一点都不像拒绝，继续说道：“不过，如果刘君邀请我的话，我也很愿意和刘君一起行动呢。”
<br/>　　作为一名连库克因都没有的新人，她在自己小组里面，压根都用不到她出手，这次刘哔倒是给了她一个机会。
<br/>　　毕竟对方也是一个刚刚加入不久的搜查官，也没得到相应的库克因，如果她能跟着刘哔外出搜查，想必会有意思许多。
<br/>　　至于违反CCG规定的后果，管它呢，那种东西能有好玩更加重要吗？
<br/>　　“啊，是这样啊，我也一直想和你在一起搜查呢，要不，我们现在就启程吧。”脸上带着笑意，刘哔调整了一下领带。
<br/>　　这该死的东西，勒的他脖子有些难受，像是有一根硬纸板卡在上面一样。
<br/>　　幸亏，他不需要呼吸，要不然，他可不会忍受这根绳子系在自己的脖子上。
<br/>　　“哇，好耶，那么，我们先计划一下，既然是私人组队的话，那我们最好避开我们原小组打算搜查的区域，emm，要不我们去20区吧，据说那边最近出来几个实力很强的喰种呢。”
<br/>　　虽说是在和刘哔商议计划，但少女却更像是已经做好决定了一般，话刚落音，就拉起刘哔的手臂，一脸迫切的向大楼外走去。
<br/>　　感受着胳膊上传来的柔软触感，刘哔大约知道这个世界是什么世界了，而面前的这个家伙，他也隐约记起了一点印象。
<br/>　　事实上在这家伙提起CCG的时候，他心中就已经有了一个猜测，但因为信息过少，他也不敢确定，直到对方提到喰种这个词语。
<br/>　　在他的记忆中，只有一部动漫出现过喰种这种生物，这个未免，应该是《东京喰种》无疑了。
<br/>　　而面前这个画着一脸奇怪妆容，身上缝满了红线的家伙，应该就是原著中有着‘最强纯种人类’之称的铃屋什造。
<br/>　　如果不是提前知道的话，他还真以为什造是一个没有发育良好的少女，毕竟男孩子可不会有这样白暂细腻的皮肤。
<br/>　　可事实上是，这家伙在原著中，是个被切掉了某一部分的男人。
<br/>　　不过这些事情，都不是重点所在，最主要的是，什造是公认最有潜力的纯种人类，成长到后期，甚至可以比肩SSS级别的高级喰种。
<br/>　　这可是一个让刘哔兴奋不已的好消息啊，他现在已经在心中下达决定，一定要找个机会将他吞噬了。
<br/>　　虽然什造成长起来之前，身体素质以及能力都很低下，也不会给他提供太多的积分，不过万一能吞出什造体内的潜力，然后融合进他的身体，那样他的战斗力又会有一截很大的提升。
<br/>　　带着不怀好意的想法，刘哔脸上的笑容却是比先前更加灿烂了，他对接下来的行程，心中自然是期待不已。
<br/>　　四周一些过往的工作人员，看和拉手奔跑在过道中的两人，皆是纷纷注目着，毕竟在这样正式的办公楼里，什造的装束实在是太过于异类。
<br/>　　这个家伙，不管在哪里都能吸引周围人的目光。
<br/>　　任由什造拉着自己的手臂，两人一路跑出了大厅，现在是下午一点多钟，外面的行人并不算很多。
<br/>　　大概是这个时间，人们都在自己的岗位上工作吧，就算是大街上的这些行人，也都是一副匆匆忙忙的样子。
<br/>　　只有他们，像是游手好闲的小混混一般，大街小巷的晃悠着，说是工作，倒不如说是游玩贴切一些。
<br/>　　……
<br/>　　一个多小时候，两人站在电车站牌前面，什造手中拿着一个牛奶味的冰淇淋，啃食着。
<br/>　　“喂，什造君，你到底要去什么地方啊，好像看起来你一点计划性都没有呢。”手中的冰淇淋一动未动，刘哔语气有些无奈。
<br/>　　这个家伙，一直带着他乱窜，喰种没找到，倒是这些零食吃了不少。
<br/>　　“哎呀呀，对女孩子不能称呼君呢，话说刘君你的日文还挺不错的样子。”什造轻轻咬着乳白色的冰淇淋，却没有回答刘哔一开始的问话。
<br/>　　还真把自己当成女孩了，这家伙。
<br/>　　耸了耸肩，刘哔没有说话，他不知道系统给他安排的什么身份，怕露出马脚。
<br/>　　“好啦好啦，你放心，我们不用刻意寻找，喰种会主动来寻找我们的。”两三口将手中剩余不多的冰淇淋吃完，什造将视线移到了刘哔手中：“话说，刘君的冰淇淋不吃吗，一会化掉可就不能吃了。”
<br/>　　看着什造一脸期待的样子，刘哔心领会神，将手臂往前一递，道：“喏，给你吃吧。”
<br/>　　这种令人恶心的东西，他才不想吃。
<br/>　　他只想吃新鲜的血肉。
