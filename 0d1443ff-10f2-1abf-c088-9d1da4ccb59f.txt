　　主府之内。
<br/>　　萧逸借着这浑厚的星光力量，快速参悟着星幻剑道。
<br/>　　事实上，来天星禁地的武者，全是冲着那极快的修炼增幅而来的。
<br/>　　毕竟那动辄百倍、千倍的修炼效果，堪称恐怖。
<br/>　　不过，对萧逸而言，这还算不得什么。
<br/>　　他那庞大得恐怖的气泉与小世界，本来就需这般快的修炼速度，方能算有修炼效果。
<br/>　　当然了，他若在此修炼数天，肯定也会有修为上的突破。
<br/>　　只是，他更愿意用来感悟星幻剑道。
<br/>　　毕竟，对于修为来说，这些星光力量只是一份用于转换元力的力量。
<br/>　　他日后大可通过大量的丹药，来达到同样的修为效果。
<br/>　　但唯独星幻一道的感悟，此时绝对是大好机会。
<br/>　　当初星幻瓶，不，准确来说，是天星盏；一个天星盏的效果，尚且效果如此惊人，让得在三月之内参悟掌控八万星光。
<br/>　　现今，十八盏天星盏加持下的星光力量，效果该惊人到何等地步？
<br/>　　萧逸边闭目参悟着，边心头带着期待。
<br/>　　周遭，浑厚到极点的星光力量，快速增幅着萧逸的感悟。‘
<br/>　　这些储存于天星洞府之内，存在千万年岁月的古老星光力量，再加十八把上品圣器的增幅，效果极度惊人。
<br/>　　萧逸对于星幻剑道的感悟，不断上升着。
<br/>　　那五千倍的修炼增幅，此刻悉数换作萧逸的武道感悟速度。
<br/>　　一天后。
<br/>　　萧逸微微睁开了眼睛，脸上，尽是喜意。
<br/>　　“突破九万道了。”萧逸自语一声。
<br/>　　还记得，他前些天离开金光禁地时，才堪堪突破八万五千道。
<br/>　　现今，在这主府之内，仅仅一天，便再度突破五千道，达到九万道。
<br/>　　也就是说，他现今对星幻剑道的感悟层次，已达圣皇境九重水准。
<br/>　　此刻的他，单凭剑道水准，便能力战圣皇境九重武者了。
<br/>　　……
<br/>　　时间，渐渐过去。
<br/>　　大半天后。
<br/>　　“九万两千道星光了。”萧逸面露满意之色。
<br/>　　对比之前，他又多掌控了两千道星光。
<br/>　　现今，他单凭剑道水准，便能力战圣皇境巅峰。
<br/>　　星幻剑道，那掌控星光降临的手段，确实厉害。
<br/>　　不过，对比之前一天掌控五千道的速度；现今大半天才掌控两千道，速度似乎在减缓。
<br/>　　对于这一点，萧逸早有所料。
<br/>　　毕竟，这是星幻一道上的武道感悟。
<br/>　　武道，越上感悟下去，难度便越大，自然的，参悟速度也开始变慢了。
<br/>　　又是半天后。
<br/>　　这个速度，还在持续减缓。
<br/>　　半天时间，他只参悟了950道星光。
<br/>　　算上之前的参悟，也就是说，他现今掌控了92950道星光。
<br/>　　还差50道，便达到九万三千的层次。
<br/>　　只是，剩下的这50道，忽然难如天堑，萧逸的感悟，再难进分毫。
<br/>　　“怎么可能，速度慢了如此多。”萧逸眉头紧皱。
<br/>　　九万三千道星光，大概是等同圣皇境巅峰中极强的武道层次。
<br/>　　萧逸深呼吸一口气，再次陷入了参悟。
<br/>　　在天星禁地之内的时间，只有三天。
<br/>　　三天后，洞府就会自动隐匿于地底。
<br/>　　故他只剩一天时间。
<br/>　　时间，渐渐过去。
<br/>　　又是一天后。
<br/>　　萧逸睁开了眼睛，脸色略微有些难看。
<br/>　　整整一天参悟，他仅仅是多掌控了50道星光。
<br/>　　算上之前的参悟，终于达到了九万三千道星光的层次。
<br/>　　而现今，距离进入主府，已整整过去了三天。
<br/>　　而这最后一天的时间，加上这18盏天星盏的星光力量增幅，他才仅仅参悟了50道星光？
<br/>　　“厉害，不愧是星幻剑道。”萧逸皱着眉。
<br/>　　当他的参悟层次达到九万三千道星光时，他打算再度参悟下去，已觉脑袋生疼。
<br/>　　再往后的武道知识，太过玄奥了，绝非他短时间内能参悟透彻的。
<br/>　　“不过也算极大的收获了。”萧逸轻笑一声。
<br/>　　三天时间，多掌控了8000道星光。
<br/>　　这8000道星光，若是放到外面，他怕是一年半载都难以参悟透彻，更别说掌控星光降临。
<br/>　　毕竟这8000道星光，可是八万五千道星光的后续层次。
<br/>　　特别是最后那50道星光，论层次，已等同是圣皇境巅峰中极为玄奥的武道知识。
<br/>　　短短三天，有此收获，萧逸足够满意了。
<br/>　　这时，萧逸看了眼石坛另一边的顾怜星。
<br/>　　此刻的顾怜星，脸上苍白之色，已然褪去。
<br/>　　本就白泽的皮肤上，添上了一抹红润。
<br/>　　原本虚弱的气息，也在不断变得平和。
<br/>　　“原来这些星光力量竟是你的治病之物。”萧逸面露恍然。
<br/>　　……
<br/>　　主府之外。
<br/>　　一众武者亦停下了修炼。
<br/>　　此时，众人的气息，对比之前刚入洞府时，起码提升了一个层次。
<br/>　　三天洞府修炼，一众武者，最差的都突破了两重修为，有些强的，整整提升了三、四重。
<br/>　　要知道，圣王境以后的修炼，每突破一重，起码需要数年，乃至更久的时间。
<br/>　　而现在，短短三天，便突破了修为。
<br/>　　可想而知天星禁地的效果之强。
<br/>　　当然，这还是归功于萧逸放下的星幻瓶，让得18盏天星盏齐全，重新启动了禁制。
<br/>　　禁制的重新发动，让得这里的增幅效果大增。
<br/>　　否则，按照以往的情况，在此修炼三天，一般武者能提升两重修为已是极限了。
<br/>　　大多都只是提升半重修为左右。
<br/>　　“三天时间已过，也该离去了。”
<br/>　　一众武者，纷纷起身。
<br/>　　主府大门，那块巨石之外。
<br/>　　公孙火舞三人，仍旧脸色急切地等待着。
<br/>　　“该死，三天了，那小贼还未出来。”公孙火舞面露担忧之色。
<br/>　　“顶多还有半个时辰，洞府就会自动下沉。”
<br/>　　“怜星。”西门孤夜握了握拳头。
<br/>　　月姐姐脸色怪异，“孤男寡女的，在里头待上三天，此刻还未出来，别是真的出什么事了吧。”
<br/>　　“荒唐，不可能。”西门孤夜冷喝一声。
<br/>　　“那小贼，他若真敢对怜星…”
<br/>　　后面的话，西门孤夜没有说下去，只是，脸庞上已杀意凛然。
<br/>　　“萧逸小贼，还不快些滚出来？”公孙火舞对着主府愤怒大吼着。
<br/>　　“三天已过，再不走，这里要埋于地底了。”
<br/>　　“你自己要死便死，别连累了怜星。”
