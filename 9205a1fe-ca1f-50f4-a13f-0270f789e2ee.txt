　　第597章
<br/>　　夕阳斜照，天空昏沉沉的。钟牧山和莫小湖斜靠在马车上，也是有气无力，打不起精神来。
<br/>　　这一路过来，钟牧山也并不是没有长眼睛，已知道商队拉着的是什么。如果齐如海只是往戎族送些布匹瓷器，锅碗瓢盆什么的，钟牧山也不在意。可马车上装着的，都是锋利的兵器。
<br/>　　把兵器送给草原上的戎族，说不得什么时候，这些兵器就会砍在帝国子民的脑袋上。
<br/>　　这叫怎么一回事。
<br/>　　钟牧山心里骂骂咧咧，已经有几分不痛快。他们是从书院出来的，学的是忠孝礼智信，可现在却做起了卖国的勾当。
<br/>　　如果是与齐如海拍桌子发怒，可对方毕竟对自己有恩。但如果不做什么，这个污点是挂在身上洗不干净了。
<br/>　　于是钟牧山心情才不痛快，两条路无论怎么选都是错的。
<br/>　　这时候有个戎族的汉子过来，粗着嗓子道：“喂，你们两个过来。”
<br/>　　钟牧山和莫小湖一头雾水，也不知找自己二人为什么事。却也站起身，随着戎族汉子走进大帐。
<br/>　　二人见到了火之部落的首领，一个独眼年轻人坐在帅案后，看上去年纪并不大。钟牧山倒是没想到，这戎族部落的首领竟是个如此年轻的人。
<br/>　　齐如海笑着站起来，道：“二位老弟，我正有件好事要与你们商量，这可是天上掉下来的福分。”
<br/>　　钟牧山奇怪的看着他，等听齐如海说完，他脸色当场就沉了下来。虽然没有开口，但心里想说什么，其实齐如海心里也明白。
<br/>　　就算普通的帝国人，让他们与戎族做事，也不是一件容易的事。更何况钟牧山这种读圣贤书长大的人，心里还是想着忠君爱国，怎么可能辅佐戎族。
<br/>　　齐如海叹道：“老弟，我知道你们看不起我，但天下大乱，你想想如果能好好端碗饭吃，谁不想着好好过日子。你们都是有本事的人，难道就想着埋没了毕生所学。”
<br/>　　钟牧山一拱手，道：“齐兄，你的收留之恩，钟某记在心里。但想让我们为戎族做事，万万不能。”
<br/>　　齐如海苦口婆心的劝着，帅案后的野原火轻咳一声，打断了他想说的话。
<br/>　　三人的目光都落在野原火上，只听他道：“你们好好考虑一下，若肯为我做事，日后未必不可在乱世留下自己的名字，若是不想……”
<br/>　　“不想怎样？”莫小湖忍不住问道。
<br/>　　野原火奇怪的看着他：“自然是杀了，否则还会怎样？”
<br/>　　钟牧山和莫小湖心中都是一惊，感觉到大帐内隐隐的杀机。
<br/>　　正这时候，一个戎族汉子大步走进帐篷，道：“大王，大事不好，北蛮部杀过来了。”
<br/>　　野原火一直养精蓄锐，扩张势力想要打北蛮部的主意。北蛮部的金问道也并不是一点风声没得到，他想灭了北蛮部，北蛮部也同样想灭掉这个突然冒出来的火之部落。
<br/>　　“打过来就打过来，慌什么。”野原火喝了一声：“列阵，与我杀。”
<br/>　　他长身而起，拔出了自己身上的弯刀，大步走出帅帐。刚才看上去还无比安静的年轻人，此时宛若一头猛虎起身。
<br/>　　齐如海心惊胆战，自己怎么这么倒霉，碰到这种事。乱杀乱砍，假若自己一条命丢在战斗中会怎样。
<br/>　　他凑到钟牧山身边，道：“二位老弟，你们可要救哥哥一救，别让哥哥被戎族杀了。”
<br/>　　钟牧山看他一心，握紧了佩剑剑，道：“放心，你跟在我们身后，我们护着你。”
<br/>　　三人走出大帐，只看到北方烟尘动荡，赫赫一群人马向这边杀过来。钟牧山在书院也研究过兵法战阵，但纸上得来终觉浅，这样的大规模战斗却也是第一次遭遇。
<br/>　　野原火跨上马匹，手中弯刀一甩，将手下人马聚拢在一起。戎族之间的战斗还是很简单的，无非是远程弓箭射杀，凑近了白刃见血。但简单之中方能看出不简单，能在短短时间内将战士聚集在一起，组织起反击力量，绝对不是一件简单的事情。
<br/>　　在这一点上，野原火做的不错。士兵很快准备妥当，人如虎马如龙，士气高昂。将为兵之胆，一位将官在士兵心中的地位如何，从士兵的士气也能看出来。
<br/>　　两支人马互相冲锋，各个都有吞狼食虎的劲头，在靠近时狠狠的撞在一起。一时间人仰马翻，喊杀声，惨叫声响成一片。
<br/>　　钟牧山和莫小湖一起，将齐如海掩在身后。齐如海吓得浑身颤抖，在二人后面躲着，连头都不敢冒。而钟牧山二人也并没有比他好多少，毕竟，这样的战斗二人是第一次亲眼目睹。
<br/>　　在书中读来的战争，某一战役，杀死多少多少士兵，无非是一串数字而已。可在真正的战场上，每一个人的惨叫都是那样响亮，每人身上流出的血都是那样清晰。
<br/>　　真正的战争原来是这样的。
<br/>　　钟牧山在心底发出一声长叹，只有亲眼目睹，方才知道以前的认识有多么浅薄。
<br/>　　而他也真切的见识到戎族的实力，尤其是野原火，在乱战丛中，他骑着一匹红马，是格外显眼的存在。
<br/>　　宛若一团红色的火焰，在杀戮场中流动，士兵簇拥在其身后，闯进了敌人的战阵中央。
<br/>　　所到之处，无人可挡，一切如摧枯拉朽一般，真的想一团会将世间一切污垢都焚烧干净的怒火。
<br/>　　“小湖……”钟牧山开口的时候，才发觉声音竟在颤抖。
<br/>　　“师兄……”莫小湖现在的感受也不太好。
<br/>　　“原来，世间当真有以一当万的名将。”钟牧山发出一声长叹。
<br/>　　这是一场奔袭战，金问道指挥的大军本想趁着敌人不备，一举将他们吞没。但是，他没想到对方的组织能力竟如此强，在很短世间内便组织起反抗力量。于是，这场攻敌不备，就成了敌人的以逸待劳。
<br/>　　如今，北蛮部已经完全处于劣势，战阵被冲散，金问道明白，今天这场战斗，己方已经很难取得胜利。
<br/>　　然后，他在乱军丛中，看到独眼的野原火。
<br/>　　“是他？”
<br/>　　“怎么是他？”
