　　马允，可是这一届新人中的佼佼者了，实力极强，有着可能晋升内门弟子的希望。
<br/>　　谁也料想不到，马允居然会突然出手，趁着楚风眠跟牛峰交手之际，突然出手向着楚风眠偷袭。
<br/>　　像是这马允，这样一尊已经算是小有名气的强者，居然是会选择偷袭，这让许多武者都有些不耻。
<br/>　　“卑鄙！”
<br/>　　“本身便是二对一，居然还偷袭。”
<br/>　　许多围观武者心中不由暗骂道，虽然他们不敢说出来，但是如今的眼神中，便是将他们的心中想法，全部暴露出来。
<br/>　　看到这许多武者的眼神，马允完全不以为然，偷袭又如何？
<br/>　　楚风眠的实力，明显比起看起来的要强大的多，能够压制住牛峰，与其去跟他硬抗，不如直接偷袭。
<br/>　　现在楚风眠正是在跟牛峰抗衡的阶段，无暇分心，这个时候偷袭，绝对是最好的选择。
<br/>　　“小子！我就先斩断了你的两条手臂！看你怎么跟我们斗！”
<br/>　　北邙学院中，贸然杀害其他弟子，还是极大的罪孽，所以这马允一刀斩出。
<br/>　　这三千刀锋的目标，却是楚风眠的双臂，只要将武者的双臂斩断，足以让他们失去九成的力量。
<br/>　　倒时候不都是任由他们去宰割。
<br/>　　“偷袭？”
<br/>　　楚风眠眼神一动，同样也是看到了马允的动作。
<br/>　　就在这三千刀锋，抵达到楚风眠的面前的一刻，楚风眠的嘴，缓缓动了。
<br/>　　“破！”
<br/>　　楚风眠口中微微一动，出现的却是一声龙吟。
<br/>　　真正的天龙八音，从楚风眠的口中，展现出来，这天龙八音，本是按照龙吟，创造而出的武技。
<br/>　　如今楚风眠拥有战龙血脉，动用这一招，简直便是轻而易举，不止龙啸，更有龙威。
<br/>　　“哗啦！”
<br/>　　一声龙吟，响彻整个山脉之中，那三千刀锋，在这一声龙吟之下，尽数破碎，全然毁灭。
<br/>　　谁也没有办法料到，本来看起来已经是进入到死局之中的楚风眠，居然是会来这样一个惊天逆转。
<br/>　　根本没有出手，只是口中喊出了一个字，便是将这马允的最强招数，三千刀锋，尽数毁灭。
<br/>　　楚风眠的实力，远远超越众人的想象，以一敌二，在马允偷袭的情况之下，居然都没有丝毫威胁到楚风眠，反而是被楚风眠直接逼退。
<br/>　　“啊！”
<br/>　　在这一声龙吟之下，突然之间，马允也是发出一声惨叫声。
<br/>　　众人随着马允的惨叫声，看了过去，便是看到了马允的胸口，居然是出现了一个巨大无比的裂痕，仿佛是炸裂开来了一样。
<br/>　　原来，楚风眠的这一声龙吟，居然是将马允胸口，都给硬生生的喝破。
<br/>　　传闻中猛虎长啸，一个个人都会被直接吓破胆，如今楚风眠这一声龙吟，便是马允的胸口，都被直接炸开，无数血液，顺着其中留了下来。
<br/>　　这天龙八音，蕴含无上玄妙，传说中由真正的强者喊出，一吼之间，方圆百里，无人生还，楚风眠虽然还远远达不到这一步。
<br/>　　但是对付这马允，还是绰绰有余。
<br/>　　“两个废物？不是想要抢占我的洞府吗？还想要来偷袭我？断我双臂？剥夺我的血脉？”
<br/>　　楚风眠一双眼神，冰冷到极点，看向那牛峰，马允二人，冷喝一声道。
<br/>　　“今日我就将你们二人身上半数的圣者精血全部抽走！让你们知道，有的人，不是你们可以招惹的起的！”
<br/>　　楚风眠话语之中，狠辣到极点，如今来到这北邙学院，不展现出实力来，这些人还真会将楚风眠当做软柿子了。
<br/>　　“楚风眠，你敢抽我们二人的精血？”
<br/>　　听到楚风眠的威胁，那牛峰丝毫不惧，更是大声出言喝道。
<br/>　　“我们二人，已经是加入了风云会！投奔了武风云师兄，你敢抽取我们二人的精血，武风云师兄不会放过你的！”
<br/>　　“什么？风云会？”
<br/>　　“这不是外门中的势力之一，传说创立风云会的武风云，可是人榜位列第十一位的超级天才，在外门中的地位，接近前十！”
<br/>　　“这种人物，想要晋升内门，只差一点点机缘，只要得到一点机缘，直接就会被邀请进入到内门之中去。”
<br/>　　“这二人，居然是加入到了风云会，那这一届的新人中，谁还敢招惹他们。”
<br/>　　听到风云会三个字，周围不少围观众人都是眼神不由一变。
<br/>　　无数人的眼神中，也是露出了几分羡慕，刚刚加入到北邙学院中，就可以依附于这样一方势力。
<br/>　　没想到这牛峰，马允二人，刚刚加入到了北邙学院中，居然就攀上了这样一棵大树。
<br/>　　北邙学院中，其中也拥有这无数势力，都是一个个的小团体，这风云会正是外门的小团体之一。
<br/>　　风云会在外门的无数势力之中，已经算是中上游，一些老牌的外门弟子，想要招惹起来，都要自己掂量掂量几斤几两。
<br/>　　放在这一届的新人之中，这样的势力，绝对是一尊庞然大物了。
<br/>　　背靠大树好乘凉，这二人如今加入到风云会中，抱上这个大腿，便是真正的有着了靠山。
<br/>　　这也是如今这二人，还敢如此嚣张的原因，他们不信，楚风眠敢跟风云会为敌。
<br/>　　只有真正北邙学院外门人榜前十的弟子，才可以有资格无视风云会。
<br/>　　但是显然，如今的楚风眠，并不拥有这样的实力。
<br/>　　“小子，既然你已经知道了我们是风云会的人，要怎么做，你应该清楚吧。”
<br/>　　牛峰看着楚风眠沉默不做声，脸上的表情又是变的嚣张起来道。
<br/>　　“这洞府，可是武风云师兄亲自指定给我们的，你还是乖乖交出来吧，不然等到武风云师兄亲自出手，你可就没这么好运了。”
<br/>　　“风云会？武风云？这又是个什么东西！”
<br/>　　楚风眠脸上露出几分嘲笑，嘴角微微一翘，开口道。
<br/>　　“你们两个，还敢来威胁我？看来给你们的教训还不够！今日楚某正是要在这外门中立威，不给你们个教训，还真以为是个什么阿猫阿狗都敢来威胁我了？”
<br/>　　楚风眠的言语，可谓是嚣张至极，居然是直接不讲风云会，甚至是人榜第十一的武风云，放在眼中。
