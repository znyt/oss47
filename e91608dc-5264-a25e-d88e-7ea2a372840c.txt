　　嘭…
<br/>　　萧逸手中，一股火焰凝聚而出。
<br/>　　大手一挥，将夜影流光蟒与荆棘圣蜥的内丹和妖兽精血收入手中。
<br/>　　妖兽精血，就此收起。
<br/>　　内丹，则扔入火焰中。
<br/>　　配合一株株天材地宝，不多时，两粒金光熠熠的金光丹炼制而出。
<br/>　　“呼。”萧逸轻呼出一口气，看了眼天空
<br/>　　天空中，那抹漆黑，在此时达到了极致。
<br/>　　他没记错的话，算上他一路前行深入，再到大战夜影流光蟒与荆棘圣蜥，差不多用去了近一晚的时间。
<br/>　　天，就快亮了。
<br/>　　如今的这抹夜空极致漆黑，不过是破晓前的最为黑暗之时。
<br/>　　萧逸摇了摇头，准备后退一段距离，开始一日的打坐感悟。
<br/>　　今晚，大部分的时间，还是用在了对战夜影流光蟒和荆棘圣蜥上。
<br/>　　特别是最后星光之力绞杀荆棘圣蜥，耗费了大量时间。
<br/>　　今晚的历练，可以告一段落了。
<br/>　　虽然战斗不多，但却收获颇丰。
<br/>　　夜影流光蟒和荆棘圣蜥，平日可是极难遇到的妖兽。
<br/>　　更别说这两头妖兽还联手为之。
<br/>　　与之一战，历练效果可比昨夜一晚大战数十妖兽还要好得多。
<br/>　　至于他选择后退一段距离。
<br/>　　实在是这里毕竟是金光险地深处，即便白日里，亦危险重重。
<br/>　　萧逸倒不惧这些危险，只不过白天的妖兽实力不强，历练效果不大，便没必要爆发这种没有意义的战斗了。
<br/>　　萧逸转过身，刚要离去。
<br/>　　忽然…
<br/>　　“嗯？”萧逸脸色一凝，“好可怕的妖兽气息。”
<br/>　　几乎是他感知到这股妖兽气息的瞬间。
<br/>　　轰…一股猛烈的破空声，疾速而来。
<br/>　　萧逸来不及看那是什么东西，脚步连忙一退。
<br/>　　轰…一声惊天巨响。
<br/>　　几乎是脚步退开的瞬间，原本所站之地，已被轰出一个巨坑。
<br/>　　一股巨力之下所产生的恐怖气流，直将萧逸后退的身影轰出百米之外。
<br/>　　单单是余威所产生的气流，竟已有如此威力？
<br/>　　萧逸脸色一变，手中冷焱剑瞬间出现。
<br/>　　锵…
<br/>　　剑尖撑地，止下了气流的冲击，亦稳下了身影。
<br/>　　待得稳下身影，萧逸看了眼百米之外，霎时瞳孔一缩。
<br/>　　“金刚白猿？”
<br/>　　萧逸脸庞抽了抽。
<br/>　　这可不是普通妖兽，而是拥有上古血脉的珍稀妖兽。
<br/>　　当年，萧逸在东域时，曾遇到过雪地金刚猿。
<br/>　　雪地金刚猿，便是极为稀有的上古血脉妖兽。
<br/>　　论品种，雪地金刚猿与金刚白猿同类。
<br/>　　但金刚白猿的血脉层次，比雪地金刚猿高了数个层次都不止。
<br/>　　其实力，更是强得发指。
<br/>　　此时，这头金刚白猿，看着地面上夜影流光蟒与荆棘圣蜥的尸体，明显露出了一丝人性化的伤感。
<br/>　　萧逸双眼一眯，随后恍然。
<br/>　　“原来如此，难怪夜影流光蟒和荆棘圣蜥会联手。”
<br/>　　正常来说，妖兽拥有极强的领地意识。
<br/>　　实力越是强，级别越是高，灵智便也越高，这种意识也越强。
<br/>　　夜影流光蟒和荆棘圣蜥，可是不同种类的妖兽，根本不可能联手。
<br/>　　可再加上这头金刚白猿，那便不同了。
<br/>　　妖兽族群，与人类一样，既存在地盘意识，自然也有势力之分。
<br/>　　有的妖兽森林，一兽为王，诸兽臣服。
<br/>　　强者为尊，同样适用于妖兽之中，且那般弱肉强食更为残酷。
<br/>　　金刚白猿，显然是极强的妖兽。
<br/>　　不过，金刚白猿喜独行，也喜收揽少数妖兽同行。
<br/>　　或者更简单来说，金刚白猿，喜欢收‘小弟’。
<br/>　　如果没有猜错的话，这头夜影流光蟒和荆棘圣蜥，便是它的‘小弟’。
<br/>　　观它现今表情，似乎三兽之间，感情还不浅。
<br/>　　吼…
<br/>　　下一秒，金刚白猿，发出了一声震天怒吼。
<br/>　　怒吼声中，充斥着惊天怒意，震颤整个金光险地。
<br/>　　萧逸感受到这股气势，脸色再次一惊。
<br/>　　“圣皇境巅峰。”
<br/>　　这头金刚白猿，显然已吸食金光力量。
<br/>　　一声气势，达到圣皇境巅峰。
<br/>　　也就是说，现今活生生站在萧逸百米外的这头妖兽，等同一个天藏十二峰长老。
<br/>　　最重要的是，这是嗜血的妖兽，且对萧逸产生了无比的杀意。
<br/>　　若是换了别人，碰上这样一头处于暴怒状态，且实力恐怖的妖兽，怕是早已溜之大吉。
<br/>　　但对萧逸而言，这才有意思。
<br/>　　俊朗的面庞，虽抽了抽，但却充满了战意。
<br/>　　锵…
<br/>　　萧逸一剑指出。
<br/>　　冰冷的剑锋，直指金刚白猿。
<br/>　　金刚白猿巨大而狰狞的眸子中，凶性更增。
<br/>　　萧逸的动作，在它眼中，无异于宣战的挑衅行为。
<br/>　　这偌大个金光险地，别说区区一个人类小子，就是那些霸主妖兽，都不敢如此对它。
<br/>　　轰…
<br/>　　金刚白猿的庞大身躯，瞬间动了。
<br/>　　巨大的双腿，重重一瞪，霎时爆发惊人。
<br/>　　地面顷刻在反震之下凹陷百米。
<br/>　　巨大的身躯，丝毫没有笨拙，反倒速度惊人。
<br/>　　“来得好。”萧逸冷喝一声，一剑劈出。
<br/>　　八万星光，从天而降。
<br/>　　剑身与金刚白猿拳头对轰的瞬间，剑影重重。
<br/>　　一剑出，20道星光长河，悉数碾压而过。
<br/>　　嘭…嘭…嘭…
<br/>　　一连20道爆响，响彻百里。
<br/>　　待得星光消散，金刚白猿的攻击虽被止下，但拳头上却毫发无损。
<br/>　　轰…
<br/>　　金刚白猿一拳轰出。
<br/>　　萧逸脸色凛然，同样一剑劈出，再次20道星光长河齐下。
<br/>　　嘭…嘭…嘭…
<br/>　　两人的交战，无丝毫花哨，只有硬碰硬。
<br/>　　金刚白猿，是出了名的力大无穷。
<br/>　　萧逸的星光长河，同样威力滔天。
<br/>　　“战个痛快。”萧逸暴喝一声。
<br/>　　虽然刚刚才经历过大战不久，体内元力匮乏。
<br/>　　但他还是一边战斗，一边大口吞服着丹药，补充元力。
<br/>　　然而，不过几秒，金刚白猿的身体，陡然一退。
<br/>　　“嗯？”萧逸眉头一皱。
<br/>　　轰…
<br/>　　一声爆响。
<br/>　　金刚白猿的身体，一跃而起。
<br/>　　高空中，数头疾风烈鸟堪堪飞过。
<br/>　　金刚白猿的身体，宛若一枚陨石，冲撞而上。
<br/>　　巨大的手掌，轻松捉住了这几头疾风烈鸟。
<br/>　　下一秒，手掌一撕开，数头疾风烈鸟，瞬间被分尸。
<br/>　　数粒散发着金光的妖兽内丹，被金刚白猿一口吞下。

<br/>　　“吞食妖兽内丹以及其中金光力量，强行增强实力？”萧逸脸色一变。
<br/>　　下一秒，脚步一动。
<br/>　　不是仗剑前行，而是脚步飞退，疾速远遁。
<br/>　　现今的金刚白猿，怕是一拳头就能将一个圣皇境巅峰强者轰成重伤。
<br/>　　他怎么可能打得过。
<br/>　　吼…一声怒吼。
<br/>　　金刚白猿，瞬间追袭而来。
<br/>　　“好快的速度，不好。”萧逸脸色一变。
<br/>　　恰在此时。
<br/>　　“孽畜。”一声暴喝，破空而来。
<br/>　　人未至，一道如云剑气，已瞬间而至。
<br/>　　锵…
<br/>　　金刚白猿袭来的身影，瞬间在如云剑气之下轰退百米。
<br/>　　一道身影，牢牢挡在了萧逸身前。
<br/>　　“云渊长老。”
