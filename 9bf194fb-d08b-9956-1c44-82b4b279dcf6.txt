　　第501章
<br/>　　一追一逃，二人一口气奔出百十余里。便是程大雷一身泼天力气，此刻也觉得有些力疲，一口气要喘三次，才能喘匀。
<br/>　　更令程大雷郁闷的是，聂隐娘依旧紧紧跟在身后，如跗骨之蛆，阴魂不散，二人之间的距离，并没有缩短一步，当然也未拉开一步。
<br/>　　日落黄昏，夜幕渐渐垂下来，二人一直从正午跑入夏夜。四周围灰蒙蒙一片，程大雷也不知身在何处，只略记得从开始就是向西逃的。
<br/>　　前方大河拦路，程大雷忽地停住脚步，回身横剑指着聂隐娘。
<br/>　　“呔，兀那女子，忒不讲道理。本当家怜你是个女子，不愿与你动手，当真以为本当家怕了你不成？”
<br/>　　聂隐娘也忽地停下，谨慎的盯着程大雷。她的状况，其实并不比程大雷好多少，一路追来，已快累得虚脱。毕竟是女儿身，这些日子也不可能好好休养，那里比得上程大雷一日三餐都由人伺候。能够追到现在，不过靠着意志力撑着而已。
<br/>　　心中其实也明白，今日若杀不了程大雷，怕以后便再没杀死他的机会。程大雷此贼恁地不要脸，甘当缩头乌龟也不露面。今天好不容易从乌龟壳中出来，聂隐娘怎可放了他。
<br/>　　十息之内，聂隐娘喘匀了气，猛地扑杀过来。掌心寒芒一闪，隔着十几步，便能让程大雷感觉到扑面而来的杀机。
<br/>　　来得好快！
<br/>　　程大雷心中惊叫一声，心道这姑娘是属狗的么，但凡有一丝力气，便会不顾一切扑杀。
<br/>　　程大雷刺出一剑，也是追星赶月的一剑，快到极致。只见聂隐娘身子旋转，侧着身子躲过这招快剑，然后借着旋转的力道，匕首稳准狠的砍向程大雷喉咙。
<br/>　　程大雷瞬间便是一层白毛汗，使了个铁板桥的架门，堪堪躲过了这一招杀机。然后身子在地上滑了半步，与聂隐娘拉开距离。
<br/>　　程大雷难以相信，在如此境地下，聂隐娘的招式还如此迅猛。刚才反应若慢个半拍，一条命便也就交待在荒原上。
<br/>　　不过，程大雷也能感觉到，聂隐娘的招式已慢下来。在一开始，自己与她过招的资格都无，但在现在，已经可以勉强对上几招。
<br/>　　没办法，继续逃吧。程大雷沿着河岸，向北方撒丫子狂奔。
<br/>　　聂隐娘本以为程大雷停下，是要与自己决一死战。而那正是聂隐娘期盼的，二人拿出看家本事，真刀实枪的拼一场，哪怕是聂隐娘被程大雷杀了，自己也甘心。
<br/>　　然而谁曾想这程大雷，天下杀名赫赫的琴川之主，竟然如此胆小。根本没要和自己决战的意思，似乎除了逃跑，根本没有其他心思。
<br/>　　二人一追一逃，沿着大河向西而去。一路上，程大雷倒也不是一味逃跑，时不时回过头来，与聂隐娘对上两剑，但他绝不恋战，偶尔刺上一剑，偶尔刺上两剑，脱离危险后，继续狂奔。
<br/>　　程大雷是在试探聂隐娘的体力，聂隐娘的耐力令他吃惊，其展现的意志力更加可怕。然而，程大雷已察觉到，随着时间推移，聂隐娘的体力在一点点被耗尽。
<br/>　　一开始，他只能和聂隐娘对上一剑，到后来，已能慢慢撑上几个回合。一招，两招……到后来十招还能撑着不败。
<br/>　　毕竟，程大雷的体力要比聂隐娘强，程大雷心中暗暗有了准备，再将时间拖久一些，要取聂隐娘性命并非做不到。
<br/>　　穿过平原，绕过大河，便进入了大山。此刻正是炎夏，树繁叶茂，密林深处，时不时响起兽吼鸟鸣。
<br/>　　进入丛林深处后，程大雷展现了出色的轻身功夫，一口气不泄，便能纵出四五丈远。程大雷也很满意自己的表现，本想靠此与聂隐娘拉开距离，可谁能想到，聂隐娘的表现更加惊人。
<br/>　　她本就是在深山密林中长大，悬崖峭壁行走如履平地，大树之上，怪石之间，一跃数丈。
<br/>　　程大雷还想借着地形把她甩开，谁料二人之间的距离竟然渐渐近了。
<br/>　　“狗贼，纳命来吧！”
<br/>　　聂隐娘忽地发出一声吼，扑身追来。程大雷汗毛倒竖，就地一个翻滚，虽然姿势极其狼狈，但却是将这一招避开。
<br/>　　他一个翻身跃起，看到聂隐娘已经拦在自己面前十步外，手中提着匕首，汗水顺着手腕从匕首上滴落，一滴滴砸在草叶上。
<br/>　　程大雷也站了，仔细打量着聂隐娘，见她不停喘息，小小的胸脯剧烈起伏。
<br/>　　程大雷心中忽然一动，直到聂隐娘已撑不下去。此时便是此时，此时不取她的性命，更待何时。
<br/>　　他深吸一口气，稳住砰砰乱跳的心脏，气走四肢百骸，陡然圆睁双目，一剑袭来，直取聂隐娘眉心要害。
<br/>　　直到此刻，程大雷终于搬出全部实力，要和聂隐娘捉对厮杀，虽不能说必然可胜了聂隐娘，但程大雷觉得自己已有和她决战的资格。
<br/>　　聂隐娘眼睛眯成一条线，借此掩饰心底的震惊。这一剑的杀威自然不容忽视，她却是没料到，程大雷刚才跑得像条丧家犬，此刻却凶得像头狮子。盛名之下无虚士，自己或许并不该小看程大雷。
<br/>　　随之，嘴角一抹冷笑轻轻抿起。身子前倾，握着匕首的右手架在身前。
<br/>　　程大雷瞥见了这抹笑，心底忽然一惊，这笑容似乎自己落入某个圈套中，她拉开的架门像张开网等着笨鸟来逃。
<br/>　　糟糕，中计！
<br/>　　程大雷想退，却依然是来不及，聂隐娘扑了过来，速度快到极致，哪里有用尽力气的迹象。无非是故意卖个拙，引得程大雷厮杀而已。
<br/>　　聂隐娘的情况的确不太好，她也看破了程大雷的计谋，再被他拖下去，自己非要在这大山中丧命不可。也就是为此，才故意佯作力疲，引得程大雷上钩。
<br/>　　看来，程大雷如今已经中计。
<br/>　　程大雷暗骂一声，自己本不该小看她。毕竟是绝世，刺杀的本事高，脑袋也不是榆木疙瘩。
<br/>　　但此时此刻，想太多也无用，退无可退，只能握剑向前，用命厮杀。
<br/>(本章完)
