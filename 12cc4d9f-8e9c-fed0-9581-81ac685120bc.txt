<br/>    十万大山，荒茫雪原。
<br/>    
<br/>        妖族大君的一只手已经是断了，悬在自己的胳膊上，只剩下皮肉还连着，鲜血混合着肉沫从他的手臂上滴下，落在白雪之上，血液的温度融穿一个小洞。
<br/>    
<br/>        妖族大君毕竟不是自身实力突出的大君，而紫霞仙子，却是古仙门立志成仙的修行者。
<br/>    
<br/>        连续的对战之下，还是紫霞仙子占了上风。
<br/>    
<br/>        紫霞仙子在妖族大君的背后，不紧不慢地跟着，此刻的妖族大君，就像是一只迷失的野兽，而迷失的野兽，最有可能会想要拼个鱼死网破，可是紫霞仙子却是没有任何搏命的心思。
<br/>    
<br/>        现在最好的办法，就是温水煮青蛙，逐渐加热温水，等到水温已经足够杀死青蛙的时候，青蛙根本不会意识到要突然跳出来。
<br/>    
<br/>        紫霞仙子就像是一个熟练的猎人，一步步将妖族大君逼到山穷水尽的地步。
<br/>    
<br/>        陈明迎着风雪行走在大地之上，在远处缓缓的跟着。
<br/>    
<br/>        说实话，这一次，他倒是对紫霞仙子另眼相看，至少紫霞仙子有着足够的耐心。
<br/>    
<br/>        紫霞仙子再次一箭朝着妖族大君射出，无尽的风雪之中，长箭撕裂半天的风雪，如同在一卷画着风雪的画卷中，一笔将其撕开。
<br/>    
<br/>        妖族大君回头，仅剩的右手挥刀，一刀斩在身前的风雪之上，一刀之下风雪倒卷数百丈，将长箭劈断在弯刀之下。
<br/>    
<br/>        妖族大君白色的眸子闪动，“每一位大君，都是大地之上的灾难，每出现一位新的大君，整个大陆的修行者们都会山呼海啸他的名字，就像你们俩一样。而一位大君的死亡，便是一场灾难的终结。我生于这里，也当死于这里，我在这里出生，同样在这里埋葬，这是一个好故事，我知道，故事里都是这么写的，垂死的大君回到自己的出生的地方，安然迎接自己的死亡。”
<br/>    
<br/>        妖族大君的眸子里忽然闪出一抹凶栗的光，“可是那只是故事，我君临十万大山八百年，可是有什么用，这一片大陆，并不是妖族的故乡，我们只是被遗弃在这片大陆上的妖族弃儿，甚至于，我修炼的功法根本不足以支撑我修行到大君，但是我还是成功了！”
<br/>    
<br/>        “在这一片大陆之上，根本没有任何人会去管妖族，所以我八百年修为未进半步，即便是投靠鬼仙又如何，死又如何，有什么区别吗？如果没有鬼仙，妖族还不都是在你们仙魔之间的缝隙中，在这十万大山的穷山恶水中挣扎着求生！”
<br/>    
<br/>        “温水煮青蛙，没有用的，我是狼，不是青蛙，我绝不会安心接受命运的安排！”
<br/>    
<br/>        妖族大君看向紫霞仙子和陈明，“我或许会死，但是你们，已经在劫难逃！”
<br/>    
<br/>        妖族大君看向远方的风雪，笑了笑道：“这妖族的天下，我不要了！”
<br/>    
<br/>        “阴阴九幽，滚滚黄泉，生亦何欢，死亦何苦？我，妖族大君，愿意奉献己身，成为仙尊马前卒！”
<br/>    
<br/>        紫霞仙子忽然暴喝道：“不好！他是要迎接鬼仙的分身降临！”
<br/>    
<br/>        陈明也是眼中露出一抹精芒，鬼仙的分身，他可是见识过的，至少是数千的战力！
<br/>    
<br/>        若是鬼仙真的降临，后果不堪设想，陈明道：“赶紧动手杀了他！”
<br/>    
<br/>        此时此刻，陈明也是不能再继续观望下去了，妖族大君却是哈哈大笑道：“没有用的，杀了我有用吗？鬼仙的力量，本就是死人才能承受的力量，你们，不过是在加快我的变化，做无用功而已！”
<br/>    
<br/>        陈明本以为眼前会出现诡异的景象，没有想到的是，妖族大君的身旁，依旧是静悄悄的，没有任何事情发生，陈明有些疑惑地看向紫霞仙子，“怎么回事？好像并没有发生什么事情啊？”
<br/>    
<br/>        紫霞仙子也是愣住了，“难到说，鬼仙并没有听到他的召唤？”
<br/>    
<br/>        魔宗之中，一具白骨忽然醒来，桀桀道：“是谁在呼唤我？！妖族大君？什么废物！我这一具分身好不容易跑了出来，就浪费在你的身上，开什么玩笑？只有无疆的那具躯体，才能够真正承受住我的力量！”
<br/>    
<br/>        妖族大君愣在了原地，风雪依旧是风雪，并没有什么特殊的事情发生。
<br/>    
<br/>        妖族大君朝天嘶吼道：“仙尊大人，我愿意将我的一切奉献给你，只求你能够赐予我力量！”
<br/>    
<br/>        回应他的只有风雪。
<br/>    
<br/>        陈明挥手，剑龙沐浴着白雪前进，龙舞苍穹，直接冲向妖族大君，妖族大君丝毫不惧，大笑道：“你们给我等着，就算是你们杀了我也没用的，你们就等死吧！”
<br/>    
<br/>        然后剑龙呼啸而过，妖族大君猖狂地大笑，“哈哈哈哈哈，我即便是死了，也依旧回活过来，所谓恶鬼，便是从泥土之中挣扎着爬出，要杀尽这世间的一切生灵！”
<br/>    
<br/>        剑龙从妖族大君的身体之中穿过，将没有任何防备的妖族大君瞬间绞杀成碎片。
<br/>    
<br/>        紫霞仙子很是小心地来到陈明的身旁，“小心一点，要提防鬼仙分身降临，鬼仙邪恶之极，什么事都做的出来！”
<br/>    
<br/>        陈明也是万分警戒，他可是见过一次鬼仙分身的霸道的，身旁剑舞不绝，小心翼翼地查看着周围。
<br/>    
<br/>        “叮，你击杀了妖族大君，你获得了1w点灵识，你获得了1000点声望。”
<br/>    
<br/>        “叮，你击杀了妖族大君，妖族的远征，进度增加30%。”
<br/>    
<br/>        “叮，你击杀了妖族大君，你获得了一次抽奖机会，此次抽检随机不可控，你获得了国器狼王双刀。”
<br/>    
<br/>        陈明愣了愣，嗯，怎么回事？这就杀了？
<br/>    
<br/>        不对，杀了之后应该会复活，还是一场苦战！
<br/>    
<br/>        紫霞仙子和陈明背对背，扫视着周围的变化，但是等了足足三十息，还是没有任何变化。
<br/>    
<br/>        紫霞仙子忽然疑惑道：“怎么回事，现在是什么情况？妖族大君是在埋伏我们吗？”、
<br/>    
<br/>        陈明看向妖族大君留下的一堆血雨，“难不成他真的死了？魂飞魄散了？”
<br/>    
<br/>        紫霞仙子道：“不会吧，意思是鬼仙并没有回应他？”
<br/>    
<br/>        陈明想了片刻道：“可能是妖族大君呼叫的鬼仙并不在服务区。”
