竟然有人闯入，夏夕绾还一副衣衫不整的样子，她吓得轻呼了一声，整个人都躲到了陆寒霆的怀里。
<br/>
<br/>        陆寒霆几乎是瞬间用单臂搂住了她，将她护住，另一只大手快速的抽了出来，他扭头看向了门边。
<br/>
<br/>        门边是那个刚被夏夕绾给赶跑的女人。
<br/>
<br/>        这个女人听说陆寒霆回来了，迅速马不停蹄的赶了过来，她也没有料到会在门边看到这一幕，一身衬衫西裤，昂贵的布料没有丝毫褶皱的男人将纤柔的女孩抱在怀里，女孩身上挂着的那件黑色衬衫都要掉了，整个沐浴间旖旎而暧昧。
<br/>
<br/>        女人一僵，她抬头正好撞上了陆寒霆那双森冷如寒潭的狭眸，她当即从骨头里打了一个寒颤，说话都不利索了，“总…总裁…”
<br/>
<br/>        陆寒霆伸手抓了一样东西，直接砸到了门边，“啪”一声，盥洗台上好几万的砚台被他砸的粉碎，还伴随着他沉冷阴鹜的嗓音，“滚出去！”
<br/>
<br/>        砚台的碎片飞溅到了女人的脸上，带着巨大而锋利的戾气，女人只觉得脸上一痛，已经被碎片划伤了一道血痕。
<br/>
<br/>        承受不住男人的雷霆之怒，女人吓得捂着流血的脸就跑了出去。
<br/>
<br/>        陆寒霆气的不轻，他身边从来没有少过这些诱惑，但是这样下作的还比较少见，他抬手摸了摸怀里女孩的长发，薄唇落到她的额头上，“不怕不怕，我把人赶跑了，我会让她滚出海城的。”
<br/>
<br/>        夏夕绾小脸通红的埋在他的怀里，怎么都不肯抬起头，捏着拳往他身上砸了两下，“都怪你，以后我还怎么见人啊，我不理你了！”
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆留在里面哄了好久，可是陆太太就是不肯理他了，陆寒霆只能先出来将那个女人给处理了，生活助理迅速找来了一个老实年长的阿姨，阿姨进去将休息室整个都打扫好了。
<br/>
<br/>        夏夕绾还待在休息室里，她换好了衣服，里面是蕾丝的小白吊带，外面罩着小香风的紫色翻领开衫，下面高腰牛仔裤，没戴面纱的她肌肤赛雪，唇红齿白，少女的清纯娇妍让她像一朵俏生生的水莲。
<br/>
<br/>        陆寒霆送来了丰盛的早餐，还有她喜欢的各种甜品来赔罪，夏夕绾还是不搭理他。
<br/>
<br/>        这时一串悠扬的手机铃声响起了，来电话了，是李文清院长打来的。
<br/>
<br/>        院长一定是为了厉老夫人的手术方案打来的电话，毕竟厉老夫人的手术就在两天后，时间相当紧迫。
<br/>
<br/>        夏夕绾迅速按键接通，“喂，院长。”
<br/>
<br/>        “绾绾，”李文清院长和蔼亲切的嗓音传来了，“你现在有没有时间过来酒店，我在酒店里订了一个商务套房，子羡已经来了，就差你了，我们研磨一下厉老夫人的手术方案。”
<br/>
<br/>        夏夕绾还没有说话，这时身后传来了稳健的脚步声，陆寒霆来了。
<br/>
<br/>        陆寒霆明显已经听到了电话内容，他的俊脸上并没有什么情绪起伏，只是不动声色的看着她，勾了一下薄唇，“你们研究院商量手术方案都会去酒店？”
