　　第694章
<br/>　　开山立柜，有两件事最为重要，一是人口，二是粮食。没有粮食就养不起兵，没有兵也无法保护粮食。
<br/>　　先让这两者平衡之后，才能想着怎样获得更锋利的兵器，更优秀的马匹。其他事都可以慢慢发展过来。
<br/>　　程大雷可以说兵不血刃拿下凉州城，凉州并未遭遇怎样的破坏。在经过最初的混乱之后，城池开始慢慢复苏，如今已经渐渐步入正轨。
<br/>　　其实拿下一座城池之后，有许多事情要做。城中的富绅大户总是要贡献出一些财物的，对于城中的小民也要安抚，除此之外，还要保持一座城池的正常运转。
<br/>　　最近，有不少大户求见程大雷，为的就是和新任城主处好关系，能在新一轮的洗牌中，依旧立于不败之地。为了达到这个目的，他们不惜送钱送物送女人。
<br/>　　对于这批人，程大雷根本懒得与他们见面。一切都交给刘悲处理，刘悲看上去两袖清风，对财和女人也没有多大兴趣，程大雷还是可以放心他的。
<br/>　　程大雷现在盘算着的，是如何从这些大户中敲出一些粮食来。要根据每家每户的实际情况分析，敲得太少也是可惜，敲得太多难免他们就会狗急跳墙。这其中的尺度，还需要程大雷自己把握。
<br/>　　崔白玉开始协助程大雷完成这项工作，凉州城还颇有一批大户，关于他们的调查资料也已经递了上来。每家每户有多少农田，多少商铺……通过这些盘算出他们有多少家产。
<br/>　　关于这项工作，崔白玉完全可以胜任。她也是第一次接触到这些资料，但用不了多长时间便可以推算出来。小鹿三女见小姐已经开始做事，她们也是无可奈何，只好在一旁配合。
<br/>　　这些事情她们已经做得熟了，很快就能接手程大雷的工作。程大雷顿感压力减轻许多，心里盘算着，自己或许早就该有个助手了。
<br/>　　他本身写字难看，虽然冒名扯了几篇诗，但文笔真的不如何。书信来往这样动笔的事，需要有人协助。这些事，崔白玉当然可以胜任，另外她的脑子也可以用，帮程大雷省了不少的力气。
<br/>　　程大雷待了一会儿，竟觉得无事可做。他站起身，道：“你们在这里做事，我去转转。”
<br/>　　说罢，在小鹿不可思议的眼神中，程大雷竟然就这样离开了。她目光左右看看，书房内的确就只剩下自己四人。
<br/>　　“小姐，他真如此信任我们？”小鹿道。
<br/>　　“信任？”崔白玉冷哼一声，道：“我们连这院子走不出，就算知道得再多又有什么用，他不是信任我们，而是知道我们坏不了什么事。”
<br/>　　说罢，崔白玉叹了口气。自己用这种语气说话，平白显得可笑。既然已答应归附，就必须证明自己的价值，就算想坏程大雷的事，现在也根本没有机会。
<br/>　　不过，程大雷对自己的信任的确有些过分。书房内的确有很多重要材料，算起来都是程大雷的机密，可程大雷就摆在这里，自己想看的话随时都可以看。
<br/>　　本来还想这是不是程大雷的试探，但想着程大雷连刺杀的事都可以容忍，看几眼所谓机密应该不算什么。
<br/>　　透过这些材料，崔白玉了解了更多程大雷目前的情报，之后更是觉得震惊。首先，程大雷怕是帝国各路诸侯中，最喜使用火药的存在。其他诸侯也并不是没有火药，但如此痴迷使用程大雷委实是第一份。
<br/>　　然后是马匹，程大雷拥有大量的马匹，事实上，一匹养熟的战马在帝国绝对是天价，马又格外娇贵些，养一支骑兵队伍需要极大的财力支撑。但很显然，程大雷并没有这方面的担忧。
<br/>　　除此之外，程大雷手中还有许多精悍武将。关鱼、赵子龙、张肥、秦蛮等人都是万人敌的好手。诸多诸侯求一而不得，可这些人对程大雷来说恍惚不要钱一般，只要他需要便随时可以出现。
<br/>　　本以为走进程大雷的机密，就能了解程大雷这个人。可是越了解他，便越来越困惑。
<br/>　　……
<br/>　　程大雷离开书房后，深吸一口气，方才卸去身上的压力。与崔白玉这样的美人同室而处，的确不是一般的折磨。
<br/>　　他在府里转了一圈，发现也无事可做，再无聊程大雷也没有兴趣让乐班为自己吹拉弹唱。索性便去找苏樱，修复一下二人的关系。
<br/>　　苏樱正在账房算账，程大雷去的时候，奥黛琳娜正与苏樱聊天。见程大雷进来，道了声告辞离去。
<br/>　　眼见苏樱脸上有些愁容，程大雷伸出手揉了揉她的眉心，道：“现在人手渐渐够了，也不用你如此操劳，真如果觉得累了，就歇一歇。”
<br/>　　苏樱笑着摇摇头，道：“不做事难道我只每天养花喂鱼，琢磨怎么讨你欢心，我好端端便养废了。”
<br/>　　“你乐意做事也好，只是别太疲惫。”程大雷说了一声，道：“这圣女找你有什么事，她可不是常来？”
<br/>　　苏樱撇撇嘴，道：“有一事正要与你商量，我说了，你可不许生气？”
<br/>　　“什么话，我向来是好脾气，尤其是对你，我怎么敢生气。”
<br/>　　程大雷只要稍微上点心，就能讨得苏樱的欢心。苏樱想了想道：“求人的不是刘悲，而是张肥，张将军麾下有一小将，名叫孙小毛，也是从青牛山过来的。进城之后，相中了一个姑娘，喝醉酒后便做了一些胡闹事。将那姑娘糟蹋了，当天那姑娘就投了井，虽然被捞出来，却依旧是……”
<br/>　　啪。
<br/>　　程大雷的手重重拍在桌上，口中怒哼一声，道：“反了天么，眼里还有没有搁我。”
<br/>　　苏樱倒是不怕程大雷发怒，她道：“那孙小毛求到张肥那里，张肥求到奥黛琳娜那里，奥黛琳娜也就求到我这里，托我向你说句情。毕竟是一场兄弟，很早就跟着你一起起事，从青牛山走到这里，上万里路，大小仗过了无数场，都侥幸活了下来。好歹留他一条性命。”
<br/>　　程大雷紧绷着牙，拳头握紧，半晌忽地长长叹了口气。
<br/>　　自己手下的兵当真需要好好整治一下了。
<br/>(本章完)
