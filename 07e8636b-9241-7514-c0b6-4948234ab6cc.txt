<br/>　　叶少阳听到这里，心中无限感动。
<br/>　　“当时我手艺也学的差不多了，就跟毛道长说，想开一个小饭馆。没想到竟然开成了这样大的一座酒楼，我现在都跟做梦似的。”

<br/>　　当天就有官府的人来找翠云，协助她找店铺，帮她用最便宜的租金拿下了现在的这栋楼，之后的程序也都是一路绿灯，酒楼开起来，生意很好，客人都夸她做的菜好吃，尤其是西湖醋鱼，当时云儿也给她帮忙，后来两个人忙不过来，又招了几个厨师和跑堂的。

<br/>　　说话间，院门被打开了，几个姑娘走进来。
<br/>　　翠云过去打招呼，询问了一些酒店的事，然后介绍叶少阳，几个妹子都好奇地打量他和小九两个，后来都走进了一个房间大门里。
<br/>　　“她们都是店里打杂的，我招来的，有些是乡下的，在城里没地方住，我就让她们住在我这了，也省的我一个人乖寂寞的。”
<br/>　　突然想起什么，问他们：“你们俩这一路上累了吧，瞧我，光顾着自己说话了，走走，我打扫个房间让你们休息一会。”
<br/>　　不由分说，带两人进了一件卧房，又要找被子他们铺床，叶少阳总算找到空闲，拉住她说道：“姐，你不要忙了，我就是过来吃你一顿饭，待会就得走了。”
<br/>　　翠云一下子愣住，“大老远的过来，怎么能就走啊。”
<br/>　　叶少阳笑了，“我是穿越时空来的，你忘了我是未来人啊。”
<br/>　　“我知道……”翠云挠着头，只是她不太能理解叶少**体是怎么来的。但听说叶少阳要走，心中难掩失望，有些伤感地说道：“那我去给你们做点饭吃，哦，你们刚吃过……”
<br/>　　叶少阳很是感动，拉着她说道：“我跟你说，我这次来，就是看看你过的怎么样，我以后想你了就来，很方便的。”
<br/>　　翠云道：“那你以前为啥不来。”
<br/>　　“以前……以前我没找到来的办法啊，现在我找到了，嗖的一下就来了。你就放心吧，以后我没事就过来看你。”
<br/>　　翠云眼睛这才亮起来，嗔了他一眼道：“你可不许骗姐。”
<br/>　　“不骗你，我隔三差五就会来。”
<br/>　　翠云这才欢喜起来，让他们坐着，自己去拿点东西。叶少阳不知道她要干啥，等了一会，翠云拿着一个包袱皮进来了，把门关好，打开来，是几条黄灿灿的金条。
<br/>　　“少阳，这是你的，姐一直给你放着，留着你当老婆本的，你俩还没成亲吧，成亲后处处要用钱，把这些拿上吧。”
<br/>　　叶少阳怔怔地看着这几根金条，心中感慨万千。
<br/>　　翠云不知道他在想什么，接着说道：“那个啥，姐知道你跟毛道长一样，会降妖捉鬼的，但那可不是个好营生，怪危险的，姐现在开大酒楼了，生意很好，每天都有银子进账，姐都给你存着呢，将来你要是做个什么买卖，来找姐要。”
<br/>　　说完又去劝小九，“我说弟媳，你也劝劝少阳，让他找个正经营生，可不要再弄捉鬼降妖那一套了。”
<br/>　　小九哭笑不得。
<br/>　　叶少阳把包袱皮包好，交给翠云，说道：“我要说不要吧，你肯定不答应，那你就先给我放着，等我想到什么好买卖，需要的时候再找你要，只有一条，你如果需要用钱，只管去花，我们是姐弟，我的钱就是你的钱！”
<br/>　　翠云听了这话也很感动，两年不见，叶少阳跟自己还是那么亲，当下笑道：“姐满足啦，要不是你带着我从乡下出来，我还住在那破院子里，别的不说，你看看我现在的房子院子，多干净，收拾一次可以管好些天，以前那房子土坯的，到处破烂，住那样的地方，连过日子的心情都没有。”
<br/>　　叶少阳心中感然，跟她说了好一会话，觉得差不多该走了，于是跟翠云道别。
<br/>　　翠云以姐姐的身份，嘱咐了他很多话，虽然是唠叨，但叶少阳也很享受地听着，最后反过来嘱咐了她一些话，然后表示自己想睡一觉再走，翠云很高兴地帮他们铺床，然后关门出去，让他们好好休息，如果晚上不走再一起吃饭。
<br/>　　等她走后，叶少阳去把窗户关好，小九问他：“你不是真要在这里睡觉吧。”
<br/>　　“不能当着她的面用山海印。”不是怕她知道，而是她一个普通人，还是不要见到太奇幻的画面为好。
<br/>　　叶少阳激活了山海印，生成时空之门，将两人送了回去，片刻之后，两人回到了自己的世界。

<br/>　　这是自己大拇指的血。
<br/>　　每次穿越之前，一定要划破指尖，在山海印上留下血迹，这是徐福严格要求的，为的是能找到自己的世界。
