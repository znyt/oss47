　　　　横渡虚空，对楚风眠来说倒并非是什么难事。
<br/>　　他带着拓跋鼎，寒月漓二人，直接是穿梭在了虚空之中。
<br/>　　一般武者，横渡虚空，只怕是过一段时间，就必须要休息很久，或者吞服丹药，来恢复灵力。
<br/>　　毕竟在虚空中穿行，每时每刻都要消耗大量的灵力。
<br/>　　就算是巅峰圣者，甚至是生死境强者，都不可能受得了一直源源不断的消耗，总有灵力耗尽的一刻。
<br/>　　所以每过一段时间，都要休息，如此一来速度便是会慢了许多。
<br/>　　对一些生死境强者，横渡这段距离，也都至少需要十天半个月的时间。
<br/>　　但是楚风眠有着神脉的加持，灵力无穷无尽，根本不需要休息，每时每刻都在从虚空中穿行。
<br/>　　横渡这片区域，比起其他同样实力的武者至少要省去一半的时间。
<br/>　　如此一来，遇到危险的可能性也会低得多。
<br/>　　“若是这样，五日时间就可以抵达宣州古城了，别有变故就好。”
<br/>　　拓跋鼎在一旁，暗叹一声，没遇见还是极为凝重。
<br/>　　显然是对地魔门，极为忌惮。
<br/>　　这许多人，宁愿是等待传送法阵修复，也不愿意去横渡虚空的原因之一，便是担心其中的危险。
<br/>　　天灾，人祸，都有可能。
<br/>　　遇到天灾的几率，百不足一，倒是还好，但是这人祸，可就不一定了。
<br/>　　像是这地魔门，便是控制了周围无数地区，臭名昭著，专门袭击横渡虚空的强者。
<br/>　　能够横渡虚空的武者，都是有着神力境以上境界的圣者。
<br/>　　这些地魔门的弟子，所打的，正是圣者精血的主意。
<br/>　　地魔门是绝对是魔道宗门，修炼的功法也都是残忍无比，足以利用圣者精血来修炼。
<br/>　　一旦是遇到地魔门的弟子截杀，只怕是能够活下来的几率，十不足一。
<br/>　　像是北邙学院中，每年都会有着不少的弟子，死在了地魔门的手中。
<br/>　　而且大部分的死因，都是因为横渡虚空，被地魔门的弟子截杀而亡。
<br/>　　所以如今拓跋鼎的脸色，一直都是极为苍白，担心，害怕。
<br/>　　地魔门的弟子，可都是极为凶残的魔头，只要是被他们抓住，死都是最轻的，很有可能还会遭遇无数折磨。
<br/>　　“前方，似乎有着几分诡异的灵力波动？”
<br/>　　楚风眠一边穿行，突然眉头一皱，身形停了下来，然后他大手一挥，便是来到了另外一处空间之中。
<br/>　　这处空间，应该是其他人刚刚穿行过的，打开的一条通路。
<br/>　　不过现在看来，这处空间中，却是空无一人，不过在空中之中，却是可以闻到浓厚的血腥味。
<br/>　　“看看到底发生了什么吧。”
<br/>　　寒月漓闻到这血腥味，也不由眉头一皱，她手一挥，一股灵力便是打出，凝聚出去。
<br/>　　在众人的面前，化为了一道冰镜，随着她灵力催动，便是出现了一副画面。
<br/>　　这画面，正是这空间通道中，刚刚发生的事。
<br/>　　只见几尊圣者，正在其中穿行，看方向，显然也是从满元古城中，准备横渡虚空前往宣州古城的。
<br/>　　这群人，有说有笑的，不过很快，周围空间便是出现了一阵波动。
<br/>　　几尊黑袍圣者，突然出现。
<br/>　　“这些的，地魔门的弟子！”
<br/>　　拓跋鼎看着那些黑袍圣者，脸色一变道。
<br/>　　这黑袍，正是地魔门的标志，果然这群人，便是遇到了地魔门的截杀。
<br/>　　这几尊黑袍圣者出现，几招便是将那几尊全部杀死，将他们的尸体，都给带走了，随后水镜破碎。
<br/>　　从这还没有散的血腥味来看，这便是一个时辰以内，发生的事情。
<br/>　　“果然地魔门的人，不会放过这个机会的，已经开始大肆在这附近活动了。”
<br/>　　如今这传送法阵破坏，又有着如此之多准备去北邙学院参加考核的武者。
<br/>　　许多人，为了赶时间，肯定是会选择横渡虚空的，这也给了地魔门无数机会。
<br/>　　足以让他们大肆截杀。
<br/>　　拓跋鼎看完这一切，有些心有余悸的说道。
<br/>　　“这些人，应该都是地魔门中的精英强者，其中为首的一位，更有可能是其中的一尊尊者，实力极强，足以跟我们北邙学院的种子弟子媲美。”
<br/>　　北邙学院的种子弟子，便是就只有力量指数达到十的武者，才可以晋升。
<br/>　　这等实力，已经是步入到了生死境的地步。
<br/>　　不过楚风眠看得出来，这其中为首的那人，其实还未度过生死大劫。
<br/>　　但是这样，才是更加可怕的。
<br/>　　没有度过生死大劫，便是拥有这十点力量指数，这是绝对的天才才能达到的。
<br/>　　“走吧，在这里等着，也有可能被地魔门的人发现。”
<br/>　　楚风眠脸色平静，直接开口道。
<br/>　　在这里等着，地魔门的人也会有可能发现他们，还不如继续走。
<br/>　　他带起拓跋鼎，寒月漓，便是再一次的催动灵力，继续横渡空间，向着宣州古城进发。
<br/>　　三日时间。
<br/>　　楚风眠已经是连续在虚空之中，横渡了三日时间。
<br/>　　换做其他武者，现在只怕是早已经力量虚脱了，但是楚风眠有着神脉，可以无时无刻的补充着灵力。
<br/>　　就是那连续横渡三十日时间，楚风眠也不会觉得累，他依然精神饱满，最好的状态。
<br/>　　这三日时间，楚风眠已经是度过了大半的距离，一路上也遇到了一些地魔门弟子的痕迹，但是却并没有真正遇到地魔门弟子。
<br/>　　“这几日，你体内的玄寒之力，倒是已经足以收放自如了，如今若是动用全力，只怕是巅峰圣者，都可以轻易灭杀了吧。”
<br/>　　楚风眠看着寒月漓，开口道。
<br/>　　这三日时间，寒月漓倒是在一直修炼，并没有提升自己的修为，而是在选择控制这身体中的玄寒之力。
<br/>　　楚风眠利用远古战龙血脉，将寒月漓身上的玄寒之力全部引了出来。
<br/>　　不过力量再强，也必须要会控制才行，这三日时间，寒月漓已经是逐渐将这玄寒之力，彻底掌控了。
<br/>　　“恩，寻常的巅峰圣者，应该可以灭杀，不过遇到那地魔门的尊者，我却是只有把握跟他一战，能不能灭杀了他，还没有把握。”
<br/>(本章完)
