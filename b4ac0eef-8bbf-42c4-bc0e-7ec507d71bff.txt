        &lt;h3&gt;第三卷第二百一十三章  既爱你的皮囊也爱你的灵魂&lt;/h3&gt;
<br/>
<br/>        郝景文的假期是快乐而惬意，陪着陈思宇走亲访友，显示着他的所有权，从乡下别墅回到城里，又招呼小伙伴们聚在一起热闹了一番。
<br/>
<br/>        原来的郝景文是一个人，现在还带着女朋友陈思宇，阿莫将几个分部的人都联系好了，他们的这个省份现在阿莫已经是总经理，管理着整个省的运营，年轻人血气方刚，虽然是快递行业但是大家热情不减，整个物流他们拿到三分之二，当然也是因为他们的口碑好，服务好，社会在发展，每个行业都会滋生一系列的需要，人们的消费水平提高了，相应的需求也会提高，服务态度就尤为重要，他们已经树立了良好的企业形象，打开了人们的认知，给人们留下了美好的印象，大家只要发货就会想到他们。
<br/>
<br/>        经过几年的发展，他们的app在不停的更新和改进中，已经能和各大网站进行对接，下单快捷迅速，他们的库房基本不会积压货物。
<br/>
<br/>        这一次回家不但是过新年，还有一项任务要交代给阿莫，今年要将租赁服务业的质量提升起来，利用旅游旺季将业绩做起来，陈子昂的公司在三月份会启动上市步伐，他们都是公司大股东，而且陈子昂也说了要去甸城，后期他还得兼顾着那边的一些事务。
<br/>
<br/>        他带着陈思宇参加大家的聚会，陈思宇说：“你这都是工作上的事情，带着我不好吧？”
<br/>
<br/>        “有什么不好，你是将来的老板娘，不带着你我带着谁？”
<br/>
<br/>        陈思宇小脸通红，心里甜滋滋的。
<br/>
<br/>        原本她那点不自信到了郝家之后就消失殆尽了，郝景文的父母亲人真的很好，一点家族的优越感都没有。而郝景文带着她走亲访友，完全是未来媳妇儿的模样，让她心里都是满足和幸福感。
<br/>
<br/>        阿莫在酒店定了三桌，大概三十多个人，大家聚在一起听郝景文安排今年的任务，年轻人在一起，说话也没有那么拘束。
<br/>
<br/>        郝景文直接开门见山的说：“大家都是我郝景文的兄弟，从一开始到现在不离不弃，当初别人不看好这个项目的时候，是大家力挺我，能走到今天的一步全是大家的功劳，我们还有更好的事情要做，我们也有更高的平台需要我们去展示，去努力。大家有没有信心？”
<br/>
<br/>        “有！”在座的三十多个年轻人整齐划一，气震山河。听得陈思宇热泪盈眶，她怎么就没有发现郝景文还有如此帅气的一面？只看到了他那张帅死人的脸，原来他还有比颜值更高的领导值，反正陈思宇已经找不出什么词还形容郝景文了，只能这样想。
<br/>
<br/>        “有信心，那么我告诉大家，今年将会有比我们现在的这个事情更高大上的业务要做，我需要大家共同努力，将这个事情做起来，后期会让人专门过来培训，还是由阿莫负责，我们是一个团队，一个整体，无论我们是处在一个什么样的状况，请记得我们的使命是服务大众，服务我们的父老乡亲。切不可骄纵，不可忘本！”
<br/>
<br/>        “是，我们的使命是服务大众，服务我们的父老乡亲！”
<br/>
<br/>        简单的几句话似乎是一种无行的力量，人们都是需要向上的动力，那一些积极的充满正能量的概念总是会带给人们一种阳光的精神。
<br/>
<br/>        我们都是普通人，都是平民百姓，都是为了温饱奔跑在路上的人，但是，只要你给予我使命，给予我力量，我们就是一股拧着绳子努力向上的群体，我们觉得我们不是为了我们自己而活，我们柔弱的双肩也是会担起责任，做社会前进的垫脚石。
<br/>
<br/>        陈思宇两眼冒着小星星，酒不醉人人自醉，看着郝景文，她感觉她的心已经沉浸在了浓浓的酒香里，不由自主的走上前，拉着郝景文的手：“我们大家一起奋斗，为我们的将来，为了我们的梦想！”
<br/>
<br/>        她只是一个普普通通家庭的孩子，从小到大接受的教育也是能找一份好工作，有一个安逸的小家，可是现在，她突然觉得她的眼界不应该局限在小家上，面对着郝景文眼前的这几十个兄弟，面对着这几十个兄弟后面看不到的无声的奉献的人们，心，充实起来了，她要和郝景文一起奋斗，不仅仅是俩个人恩恩爱爱，而是要更多的家庭有资本恩恩爱爱，过上富足的生活。
<br/>
<br/>        像她现在所处的陈子昂的公司和郝景文的公司，他们是私人企业，他们同样都担负着一种义不容辞的社会责任，人，活着不仅仅是吃饱穿暖！
<br/>
<br/>        这一场聚会刷新了陈思宇的认知，在她的心里有一颗种子悄悄的发芽，此后的人生，陈思宇和郝景文一直努力的做着各种有益的活动，并且乐此不彼。
<br/>
<br/>        年轻的心被热情点燃，在奔梦的路上努力的奔跑，无论我们处在什么地位，什么职业，我们都有一颗快乐上进的心。
<br/>
<br/>        郝景文拉着陈思宇的手坐回到座位上：“今天像极了老板娘！”
<br/>
<br/>        陈思宇：“爱情让人忘记烦忧，梦想使人努力向上，文文，感谢上苍让我认识如此优秀的你！”
<br/>
<br/>        郝景文：“这话说的，好像你今天才认识我？”
<br/>
<br/>        陈思宇：“之前是因为你的皮囊，现在是因为你的灵魂！”
<br/>
<br/>        郝景文：“那你到底是馋我的皮囊还是馋我的灵魂？”
<br/>
<br/>        陈思宇看着郝景文认真的回答：“先是皮囊，因为好看的皮囊总是让人赏心悦目，现在是灵魂，因为有趣的灵魂仅此一个，所以我既馋你的皮囊也馋你的灵魂！”
<br/>
<br/>        郝景文哈哈哈大笑，能把情话说的如此认真的只怕只有他的小甜心陈思宇了：
<br/>
<br/>        “你不但拥有好看的皮囊你也拥有有趣的灵魂，所以我们是相互吸引，我馋你的所有！”
<br/>
<br/>        阿莫走过来敬酒听见了郝景文的话：“馋了就吃啊，桌上这么多菜，不吃都是浪费！”
<br/>
<br/>        陈思宇和郝景文相视一笑！
<br/>
<br/>        在福建的日子很快就过去了，郝景文得带着陈思宇回扬州，已经将人家的女儿拐回来好多天，再不回去实在是说不过去，而他的爸爸妈妈早都准备了很多礼品，让郝景文带给陈思宇的家人。
<br/>
<br/>        从扬州回来他们就要进入到紧张的工作中，今年的目标还很大，一切都是未知的，需要他们更加努力，更加用心！
