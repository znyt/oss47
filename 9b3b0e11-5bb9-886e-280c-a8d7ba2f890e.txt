        “你知道我的师父是谁吗？是国服第一的‘余寒犹厉’噢！我超厉害吧！曾是他唯一的女徒弟！”连星儿自顾自地跟明司寒炫耀。
<br/>
<br/>        明司寒嘴角微扬，只笑不语。
<br/>
<br/>        从她口中听到她夸他，他心里还真是挺美滋滋的。
<br/>
<br/>        “明司寒，你说我师父是不是交女朋友了？他不仅跟我解除了师徒关系，还删除了我的好友，甚至把我拉黑了！”连星儿紧接着又郁郁寡欢道。
<br/>
<br/>        明司寒忍不住腹语起来：“明明是你脚踏两条船，被你师父抓包了！”
<br/>
<br/>        “你说什么？”连星儿侧过头去，一脸疑惑地看着坐在身旁的明司寒。
<br/>
<br/>        明司寒这才一本正经地说道：“这种事情，我哪知道，说不定是你得罪了你师父，惹你师父不高兴了。”
<br/>
<br/>        “我跟他又没见过面，也没开过语音，更没在线下聊过，我哪能得罪他啊？”连星儿嘟囔道。
<br/>
<br/>        明司寒会心一笑：“你可以用小号加他，问问他。”
<br/>
<br/>        “我没小号啊！”连星儿一本正经地回答。
<br/>
<br/>        “你居然没小号？”明司寒有些不可思议地看着连星儿。
<br/>
<br/>        “我要小号做啥啊？”连星儿反问。
<br/>
<br/>        明司寒忽然觉得，其实自己跟这丫头之间还是有代沟的。
<br/>
<br/>        “那当我没说。”明司寒只好先妥协。
<br/>
<br/>        连星儿没多想，似乎也没有要重新注册小号去问缘由的意思，两人随后只能一起双排静默地打游戏。
<br/>
<br/>        两人双排，明司寒刺客打野，连星儿法师中单，强强联手，一起打了十二局，也赢了十二局，两人的默契，在当面沟通之下，几乎超神。
<br/>
<br/>        连星儿丝毫没有怀疑明司寒的身份，只是觉得这游戏里果然是“天外有天，人外有人”。
<br/>
<br/>        玩了三个小时后，一个上午就这么过完了。
<br/>
<br/>        明司寒本想带连星儿去下馆子吃午饭，但是连星儿执意要回家，他不得不先送这丫头回去，自己最后才一个人回到家中。
<br/>
<br/>        一进家门，就闻到饭菜的香味，让明司寒心里莫名其妙地拉起了警钟。
<br/>
<br/>        当他准备向厨房走去时，只见起初那个女人，身上系着围裙，端着一盘烤鸡从厨房门内走了出来。
<br/>
<br/>        “回来了啊！可以吃饭了噢！”女人此刻的脸色，比早晨那个时候要好了许多。
<br/>
<br/>        明司寒却一脸不悦地皱了眉头质问：“你是怎么进来的？”
<br/>
<br/>        “楼上房间的窗户，我特意没关。所以，就爬进来了啊！”女人会心一笑，将烤鸡端上餐桌后，解开了身上的围裙，搭在了一旁的椅背上。
<br/>
<br/>        明司寒随之坐了过去，看着坐在自己对面的女人，有些不耐烦地问：“你到底想怎样？”
<br/>
<br/>        “自我介绍一下，我叫裴萱妍，谢谢你的救命之恩，无以为报，我会以身相许。”裴萱妍落落大方地说道。
<br/>
<br/>        明司寒扶额，一脸不想跟她说话的姿态。
<br/>
<br/>        “不过，既然你已经有未婚妻了，那么我以身相许就显得有些多余。所以，从今以后，我会护你周全。”裴萱妍红唇微扬。
<br/>
<br/>        明司寒放下手来，看着裴萱妍，冷冷地说道：“你若是真的想报恩，就立即马上从我眼前消失，永远不要再出现在我面前即可。”
<br/>
<br/>        “是因为你已经有喜欢的女孩的缘故吗？”裴萱妍不痛不痒地反问，不仅如此，反而像是在明司寒的伤口上故意撒盐地接着说道，“可是，据我观察，那个叫‘星儿’的女孩子，对你没那种感情噢！”
<br/>
<br/>        “我和她，有没有感情，不重要。反正，是两个豪门之间的联姻。没感情，于她而言，更好。”明司寒虽然被说中了心事，心里不爽，但也不会在一个外人面前表现出来。
<br/>
<br/>        裴萱妍会心一笑：“既然如此，以后我就可以放心大胆地待在你身边了。你放心，你的家，我是不会白住，洗衣服做饭，我样样都行，一定比那位千金大小姐要强。此外，如果你还希望我帮你暖床，也是可以的。”
<br/>
<br/>        “看样子，你是赖上我了？”明司寒额冒青筋。
<br/>
<br/>        裴萱妍微微一笑，心里坦荡地点了点头。
<br/>
<br/>        明司寒已经见识了这个女人阴魂不散的本领，冷冷地接着说道：“既然你这么想住在这里，我就把这别墅让给你住好了。”
<br/>
<br/>        “？！”裴萱妍还未反应过来。
<br/>
<br/>        明司寒便已经起身，连午饭都未吃，便出了门。
<br/>
<br/>        下午，他约了连星儿一起返校，也因此引起了小轰动。
<br/>
<br/>        学校里不知何时疯传，说连星儿跟明司寒在一起了。
<br/>
<br/>        明司寒这边不否认也不承认，连星儿却苦不堪言地极力否认也没人相信。
