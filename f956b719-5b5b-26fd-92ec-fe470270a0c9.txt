　　“你，有资格让我全力出手！”
<br/>　　这一句话一出，在场的无数老祖，都是愣住了。
<br/>　　在这个时候，楚风眠居然还有底牌，或者说，楚风眠居然还没有动用全力？
<br/>　　一尊生死境武者，居然在隐藏实力，对付无暇之境的老祖？
<br/>　　就连夜众星在听到这一句话的时候，都是目光急速收缩了一下，显然也是忌惮无比，但是很快，夜众星便是露出了几分冷笑。
<br/>　　楚风眠就算是有着再多的奇遇，在妖孽，他终究只有生死境八重的境界。
<br/>　　一尊生死境的武者，如何可以跟他抗衡！
<br/>　　“死到临头，你以为你还能嘴硬！死吧！”
<br/>　　夜众星一边说着，一边心神一动，他的刀锋便是更快，更加锋利，直接冲着楚风眠的胸口，斩杀而去。
<br/>　　“无上剑气，出！”
<br/>　　面对这刀锋的袭来，楚风眠只是淡然的吐出了几个字。
<br/>　　从他的身边，一道道剑气，突然出现，这剑气出现的一刻，楚风眠整个人的样子，看起来便是跟刚刚，截然不同。
<br/>　　刚刚楚风眠所展现出的剑意，便是锋利无比，人人看见，都不由的想要退避三舍。
<br/>　　可是现在，这无上剑气出现的一刻，楚风眠的剑意，却是要比起刚刚还要恐怖十倍，百倍，这剑意，简直是达到了天下无双的地步。
<br/>　　在无数老祖的心中，都从未想到过，一个人的剑意，居然会达到如此恐怖的地步。
<br/>　　尤其是这剑意的主人，还仅仅是他们北邙学院的一尊弟子，一个生死境的武者！
<br/>　　“破！”
<br/>　　楚风眠冷喝一声。
<br/>　　这四百道无上剑气，突然是合为一体，化为了一道剑阵，猛然上前刺去。
<br/>　　这夜众星斩来的刀锋，在这瞬间，便是都被击碎。
<br/>　　在这无上剑气的面前，仿佛不管是什么力量，都会被直接斩灭。
<br/>　　“这是什么东西？”
<br/>　　夜众星看到这四百道无上剑气的一刻，目光中也是深深的忌惮。
<br/>　　他也看得出，现在楚风眠身上的气势已经是发生了翻天覆地的变化，而且不单单是气势，楚风眠身上的实力，也几乎暴涨了数十倍，甚至是上百倍的地步。
<br/>　　这一切的变化，正是因为这无上剑气的突然出现。
<br/>　　这任何一道无上剑气，夜众星都能感觉到其中恐怖的剑意，蕴含有着登峰造极的剑意，这样的无上剑气，居然是足足有着四百道。
<br/>　　“夜众星，你以为你真的是我的对手？我不过只是借你的手，来看看我现在的实力到底是达到了什么地步罢了，现在，你已经是没有任何的意义了，所以你可以死了！”
<br/>　　楚风眠看着夜众星，冰冷的开口道。
<br/>　　要杀夜众星，楚风眠一开始就可以展现出无上剑气，彻底的斩杀他，不过楚风眠却是没有那么做。
<br/>　　因为楚风眠也想要看看，他现在的实力到底是达到了哪一步，所以他才没有动用无上剑气。
<br/>　　现在看来，面对生死台阶前三重的武者，楚风眠就算是不动用无上剑气，都可以将其击败，而若是动用无上剑气。
<br/>　　这夜众星，都不是他的对手。
<br/>　　“嗖！嗖！嗖！”
<br/>　　随着楚风眠开口，这四百道无上剑气，便是再次变化，全部融入到了楚风眠手中的剑锋之中，令他的剑锋，要比起刚刚要恐怖了数百倍的地步。
<br/>　　“轰隆！”
<br/>　　这一剑，凌空斩出，周围的层层空间，都被彻底的斩灭。
<br/>　　同时这一道剑光，已经是锁定了夜众星的身躯，就算是夜众星想要逃走，都无法做到。
<br/>　　“杀戮天地！”
<br/>　　夜众星见到这剑光的袭来，脸色都是大变，隐约之间，他都可以看得出来这的剑光的可怕之处。
<br/>　　现在的楚风眠，已经是跟刚刚，既然不同。
<br/>　　他手中的长刀，再次动了，漫天杀意，全部都是融入到了他的刀锋之中，又是一刀，突然斩了出去。
<br/>　　这一刀便是他的无上杀招，一旦打出，漫天血海突然出现，这血海的力量，向着楚风眠碾压而去，以这漫天血海，恐怖的杀意，要将楚风眠彻底碾碎。
<br/>　　“没有用的。”
<br/>　　楚风眠看到夜众星的举动，嘴角只是冰冷一笑。
<br/>　　他的剑光，便是贯穿如虹，一剑刺了过去，这漫天血海，在这一剑的面前，都是被直接分开，在空中，这血海几乎是无穷无尽的庞大。
<br/>　　可是在这一道剑光的面前，却还是如此的不堪一击，直接被分开。
<br/>　　这一剑，已经是刺向了夜众星的胸口。
<br/>　　“救我！老祖宗救我！”
<br/>　　夜众星在危难之际，突然冲着北邙宫的深处，大喊起来。
<br/>　　在这生死关头，夜众星已经是顾不得自己副院长的颜面了，他可不想要死在楚风眠的手中。
<br/>　　这一幕，让周围老祖看到，都是倒吸了一口凉气。
<br/>　　北邙学院的副院长，拥有杀戮之体的夜众星，都被楚风眠逼迫到了如此地步，甚至是都要被斩杀，这一切，居然只是一尊生死境武者做到。
<br/>　　就算是在九域的历史之中，像是楚风眠这样的绝代天才，都没有出现过。
<br/>　　就在众人这议论之际。
<br/>　　从北邙宫深处，似乎是一尊极为古老的存在，突然听到了夜众星的话，苏醒了过来，猛然一掌，便是冲着夜众星的面前轰击了过去。
<br/>　　这一掌，来到了夜众星的面前，将夜众星直接挡在了后面。
<br/>　　“得饶人处且饶人，小辈，众星毕竟是北邙学院的副院长，今日不可能让你杀了他，住手吧。”
<br/>　　这声音从北邙宫之中传来，虽然极为苍老，但是其中的声音中，却是蕴含这一种无法质疑的声音，每一句话，都是至高无上的命令一般。

<br/>　　楚风眠听到这声音的一刻，自然也明白过来，这声音的主人，绝对是一尊真正的老祖宗一般的人物。
<br/>　　比起夜众星的境界，不知道是要恐怖了多少倍的一尊老祖宗，是北邙学院之中隐藏的超级老古董。
<br/>　　“我是谁不重要，老夫出现，不过是告诉你，人外有人，天外有天，你虽然有着几分实力，但是还是尽快收手为好，免得招惹祸端。”
<br/>　　这古老的声音，再次传来，似乎是没有想到，楚风眠居然还敢质疑他的话，言语之中都蕴含了几分愤怒。
<br/>　　“放人吧。”
