　　武牧毕竟是上了年纪，身体机能不能跟年轻人比，从登机梯上走下来的时候，需要人搀扶着才行。
<br/>　　武牧走下飞机后，对赵天宇说道：“小赵，那我们就先走了，这东西一直都是我心中的一根刺儿，不把它给攻破了，我睡觉都不踏实。”
<br/>　　赵天宇也知道武牧是那种盯准了就一定要把东西研究透彻的性子，所以也就不再多说什么，只是叮嘱武牧一定要注意自己的身体。
<br/>　　武老可是国家著名的生物毒物专家，是享受国务院特殊津贴，得到过两任一号首长亲自接见的，可不能大意。
<br/>　　武牧的科研小组离开后，机场上也就剩下战术中队这些人了。罗昊见周围也没有其他人，就向赵天宇询问起慕容飞的情况。佟千雪的意外流产对慕容飞没有造成影响是不可能的，如果造成的影响严重，慕容飞就必须要接受相关的心理评估，确认他是否还适合在作战部队服役，如果慕容飞心理评估不过关，那他就只能被从作战部队强制退役。
<br/>　　“无常情况目前来说还算正常，没有异常表现，我会盯着他，暂时不会安排他作战行动，必要的时候，我会要求他接受心理评估测试。”
<br/>　　赵天宇跟罗昊坐同一辆车，罗昊和赵天宇都坐在副驾驶上，赵天宇把慕容飞和佟千雪的情况详细地跟罗昊说了一遍，“佟千雪那边我也打算安排她到基地的医疗中心工作，其实我跟她第一次见面的时候就有这个打算，但是当时她自己放不下她在杭山市儿童医院的工作，所以我尊重了她的选择，不然她也不会遇到危险。”
<br/>　　“那她的身体？”罗昊对赵天宇问道。
<br/>　　流产对女人身体的伤害很大，当时云熙因为生产的时候遭遇到了意外情况，从而落下了病根，佟千雪这次流产对她造成的影响不会小，更何况不管是在普提查还是在布克尔特那边，医疗卫生条件都很差，这很容易落下病根。
<br/>　　说起佟千雪的情况，赵天宇忍不住叹了口气，说道：“他们回国后，我就安排了最好的医院让佟千雪去接受检查，我也亲自去问过情况，佟千雪的情况不是很好。她是在遭遇到外力冲击下导致的流产，再加上流产后没有得到专业的护理，对她的身体造成了很大的影响，而且她本身年纪也不小，最坏的情况，就是她有可能终身不孕。”
<br/>　　“当然，这也只是说她有可能，并不一定。如果经过细心的调理和养护，她还是有可能再次怀孕，成为妈妈的。”
<br/>　　“无常，他知道这个情况吗？”罗昊对赵天宇问道，他现在很担心慕容飞的情况。
<br/>　　赵天宇摇了摇头回答道：“他暂时还不知道，我也不敢让他知道，不但他不知道，就连佟千雪也不知道。不过佟千雪本身就是医生，这种事情要瞒住她几乎是不可能的，但是她很默契的保持了缄默，应该也是担心慕容飞的情况。”
<br/>　　“可是这个情况瞒得了无常一时，瞒不了他一世啊。”罗昊皱眉说道：“这件事情总归是要跟无常挑明的，而且必须是要在他归队之前说明，这直接影响到他能不能继续在战术中队服役。”
<br/>　　赵天宇点了点头，沉默不语。罗昊能想到的事情，赵天宇又何尝会想不到，只是这跟赵天宇、云岳和朱致阳一起制定的代号“追魂”的行动有关。
<br/>　　“追魂”行动，是“毒皇后”玛丽科娃跟小丑的关系被那张从仙岛夫硬盘中所发现的照片曝光，并且抓捕到李霄力，以及“crystal-ball”事件曝光后，由云岳所提出来的。
<br/>　　玛丽科娃跟小丑之间必然是存在着特殊的关系，而小丑这个家伙又藏得很深，现在“暗月”活动基本都看不到他的身影，所以要想找到小丑，就必须要牢牢抓住玛丽科娃这根线！
<br/>　　李霄力是玛丽科娃手下的一个马仔，他不可能会对玛丽科娃有很多的了解，而玛丽科娃对李霄力最多也只是知道有这么个人，但是却并不知道他详细地情况。
<br/>　　马仔也有马仔的价值，云岳制定的“追魂”行动就是要以李霄力作为突破点来进行运作。这个行动经过三个人的评估，风险程度很高，稍有不慎，执行行动的人员就会遭遇不测，行动失败，这也是为何这个行动计划被提出之后，经过一次又一次的完善，却始终没有被执行的原因所在。
<br/>　　也正是因为介于这个原因，赵天宇并没有像罗昊提及“追魂”行动。
<br/>　　“这次行动的情况，我已近从无常那边有所了解，你们做的不错。”赵天宇对罗昊说道：“剩下的事情我会跟上面进行反馈，至于要如何处理，那就是他们的事情了跟我们没有关系。不过这次能跟瓦斯汀争取到这样一笔大额交易，你们可谓是功不可没，我们国家的铜矿不足，而铜又是重要的战略资源，所以能敲定下这笔订单，几个首长都很高兴，要亲自嘉奖你们。”
<br/>　　还没回国的时候，罗昊就把瓦斯汀的那些话跟赵天宇进行了转述，而赵天宇直接通过电话跟赵国邦进行了汇报。
<br/>　　“嘉奖就免了。”罗昊笑了笑说道：“个人一等功奖章，其他人奋斗一辈子都未必能拼到的东西，我都有仨了，总不能直接给我挂个将星吧？还是来点实际的，给我放几天假把。”
<br/>　　“就知道你小子会这么说。”赵天宇用手指点了点罗昊，说道：“一会儿直接在小区门口放你下来，小熙已经回家了。只是你身上这样，回去又要被小熙唠叨，想好了要怎么说吗？”
<br/>　　“最多就是被她埋怨几句，反正都习惯了。”罗昊看了眼自己手臂上的绷带，说道：“行动哪有不受伤的，小熙也就嘴上埋怨几句，没事儿。她总不能因为这事儿就让我睡客厅沙发吧，我可是伤员，她不能这么对我。”
<br/>　　“这是你们夫妻间的事情，跟我没关系。”赵天宇一副事不关己的样子，说道：“你们夫妻俩的事情，你们自己去解决。”
