
<br/>　　<br/>　　    两对男女回来后心情挺沉闷的就钻进了帐篷里，三人睡一顶，但王惊蛰却没进去，他坐在一堆篝火旁说是要守夜，权志龙挺鄙夷的跟其他人说道：“这孩子脑袋秀逗了吧……”<br/>　　<br/>　　    之前登了一天的山，体力耗费的都挺大，人钻进帐篷里之后，很快就全都睡了过去，守夜的王惊蛰状态还算行，但一晃几个小时过去后，他也泛出了困意。<br/>　　<br/>　　    凌晨一点左右，是一个人最容易犯困的时候，这个期间几乎你困了后上下两眼皮一合就很容易睡死了，白天的时候王惊蛰也赶了不少的山路，夜间他的精神也一直在绷着，笼统的讲他可能比别人累多了，只是没人知道罢了。<br/>　　<br/>　　    不知道睡了多久，忽然间隐约有一阵阵哭哭啼啼的动静传了过来，那声音虽然断断续续的但听的很清楚，并且哭声相当的怪异，听起来有点像是人在哭，可声调却不太对，显得特别尖锐和刺耳，就像是压着嗓子在哭一样，让人听着耳膜都很难受。<br/>　　<br/>　　    王惊蛰是最先醒过来的，他虽然睡得很实诚但警惕性在呢，哭声一想起来没多久，他就从熄灭了的篝火旁一跃而起，然后谨慎的盯着树林那边，哭声就是从那里传过来的。<br/>　　<br/>　　    一顶帐篷里，李红拂和王清雪睡在一起，她突然间“唰”的一下就睁开了眼睛，侧着脑袋倾听了一会后，就打开了帐篷，睡在旁边的王清雪揉了揉眼睛，迷糊的问道：“红拂，怎么了？”<br/>　　<br/>　　    “嘘”李红拂伸出一根手指示意她别吭声，她爬出了帐篷就看见王惊蛰站在河边目视远处，脸上的表情很浓重。<br/>　　<br/>　　    “出了什么事了，这什么动静……”李红拂走过去后问道。<br/>　　<br/>　　    “孙儿啊，你死的好惨呢，是哪个挨千刀的将你活活的给打死了，眼睛都瞎了”断断续续的哭声还伴随着几声尖锐，愤怒的哀嚎，语调刺耳抑扬顿挫感十足，在这黑夜里清晰的传过来后，让人头皮都麻了。<br/>　　<br/>　　    王惊蛰皱眉说道：“这是在哭丧，有人，也不是，有东西死了”<br/>　　<br/>　　    李红拂愕然一愣，说道：“这是精怪？还是……”<br/>　　<br/>　　    哭声的动静越来越大，蔓延在河边的山涧里，片刻过后帐篷里睡着的人全都醒了，徐闯和权志龙还有赵文两个跟另外两个女生钻出来后，听见这动静脸色都被吓白了。<br/>　　<br/>　　    徐闯结巴的问道：“大半夜的，这是谁在哭啊，好像谁家死人了呢？”<br/>　　<br/>　　    “孙啊，你死的好惨呢……”哭声传来的时候，翻来覆去的就是那么几句，而且语调声音听起来十分的怨毒，听着就跟牙咬切齿似的。<br/>　　<br/>　　    大半夜的，有人莫名其妙的在哭丧，要是能看见确实是人在哭那还好点，只闻哭声却不见人，这场景太渗人了。<br/>　　<br/>　　    王惊蛰眯着眼睛看向赵文亮，徐闯还有王清雪和小雯，之前他们四个就是从那边的林子里过来的。<br/>　　<br/>　　    此时的四人脸上印堂一片漆黑，这明显是要撞邪了的征兆。<br/>　　<br/>　　    “该来的，总归还是要来的”王惊蛰叹了口气，问向了徐闯：“你们之前在林子里，遇到什么事了？”<br/>　　<br/>　　    “啊？”徐闯被问的有点发愣，回过神来后摇头说道：“我们呆了一会就出来了，没碰到什么事啊”<br/>　　<br/>　　    “我说有事就肯定有事，不然这哭声怎么会奔着你们来的？好好想想，在树林的事一点一滴都说出来，千万别漏了什么，特别是有没有碰到什么东西”<br/>　　<br/>　　    被王惊蛰这么一说，徐闯他们四个顿时都被吓麻爪了，特别是两个女生攥着手嘴唇都哆嗦了。<br/>　　<br/>　　    徐闯想了想，看着赵文亮说道：“碰见的那只大耗子？”<br/>　　<br/>　　    赵文亮说道：“我们在树林里碰见一只黄色的耗子，挺大的，尾巴也很长，它当时把我们吓了一跳，徐闯还踢了它一脚，没想到那耗子居然不怕人也不跑，还冲着我们呲牙咧嘴的，于是我跟徐闯就一人拿了一根棍子，把它给打了……”<br/>　　<br/>　　    王惊蛰幽幽的叹了口气，有些头疼的说道：“理工大学的，这些年你们的书都读到哪去了，连耗子和黄鼠狼都分不清么？你们家耗子能大到这种程度，尾巴能那么长，就是西游记里的老鼠精也长不到这样吧”<br/>　　<br/>　　    王惊蛰一说，徐闯和赵文亮都顿时明白了，他拍着脑袋说道：“我说的怎么看着有点眼熟，以前小时候家在农村，我就见到过”<br/>　　<br/>　　    李红拂的脸顿时就阴了，拧着两条长长的眉毛说道：“你们把它给打死了？”<br/>　　<br/>　　    徐闯摇头说道：“不知道啊，就打了几下，然后它跑没影了，我们就追了过去”<br/>　　<br/>　　    王惊蛰说道：“要是你们打死的那就麻烦了……”<br/>　　<br/>　　    狐黄白柳灰中，黄皮子是最记仇，报复心特重，也是最容易成精怪的，特别是在东北兴安岭一带深山老林下的村子里，关于黄皮子的传说，一般老人都听过好几件。<br/>　　<br/>　　    几十年前的时候，一户农家养了不少的鸡鸭鹅，有天晚上发现家畜被祸害死了好几只，死的家畜全都是被从肛门后面把内脏了掏了出来，鸡毛鸭毛到处都是，有的被啃的支离破碎，有的则是被咬死就完了，一连两天晚上都是的，后来这户农家的人在第三天晚上守夜然后撞见了山上下来的黄皮子过来祸害家畜，当场就把这黄皮子给打死了。<br/>　　<br/>　　    没想到就在这天晚上，从山上下来一百多只黄鼠狼，在这户农家的周边就跟黄皮子围城似的，一百多只全都冲到了家里，把所有的家畜全给咬死了。<br/>　　<br/>　　    “什，什么麻烦啊？”徐闯脸色雪白的问道。<br/>　　<br/>　　    “谁打死的就缠着谁不放……”王惊蛰指着徐闯和赵文亮说道：“你俩打死的，这黄皮子就天天找上你们，一直到把你俩给耗死了为止。”
