　　第417章
<br/>　　皇宫大门被攻破，接下来面对的是辉煌气派的宫殿，以及精锐的鱼龙卫。
<br/>　　起义军和鱼龙卫正面碰撞。
<br/>　　羽箭、弩机、投石机、火车……正义教拿出了攻城的势头，攻击这支号称帝国最强的军队。羽箭穿透喉咙，投石机将脑袋砸成稀巴烂，燃烧的火车冲进人群，带来阵阵惨叫。
<br/>　　人是强大的生物，能造出强大的武器，然而人又是那样脆弱，在武器前不堪一击。
<br/>　　然而，这还并不是最可怖的，最可怖的依旧是人。
<br/>　　林少羽黑袍黑甲，座下一匹黑得发亮的战马，当然，杀戮进行到现在，他身上已挂了太多鲜血，有敌人的也有自己的。黑色披风在阳光下浮现一种诡异的紫色。
<br/>　　他一杆铁枪，所到之处尽皆披靡，带着手下精锐直扑皇宫正殿。如一柄锐利的矛，突破密密麻麻的人群，身后大旗飘展，义字当头。
<br/>　　人和马冲进金殿，鱼龙卫乌压压冲来。
<br/>　　林冲长枪挡开敌人，道：“首领，你去做事，我拦住他们。”
<br/>　　马蹄踩在金砖上，发出哒哒的响声，此刻听上来感觉无比沉重。伴随着哒哒的马蹄声，林少羽骑马步入金殿，落日的余晖照下来，将他的影子投在地上，狠狠拉长。
<br/>　　明帝坐在龙椅上，抬起头，目光盯着林少羽。于此同时，林少羽的目光也看着他，两人的目光在空中撞在一起。
<br/>　　一个苍老，一个年轻。
<br/>　　一个行将就木，一个恍若新生。
<br/>　　大殿是如此空旷，却恍惚已容不下目光碰撞掀起的波澜。
<br/>　　“逆贼，找死！”
<br/>　　刘阿吉破口大骂，身子像离弦的剑弹射出去，手中的拂尘拂过空气时，发出锐利的沙沙声，很像剑拂过冰面的声音。
<br/>　　大殿很是空旷，从龙椅到金殿正门，有极远的距离，但刘阿吉苍老瘦弱的身躯，此刻却矫健如白猿。如果白元飞在此，怕会惊得瞠目结舌，这个老太监的轻身功夫，绝对不弱自己。
<br/>　　你以为他是个太监，他其实是个战士，一个很强的战士。
<br/>　　刘阿吉拥有极好的身手，在大内皇宫，甚至放眼整个帝国都是屈指可数的高手。他在帝王之侧，一身功夫却是来源于江湖。
<br/>　　江湖对决和战场厮杀是两回事。战场上讲究大开大合，用沉重的兵器砸烂敌人的脑袋。然而江湖上却讲究拳打卧牛之地，胜负在方寸之间。
<br/>　　虽然不同，但终究大同小异。同样讲究速度、力量以及瞬息之间的反应。
<br/>　　林少羽骑在马上，铁枪轻描淡写的挥动，长枪逼得刘阿吉不能近身，将他生生拦在自己的枪围外。
<br/>　　破浪七叠枪，本身就属于花枪一路，用繁复的枪法诱敌深入，一招毙命。
<br/>　　刘阿吉打得很苦，他展现出了自己全部实力，甚至超水平发挥。但依旧没办法突破林少羽的枪围，同时，林少羽随意一枪他却必须拼尽全力躲避。
<br/>　　决定动手前，刘阿吉并不是完全没有胜算。以他得到的情报分析，从战斗开始到现在，林少羽已衣不解甲多日，持续的高强度战斗，对任何一人的体力都是巨大的消耗。自己就算一时胜不过对方，可慢慢将战斗时间拉长，总能找机会杀掉他。
<br/>　　然而，到现在为止，刘阿吉还没发觉林少羽露出任何疲态，他就像一块燃烧的火炭，不把自己烧成灰烬，绝不会熄灭。
<br/>　　破浪七叠枪，一枪七叠，力气叠着力气，像层层波浪般向刘阿吉拍来，正应了那句话，一浪更比一浪浪。
<br/>　　刘阿吉展现了风骚的走位，一退再退，直到退无可退，然后他没有林少羽那么浪，一枪被林少羽洞穿肩膀。
<br/>　　明帝坐在龙椅上，高高在上，宛若一条睡龙，眯缝着眼不知道在想些什么。
<br/>　　在林少羽进攻决定杀掉刘阿吉时，一杆长枪突至，生生将林少羽逼退。
<br/>　　林少羽眼底闪过一抹惊诧，抬起头，发现这杆枪竟然是从明帝手中投出。
<br/>　　明帝此刻已长身而起，身上的龙袍张牙舞爪，宛若要破壁而出。
<br/>　　望着林少羽、依稀能看见其父林问天曾经的影子。那时林问天还年轻，明帝也正少年，二人鲜衣怒马，时而饮酒狂欢，时而牵犬引苍，踏破长安繁华。
<br/>　　当然，这都是曾经的事情了。曾经的长安少年，如今已是九五之尊，垂垂老者，而林问天，则是被明帝亲自下旨处死在街头。
<br/>　　明帝自然不会觉得后悔，只不过再无可通宵达旦彻夜长谈之人，终究有几分落寞。
<br/>　　江山代有人才出，曾经的长安少年变成白发老贼，年老的总会被比更年轻的取代。
<br/>　　林少羽立马横枪，已胜过当年的林问天太多。让明帝唏嘘的是，自己却没一个儿子能站出来，去挡林少羽的枪。
<br/>　　总是有几分难过的。
<br/>　　林少羽似乎读懂了明帝此刻的心情，挑起地上的枪，拨还给明帝。明帝抄手接住，越过龙书案，站在金玉台阶上。
<br/>　　“陛下！”刘阿吉失声道。
<br/>　　明帝摇摇头，抬起长枪，遥遥指着林少羽。
<br/>　　“你来！”
<br/>　　这是新王与旧王的战斗，胜者的奖品是帝国九万里江山，一个太监根本没有插手的资格。
<br/>　　林少羽对明帝保持了足够的敬意，他从马上跃起，身子向前突进，在靠近明帝时狠狠挑起，长枪向明帝扎来。
<br/>　　如一个勇士要捅穿灼目的太阳。
<br/>　　明帝同样跃起，两杠枪在空中相接，发出沉重的碰撞，然后各自退后，如被丢进斗兽场的野兽，绕着对方转圈，寻找敌人的破绽。
<br/>　　林少羽看错了明帝。
<br/>　　这个行将就木的老者，展现出难以想象的体力和悍勇，像垂死的野兽对着苍穹发出最后的咆哮。
<br/>　　长安不仅有林家枪，也有李家枪。
<br/>　　李家盘龙枪。
<br/>　　时间过去得太久，久到曾经发生过什么事，已没人记得。李家并非生来就是皇族，他们的江山也是一寸一寸打下来的。
<br/>　　大武的开国之君，号称天下武尊，靠着一杆铁枪收拾山河。
<br/>　　他当年使得枪法，便是此刻明帝用的。
<br/>　　李家盘龙枪，又称盘龙八打枪。
<br/>
