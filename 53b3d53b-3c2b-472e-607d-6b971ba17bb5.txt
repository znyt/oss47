        到时候，傅夜沉打算随便找个理由，说跟苏漫雪性格不合，分手了便是。
<br/>
<br/>        而苏漫雪，明知道傅夜沉不会放过自己，所以赶紧抱紧傅老太爷这条大腿，好为自己找个靠山来压制傅夜沉。
<br/>
<br/>        晚饭过后，傅昼景并未留在傅家大庄园里过夜，而是因为公司里有事，急着赶回了滨江城。
<br/>
<br/>        傅夜沉还未正式上任成为傅氏集团的新任总裁，所以，他一点都不忙。之前去集团公司，也只不过是去提前了解集团公司里的情况罢了。
<br/>
<br/>        而且，傅老太爷也有意把自己的人脉圈，一一介绍给傅夜沉。所以，傅夜沉基本上是主外，而傅昼景则主内，完全作为傅夜沉的辅佐人而存在。
<br/>
<br/>        童筱颖恋恋不舍地送走了傅昼景，站在大门口，一转身，就遇见了不知何时出现在她身后的大哥傅夜沉。
<br/>
<br/>        晃了一会儿神，童筱颖下意识地后退了一步，微微低着头，轻轻地唤了声：“大哥。”
<br/>
<br/>        傅夜沉听着这一声“大哥”，心里十分不是滋味，却只能装作若无其事的样子，应了一声“嗯”，然后有话没话地找话题关心道：“外边风大，早些进屋休息吧！”
<br/>
<br/>        “好！”
<br/>
<br/>        童筱颖微微点了点头，躲避着傅夜沉的目光，从他身旁快速跑过。
<br/>
<br/>        回想起之前他对她所做过的那些亲密的事情，她现在一看到他就尴尬不已。
<br/>
<br/>        一阵微风拂过，风中留下了她身上的柠檬香味。
<br/>
<br/>        这样的香味，令他既熟悉又怀念，只怕，从今以后，他再也闻不到她身上的清香了吧！
<br/>
<br/>        傅夜沉剑眉微蹙，回眸之际，他深情款款地凝视着童筱颖离去的背影，心口在隐隐作痛。
<br/>
<br/>        从此以后，他对她的爱，就只能这样默默地藏匿在内心深处。
<br/>
<br/>        童筱颖回到了自己那被傅老太爷安排的偏院里。
<br/>
<br/>        这一入夜，院内萧条，虫鸣鸟叫，月光下，树影婆娑，这景象令人有些感到渗凉。
<br/>
<br/>        童筱颖加快步伐，穿过庭院，跑进了屋内，将木门关上后落上了木栓。
<br/>
<br/>        这百年老宅里，为何一到晚上就这般阴森？！
<br/>
<br/>        童筱颖坐在屋子里的圆桌前，忍不住掏出手机给傅昼景发了条短信，问他有没有平安到家，但是傅昼景并未回复。
<br/>
<br/>        傅夜沉回到自己的寝居室，刚关上门，一转身，就看到苏漫雪长发披肩，身上只裹着一条粉色浴巾，酥胸半露，呼之欲出，雪白的长腿，赤脚着地。
<br/>
<br/>        苏漫雪撩起珠帘，妖娆妩媚地向傅夜沉走了过来。
<br/>
<br/>        她跟筱颖有着一张一模一样的脸，就像一对双胞胎，若不细看，压根就分辨不出她两谁是谁。
<br/>
<br/>        看到这样的苏漫雪，傅夜沉心里只觉得恶心。
<br/>
<br/>        若不是这个女人贪慕虚荣，鱼目混珠，他又怎么可能失去筱颖？
<br/>
<br/>        都怪他不好，他当初，就应该抽空回依山别苑去确认一下，该有多好！
<br/>
<br/>        至少，那个时候，他还有机会得到筱颖。
<br/>
<br/>        如今，他真的一点机会也没有了！
<br/>
<br/>        筱颖成为他的弟妹，都拜这个女人所赐！
<br/>
<br/>        一想到自己失去了筱颖，傅夜沉心里疼痛难耐，又是一阵怒火袭来，这些抑郁积压于心，他一时失控，跨步上前，挥手便给了苏漫雪重重的一巴掌。
<br/>
<br/>        傅夜沉是练过底子的，曾经徒手打死过歹徒饲养的一只猛虎，他这重重的一巴掌，苏漫雪哪能受得住，“啪”地一声，她便被打趴在地上。
<br/>
<br/>        “呜呜——”苏漫雪跪坐在地上，捂着疼得火辣辣的脸颊，痛得抽泣了起来，嘴角还冒出了一丝鲜血。
<br/>
<br/>        “苏漫雪，你给本少听清楚了！从今以后，你不要妄想再在本少身上捞到一丁点的好处！还有，在这之前，你花了本少的那些钱，我会让刘管家列账单出来作为你对本少的借款，将来，你得一分不少，连本带利地还给本少。此外，你拿本少的钱，用你的名字购买的住房、门面、公司和豪车，这几日，我会让刘管家带你去办理转移手续，全部过到筱颖的名下！”傅夜沉冷冷地呵斥，丝毫不给苏漫雪留任何颜面。
<br/>
<br/>        苏漫雪泪眼婆娑地抬起头来，看着面前这个尊贵如神，俊美无暇，让她可望而不可即的冷酷男人，哽咽道：“夜沉，你何必要这样对我？我有什么错呢？”
<br/>
<br/>        “你有什么错，你会不知道？”傅夜沉黑眸微眯，正眼都不愿瞧苏漫雪一下，双手背在身后，冷酷无情地冷哼。
<br/>
<br/>        苏曼雪一边抹眼泪，一边接着哭诉：“我和筱颖情如姐妹，我是她最好的朋友，也是她的闺蜜。你为何要这般对待我？是筱颖不要你！是筱颖把玉坠给了我！这一切的一切，都是筱颖自己的选择。其次，是刘管家冲着玉坠的来头亲自将我接走的！所以，我没有错！你为何要把所有的错，都怪罪在我的身上？筱颖她不爱你，可我爱你啊！你怎么能这样对我呢？对待一个爱你的女孩……呜呜——”
<br/>
<br/>        “呵？苏漫雪，收起你那廉价的眼泪，别在我这儿博什么同情心！你是爱我，还是爱我的权利和金钱，我比谁都要清楚！苏漫雪，我警告你！这段日子，你最好给我安分点！既然爷爷想留你在这傅家大庄园里，你就先在这住下。但是，你别想耍什么花招！我会派人在暗中盯着你！”傅夜沉呵斥道，看都不想多看这恶心的女人一眼，便直接踱步离开了房间。
<br/>
<br/>        苏漫雪看着傅夜沉离去的背影，顿时心里苦不堪言。
<br/>
<br/>        她如今落到这种地步，都是拜那个童筱颖所赐。
<br/>
<br/>        当初，她就不应该对童筱颖心软的。
<br/>
<br/>        当初，她就应该暗中雇个杀手，把童筱颖杀了不就了事了吗？
<br/>
<br/>        她为什么要心软？
<br/>
<br/>        她为什么要让童筱颖活着啊？
<br/>
<br/>        童筱颖若是死了，傅夜沉就是她的了，傅氏集团的未来也是她的，这里的一切的一切，都是她苏漫雪的！
<br/>
<br/>        苏漫雪不甘心，她得想办法让自己翻身才行。
<br/>
<br/>        童筱颖躺在床上抱着手机，半晌都未睡着。她在等傅昼景的回信，但是，傅昼景迟迟未回短信过来。
<br/>
<br/>        另一边，滨江城高铁站。
<br/>
<br/>        傅昼景出站后，是季思妍开车到站口接的他。
<br/>
<br/>        季思妍接到傅昼景后，并未送傅昼景回家，而是开车载着他去了自己的公寓。
<br/>
<br/>        “阿景，我昨天得到了一个关于你亲生母亲的资料，放在家里了，我一直在犹豫着要不要给你看。”途中，季思妍慢条斯理地说。
<br/>
<br/>        她若真的是在犹豫，就不会开车载着他往她家的方向去。
<br/>
<br/>        傅昼景微微凝眉，只是抿着薄薄的嘴唇，沉默不语。
<br/>
<br/>        “我给你做宵夜吃吧！”一回到家中，季思妍就像一个小媳妇一样伺候着傅昼景。
<br/>
<br/>        傅昼景的所有喜好，季思妍一清二楚。
<br/>
<br/>        所以，傅昼景被季思妍伺候得非常舒适也非常地顺心。
<br/>
<br/>        季思妍不像童筱颖，不了解他的喜好，也不会跟他无理取闹。
<br/>
<br/>        童筱颖的不懂事，反而让傅昼景觉得季思妍更加贴他的心。
<br/>
<br/>        季思妍做的宵夜，是生滚鳕鱼粥，清淡口味，是傅昼景喜欢的味道。
<br/>
<br/>        傅昼景吃得很舒心，胃里也暖暖的。
<br/>
<br/>        但是，他没忘记，他来她家的正事。
<br/>
<br/>        “关于我妈妈的资料呢？”傅昼景直接开门见山地问。
<br/>
<br/>        “阿景，你想不想自己当老板？”季思妍坐在傅昼景的对面，双手交叠平放在餐桌上，一边看着傅昼景喝粥，一边试探性地问。
<br/>
<br/>        傅昼景怔了一下，抬眸看向季思妍，默默地吃粥，不吭声。
<br/>
<br/>        季思妍抬起手来，一手托腮，凝视着傅昼景，却转开了话题：“阿景，你现在在傅氏集团旗下地利置业当总经理，年薪多少？”
