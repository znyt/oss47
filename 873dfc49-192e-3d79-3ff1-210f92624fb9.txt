    
<br/>　　<br/>　　抚仙湖养尸地的传说历来就有，只不过多数人都没当真，只在本地有些流传，大部分抚仙湖周边的居民都传言湖下连通九幽，阴气很重可以滋养尸体，可以保证不腐，但至于这些尸体怎么来的，为何会出现在湖里就没人知晓了。<br/>　　<br/>　　至于说有没有人亲眼见过抚仙湖水下的尸体呢，本地人似乎都没见过，也有不少年轻人始终觉得这不过是个故事罢了，只有那些上了年纪的老人才对此深信不疑，而湖底养尸的传言大概是在二十几年前的时候，有一名资深的潜水员曾经带着精良的装备下过湖底，然后看见了水下密密麻麻的尸体直立在湖下的沙泥中，更诡异的是男尸全都朝前仰着，女尸无一例外是向后倾斜的。<br/>　　<br/>　　不过那名潜水员从湖底回来，上岸之后没多久就消失了，在他消失之前曾经在湖边的村子里打听过，问一些上了年纪的老人，湖下为什么会有尸体，这才让抚仙湖养尸地的传闻，渐渐的传言开来，不过在那以后这名潜水员就再也没出现过了。<br/>　　<br/>　　王惊蛰也是知道抚仙湖养尸地的，他倒是没有来过，而是以前听王仙芝跟他讲述的，不过说的也不怎么详细，只说抚仙湖的阴气确实挺重，如果将尸体沉入水下，可能确实会起到养尸的作用。<br/>　　<br/>　　陈三岁落脚的地方，在抚仙湖区域澄江县下面的一个村子里，大概有几十户人家，多数都是留守儿童和老人，妇女，基本青壮劳力都出去打工了，他就在村子里租了个民房，一连住了三个多月。<br/>　　<br/>　　皮卡开到村子里，直接开进一农家院，两人下车后，陈三岁就说道：“我平时吃住都在这，大部分的时间都不会出门，偶尔每隔几天去一趟湖上，看看诗远……”<br/>　　<br/>　　民房收拾的还算干净，比较整洁，看的出来这货生活的也不是特别懈怠，让王惊蛰比较欣慰的是，民房里没有什么变态的状况，比如尸体或者尸油味什么的。<br/>　　<br/>　　“还行，这里勉强说的过去，我还真怕一进来就看见满地的尸蟞什么的呢”王惊蛰打量几眼后，还算满意的点了点头。<br/>　　<br/>　　陈三岁尴尬的说道：“王哥你别开玩笑行么……”<br/>　　<br/>　　由于两人到抚仙湖的时候时间已经很晚了，多少也不差这一天的时间，当天就没有去湖上，先休息了一夜。<br/>　　<br/>　　晚间，吃饭的时候，陈三岁做的，饭就是普通白米饭，菜就比较简单了，一个炒青菜和一份腌萝卜。<br/>　　<br/>　　王惊蛰端着饭碗，斜了着眼睛用筷子夹起一片萝卜皮，然后慢条斯理的松紧了嘴里，可能是看他吃的挺不愉悦，陈三岁就抱歉的说道：“不好意思王哥，明天我出门在买点菜回来，平时就我自己在吃上面也不太在意，经常随便对付一顿就行了”<br/>　　<br/>　　王惊蛰随意的摆手说道：“没事，过年吃的油水太足了，吃的简单也行，我正好刮刮肠子”<br/>　　<br/>　　陈三岁顿时笑了：“那就行，你不嫌弃就好”<br/>　　<br/>　　王惊蛰无语的看着他，都给陈三岁看懵住了，他说道：“你就是这么求人办事的？我不在意是我不在意的问题，但你的态度必须得有啊，你当我是兔子呢？这都啥玩意啊，萝卜皮子和鸡毛菜？这我就不说什么了，但你倒是多放点油啊，这青菜让你愣是给炒成了干煸，一点口感都没有”<br/>　　<br/>　　陈三岁抹了把冷汗，连连点头说道：“下次注意，明天我先出门买菜，王哥我真是没心思考虑啥生活水准啊”<br/>　　<br/>　　“行了，边吃边聊吧，我就随便对付一口得了……”<br/>　　<br/>　　陈三岁和王惊蛰一边吃饭，一边说他这三个多月来在抚仙湖的经历。<br/>　　<br/>　　初时，大概刚来一个月左右吧，诗远被他沉在湖底后，一切都挺正常的，应该是水下七八米的深度，尸体沉到水底后果然就如传闻中那样，脚踩着沙泥立足了，然后身体随着水波来回的晃动，但始终都没倒下，于是往后每隔几天陈三岁都会过来查探一番。<br/>　　<br/>　　因为诗远的魂被他从阴间带出来封在了尸体里，每间隔几天都得需要重新再封一次，不然魂魄是无法长期停留在尸体里的。<br/>　　<br/>　　到了二个多月的时候，有一天陈三岁照例去了抚仙湖，然后下到水底，这时候他却发现诗远的尸体有了一些变化，头发乌黑有光泽，皮肤上也焕发出了纹路，刚开始他还没太在意以为是光线的问题看错了，等再过几天的时候变化又明显了不少，原本诗远脑袋上是有白头发的，可这次去了后她头上白发全没了，并且两手的指甲也长出了不少。<br/>　　<br/>　　陈三岁这人心里可能一根筋，但脑子并不笨，他察觉到这个变化后就留了个心眼，当天回到村子又在来了一次，于是将诗远的指甲全都给剪短了，还把她的长发也给削了下去。<br/>　　<br/>　　陈三岁说道：“以前我一直用尸油为诗远擦身可以保持尸体不腐维持原样，但头发和指甲什么
