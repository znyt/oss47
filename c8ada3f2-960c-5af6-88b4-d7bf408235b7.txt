　　轰隆隆！
<br/>　　乌云密布，风云倒卷，风暴不断，灵气狂涌而来，九天之上，电闪雷鸣。在在昆仑山的上方，雷云正在不断的汇聚，同时，九天之上，一道紫色的雷罚神眼出现，眼眸之中，是无尽的冷漠，它紧盯着下方的昆仑山。
<br/>　　自从元始从元始印中出来，它便出现了。显然，元始突破混元大罗金仙境界，天道要降下雷罚，若是挺过去了，那么，从此风平浪静，前路一路平坦。相反，若是不能挺住，那便自然是万事皆休。
<br/>　　天空之上的雷劫依旧在不断汇聚，但元始却丝毫不放在心上，在没有成为混元之前，还可能会有些麻烦，当然，也仅仅只是麻烦而已。但如今元始都已经突破了，雷劫才来，此时来了吾又有何惧？
<br/>　　事实上的确如此，九天之上，不断降下灾劫，无论是多大，元始就站在那里，它都不能威胁元始分毫。而且，不管来多少，元始都全部吞下。
<br/>　　不管是雷劫、风灾、火灾等各种灾劫，都不能伤害元始分毫，元始大道一出，什么雷劫、火劫，不管来多少，都被他吞噬同化一空，用来增加自己的底蕴。
<br/>　　见此，九天之上的雷罚也无奈了，在降下九九八十一道灾劫之后，便无奈地收了回去。九天之上，再次回复平静。若不是昆仑山附近有一地的雷罚痕迹，这雷罚就仿佛是没有出来过似的。
<br/>　　“怎么回事？昆仑山又发生什么事了？”洪荒之中，有人看见这惊天雷劫，纷纷惊呼，猜测昆仑山发生了何事。
<br/>　　“又是昆仑山？又是三清？他们到底要干什么？”有人想起之前昆仑山发生的种种异象，猜测到三清。
<br/>　　“莫非是有顶级灵宝出世？”有人这样猜测。
<br/>　　“不对，还可能是顶级仙丹出世，要知道，太清老子可是炼丹高手啊。”有人想到老子的身份，猜测这是仙丹出世。
<br/>　　“仙丹不可能招来如此强大的雷劫，这雷劫已经可以劈死准圣高手了，恐怕就连一般圣人都不敢抵抗？”
<br/>　　“那又会是什么？”
<br/>　　“不知道？可能真是有人成圣了？”有人略带感慨道。
<br/>　　“啊？”
<br/>　　……
<br/>　　洪荒之中，在众人纷纷猜测这雷劫降临的原因之时，元始却是在思考，该如何让白发原始成圣。不，准确的是在说，是在考虑用何种方式成圣。
<br/>　　成圣之法三道，不管是以力证道，还是三尸证道，亦或者是功德成圣，元始都有把握。他如今要考虑的是这其中的利弊。
<br/>　　“三尸证道，不是我的路，更何况恶尸万劫魔尊与善尸永恒道人都已经在外开山立道，他们有他们的路，不可能再召回。况且，就算召回来了，我也不会考虑此路。三尸之道自然是不成了，那么，功德成圣呢？”元始在默默思索道。
<br/>　　“以我如今的积累的功德，就算不够成圣，但也差之不远了，只要谋算一番，或者立教，便会立刻有大量功德，也足够成圣了。不过，功德成圣终究是有些取巧的嫌疑，是最弱的圣人，那也不是我的路。”
<br/>　　“那么，便只剩下以力证道这一条路了。以力证道，说的是以力证道，实则就是法则之道。以自己强大的法则之力破开天道空间，在那道属于你自己领悟的天道法则之上刻印上你的烙印，将其炼化掌控之后，你便为那道天道法则的执掌者，可以调用那天道法则作战。
<br/>　　同时，你也绝了修炼此道法则的修士成圣的路。故而有‘一条路，只可能有一人成道’这么一句话，说的就是这种情况。还有就是‘要走出自己的道’这类说法。领悟出一条完整的天道法则，便可以此成圣。”
<br/>　　“我如今领悟到法则之道，可以成圣的有好几种。如因果、吞噬、雷罚、五行、阴阳、四象、一元等。不过，对于日后好处最大的还是因果之道。
<br/>　　先不说因果之道自己只需要一步一步的解开封印，便可不断上升。就是因果之道的诡异，对自己也最为有利。若是能以因果之道成圣，到时便可直接执掌天道因果。”
<br/>　　“因果无处不在，无处不可寻，贯穿古今未来，连接前世今生与来世，轮回不能阻，天道不能灭，为最强大的道之一。若是能够执掌因果之道，不管是对于今后洪荒的掌控，还是对于鸿钧天道的防备，对他都是极为有利的。更可况天道的构成之中，因果绝对是占据绝大一部分。若是以此成圣，他的实力在圣人之中也是最为强大的。”
<br/>　　“那么，便让白发原始借成圣之基鸿蒙紫气再以因果之道成圣。若是在加上雷罚一道，那么以后吾便是执掌天下因果与天罚的圣人，为众生降下雷罚，说不定还能更近一步。领悟天道秩序一道。”
<br/>　　“如此，那便以此成圣吧。”元始对着白发原始道，“这也算是偿还了洪荒的因果吧。就算盘古以后真的还活着，也不能改变什么。”
<br/>　　白发原始变了点头，接着盘坐在一静室之中，将自己心灵放空，灵魂意识开始和弥漫宇宙各处的至高天道规则靠拢。
<br/>　　终于，借助冥冥之中的因果一道的诡异，原始隐隐感受到了至高规则中存在一处奇特空间，他似乎无限小，小到原始的各种手段都探查不到。又仿佛无限大，大到整个洪荒宇宙处处都存在这“它”。
<br/>　　“嗡！”
<br/>　　白发原始心念一动，意识便轻易进入了一处奇特的空间。
<br/>　　这是洪荒世界最核心的“本源空间”，亦称“天道空间”。这处空间十分神秘，唯有灵魂意识方能进入，其他的东西，任何实质性的物质，什么也不能进入。它的大小，无法用正常物质来判断。原始只感觉这神秘的本源空间中，五颜六色的一团又一团本源彼此交错，看似混乱，实则形成整体在运转。每一团本源都代表着一条道！
<br/>　　还有密密麻麻的如锁链般的规则法链，一层又一层地贯穿了整个空间。
<br/>　　洪荒宇宙的本源力量，在盘古开天之时，受过重创，天道法则本源不太完善，缺少较为关键的轮回大道以及人道。
<br/>　　世界的本能，是要孕育出足够强大的完美世界来。
<br/>　　这洪荒宇宙虽然力量有损，但规则依旧至高。更何况，缺失的这些法则，以后又不是没有机会补全。
<br/>　　天道不全，人道补之。
<br/>　　轮回不全，以身补之。
