　　“石族天生异禀，的确不凡！”龙腾点了点头，高个青年名为龙烨，其奴仆乃是石族之人，防御极为可怕，力量也非常强大。
<br/>　　而这位石族奴仆的修为，和龙烨相当，同样是三品高级。
<br/>　　“石崐见过公子！”石族奴仆非常客气，立刻向龙腾躬身行礼。
<br/>　　“不必多礼！”龙腾摆了摆手。
<br/>　　“龙腾，你天赋极高，不知收的奴仆符不符合你的身份？”龙烨打量着苏莫和司空炎。
<br/>　　对于他们龙族的万古天骄而言，要么就不收奴仆，要收就收最极品的奴仆。
<br/>　　“龙腾，上次你侥幸胜了我，不如这次我们再比一次如何？”矮个青年开口了，目现挑衅之色。
<br/>　　上次他和龙腾同阶切磋，折损了颜面，这一次准备找回来一些面子。
<br/>　　“龙城，你还想要打？”龙腾看向矮个青年。
<br/>　　“并不是我们打，是让我们的奴仆切磋一番，输者的主人拿出百万中品源石，你觉得如何？”矮个青年龙城微笑着说道。
<br/>　　言罢，他身后的奴仆，向前踏出一步。
<br/>　　这位奴仆居然是巫族之人，身材瘦小，身穿黑袍，一双眼眸犹如刀锋般冷厉。
<br/>　　此人的修为是三品中级，和苏莫相当。
<br/>　　正因为这名奴仆，和苏莫相当，龙城才会提出比试，不然修为不同，比试的意义也不大。
<br/>　　苏莫和司空炎两人，面无表情，他们默不作声，不管有什么事，自有龙腾安排。
<br/>　　“龙城，奴仆们马上就要争夺宝物了，那有闲心比试！”龙腾摇了摇头说道。
<br/>　　“你不敢？”龙城闻言嗤笑，继续道：“不仿告诉你，我这个奴仆巫铨，实力绝不是你能想象的，以我的越级战斗实力，都不是他的对手，你不敢也在情理之中，我不会笑话你！”
<br/>　　言语虽然如此，但龙城的面上，尽是嘲弄之色。
<br/>　　他说的都是实话，他的修为是二品低级，对上一般的三品低级武者，都有抗衡之力。
<br/>　　但是，他若是和三品低级的巫铨交手，挡不住巫铨一招。
<br/>　　当然，以他的实力，是不可能制服巫铨的，巫铨是他的父亲为他制服的奴仆。
<br/>　　但是奴印是他掌控，巫铨就受他掌控。
<br/>　　“井底之蛙！”司空炎嗤笑一声，面上露出不屑之色。
<br/>　　就这种货色还来比试，和他比试还差不多，若是和同阶的苏莫交手，还不够一个手指头碾压的。
<br/>　　“放肆！”
<br/>　　龙城见到司空炎不屑的声音，顿时大怒，面带寒光的冷喝道：“区区奴仆，卑微如狗，居然敢以下犯上？”
<br/>　　龙城暴怒，其身边的奴仆巫铨，手掌源力涌动，随时准备出手。
<br/>　　龙腾和苏莫的脸色均是一沉，这个龙城居然如此侮辱司空炎！
<br/>　　而司空炎的脸色，则是瞬间铁青了下来，身躯都因为愤怒，而微微颤抖了起来。
<br/>　　他还从未受过如此侮辱，当真是怒不可竭。
<br/>　　不过，他在拼命忍耐，他知晓苏莫身份敏感，不宜闹出大乱子，所以在拼命压制自己的怒气。
<br/>　　“好了龙城，打狗还要看主人，你怎么也要给龙腾一点面子！”
<br/>　　这时，一旁的龙洌开口了，微笑着说道。
<br/>　　虽然他也乐意看到龙腾的奴仆的羞辱，但是，他知道龙城各方面都不是龙腾的对手，所以适时出言圆场。
<br/>　　“龙腾，不是我不给你面子，你这个狗奴，居然如此胆大包天！”龙城面色阴冷的说道。
<br/>　　“龙城，不要太过分了，我的忍耐是有限度的！”龙腾面色难看，心中也是生出了怒火。
<br/>　　“怎么？你的狗奴出言不逊，难道还有理了？”龙城冷笑着说道，他心中丝毫无惧。
<br/>　　因为，他是龙族之人，在龙族之中，奴仆就是蚂蚁一般的存在。
<br/>　　奴仆的命，是不值一提的。
<br/>　　此事若是闹大，这出言不逊的奴仆，免不了被处死的命运。
<br/>　　“龙腾，让你的奴仆给龙城道个谦，此事就此揭过吧！”
<br/>　　这时，一直没有开口的那名龙族女子开口了，声音淡雅，颇为柔美。
<br/>　　“双双姐，我的奴仆我自己会管教！”龙腾对那名龙族女子说道。
<br/>　　这名龙族女子名为龙双双，算是他们之中修为最高的，但是龙腾并不会听对方的话。
<br/>　　他不会让司空炎道歉的，他知道司空炎的脾气，就算受罚都不会道歉。
<br/>　　“呵呵，龙腾你倒是会偏袒！”
<br/>　　龙城怒急而笑，随即眸中杀机一闪，转头对身旁的巫铨道：“巫铨，等会争夺宝物，你帮我杀了这个狗奴！”
<br/>　　“是！”巫铨躬身领命，面无表情。
<br/>　　“龙城，你是要找不自在吗？”龙腾闻言，双眸微眯了下来。
<br/>　　“一个狗奴而已，死了也就死了，你若是怕他死，可以带他离开！”龙城不屑的笑道。
<br/>　　“你……！”龙腾正欲再言，突然又顿住了，随即冷冷的道：“那我拭目以待。”
<br/>　　龙腾之所以顿住了，是因为苏莫给他意念传音了。
<br/>　　苏莫面色平淡，冷漠的看着这一切，他让龙腾暂忍一口气，也让司空炎暂忍一口气。
<br/>　　他知道，在龙族之人，奴仆的命低贱不已，命是属于龙族的。
<br/>　　所以，他没有让龙腾去多言。
<br/>　　既然龙城想让巫铨杀司空炎，他们接着就是，一个小小的巫族，他反手就能灭杀。
<br/>　　“那我们走着瞧！”龙城冷笑一声，随即转过了身，不再理会龙腾。
<br/>　　龙双双眉头微皱，龙腾没有听她的话，让她心中有些不悦，但是，她也没有多言。
<br/>　　毕竟，这是龙腾和龙城之间的过节，与她无关。
<br/>　　龙烨、龙洌两人，均是轻笑一声，看来等会奴仆们争夺宝物，要有点意思了。
<br/>　　随后，众人都不再多言，继续等待了起来。
<br/>　　司空炎心中憋了一口闷气，一直阴沉着脸，让苏莫有些无奈。
<br/>　　苏莫在打量四方，观察这众多的奴仆，这些奴仆之中强者太多了，而且，有很多奴仆，气息极为强大，远超本身的修为境界，明显都是天赋不凡之辈。
<br/>　　若是等会规则不公平，那他和司空炎基本上得不到什么宝物。
<br/>　　时间流逝，约莫半个时辰之后，终于有龙族高层人物到来了。
<br/>　　只见高空之中，光芒一闪，出现了两名龙族中年人。
<br/>　　这两人均是中年男子，一人身穿白袍，一人身穿青袍，气息均是浩大无边，但比神王级强者要稍弱一些，应该是龙族的执事。
