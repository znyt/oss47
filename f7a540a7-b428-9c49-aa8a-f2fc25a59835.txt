　　叶尘顿时眉头一皱，
<br/>　　他本不想招惹事端，可是眼前这个锦衣少年，明显是挑衅他的耐性啊！
<br/>　　虽然，他和这个小兰刚刚认识不久，并没有什么感情，可是叶尘也不可能让这么一个乖巧可爱的小丫头，落入对方的手里。
<br/>　　小兰听到这话，小脸顿时更加白了，不由得死死攥住了叶尘的手腕，低声带着哭腔，道：
<br/>　　“少爷，小兰不要和你分开！”
<br/>　　叶尘扭头冲着小兰微微一笑，
<br/>　　“放心！小兰，有少爷在呢！”
<br/>　　一旁的那名锦衣少年，听到两人的对话，顿时冷笑了起来，
<br/>　　“啧啧！还真是主仆情深呐！可惜，这是家主的决定，可由不得你们！”
<br/>　　说完这话，锦衣少年指了指身后的一名随从，
<br/>　　“你！去把这小子，和他那死鬼老爹的尸体，全部给本公子丢出去！”
<br/>　　“是！成公子！”
<br/>　　那名随从当即应了一声，猛然一步迈出，闪电般冲到叶尘跟前，然后手臂一伸，便朝叶尘的后颈抓去！
<br/>　　叶尘见状，不由得摇了摇头，自言自语道：
<br/>　　“修为倒是比我强不少，可惜身手太差，破绽太多！”
<br/>　　叶尘说出这话的同时，眼看那人的大手，就要落在叶尘的后颈之上，叶尘不退反进，仿佛主动扑进那人的怀中一般，不但从容躲过了对方这一抓，同时双指作剑，狠狠刺在了那人的檀中穴上！
<br/>　　那人显然没有想到，眼前这个平日里，众人眼中的废物，竟然敢主动攻击他，而且还一举击中了他的要害部位，顿时感觉胸口猛然一阵剧痛，竟然连一丝力道也使不出来了。
<br/>　　紧跟着，
<br/>　　蹬蹬蹬！
<br/>　　那人一连倒退了四五步，一屁股坐倒在了地上，浑身酸软无力，竟然一时间怎么也爬不起了！
<br/>　　“什么？”
<br/>　　这突然发生的状况，顿时让所有人，全都惊呆当场。
<br/>　　这个公认的叶家第一废物，居然可以打败叶家的护卫？
<br/>　　那名锦衣少年，先是两眼一瞪，继而一脸怒色，朝地上的那名随从狠狠踹了一脚，大骂道：
<br/>　　“丢人现眼的东西！居然连一个废物都搞不定，本公子养你何用？”
<br/>　　说完这话，锦衣少年又冲着另外两名随从怒吼道：
<br/>　　“都特么还愣着做什么？还不快把这小子给本公子拿下！”
<br/>　　“是！”
<br/>　　那两名随从听到锦衣少年的命令，当即纷纷朝叶尘扑了过去，不过有了之前的那一幕，两人自然不敢再大意轻敌，一左一右，同时挥拳，朝叶尘攻了过去。
<br/>　　若论修为，这两人都是胎动期六重，而叶尘如今才不过胎动期三重，实力根本不在一个档次。
<br/>　　可惜，他们的战斗经验，和活了八百多年的叶尘相比，简直天差地远，犹如婴儿一般可笑。
<br/>　　看似气势十足的攻势，落在叶尘的眼中。却破绽百出，
<br/>　　只见叶尘身体突然一矮，从容躲过两人的拳头，同时双手穿到两人的胸膛之前，犹如灵蛇出洞一般，狠狠推在了两人的下巴之上，速度快如闪电！
<br/>　　那两名随从根本没有反应过来，下巴遭到重击之后，身体瞬间就倒飞了出去，狠狠栽落在地面之上，脑袋一阵眩晕，连牙齿也脱落了好几颗，躺在地上哀嚎不止，一时半会显然是起不来了！
<br/>　　蹬蹬！
<br/>　　后方的那锦衣少年，看到这一幕，也顿时吓了一跳，不由得倒退了数步，满脸不可思议的望着叶尘，犹如活见鬼了一般，
<br/>　　“你、你真的是叶枫？这怎么可能！！”
<br/>　　不过，在惊愕过后，锦衣少年很快就又镇定了下来，
<br/>　　“哼！看来本公子之前是小瞧你了！不过就凭你胎动三重的修为，你以为你是本公子的对手吗？本公子如今可是胎动九重！”
<br/>　　轰！
<br/>　　锦衣少年说完这话，身上的气息猛然释放了出来，果然远远超过了之前的那三名随从。
<br/>　　“废物！上一次是本公子出手太轻了，这一次本公子会让你清楚，违逆本公子的下场！！”
<br/>　　说完这话，锦衣少年大喝一声，一步猛然迈出，瞬间便冲到叶尘的跟前，狠狠一拳砸落下来！
<br/>　　轰隆！
<br/>　　这锦衣少年的实力，明显比之前那三名随从要强上不少，无论是速度还是威力，都要高出一大截。
<br/>　　若非叶尘早就有所防备，在对方动手的同时，直接施展出了地球上的武功身法，恐怕根本躲避不料。
<br/>　　可即便如此，在锦衣少年的拳劲冲击之下，叶尘也不由得连连倒退。
<br/>　　“咦？居然躲开了？”
<br/>　　锦衣少年显然没有想到，这个公认的废物，居然躲过了自己的全力一击，顿时大为惊讶。
<br/>　　轰轰轰！
<br/>　　锦衣少年一连几拳打出，全都被叶尘用精妙的身法躲开，但是两人的修为，实在相差太多，而且叶尘如今的这具身体也实在太弱，才打了一会，就开始有些气虚喘喘。
<br/>　　“不行！我修为太弱，必须速战速决！”
<br/>　　一念及此，叶尘将身法施展到极致，以一个无比诡异的角度，闪到锦衣少年的背后，狠狠一掌拍在了他的大椎穴上，
<br/>　　大椎穴，乃人体大穴，叶尘现在修为不占优势，所以只能兵行险着。
<br/>　　轰！
<br/>　　叶尘想要速战速决，所以这一掌没有任何留手，不但将体内的造化之力完全灌入掌上，而且还运用了无比精妙的发力技巧。
<br/>　　“噗！！”
<br/>　　锦衣少年顿时狂喷了一大鲜血，整个人重重栽落在地上，显然已经受了不轻的伤。
<br/>　　“你！你居然敢伤我？本公子要杀了你！杀了你！！”
<br/>　　锦衣少年翻过身来，羞怒交加，指着叶尘怒吼起来，身上升起浓浓的杀意！
<br/>　　其余三名随从，此刻也已经缓过劲来，看到眼前的这一幕，也纷纷怒吼起来，
<br/>　　“叶枫！你竟然敢打伤叶成公子，你就等着被抽筋扒皮吧！”
<br/>　　“好你个废物！真是好大的胆子！”
<br/>　　“我这就去请家族的执法长老，今天非得活剥了你这废物不可！”
<br/>　　听到几人的叫嚣声，叶尘顿时眉头紧皱，
<br/>　　他也没有想到，这才刚刚进入这个世界，就招来了杀身之祸，
<br/>　　“看来这最后一关的考验，果然没有那么简单！也罢，当断不断，必受其乱！杀！”
<br/>　　一念及此，叶尘直接身体一晃，突然冲到叶成跟前，在其余众人无比震惊的目光中，狠狠一掌拍在了他的脑门之上。
