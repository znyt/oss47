　　楚风眠的力量，虽然强，甚至也足以说是碾压同境界的武者。
<br/>　　但是葬天的力量，才更是强的吓人，百绝榜第四位的沧澜惊，甚至都接不下他的一拳。
<br/>　　楚风眠跟葬天选择硬碰硬，在众人看来，绝不是一个明智的选择。
<br/>　　“轰！”
<br/>　　两拳相撞。
<br/>　　猛烈的灵力肆意，横扫这整个擂台，这擂台上都在震动。
<br/>　　就算是在擂台之外，无数众人都感觉到了其中爆发相对的力量，一个个不由的催动灵力来抵挡着。
<br/>　　这擂台上的禁制，居然都无法彻底将他们二人的力量给化解，甚至是有些要撑不住。
<br/>　　这力量的碰撞，是这一届武斗大会之中，最为激烈的一次交锋。
<br/>　　灵力碰撞，在瞬息之间便是结束，一道身影猛然退去。
<br/>　　众人顺着目光看了过去，这退去的身影，居然是葬天。
<br/>　　在这力量碰撞之中，居然是葬天败退了，而楚风眠，则是站在原地，一脸轻松的样子。
<br/>　　“我不相信，今日我杀不了你！”
<br/>　　葬天被逼退，但是他的脸色杀意却是更浓。
<br/>　　这葬天，对楚风眠恨之入骨，曾经在武胜国度，他便是被楚风眠死死的压住，一直抬不起头来。
<br/>　　他来自七大宗门，本来便是心中极为自傲，没想到在一个小地方，居然都被人死死的踩在脚下。
<br/>　　如今他又得奇遇，来到北域，本来便是要报仇楚风眠。
<br/>　　在这武斗大会上，是他斩杀楚风眠最好的机会，因为在这里，不管是楚风眠的背后是谁，都无法出售，参与其中。
<br/>　　葬天的心中已经是无数次的计划好了，如何斩杀楚风眠。
<br/>　　但是却万万没有料到一点，便是楚风眠的实力，居然会强大到了这一步。
<br/>　　他的大心魔决用来对付楚风眠，却是没有任何一点作用，这已经是让葬天的心中产生几分慌张了。
<br/>　　接下来，在他最为自信的力量上，居然都是输给了楚风眠，这让他如何能够承受，他的脸色都变的极为难堪起来。
<br/>　　但是葬天毕竟也是一代天才，他的心中没有任何的慌张，就算是如今处于劣势，他的心中所想的，也是如何可以对付楚风眠，动用如何手段杀了楚风眠。
<br/>　　他目光的杀意，越来越浓，甚至是脸色都是变的极为阴狠起来。
<br/>　　突然就在葬天的手中，一枚拳头大小的果子，突然出现。
<br/>　　这果子上面蕴含的纹路，都是怪异到了极点。
<br/>　　一些武者，看向这果子上面的纹路，都是一下子有些面露呆滞，似乎是被心魔入侵。
<br/>　　“心魔道果？”
<br/>　　楚风眠看到这果子的一刻，心中低声自言自语一声。
<br/>　　这果子，正是心魔道果，相传是一些修炼大心魔决的强者，抽取其他人心中的心魔，培养出来的果子。
<br/>　　这心魔道果，对武者来说，是绝对是致命之物，一些心智稍弱的武者，看到这心魔道果的时候，都会有可能会受到影响，引发心魔来。
<br/>　　但是对修炼大心魔决的武者来说，这心魔道果可是绝对的大补之物，对他们来说的效果，比起天级灵丹都要强大。
<br/>　　一枚心魔道果，想要凝聚，都不知道要耗费多少时间手段才行，这心魔道果，应该是葬天打算在度过生死大劫的时候动用的。
<br/>　　如今在楚风眠的逼迫之下，却是直接动用出来。
<br/>　　“小子，你让我失去了一次度过生死大劫的机会，不过没事，等我杀了你，得到了你的奇遇，这一切损失，都可以补回来！”
<br/>　　葬天看着楚风眠狰狞一笑。
<br/>　　他手一动，这心魔道果，便是直接落入到了葬天的口中。
<br/>　　只见葬天的身形，开始猛然膨胀起来，他身上的魔意，也是变得恐怖无比。
<br/>　　葬天看着楚风眠大声的笑了起来。
<br/>　　“楚风眠，今日你是必败的，就算是你的实力，超出了我的计算，哪又如何？今日你必死无疑！”
<br/>　　这心魔道果服下，葬天身上的气息，比起刚刚要猛然暴涨十倍，他的脸上，也是流露出了极端的自信。
<br/>　　“你手中的祖龙至尊剑，这样的宝物，根本不是你配得到的，还有你的一切奇遇，都将是会属于我的！”
<br/>　　葬天看着楚风眠，眼神中尽是贪婪。
<br/>　　楚风眠身上的无数奇遇，都是一个个的宝藏，杀了楚风眠，这些宝藏，便是都全部属于他。
<br/>　　“是吗？葬天，你以为你的实力暴涨，就是我的对手了？”
<br/>　　楚风眠看着葬天平静的开口道。
<br/>　　“一个小魔头，你的实力别说是强上十倍，强上百倍，在我的面前，也终究只有陨落。”
<br/>　　面对这葬天暴涨的实力，楚风眠的脸色平静极致。
<br/>　　这平静，并非是刻意去装出来的，而是楚风眠的心中，正是毫无波动。
<br/>　　如今在强大的力量，在楚风眠的面前，他都不会有着任何波动。
<br/>　　他的九域剑术，如今在这武斗大会上，随着他观看无数战斗，早已经再度升华。
<br/>　　现在的楚风眠，就算是面对曾经那陆家的老祖，达到碎命之境的陆千刃，楚风眠虽然无法战胜，但是却也有着跟他一战的自信。
<br/>　　这葬天的实力在暴涨，也远远无法达到碎命之境的层次，这种实力，虽然棘手。

<br/>　　今日这葬天，依然只是一个死人。

<br/>　　葬天猛然怒喝一声。

<br/>　　他的这一剑，没有丝毫的技巧，但是这纯粹的力量，都足以是让生死境巅峰的强者，直接陨落。
<br/>　　“纯粹的力量，你的剑，简直就是对剑道的侮辱。”
<br/>　　楚风眠冰冷的开口道。
<br/>　　“就你这种手段，也想要废了我？可笑，今日我便是告诉你，什么叫做真正的剑道！”
<br/>　　随着楚风眠话音的落下。
<br/>　　他手中祖龙至尊剑动了，一剑猛然冲天而起。
