　　神龙岛上空，帝释天目光极其冷冽，望着满脸讥讽的慕清霄，前所未有的暴怒，心中杀意凛然。
<br/>　　数千年以来，他曾何几时让人这样嘲讽过！
<br/>　　“慕清霄，你真是让本座太失望透顶，原来本座还想好好培养你，让你成为本座的左膀右臂，既然如此，那你就去死吧！”
<br/>　　话语刚落，天境九层的气息自帝释天身上扩散而出，圣心诀参杂着狂暴无比的灵气，瞬间扑面而来。
<br/>　　（天境对应的是金丹，神境对应的是元婴。）
<br/>　　“哼！”
<br/>　　慕清霄冷哼一声，金丹八层巅峰的气息爆发而出，两股力量相撞，四周狂风四起，尘埃弥漫，乱石滚滚，风云骤变。
<br/>　　“你......你居然是天境八层，真是能够隐忍，我帝释天真是小瞧了你。”
<br/>　　慕清霄散发出天境八层的气息，当真让帝释天震撼不已，心中也升起浓浓的忌惮与杀意，如此年轻的绝世强者，龙元要留下，他的命更要留下。
<br/>　　否则，后患无穷！
<br/>　　“慕清霄，难不成你以为，凭你天境八层的实力，能够与本座抗衡，本座也很久没有动用全力，能死在本座的圣心诀下，是你的荣幸。”
<br/>　　话语刚落，帝释天圣心诀运转，浩荡的灵气化作成千上万的冰刃，随着他手臂向前一推，冰刃快如闪电，直接朝后者穿刺而去。
<br/>　　“圣心四诀.万仞穿云！”
<br/>　　冰刃迎面而来，慕清霄不闪不避，将龙元收入储存空间中，凝气化剑，一柄长剑出现在手中，寒芒闪烁间，数千道剑气飞射而出。
<br/>　　“铿锵......”
<br/>　　剑气与冰刃在空中相撞，发出一连串清脆的撞击声，最终两股力量相互抵消，化作灵气消散于天地之间。
<br/>　　“帝释天，你这千年真是活到狗身上去了，难道就这点能耐吗？”
<br/>　　待冰刃与剑气消散，帝释天发现慕清霄不仅毫发无损，还能开口讽刺自己，双拳紧攥在一起，心中的杀意更加浓郁。
<br/>　　“很好，非常的好，你越是强大，待本座将你击杀时，越是解恨。”
<br/>　　“噼里啪啦......！”
<br/>　　忽然，帝释天身上涌出无尽的雷弧，这是圣心四决中的『帝天狂雷』，狂暴的灵气由冰化作雷，欺身而上，瞬间攻至慕清霄面前。
<br/>　　霎时间，帝释天身体周边雷球乍然而现，当双拳即将攻击到慕清霄的时候，豁然如雷般爆炸。
<br/>　　“轰轰轰......！”
<br/>　　慕清霄剑眉一挑，感受到雷球中的狂暴气息，身形闪烁急速爆退，当离开攻击范围后，白色的长袍上已经满目疮痍，身躯也出现不少的血痕。
<br/>　　帝释天千年的修为，凤血的灼热气息，以及圣心诀的异变，攻击力当真变态，根本不是一般金丹强者能够抵抗的。
<br/>　　与此同时，在两人交战的左侧，步惊云，聂风，以及神母，怀灭，怀空互相对峙着。
<br/>　　但谁都没有动手，而是看着上空，那令人震撼的战场。
<br/>　　“慕清霄，知道本座的力量了吧，本座刚刚也只是动用了七层的实力，若是动用全力，你早就死无全尸，本座在给你最后一个机会，臣服或者死！”
<br/>　　站在高空之上，慕清霄长发随风而动，瞥了眼身上的伤痕，嘴角牵起一丝不屑。
<br/>　　“七层的实力，呵......我刚刚只是试探你的力量有多强，还真能往自己脸上贴金。”
<br/>　　“你非常的好，本座还是第一次遇到你这种冥顽不灵的家伙，你有没有感觉到浑身发软，完全使不上任何力气，心腑剧裂，痛不欲生？”
<br/>　　闻言，慕清霄挥挥长袖，嘲讽道：“你是说半年来，给我饭菜中下的罗厄粉吧，我可从未中过那种东西。”
<br/>　　听到慕清霄的话语，帝释天怔住了，真是出乎意料，他百思不得其解，后者为何如此清楚，甚至能够提前警觉，将罗厄粉从体内清除。
<br/>　　他当然不知道，慕清霄修炼《御女心经》，对任何东西都异常敏感，罗厄粉无色无味，其他金丹期修士恐怕会中招，但对他却完全不管用。
<br/>　　此刻，帝释天已经清楚。
<br/>　　面前的青年比他遇到的任何人，都难对付的多，甚至已经威胁到自己，所以他决定动用全力，尽快将他轰杀，以绝后患。
<br/>　　霎时间，帝释天手中光芒大作，紧接着一柄带着骇人气息的长剑，乍然而现。
<br/>　　“圣心四劫.极神劫，这柄剑是本座的元神所化，灌注着本座最强大的力量，本座千年以来，第一次使用这招，你能死在这招下，应该觉得荣幸。”
<br/>　　望着帝释天手中，散发着狂暴且锋利气息的长剑，慕清霄嘴角勾起一丝邪异的笑容。
<br/>　　元神所化的长剑，倒是和魂剑有些相似，但也完全不同。
<br/>　　元神非魂非魄，魂为阳神，魄为阴神，阳神可分为三部分，即天地人三魂。
<br/>　　三魂统一中有一点真阴，此为元神，它是通过修行人修炼而逐渐掌握的可以控制魂魄的物质，它也是人类生命的真正意义与一切精华。
<br/>　　而灵魂即是主魂，主宰人的意识，慕清霄的魂剑就是用主魂所铸，帝释天在他面前展现元神之剑，真是班门弄斧。
<br/>　　“活了上千年，你难道不懂人外有人，天外有天的道理，任何事情都不能够妄下定论，就让你看看，什么叫做真正的魂剑。”
<br/>　　“嗡嗡嗡......！”
<br/>　　慕清霄话语声刚落，一柄霸气无双的长剑，带着贯彻九霄的剑意，破空而出。
<br/>　　无上剑意将整个神龙岛都笼罩在内，步惊云，怀空，以及帝释天的元神之剑都颤抖起来，仿佛在恐惧在朝拜。
<br/>　　慕清霄手握魂剑，也不再啰嗦，剑意凝聚成一柄长达数百米的剑气，携着斩破虚空的恐怖力量，径直朝帝释天斩下。
<br/>　　剑气已经迎面而来，感受到其中参杂的恐怖力量，帝释天急忙握紧元神之剑，圣心诀全速运转。
<br/>　　霎时间，狂暴的灵气全部凝聚在元神之剑上，想要以此来抵挡住这一剑！
<br/>　　“嗡嗡！”
