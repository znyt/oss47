　　一行四人，先后进入了办公区域，真户随手将手提箱放在桌子上，转过头来。
<br/>　　“亚门，你通知检验部门，让那些家伙把尸体带走，顺便，帮我拟写一份任务报告。”对亚门吩咐了一声，真户拉开自己小组办公室的推拉门。
<br/>　　在CCG，每个小组都有自己的办公区域，平日的时候，除非有需要合作的任务，不同小组的成员也很少聚集到一起。
<br/>　　至于刘哔和什造两人，已经属于比较特殊的情况了。
<br/>　　“我还有事情要做，你们回吧，如果没什么意外的话，库克因这周可以做出来。”看到亚门离去，真户转过了身，话语中已经带上一些逐客的意思了。
<br/>　　嘀咕了一声，什造和刘哔告了一声别，转头向自己小组的办公区域走去。
<br/>　　还站在原地的，就只剩下有些蒙圈的刘哔了，因为，他压根不知道自己属于哪个小组。
<br/>　　看着眼前紧闭的大门，刘哔皱起了眉头，仔细琢磨了一会后，走出了办公区域。
<br/>　　与其待在CCG，他还不如趁着这段时间去20区看看，他要确认这个世界现在的时间线，也好对接下来的事件作出相应的对策。
<br/>　　他来这个世界，可不是为了游玩的。
<br/>　　……
<br/>　　接下来的三天里，刘哔都没有回到CCG，他一直在20区寻找，那家名为古董的咖啡店。
<br/>　　当然，在搜寻咖啡店的时候，他也顺手击杀了一些喰种以及搜查官，虽然这些家伙的等级都不算很高，但也给他提供了一千多的积分。
<br/>　　似乎是被刘哔毫不收敛的做法激怒，CCG很快对此做出反应，派遣了大量的搜查官进入20区，各种有关他的新闻报道也从电视上传播了起来。
<br/>　　来自东方的‘修罗’，CCG在新闻中，是用这样的词语来称呼他的，这个代号的起源，还是来源于他脸上的那张鬼脸面具。
<br/>　　这张出自系统的特殊面具上，绘着一张带有浓郁东方色彩的小丑脸庞，几乎将他的整张脸给覆盖住，只在眼睛的部位留下了两个空洞。
<br/>　　虽说小丑脸上的表情，是在做一个滑稽的鬼脸，但是给人的感觉，却丝毫没给人带来一丝好笑的意味，看上去，反倒让人感觉有些诡异和狰狞。
<br/>　　当然，也只有在夜晚的时候，他才会将面具戴上，平日里，他依旧是往常那般人畜无害的模样。
<br/>　　第三天的下午，刘哔来到一间装修精美的咖啡厅内，或许是因为在上班时间，咖啡厅内没有什么客人，只有几个服务人员在柜台前闲聊着。
<br/>　　感受到四周蛰伏的强大气息，刘哔轻笑一声，来到一个靠窗的位置旁坐下。
<br/>　　终于找到了，这几天他都跑了二十多家咖啡厅了，如果再找寻不到，他都要考虑换个目标了。
<br/>　　这个世界不缺喰种，但高级别的喰种数量却并不太多，那些低级的喰种，不光给他提供不了积分，更吞噬不出什么有用的属性。
<br/>　　故此，他才会执着寻找到这里，毕竟这个地方可是聚集着大量的高阶喰种，而且每个喰种的能力，都有自己的特殊性。
<br/>　　吞噬了这么多能量体，刘哔也找到了一些规律，一般越特殊的能量体，越是能吞噬出好的属性。
<br/>　　像寻常的那些低级喰种，最多也只能给他增加一下体内的能量罢了。
<br/>　　“先生您好，请问要点些什么？”刘哔刚刚做到座位上，一个娇小可爱的少女就走了上来，脸上挂着标准的笑容，向他递出了手中的菜单。
<br/>　　虽然外表上是一副人畜无害的模样，可面对着这样一个女孩子，他依旧感觉到一股压迫感。
<br/>　　是S级喰种么，按照年龄来区分，应该是董香吧。
<br/>　　视线在少女身上停留了几秒，略微有些放肆，随手，刘哔收回了自己的目光。
<br/>　　“不好意思，真是失礼了，毕竟是第一次看到这么可爱的喰种呢。”带着绅士的微笑，刘哔接过了菜单，口中的话却让董香脸上的表情瞬间凝固。
<br/>　　随手翻看了一下，都是些人类的食物，他心中也失去了兴趣，他还以为喰种开的咖啡厅，有什么不一样的地方呢。
<br/>　　“给我来杯白水吧，喰种不能吃人类食物，想必你也应该了解。”将菜单推到一边，刘哔随口说道。
<br/>　　董香一时间没有接话，看着刘哔的目光有些戒备，脸上的表情也开始逐渐消失。
<br/>　　“你这家伙，到底是什么人啊，喰种或者人类，都不会有你身上这股味道啊。”董香后退了一步，面无表情的看着刘哔。
<br/>　　似乎是察觉到这边的气氛不对，一旁在柜台后面闲聊的两男一女也走了上来。
<br/>　　这家咖啡厅内店员的身份，倒是很好区分，那个身材有些微胖的大鼻子，应该就是原著中被称为‘魔猿’的古间圆儿吧。
<br/>　　他身边那个有些妖娆的少妇，应该就是‘黑色杜宾’的首领入间萱，倒是比他印象中的妩媚许多。
<br/>　　最后那个看起来有些呆呆的男孩，吸引了他绝大部分的注意力，金木研，这个原著中的主角，基本上大部分的事件，都是围绕着他展开的。
<br/>　　一间小小的咖啡厅里，却聚集了如此豪华的阵容，也着实让人意想不到。
<br/>　　目光在金木身上停留了一会，刘哔压下了心中蠢蠢欲动的想法，这家店里的高手太多，凭借现在的他，很难应对。
<br/>　　他可不会忘记，这个地方的店长可是一名SSS级别的赫者，那家伙，指不定在什么地方观察着他们呢。
<br/>　　金木研现在还在古董，这样说来，现在的时间线应该只是刚开始的时候，很多剧情，应该还没有展开。
<br/>　　也就是说，他还有很多事情可做。
<br/>　　“呐，你们对待客人的态度，有点不太友好啊。”看着围在身边的几人，刘哔脸上的微笑开始逐渐消失。
<br/>　　不愧是身经百战的高阶喰种，古间圆儿，入间萱两人走上来的时候，就有意和董香站立的位置形成了一个犄角，一旦动起手来，他立刻就要受到三个人的围攻。
<br/>　　至于金木研，那个家伙，现在就是一个废物，根本还不知道发生了什么事情。
