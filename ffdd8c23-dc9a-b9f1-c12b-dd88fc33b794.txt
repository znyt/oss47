　　除了这身代表着北邙学院的衣服，还有另外两件东西，便是代表身份的身份牌，以及那北邙大典。
<br/>　　这北邙学院的身份牌，跟武胜学院的极为类似，第一便是可以证明身份，第二则是可以从中记载功勋值。
<br/>　　这北邙学院，也有着功勋值，只要是完成其中的任务，便是就可以换取珍贵的奇物。
<br/>　　在北邙学院中的许多奇物，都是外面所无法得到的，只要是攒够了足够的功勋值，都可以兑换。
<br/>　　楚风眠第一滴血，在这身份牌上，便是显现出来了楚风眠的名字，以及一个零的数字。
<br/>　　这零的数字，便是代表着如今楚风眠的功勋值。
<br/>　　像是楚风眠这样，刚刚来到北邙学院的武者，都是没有任何的功勋值的，想要得到功勋值，换取奇物，都必须要去接取任务，为北邙学院做事才行。
<br/>　　至于最后的北邙大典，则是一本书，楚风眠打开一看，其中便是记载了无数北邙学院的历史，规矩，还有这如今北域的势力分布种种。
<br/>　　这关于北邙学院的历史，规矩，楚风眠便是没有太过理会，只是随意一看，记得差不多便是。
<br/>　　他最为在意的，是后面所记载的，如今北域的势力分布，还有这如今九域的势力分布。
<br/>　　虽然写的极为笼统，但是对楚风眠来说，也还是宛如珍宝。
<br/>　　他重生万年，这万年间发生的种种变化，楚风眠一概不知。
<br/>　　像是北邙学院，这样的庞然大物，都是在万年间崛起的，万年间，已经足以让九域中的势力大洗牌一次了。
<br/>　　如今看着北邙大典中的记载，楚风眠便是发现了许多陌生的名字。
<br/>　　当然其中的不少名字，楚风眠还都是极为熟悉的。
<br/>　　“天剑宗，荒古门，圣王宗，天枢殿，九魔城，古月峡谷，药王谷。”
<br/>　　这七大宗门的名字，也赫然在列，如今依然是位于九域顶端的庞然大物。
<br/>　　“等着，很快了，我要将这七大宗门，从九域中，彻底除名！”
<br/>　　楚风眠的眼神中，都露出了几分杀意，这股杀意，不同以往，乃是楚风眠心中达到极点的恨意。
<br/>　　“怎么了？”
<br/>　　在一旁的寒月漓，也察觉到了楚风眠的变化。
<br/>　　这种惊天杀意，可是她从来没有遇见过的，一向是性格极为淡漠的楚风眠，居然会有如此可怕的杀意。
<br/>　　她急忙拍醒了楚风眠，才让楚风眠身上的杀意，收敛了起来。
<br/>　　楚风眠清醒过来，也是一下子冒了冷汗，他差点都忘了，这里是在北邙学院中。
<br/>　　众目睽睽之下，若是表现的太过异样，很有可能楚风眠都会被抓起来审问的。
<br/>　　“没事，只是想到了以前的一些事情。”
<br/>　　楚风眠感激的看了一眼寒月漓道。
<br/>　　若是没有寒月漓在一旁提醒，惊醒了楚风眠，楚风眠差一点都被仇恨迷失了双眼。
<br/>　　随着楚风眠的实力的提升，他心中的恨意非但没有消失，反而是越来越浓。
<br/>　　平日里没有受到影响，但是当看到这七大宗门名字的一刻，楚风眠心中的杀意，还是涌现了出来。
<br/>　　“不行，现在我的实力，在七大宗门前，还很弱小，只能隐忍，等到我实力足以抗衡七大宗门的一刻，便是七大宗门在九域除名之时。”
<br/>　　楚风眠的心中，暗自想到，将情绪，便是调整了回来。
<br/>　　“好了诸位，诸位已经是通过了考核，便已经都是我们北邙学院的弟子了，一旦进入我们北邙学院，便是终生不能退出，必须要以我们学院为荣，谁敢退出，便是死路一条。”
<br/>　　苏凌长老，冷声开口道。
<br/>　　在他开口的瞬间中，便是也蕴含着一股极为恐怖的杀意，令人听到，都不由有些不寒而栗。
<br/>　　宫殿中的气氛，瞬间凝到了冰点。
<br/>　　苏凌长老笑了笑开口道。
<br/>　　“不过，既然诸位已经成了我们北邙学院的弟子，我们北邙学院自当好好培养诸位，不用担心，只要不背叛门派，学院也不会限制自由的。”
<br/>　　这一句话，倒是打破了如今冰冷的僵局。
<br/>　　这是一种震慑，楚风眠自然清楚，他早已经见怪不怪了，淡然的看着这一切。
<br/>　　“现在所有人听令，全部离开宫殿，在外，已经准备好了方舟，将所有人全部送到我们北邙学院真正所在的地方。”
<br/>　　苏凌长老说了一句，便是直接转身离开。
<br/>　　楚风眠跟寒月漓，随着人群走到外面去，便是看见了足足十艘巨大无比的方舟。
<br/>　　这方舟，比起武胜学院的，足以大上无数倍，其中方舟上刻画的无数禁制阵法，也是要玄妙的多。
<br/>　　楚风眠跟寒月漓，踏上其中一艘方舟。
<br/>　　这方舟，便是直接突破空间，进入到了空间隧道之中，不一会，眼前便是出现了一片群山。
<br/>　　在这群山中，高楼环绕，宫殿林立，足足是有着无数国度大小。
<br/>　　这群山中的一切，都是属于北邙学院。
<br/>　　“这北邙学院，居然是在一处独立的空间之中。”
<br/>　　楚风眠看到眼前的一幕，也是微微有些惊讶。
<br/>　　如今这北邙学院的所在之地，居然不是在北邙城中，而是在另外的一处空间之中。
<br/>　　能够开辟出如此之大的一片空间，用来当做北邙学院的所在地，这北邙学院曾经开宗立派的强者，比起楚风眠想象中的还要强大许多。
<br/>　　能够做到这一点的，只有在九域中都位列巅峰的人，才足以。
<br/>　　不止是楚风眠，方舟上的许多人，都是更加震惊，他们还从未想过，居然会有人在虚空之中，开辟出这样一道巨大无比的空间来，开宗立派。
<br/>　　苏凌长老看着众人羡慕，震惊的眼神，脸上也是露出了几分骄傲道。
<br/>　　“我们北邙学院的驻地，就在这虚空之中，是曾经我们北邙学院的开宗祖师，以无上神通，从虚空中开辟出来的。”
<br/>　　“你们在其中修炼，安全无比，完全不用担心任何外来的威胁。”
<br/>　　在这北邙学院中修炼，倒是的确安全无比。
<br/>　　毕竟是一处单独封锁的空间，楚风眠感觉的到，就算是生死境的强者，都不可能硬闯的了这片空间。
