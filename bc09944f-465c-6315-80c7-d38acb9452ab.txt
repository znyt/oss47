　　“轰隆！”“轰隆隆！”“隆隆……！”
<br/>　　九天之上，随着万劫魔尊证道混元，风云倒卷，日月暗淡，星辰无光，乌云汇聚而来，遮天蔽日，滚滚神雷，震耳欲聋的响声，轰天彻底，雷电狂爆，天劫疯狂，一场大劫在酝酿。
<br/>　　天劫！
<br/>　　自从原始圣主成圣后，便立下了修士突破要渡劫的规定，如今，万劫魔尊突破混元，没想到一样有天劫降临。
<br/>　　原本朗朗乾坤，碧空如洗，而此刻突然间雷光万丈，电芒如海，淹没九天十地八荒，这是一种吓人的场面。
<br/>　　仅仅一刻钟的时间，天穹便像是要坍塌下来一般，乌黑的雷电蛇蟒在乱舞，有数不尽的雷电蛟龙在冲腾。
<br/>　　“轰！”
<br/>　　声音震耳欲聋，若是凡人在此，耳膜都要被击穿，巨大的声音似麒麟在吼动山川，要将大地翻了过来。
<br/>　　远远望去，那里沉闷而可怖，一片乌光如平瓢大雨一般倾泻而下，密密麻麻到处都是暴雷。
<br/>　　“轰隆隆！”
<br/>　　其中还有不少血色的闪电如一条条血河在入海，这番景象骇人，这片天空的寂静一下子被打破。
<br/>　　六合八荒，无远不届，压抑的气息降临，挤满了这片宇宙，那是无穷无尽的雷海，化成了万劫魔尊的劫。
<br/>　　轰隆隆！
<br/>　　沉闷的声音像是上苍的战车在隆隆作响，碾压过来，主宰者像是降临了，要亲手对付万劫魔尊，将他镇杀。
<br/>　　万劫魔尊全身上下的因果之力被引了出来，无数因果怨力充斥其身，将整片天空渲染成墨色，无尽冤魂在咆哮，天地之间，九天之上的大劫像是被激怒了，大劫变得更加狂暴了。
<br/>　　众所周知，身上因果之力越多者，渡劫时面对的大劫就越是狂暴，而万劫魔尊身上，更是因果无数，可见他一生造成多少杀孽。
<br/>　　“嘶！”
<br/>　　看到这一幕，所有人纷纷倒吸一口气，头只觉得皮发麻，不敢置信，这是要杀多少人啊，才有如此多的因果业力。这一刻，他们看向万劫魔尊的目光，就仿佛是在看一个绝世大魔头。
<br/>　　同时，也有人觉得，万劫魔尊就是一大团功德，只要杀了万劫魔尊，天道便会降下无尽功德之力。于是，有人看向万劫魔尊的目光之中，带着浓浓的火热。
<br/>　　当然，也有人十分清晰，认识到了万劫魔尊的实力，这种人连圣人都敢拼杀的，天地间哪还有他不敢做的事情，想要杀他，别做梦了，还是洗洗睡吧。
<br/>　　“还是天罚圣主狠啊，一道天劫降下，看看有几人能够度过？”
<br/>　　“是啊，元始圣人打得好算盘，修士渡劫，天经地义，可是，这万劫魔尊的劫也太狠了吧，简直是十死无生啊。”
<br/>　　“嘿嘿！”
<br/>　　“……”
<br/>　　有人认为是元始不想让万劫魔尊证道混元，所以降下雷罚，要灭了万劫魔尊。不止一人这么认为，洪荒中大多数人都这么认为，就连圣人都认为如此。
<br/>　　要知道，掌控天道因果天罚之力的可是元始圣主啊，如今他老人家降下如此灭绝天地的天劫，不是要灭了万劫魔尊又是如何？
<br/>　　顿时，无数人看向元始的目中之中，带着深深的敬畏，惧怕，要知道，他们也是修士啊，若是哪一天得罪了元始，渡劫之时，人家随便放个大劫下来，你还不是要身死道消，无尽修为，一朝散尽。众人想想就觉得十分可怕。
<br/>　　其他接近混元的修士，更是脸色悲苦，因为他们想到了，他们以后成就混元之时，若是元始也降下如此大劫，那他们还怎么混啊。
<br/>　　一个个目光闪烁，脸色阴沉的看着大劫之下的万劫魔尊。
<br/>　　此刻，就连准提圣人都躲避开了，来到远处，饶有兴味的看着万劫魔尊，他一样认为那是元始在阻止万劫魔尊成道，对此，他乐见其成。
<br/>　　他不认为万劫魔尊能够在如此天劫之下活下来，因为他能够感受到九天之上的那道天劫，蕴含着狂暴的力量，那股力量，甚至能够威胁圣人。再说，就算万劫魔尊侥幸不死，到时候也肯定会陷入虚弱之中，他在旁边，到时随手便可灭了万劫魔尊。
<br/>　　不管其他人怎么想，在天劫之下的万劫魔尊，看着九天之上还在酝酿着的狂暴雷电，他面无表情，毫不畏惧，甚至，心中隐隐还有些兴奋。
<br/>　　万劫魔尊修炼的天经为《万劫魔典》，需要不断吞噬天地劫气。而天劫、雷电，又何尝不是一种劫气呢？
<br/>　　“吼，本尊身为魔尊，魔屠苍生本就是天经地义，因果之力又能奈我何？”
<br/>　　万劫魔尊看着身上的无尽因果业力，牵引出更多的天劫，甚至虚空之中，连因果业火都出现了，他毫不畏惧，壮志凌云，对天大吼道。
<br/>　　不过，雷劫还在酝酿之中，虚空之中最先出现的灾劫是风劫。
<br/>　　风、火、雷，这三劫乃是磨练肉身的。
<br/>　　对修士而言，若是度过了，便会凝聚出更强的仙体，使得肉身更加完美，更加契合大道。越强大的天劫，造就出来的仙体也更完美。
<br/>　　毕竟，仙体也有强弱之分，一些弱的仙体，单单要将仙体修炼圆满，就需要花费无尽岁月。仙体的完美，乃是踏入仙道的其中一个门槛。
<br/>　　“隆隆隆……”
<br/>　　狂风呼啸。
<br/>　　无数风刃形成了巨大的旋涡，万劫魔尊就在旋涡的中央。
<br/>　　“可怕。”
<br/>　　“单单是这风劫，就已经可以威胁准圣高手了。”在远处观看的大神通者们，纷纷色变，这风劫的威能太强了。
<br/>　　而且，这些风不仅作用在人体的肉体之上，更是能够吹进人体的五脏六腑之中，作用在人体内部，让人防不胜防，真是恐怖。
<br/>　　但万劫魔尊毫不在意，面对这些可怕的风灾，他仿佛完全放弃了抵抗，坐在旋涡中心，宛若一座大山，岿然不动，任他雨打风吹，我就是不动分毫。而且，他还运转起《万劫魔典》，全身上下形成三百六十五道黑洞旋涡，吞噬着这些风灾之力，增强着他的法力。
<br/>　　看到这一幕，众人又是一阵惊讶，一阵议论。
<br/>　　三年过去了。
<br/>　　“哗……”
<br/>　　但随着最后一道无形无相的“无相罡风”从头顶的卤门进入身躯，让万劫魔尊身躯的每一处都在颤动，可万劫魔尊的内心却静如止水，丝毫不为所动，反而还有些兴奋。在这种震动中，万劫魔尊的万劫魔体也在悄无声息发生这一些变化，这些变化毫不起眼，可等到雷劫结束之后才会真正有一个质变。
<br/>　　“风劫，过了。”不少人心中都很清楚，风劫根本奈何不了万劫魔尊。
<br/>　　万劫魔尊抬头，目光平静的看向高空。
<br/>　　一眼看去，只见高空之上，出现了巨大的炙热无比的旋涡云团，旋转的云层中央可以肉眼可见泛着金色的火光在流动，但随着云团的下降，周围温度迅速飙升。
<br/>　　到后来，连空间都变得滚烫起来，甚至，有些空间直接经受不住如此高的温度，变成了虚无。
<br/>　　万劫魔尊周围的虚空，在一瞬间变得通红无比，甚至随着云团的下落，更是直接化为虚无。空间在震动，好似在膨胀，想要爆炸。
<br/>　　万劫魔尊则是虚空而立，丝毫不为所动，抬头看着云团。
<br/>　　……
