　　第679章
<br/>　　在城主府尚未被攻占的时候，宋伯康就已经携带者家小逃生。同行还有戴白、陆洛这样的凉州城文武官员。
<br/>　　当程大雷知道这一点时，顿时觉得匪夷所思，开口问道：“那如今守在营中的是谁？”
<br/>　　“不知道，没听说过。”赵子龙答道。
<br/>　　蛤蟆城事先已经收集过关于凉州城的情报，数得上号的人物大家都有所了解。现在赵子龙说不知道，定然是营中只剩下些小鱼小虾，或者说，营中根本没有主事者，如今是一锅乱粥。
<br/>　　程大雷长舒一口气，到此刻几乎已经可以宣布，自己已经占领凉州城。
<br/>　　他骑着黑牛出列，立在兵营的大门前，深吸一口气，口中喝道。
<br/>　　“营中的人马听着，某乃是凉州程大雷，宋伯康罪大恶极，本当家替天行道，与诸位兄弟无干。”
<br/>　　嗖！
<br/>　　一柄羽箭凌空向程大雷射来，程大雷抄手接住，随手丢在一旁。
<br/>　　“诸位兄弟都是帝国的好男儿，何必为虎作伥。想一想你们的妻儿老小，想活命的空手走出营门，本当家既往不咎，送你们回家团聚。”
<br/>　　连环羽箭射来，程大雷便说边接，拿住箭后便丢在一旁，看上去轻描淡写至极。
<br/>　　“本当家给你们一条活路，天亮之前还不投降，莫怪我大开杀戒。”
<br/>　　手下人齐声怒吼，将『降者免死』四个字喊得气冲斗牛。
<br/>　　程大雷退了回来，面对箭雨还是蛮危险的，稍有疏忽自己一条命也就报销了。
<br/>　　“大当家，他们会投降么？”秦蛮问。
<br/>　　“谁知道呢。”程大雷随口道了一声：“有人不怕死，但没人不想活。给他们腾出点地方，放他们出城。”
<br/>　　程大雷如今也是虚张声势，一面围着兵营，一面还得安抚城中的事，如果对方大军冲出来，程大雷就只能转身逃跑。
<br/>　　他现在欺得就是对方群龙无首。
<br/>　　命令手下人大呼小叫，冲着军营喊话。
<br/>　　说些宋伯康已经逃走了，大家不要做没必要的抵抗。程当家向来唯才是用，跟在程当家身边也能有一份前程。
<br/>　　程大雷看着东方，战斗持续到现在，东方已经露出鱼肚白。若天亮之前，对方还是不肯投降，程大雷就必须杀进去了。
<br/>　　关键是自己不知兵营中现在是什么情况，程大雷如今也不敢打进去。
<br/>　　“大当家，大当家……”
<br/>　　刘悲骑着一匹快马，身后跟着来自蛤蟆城的大军。程大雷是先头部队，只有三千人，刘悲率领的有一万人。如果今天程大雷没有拿下凉州城，他便负责支援断后。
<br/>　　但如今程大雷已经攻进城，刘悲便领着大军冲了进来。
<br/>　　“来了。”程大雷一激灵，有了这一万兵，自己做事就更顺手些。
<br/>　　“来人，与我围上。”
<br/>　　程大雷大喝一声，拔出匹夫剑下令。没用多少时间，手下便将兵营大门堵住。程大雷用剑指着前方，口中喝道：“呔，活路你们不走，偏偏要自寻死路，再不投降，本当家便杀进去，勿谓言之不预也。”
<br/>　　“来人，给我摆上。”
<br/>　　刘悲不仅带着兵来，也带着蛤蟆城生产的各种攻城器械。两架投石机摆在兵营前，这次投得不是石头，而是可以产生爆炸的燃烧弹。
<br/>　　“把他们的大门给我轰开！”
<br/>　　只用了两发炮弹，就将营门炸开，一时声势动天。
<br/>　　程大雷冷哼一声，他手上早已沾满鲜血，还在意再杀一批人么。
<br/>　　只要这样人仍坚持不投降，他就指挥大军杀进去，给他们来个一网打尽。
<br/>　　“程当家，我们投降，我们投降……”
<br/>　　营门的烟雾散去，一队人举着双手从里面走出来。
<br/>　　“大当家，小心有诈。”刘悲提醒道。
<br/>　　程大雷点点头，手下现在没有放松警惕，满张之弓、投石机都做好准备，只要这些人稍有异常，便会攻击。
<br/>　　不过，这队人明显被吓破了胆，已经没有反抗的意志。他们出来时，已经做好我为鱼肉，人为刀俎的心理准备。
<br/>　　天亮之前，和珅、关鱼、张肥带着五千人马从朔方城赶过来。程大雷攻打凉州城之前，已经通知他们。一旦凉州城被攻破，必然有许多事情要处理，也是和珅发挥用武之地的时候。
<br/>　　天亮之后，程大雷已经宣告自己是凉州城的主人，将近两万人接管这里。
<br/>　　程大雷出现在城主府中，诸多事忙得不可开交，还没时间品尝胜利的滋味。
<br/>　　第一件事，就是发布安民告示。士兵满大街张贴，说我们程当家乃是替天行道，解救黎民百姓与水火之中。蛤蟆城的兵绝不会做任何有损城内百姓的事，否则自有程当家的军法伺候。
<br/>　　当然，若有人敢趁机生乱，想要胡作非为者，程当家也绝对饶不了他们。
<br/>　　第二件事，便是降兵的处理，这件事由刘悲负责。
<br/>　　“问问他们的，想留下的，便是我蛤蟆城的兵，想走的，发给他们盘缠路费，让他们带着妻儿老小离开。”程大雷道。
<br/>　　“还发给他们盘缠？”刘悲觉得不可思议：“这样是不是有些太仁慈了？”
<br/>　　程大雷白他一眼，道：“你懂什么，我的名声已经糟糕到极点，放他们出去，正好给我扬名呐。”
<br/>　　守军中要么是贫苦子弟，走投无路才会去当兵。对他们来说，指挥官究竟姓程还是姓宋，其实没多大区别。
<br/>　　另外一部分，就是在朝廷有些关系的，他们自然不愿为程大雷做事，留着也是祸害。
<br/>　　还有另外一部分人，乃是兵油子。杀人放火你去，升官发财我来，这波人见风使舵的本事是高的，估计也不会留在蛤蟆城，正好用点盘缠送他们滚蛋。
<br/>　　而这些人一旦散出去，就会传说程当家的仁慈之名。成为程当家的俘虏后，不打不骂，反而会发给盘缠。如此，多多少少，就有人更愿意做程大雷的俘虏。
<br/>　　另外，也能稍微帮程大雷洗一洗恶名。
<br/>　　这两件事多多少少都好办些，只是有些忙碌。关键是第三件事——凉州的另外三座城。
<br/>　　金城、武威、浩甲。
<br/>(本章完)
