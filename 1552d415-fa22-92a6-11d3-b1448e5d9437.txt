　　刚刚少帝跟五行刀圣为了人皇罐争夺交手。
<br/>　　正好是给了楚风眠一息喘息之机。
<br/>　　虽然只是一息时间，凭借着神脉的恢复力，也足以是让重振旗鼓，凝聚灵力。
<br/>　　楚风眠将他身躯之中的力量全部注入到了人皇罐之中，在这少帝逼退五行刀圣，打算来抢夺人皇罐的一刻。
<br/>　　人皇罐当即变幻成人皇之矛，变成了楚风眠手中最为锋利的利刃，对着少帝当即轰击而去。
<br/>　　换做是其他任何一位武者，纵然是半步天人境，在受到了如此严重的伤势之后，一息之间的喘息，也难以恢复过来。
<br/>　　但楚风眠却是做到了，楚风眠的力量全部凝聚在人皇之矛上，轰然对着少帝轰击而去。
<br/>　　人皇之矛，乃是上古人皇最为锋利的兵器，现在以楚风眠力量所凝聚出的人皇之矛，已经是有着几分真正人皇之矛的锋芒了。
<br/>　　刹那之间，人皇之矛轰然刺在了这少帝的手掌心之中，一矛刺穿而去，顿时少帝的手掌都是被刺穿而出的一道血洞。
<br/>　　随即澎湃的灵力更是猛然打入到了少帝的手心之中，在这力量的轰击之下，少帝的手掌都被撕裂开来，当即打爆。
<br/>　　就在这天空之上，少帝的手掌被打爆，化为了一团血雾。
<br/>　　“可恶！可恶！”
<br/>　　少帝都没有想到，再这么短的时间之内，楚风眠居然是足以恢复伤势，向他反击。
<br/>　　人皇之矛的威力，也超出了少帝的计算之中，少帝的实力虽强，可毕竟没有踏入真正的天人境。
<br/>　　而这人皇罐，哪怕是残破，拥有的力量依然是极道仙兵，人皇之矛刺入到少帝手心的一刻，少帝的手掌就被直接打爆。
<br/>　　在少帝的眼中，就楚风眠的实力，也就是跟蝼蚁一般，甚至是就连蝼蚁都不如，是他可以随意捏死的存在。
<br/>　　可就是这样一只蝼蚁，居然是三番两次的坏了他的事，这让少帝已经是忍不住了。
<br/>　　要不是因为楚风眠的身躯之中，曾经炼化了青铜天魔骸骨，有着踏入不朽真身的秘密，现在的少帝早已经是忍不住要轰杀了楚风眠了。
<br/>　　“本帝倒是要看看，你还能挣扎到什么时候！给我镇压！”
<br/>　　少帝盯着楚风眠长啸一声，似乎是动了真怒了，魔意纵横，心神一动，他那破碎的右手再度凝聚成形，凌空一掌对着楚风眠猛然抓了过来。
<br/>　　这一掌之力，比起刚刚足足强横了无数倍，死死的镇压这楚风眠的身躯，不打算给楚风眠第二次再度反抗的机会。
<br/>　　“快逃！”
<br/>　　楚风眠心中也清楚无比，以他的实力还远没有达到足以跟少帝抗衡的机会。
<br/>　　刚刚楚风眠可以打破少帝的手掌，也是因为少帝太过大意了，没有注意到到楚风眠。
<br/>　　而且这还是少帝已经是为了逼退五行刀圣，浪费了不少的力量了。
<br/>　　要是真正交手，哪怕是楚风眠手持人皇罐，也不可能是少帝的对手，如此巨大的境界差距，已经不是奇遇可以弥补的了的。
<br/>　　除非是楚风眠在进一步，境界踏入九劫古帝，不然在这少帝的面前，楚风眠永远没有一战之力。
<br/>　　“天龙变！”
<br/>　　楚风眠在刚刚出手的一刻，身躯之中也积蓄好了力量，就在少帝出手再度轰击而来的一刻。
<br/>　　楚风眠的身形闪烁，猛然直接向着佛寺入口处飞掠而去。
<br/>　　两三里的距离，没有少帝的限制，楚风眠的身躯刹那之间飞入到佛寺的入口前。
<br/>　　“五行封锁！”
<br/>　　天空之上的五行刀圣，看着楚风眠的身躯即将飞入到佛寺之中，立刻出手，凝聚灵力。
<br/>　　五行之力，在五行刀圣的手心之中凝聚，打入到了佛寺入口前，凝聚成了一道封锁的阵法。
<br/>　　“晚了！三阳烈炎剑！”
<br/>　　楚风眠面对这五行封锁，身躯没有半分后退，双剑齐出，一股无色火焰凝聚在上方，刹那之间一剑斩出。
<br/>　　熊熊燃烧的烈焰，包裹着两道剑气，轰然斩在了那五行封锁之上，将其中斩出了一个裂痕。
<br/>　　唰！
<br/>　　楚风眠的遁光紧随其后，猛然钻入到这一道裂痕之中，身形一下子冲入到了佛寺之中。
<br/>　　只见少帝跟五行刀圣的身形追过来的一刻，楚风眠的身形已经是踏入到了佛寺之中。
<br/>　　“追！”
<br/>　　少帝都没有看五行刀圣一样，身形闪烁，同样是一步踏入到了佛寺之中。
<br/>　　五行刀圣，也是紧随其后。
<br/>　　随着五行逆转大阵的破碎，无数的九劫古帝，也争先进入其中。
<br/>　　佛寺之中。
<br/>　　楚风眠的身躯连连闪烁，丝毫不敢停留。
<br/>　　楚风眠的心中也清楚，那少帝，五行刀圣，不会如此轻易的就放过楚风眠的。
<br/>　　在楚风眠踏入到佛寺不久后，无数气息便是降临到了这佛寺之中，其中的一尊尊强者之中，少帝，五行刀圣，赫然在列。
<br/>　　而且还有更多的九劫古帝，也已经是步入到了佛寺之中。
<br/>　　“走！”
<br/>　　楚风眠的身躯连连闪烁，足足是在其中穿行了数千里，这才缓缓的停了下来。
<br/>　　在这方圆十里之中，楚风眠已经是察觉不出任何武者的存在了。
<br/>　　进入到了这佛寺之中，楚风眠的灵识便是受到了极大的压制。
<br/>　　在这佛寺之中，仿佛是天然存在这一道禁制，足以压制武者的灵识。
<br/>　　在外的话，楚风眠的灵识足以轻易横扫周围的数百里，但是在这里，也就只能探察方圆十里。
<br/>　　虽然不便，但是对现在的楚风眠来说，却是一个好事。
<br/>　　灵识全部收到压制的话，那少帝跟五行刀圣，想要在找到楚风眠，也并没有那么容易。
<br/>　　足以是让楚风眠喘一口气。
<br/>　　“这佛寺之中，居然如此荒凉。”
<br/>　　楚风眠停下时，终于是可以仔细的观察着周围的一举一动。
<br/>　　这佛寺，在外看去，气势磅礴，辉煌无比，可一旦是进入其中，楚风眠看着方圆四周，居然是一片荒原。
<br/>　　在这荒原之上，方圆十里，都没有任何武者的存在。
<br/>　　别说是武者，甚至是就连生灵，草木，都一概没有，荒凉到了极致。
<br/>　　至于之前显现而出的无数宝光，更是一点也察觉不到。
<br/>　　如果不是楚风眠确定，这里就是佛寺之中的话，楚风眠都无法想象，在这外面气势磅礴的佛寺内部，居然是如此的荒凉。
