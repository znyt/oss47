<br/>?    姜王陵地道之外。
<br/>    
<br/>        一条长长的石廊之中，一群手上绑着白色布条的杀手朝着地道靠近，墙壁上的灯火映照着他们平静的脸,守护着地道入口的守卫看见最前面的那人，笑道：“原来是夜鹰堂主，不知夜鹰堂主来这里做什么，可是殿主大人命夜鹰大人前来检查地道？”
<br/>    
<br/>        夜鹰的身材偏瘦，看起来并不威武，但是杀生殿堂的人都知道，夜鹰可是一位入道境的修行者，而且为杀生殿堂征战了十年，可谓是劳苦功高。不过现在夜鹰并不想多话，他伸手从怀里摸出一条白色的布条绑在了手臂上，守卫感觉到有些不对，但是已经晚了，夜鹰已经拔出了腰间的长刀，大喝道：“杀，不留活口！”
<br/>    
<br/>        夜鹰乃是一个入道境的修行者，本应守护地道的入道境高手已经被殿主召集过去商议大事，什么大事，夜鹰自然是知晓，自然是集结大军，攻入都城。不过这也给了夜鹰机会，让他有一个最容易下手的时间。
<br/>    
<br/>        夜鹰挥动着手中森寒的长刀，一刀将阻挡自己的守卫劈成两半，斜持着滴血的长刀继续向前，他的刀法以诡谲著称，即便是入道境的高手，和他交手都心有余悸，此刻杀这些不过闻道境的修行者简直如同虎入狼群。
<br/>    
<br/>        夜鹰再次挥刀解决了三人，将三人分尸，朝着身后大喝道：“速速解决掉这些人，不然我们将没有任何撤走的机会！”
<br/>    
<br/>        身后的一百多人迅速向着地道逼近，手中长刀不断地与五十多守卫交锋。
<br/>    
<br/>        有着夜鹰在，对方根本没有任何抵挡之力，不过一盏茶的功夫，地面上已经是血流成河。
<br/>    
<br/>        夜鹰看向地道，为了方便大军迅速撤离，这些地道可不窄小，有着木头支撑，夜鹰看了一眼，挥手道：“将这些承重木全部劈断。”
<br/>    
<br/>        他此刻带来的都是精锐，劈断这些承重木不过也就是一刀的事。
<br/>    
<br/>        看着这些承重木一根根被劈断，足足摧毁了承重木长达一百丈的距离，手下的杀手都撤了出来，夜鹰看着已经有着裂痕的地道顶，飞身而上，朝着地道顶便是一掌挥出，轰隆！
<br/>    
<br/>        没有了承重木，本就摇摇欲坠的地道在此刻轰然倒塌。
<br/>    
<br/>        夜鹰满意地点点头，挥手道：“走，我们突围！”
<br/>    
<br/>        夜鹰身后，一个心腹道：“夜鹰大人，此战，我们真的能胜吗？本来杀手殿堂被发现，还可以撤走，但是此刻，地道已毁，他们已经没有了退路，而世子殿下，最多不过汇聚了三万大军。”
<br/>    
<br/>        夜鹰冷冷地看了一眼这个心腹，喝道：“记住，我们是军士，作为军士，只需要听从命令，至于打不打得赢，那都不是我们应该考虑的事情，走！”
<br/>    
<br/>        杀生殿堂主墓室之中。
<br/>    
<br/>        坐在主位上的殿主看向周围的堂主们，“堂主都到齐了没有？”
<br/>    
<br/>        身边的侍卫道：“只差夜鹰堂主了。”
<br/>    
<br/>        殿主并不怎么在意，道：“等不了他了，待会儿再告诉他吧。”
<br/>    
<br/>        殿主起身，笑道：“我知道，你们一直很好奇我的身份，今日，我便告诉你们我的身份！”
<br/>    
<br/>        殿主伸手摘下了脸上的青铜面具，露出他的脸颊，下面的堂主看向殿主的脸，一阵骚动，他们怎么也没有想到，他们杀生殿堂的殿主，居然是太子！
<br/>    
<br/>        殿主笑了笑，道：“没错，我便是燕国太子，摄政王陈王犯上作乱，入朝不拜，不尊皇命，乃燕国开国以来最大的奸贼，不过恶有恶报，他的亲生儿子，居然和他一模一样，也想要犯上作乱，造了他的反，人算不如天算啊，如今摄政王和大将军斗得两败俱伤（蛊惑堂主，并不是真的），正是我等拨乱反正的最好机会！”
<br/>    
<br/>        下面的堂主本以为自己的身份一辈子都见不得光了，但是现在，他们似乎有着一个机会可以洗白，他们效忠的，原来是太子殿下！
<br/>    
<br/>        而且现在，似乎有很大机会可以拨乱反正，让皇室重掌大权，那个时候，他们可是勤王功臣！勤王功臣，可比他们现在耗子一般的身份好多了，简直是天差地别！
<br/>    
<br/>        堂主们齐齐下跪道：“我等必定助太子殿下拨乱反正，让皇室荣登大宝！”
<br/>    
<br/>        殿主欣慰地点点头，我等了多少年了，十年了，终于等到了这一天了，“哈哈哈，好好好，我等同饮此杯！”
<br/>    
<br/>        道完，殿主举起手中的酒杯，和堂主们一同饮了下去，就在这时，忽然有侍卫急匆匆进来禀报道：“禀报殿主，不好了，夜鹰堂主带人将地道毁了！”
<br/>    
<br/>        殿主惊了惊，瞬即大怒，将自己手中的酒杯摔了个粉碎，大骂道：“好你个夜鹰，居然敢反叛，来人，速速去将夜鹰抓来！”

<br/>    
<br/>        殿主呵呵一笑，：“陈明是吧，还真是好胆啊，你确定只有三万大军左右？”
<br/>    
<br/>        斥候道：“其他的飞虎军并没有动向，而且，就算是其他的飞虎军敢来，也需要至少三五日准备。”

<br/>        殿主哈哈大笑道：“地道，我还要地道作甚，他还以为我杀生殿堂只有三千人吧，见了他的三万大军必定要逃走吧？不，我有着整整十万大军，而且还是占尽了地利，三万攻打十万，这陈明莫非是活的不耐烦了，传我命令，速速集结大军，准备捉拿陈明，无论死活，谁抓住了陈明，封王！”
