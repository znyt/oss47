　　“他们放弃了影天使，要跑！”
<br/>　　罗昊立刻反应过来，明白了智天使和吴云的打算。以眼下的情况来看，影天使想要活着离开自己眼前这栋写字楼基本是不可能的事情，而智天使在清楚自己无法带走影天使后，果断的选择了将影天使，选择撤离。
<br/>　　果决！冷酷！残忍！
<br/>　　罗昊脑海中冷不丁冒出这些词来，而这些词使用在智天使这个人身上却是非常的贴切。
<br/>　　这家伙无论在什么时候都能冷静的做出对他而言最优的选择，即便这样的选择是会牺牲自己的同伴，智天使也是在所不惜。
<br/>　　写字楼，地下一层。
<br/>　　这里除了智天使和吴云外，还集结着近百人，他们衣着打扮一致，整齐的站在智天使和吴云面前。
<br/>　　“出发吧。”智天使眼神冰冷的扫过站在自己面前的这些人，冷声说道：“你们的死亡会给你们的家人带去财富，钱已经打入你们的账户，你们现在就可以查账。”
<br/>　　这些人立刻从口袋中取出自己的手机，进入自己的银行账户进行查询，当看到自己户头上所多出来的钱后，他们收起手机，眼中闪过一丝坦然和如释重负。
<br/>　　这些人是智天使从非洲贫民窟中找来的，买通了K国的海关，以偷渡的方式进入到K国，作为智天使所制定的撤离计划中的牺牲品。智天使承诺给他们每个人一笔钱，这笔钱在发达国家来说，或许并不算很多，但是对于这些来自于非洲最贫穷的地方，连吃饭都难以解决的人来说，这就是一笔巨款，能让除了他们自己以外，家里人吃饱饭。
<br/>　　有人追求的生活是住豪宅、开豪车，生活的光鲜亮丽，被人羡慕；而有的人追求的生活则是能吃饱饭，有多余的钱让自己的孩子上学、看病，为了这样简单的生活要求，甚至不惜付出自己的生命，就比如智天使眼前这些人。
<br/>　　收起手机后，停车库里这些人分散走上停在停车库里的汽车，这些汽车绝大部分都是以低廉的价格从二手汽车市场买来的二手车，其中有一部分已经属于报废车，只是被人修改了里程数，然后作为为报废车继续销售。
<br/>　　智天使和吴云也上了一辆汽车，智天使拿起放在中控台上的对讲机，说道：“炸药爆炸后出发！”
<br/>　　写字楼顶楼，两个房间中塑胶炸药上的倒计时还剩下最后三十秒，影天使还在不断的敲打着那扇用轻型装甲打造出来的房间门，发出“咚咚咚”的声响。
<br/>　　而相较于影天使所表现出来的急躁，子午则是显得相当平静，看着墙壁上液晶屏幕中的影天使，说道：“SHADOW，现在知道为什么我说这次对决你输定了吗？我有学生，我有能够继承‘GHOST’这个名字的人选，但是你没有！”
<br/>　　“GHOST，我现在立刻解开你房间门锁上的锁定程序，你也把我房间门锁上的锁定程序解开，我们还能逃出去！”影天使盯着液晶屏幕中嘴里咬着一根烟的子午，大声吼道。
<br/>　　这是影天使目前唯一能想到的，也是可行的逃脱办法。
<br/>　　子午抽了一口烟，缓缓吐出眼圈，看了眼塑胶炸药上飞快跳动的倒计时，说道：“已经来不及了，还剩下不到二十秒的时间，就算是我们能够从房间里逃出去，两个房间加起来十公斤的塑胶炸药爆炸所引发的冲击波都能震死我们，你认为是你跑得快，还是冲击波的冲击速度快？”
<br/>　　塑胶炸药上，倒计时还在飞快跳动着……
<br/>　　5秒，4秒，3秒，2秒，1秒，轰！
<br/>　　指挥中心大屏幕上突然失去了影像，变成一片雪花，站在指挥中心房间里，也能清楚的听到从写字楼方向传来的爆炸声，甚至，放在桌子上的茶杯也都是因为这爆炸而晃动。
<br/>　　“老师！！”
<br/>　　云熙看着发生爆炸的写字楼顶楼，发出撕心裂肺的喊声，紧接着云熙只觉得自己身体不受控制的软倒下去，模糊间她能看到有人正着急的朝着自己的方向跑来。
<br/>　　写字楼顶层，火焰从玻璃墙背后席卷而出，被震得粉碎的玻璃墙碎片朝着地面砸落下来，不仅是写字楼顶层的玻璃墙，20到24层的玻璃墙上也是出现了裂缝。
<br/>　　咔嚓！咔嚓！咔嚓！
<br/>　　玻璃墙上的裂缝蔓延开来，裂缝也越来越粗，紧接着这些楼层的玻璃墙也是发生了碎裂，玻璃碎块在地心引力的引导下朝着地面坠落下去。
<br/>　　这些锋利的玻璃碎片在重力加速度的作用下，瞬间成了一柄柄能刺穿人身体的杀人武器，如果此时有人站在这栋写字楼下的话，无疑就是一场灭顶之灾，汽车即便是停在了马路对面，但是有几块玻璃碎片打中了汽车，车身上瞬间出现了几处凹坑。
<br/>　　写字楼下花坛中的植被或是被玻璃砸倒，或是被玻璃拦腰切断，或是被细小的玻璃碎片剔成了光杆……
<br/>　　片刻间，原本打理有序的景观花坛在玻璃雨摧残下一片狼藉，落下的玻璃碎片砸落在地面上，四分五裂，溅射而起的玻璃碎片向四周进行着无差别攻击，令得周围K国的快速反应部队和龙隐部队队员不得不藏身在汽车背后，进行躲避。
<br/>　　玻璃碎片打在车窗上，啪的一声，车窗上瞬间出现一道裂纹。
<br/>　　就在这个时候，写字楼地下停车库传来一阵阵汽车引擎的轰鸣声，紧接着一辆接一辆的汽车冲出了地下停车库。
<br/>　　汽车车窗上都贴着深色窗膜，加上汽车又是在高速行驶状态，所以根本无法准确分辨出坐在汽车中的人是谁！
<br/>　　“狙击手，无差别射击，拦住他们！”
<br/>　　罗昊果断下令，拦截这些从地下停车库中冲出来的汽车。
<br/>　　乒！
<br/>　　砰！
<br/>　　砰！
<br/>　　所有狙击手集体开火，子弹朝着那些从地下停车库冲出来的汽车招呼过去。
<br/>　　李毅峰用tac-50一枪打中了一辆蓝色轿车的引擎盖，子弹击穿了汽车机油箱和引擎，机油从弹孔中喷涌而出，喷溅在汽车前挡风玻璃上……
