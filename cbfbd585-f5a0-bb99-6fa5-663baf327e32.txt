　　刘弘基真的无话可说了，听完李绩的话之后，他也觉得留着城墙没啥用，如果真遇到具有这样远程攻击力度的军队，城墙除了可以限制自己人逃走之外根本无力阻档敌人。
<br/>　　当然，他也想过，大不了你来打我的时候我的人全都躲到城墙下面，这样不就没事儿了？就像以前对付投石机的时候一样。
<br/>　　可是再转念一想他又觉得不对了，因为大唐最大的城池就是长安城，可长安城又有多大？长九千多步，宽八千多步，那么那炮打不到这边难道还打不到另一边么？把炮拖到城墙下面打另一面的城墙总行了吧？
<br/>　　想到这里，刘弘基不再说什么了，苦笑着摇摇头对李绩说道：“还有什么更狠的么？一起说说看。”
<br/>　　“枪，我们这里新到一批枪，射程四百步，你觉得如何？而且射速十分快，不用像以前那样往里塞火药，然后再塞铁沙。”李绩嘿嘿怪笑着说道。
<br/>　　“射程四百步的枪么？这东西……”刘弘基已经不知道应该说什么了。
<br/>　　就他所知，现在射程最远的弓或许可以射到四百步之外，但那必须是特制的，正常的弓如果能射到两百步都可以算是厉害到不行，甚至就算是弩，也没有这么大的力量。
<br/>　　大唐正在逐步走向强大，也逐步走向未知。
<br/>　　刘弘基突然发现，自己似乎来都没有好好认识一下自己的国家。
<br/>　　这几年来大唐的变化已经可以用日新月异来形容，那些庞大的机器让以前原本需要很多人劳作很久的工作变的简单，只要动几下操作杆，一切便已经完成了。
<br/>　　人的力量在机器的面前一次又一次被击败，这些都是在近几年才出现的情况，但却足以改变人的三观。
<br/>　　……
<br/>　　而与此同时，伟大的大唐皇帝陛下同样也在吃惊。
<br/>　　“六千步的炮也研究出来了？”看着面前神情自若的儿子，李二只觉得牙疼不已。
<br/>　　什么炮能打六千步？或者说能打六千步之外的目标，那特么还叫炮么？
<br/>　　“父皇，六千步是极限了，如果没有材料上的重大改进，基本上十年之内不会再有更先进的贞观炮出现。”李承乾的神情平淡如说，就好像在说昨天晚膳吃的是什么一样。
<br/>　　可是李二却接受不了这样的现实，看着李承乾在平静的样子，老头子越想越生气，终于还是忍不住喝斥到：“六千步还不行，你还要怎么样？！”
<br/>　　“父皇，如果有足够的技术支持，我们可以从西安一炮打到倭国。”李承乾老神在在的说着，脑子里却在想着后世的那些可以飞过半个地球的‘大杀器’。
<br/>　　不想老头子竟然犹豫了，沉默半晌才斩钉截铁的说道：“不，六千就已经足够了，今后这东西别在搞了。”
<br/>　　“为，为什么啊？”李承乾有些不解的问道。
<br/>　　“不为什么，六千就是级限，所以你不用再研究下去了。”李二瞪了儿子一眼，随后说道：“以后就以这个数字为标准，任何武器只要射程超过六千，便是逾制。”
<br/>　　啥？李承乾眼珠子瞪的老大。
<br/>　　没这个必要吧？弄个炮射程超过六千步就是逾制？有没有必要这么严格？啥时候老头子也开始向辫子皇朝学习了？
<br/>　　想到这里，李承乾连忙解释道：“不是，父皇，您不能这样啊，以后这武器的威力越来越大，射程越来越高，六千步根本解决不了问题啊。”
<br/>　　听到李承乾的解释，老李也怒了，拍着桌子吼道：“你解决个屁，老子问你，你搞了那么多的武器到底要打谁？现在除了咱家，还有谁家的武器最大攻击范围能超过五百步的！”
<br/>　　“呃……”李承乾被老头子吼的一呆。
<br/>　　事实上他自己都不知道如果现在给他一颗DF他要去打谁，因为完全没有必要。
<br/>　　在大唐这个时代，好像五百步已经是极限了，搞出射程三千步的火炮已经是在欺负人了，六千就等于是在打小孩，如果射程再高一些……。
<br/>　　好吧，射程再高的话，连准确度都保证不了，通讯也联系不上。
<br/>　　毕竟炮弹打出去了需要校正，在后世有对讲机，有卫星可以直接联系，但是在大唐这个通讯完全靠吼的时代，射程再远都特么是在扯蛋。
<br/>　　想到这里，李慕去也不纠结了，六千就六千吧，老头子怕半夜里被炮击，咱家也怕啊，大明宫一共也没多大，射程六千米的炮摆在两里开外就可能把皇宫打个对穿，为了安全着想，还是算了吧。
<br/>　　……
<br/>　　不过算了归算了，该搞的东西还是要搞，比如那个迫击炮，有了无烟火药，那东西就可以研究了。
<br/>　　反正其原理简单的很，就是一个大号的子弹，利用自由落体的原理撞击其底火，然后击发的东西，再配上一个合金的炮筒就好了，比那个火箭筒还要简单。
<br/>　　唐善识在听完李承乾的解说之后，没用多少时间便已经搞出了炮弹的原型，也就是一个大号的子弹，再用两天时间安排人制作了合金钢的炮管，一切就算是大功告成。
<br/>　　李承乾并没有想过事情会如此简单，所以当他看到小唐命人拿出那个所谓的炮弹时，整个人都是傻的，呆呆看着那圆滚滚的东西问道：“我说唐善识，你这是什么东西？”
<br/>　　“炮弹啊，按你说的，大号子弹么。”小唐理直气壮的说道。
<br/>　　“这就是炮弹？我看着怎么根个茄子似的！”回到长安已经完婚的程处默接过那颗圆圆的炮弹，放在手里掂了掂，闷闷的说道。
<br/>　　这家伙虽然已经大婚了，可是人却还是一副靠谱的样子，一点长进都没有，瞅着那炮弹在他手上上下翻飞的样子，李承乾吓的心差点没有跳出来。
<br/>　　要知道，那东西虽然长的像茄子，但却并不是真的茄子，如果掉到地上，一个不好就有可能把在场的所有人都送上西天。
<br/>　　唐善识显然也被程处默吓的不轻，但是却没敢喊出来，生怕惹的丫分心，不小心把那个‘茄子’掉到地上。
