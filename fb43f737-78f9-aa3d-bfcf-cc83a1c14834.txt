        当妹妹季思妍把童筱颖的相片放到他眼皮子底下时，他还一脸不耐烦地推开妹妹季思妍的手，拧着眉头说道：“去去去，别打扰我泡妞！”
<br/>
<br/>        “哥，我手里的这个女人，可是傅昼景的正妻！你确定对她没兴趣？”季思妍故意把傅昼景搬出来。
<br/>
<br/>        一提到傅昼景，季运恒立即放下手机，将妹妹季思妍手中的照片夺了过来，仔细地打量了一番。
<br/>
<br/>        “呦！不错嘛！是个素颜美女！”季运恒一边欣赏，还不忘一边赞赏。
<br/>
<br/>        “那你到底玩不玩这个女人？”季思妍双手抱臂，站在沙发旁，居高临下地睨着躺在沙发上的哥哥季运恒。
<br/>
<br/>        季运恒却将照片一甩，继续捧起自己的手机和女人聊天，嘴里叼着烟，口齿不清地说道：“我对已婚女人没兴趣！”
<br/>
<br/>        “我想转正，不想给阿景当一辈子的地下情人！”季思妍一脸认真地说。
<br/>
<br/>        季运恒顿时从沙发上起身，将嘴里的烟拿出来，摁灭在一旁矮柜上的烟灰缸中，没好气地谩骂：“我说思妍，你是不是脑子进水了！”
<br/>
<br/>        “阿景现在的实力你应该已经看到了，我只不过是为他搭桥引线，提供他资金，他就能在这么短的时间内，把我们的新公司给做了起来。而且，他做得比任何人都好，不是吗？”季思妍振振有词地说。
<br/>
<br/>        季运恒忍不住对季思妍翻了个白眼：“呵！我算是看明白了，你现在是把自己的心也搭进去了！当初，我是怎么警告你的？让你不要丢了心！”
<br/>
<br/>        “对！当初你是警告过我！但是，早在你警告我之前，我就已经喜欢上阿景了！”季思妍说得条条是道。
<br/>
<br/>        季运恒在沙发上盘腿而坐，不禁对自己的妹妹季思妍嗤之以鼻起来：“你压根就不是真正的喜欢傅昼景吧！你若是真的喜欢他，高中毕业之后，你就不会为了……”
<br/>
<br/>        “哥，什么话该说，什么话不该说，你应该比我要更懂得分寸！”季思妍疾言厉色地呵斥住哥哥季运恒对自己的冷嘲热讽。
<br/>
<br/>        季运恒啧啧嘴，目光无意间扫视到刚刚那张被他甩到地上的照片时，蓦地身心一怔。
<br/>
<br/>        女人的照片，有一半截掉进了茶几底下，只露出了鼻子、嘴巴和上半身。
<br/>
<br/>        “小谢？！”季运恒惊怔，连忙起身将童筱颖的照片从茶几底下捡了起来，然后在茶几上抽了一张纸巾，将纸巾叠成长条状，盖住了照片上的女人的眼睛。
<br/>
<br/>        他总觉得这样还不够他准确辨认，又拿起茶几上置物盒里的剪刀，将长条状的纸巾剪了两个小洞，再往照片上的女人的眼睛上一贴。
<br/>
<br/>        “我靠！真是小谢！”季运恒就像发现新大陆般，拍了下手，身子弹了起来，直指童筱颖的照片，“夜惑里小谢！”
<br/>
<br/>        “小谢？！什么夜惑里的小谢？”季思妍被季运恒这一举止给惊得一脸莫名其妙。
<br/>
<br/>        季运恒忽然仰天大笑：“哈哈哈——万万没想到！这傅昼景的老婆，竟然是夜惑里的那个小谢！”
<br/>
<br/>        “小谢是谁？”季思妍纳闷地问。
<br/>
<br/>        季运恒举起童筱颖的照片，弹了弹，自言自语道：“好你个小谢，真的是让我好找啊！你把我害得那么惨，差点让我老二抬不起头，哼，这回被我逮着了真身，我看你还往哪里逃！”
<br/>
<br/>        “哥，你到底有没有听我在说话？”季思妍有点儿恼火地质问。
<br/>
<br/>        季运恒这才回过神来，解释道：“她是傅昼景的老婆是吧！我告诉你，她还是‘夜惑’里的歌女！之前，有个叫‘连少’的男人，为了跟我抢她，把我的人给打了！”
<br/>
<br/>        “童筱颖在‘夜惑’里卖唱？！”季思妍惊诧了。
<br/>
<br/>        季运恒接着说道：“之前是在‘卖’，现在好像已经不在‘夜惑’里‘卖’了。而且‘夜惑’里的人，对外从来不透漏任何一点关于她的资料信息。”
<br/>
<br/>        “她除了卖唱，还有没有卖身？”季思妍十分敏感地八卦了起来。
<br/>
<br/>        季运恒口无遮拦地回答道：“肯定是卖了！小谢这女人，和那个‘连少’关系匪浅。‘连少’给她打赏了那么多钱，她一定是跟那个‘连少’睡过了！”
<br/>
<br/>        “连少？什么连少，他又是谁？”季思妍听得一头雾水。
<br/>
<br/>        季运恒摆了摆手，开始信口雌黄地胡说八道：“那个‘连少’，一看就不是什么好男人！上次，我在‘夜惑’里跟朋友喝酒，雅姗就在我邻座的雅座里，然后没过多久，就被‘连少’给带走了！”
