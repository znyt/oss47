　　嗖，比武台上，一道人影凭空消失。
<br/>　　当他再次出现时，正狂揍着萧若狂的萧逸已经被制住。
<br/>　　人影正是裁判，家族中最强的后天执事，后天境九重武者，地位仅次于家族长老。
<br/>　　萧逸目光一凝，感受到被紧紧捉住的双手，一股无力感油然而生。
<br/>　　“这就是后天境武者的实力吗，果然厉害。”萧逸暗暗惊讶。
<br/>　　萧逸的境界不过是凡境九重，即使配合升龙提升的实力，也不过勉强达到后天二、三重左右的战力，自然在裁判手上无法反抗。
<br/>　　升龙是一种暂时大幅度提升修为的招式。
<br/>　　不过以他现在体内的微末真气，无法完全发挥出威力罢了。
<br/>　　假以时日，当他能完全驾驭升龙这一绝时，战力必然达到一个骇人的地步。
<br/>　　萧逸在惊讶，殊不知，裁判脸上也是一阵讶异之色一闪而过。
<br/>　　“好小子，好强的手劲，区区凡境九重，就有如此力量。难道，这些年来，这个人前的废物，一直都只是在韬光养晦，等待着一飞冲天之日？”裁判心中暗惊，却又感到疑惑。
<br/>　　忽然，裁判心中一阵恍然，暗道，“也是，家主当年是何等惊采绝艳之辈，他的儿子，岂会是窝囊脓包。或许，萧逸一直是个天才，只不过家族这些年对他的关心太少罢了。”
<br/>　　心中想罢，裁判轻声道，“萧逸，比武已经结束了。”
<br/>　　萧逸点点头，退到了一边。
<br/>　　而此时，地面已经出现了一个大坑，坑中是凄惨无比的萧若狂，浑身是伤。
<br/>　　原本张狂的嘴脸，此刻鼻青脸肿，像个猪头似的。
<br/>　　原本满是嚣张言语，此时只能忍着疼痛咬牙切齿。
<br/>　　五长老这时已经迫不及待地来到比武台，将他扶起。
<br/>　　三长老则在长老席上站起，自豪地高声说道，“萧逸连败萧若狂、萧若狂，且皆是完胜，可还有人质疑他当少家主不够资格？”
<br/>　　“没有。”族人们的声音异常整齐。
<br/>　　“萧逸的天赋，我闻所未闻，从未所见，年仅16岁，却连败家族两大天才，如此天赋，毫不逊色于当年的家主。”
<br/>　　“不错，年仅16岁，却打败后天境武者，这样的天才，我萧家已经很多年没有出现过了吧。”
<br/>　　族人们的赞扬之声不绝于耳，他们看向萧逸的目光完全变了，仿佛已经可以看到日后的萧家，在眼前这个天才少年的带领下，必将更加强盛。
<br/>　　炎龙大陆，以武为尊，一切以拳头、实力为基准。
<br/>　　强者，是这个世界的人们所崇拜和信服的对象。
<br/>　　强者，能得到足够的权利和尊严。
<br/>　　“呵。”萧逸淡淡一笑。
<br/>　　来到这个世界的这些天，他已经充分见识到这个世界的残酷。
<br/>　　没有实力，哪怕自己拥有少家主的头衔，也只如地底泥，谁都可以踩上一脚，肆意欺凌。
<br/>　　同族之间，为了争名夺利，不惜多番陷害，甚至雇佣杀手，意欲置自己于死地。
<br/>　　其他族人们哪怕知道很多事对自己不公，也不会为自己说半句好话，冷眼旁观。
<br/>　　然而，在这残酷的同时，却有着改变这一切，扭转乾坤的办法，那就是修为与实力。
<br/>　　拥有实力、天赋，所有人的看法都变了，赞扬之声不绝于耳。
<br/>　　五长老多番设计，连大儿子萧若狂也大老远地从玄火门回到萧家，意图打败萧逸。
<br/>　　其结果，却只是萧若寒这个萧家年轻第一人，萧若狂这个早已闻名紫云城的天才，双双成为萧逸一鸣惊人的垫脚石。
<br/>　　“如果没有其他人挑战的话，我下去了。”萧逸淡淡地说了一句，看向裁判。
<br/>　　裁判翻了个白眼，暗道，“连后天一重的萧若狂都被你打得跟个猪头似的，谁还会上来挑战你。”
<br/>　　不过，他嘴上还是和颜悦色地说道，“可以了，下去吧。”
<br/>　　“慢着。”
<br/>　　忽然，一道气急败坏的声音响起。
<br/>　　“将我儿若狂打伤，你还想就此大摇大摆地离开？”
<br/>　　说话地正是五长老。
<br/>　　他的声音中，充满了愤怒与憎恨。
<br/>　　然而，萧逸却敏锐地从他的眼光中捕捉到，他的眼中藏着阴鸷和得意，他似乎早有准备，甚至是把握十足。
<br/>　　“这个阴险的家伙，又想耍什么花样？”萧逸暗暗皱眉。
<br/>　　三长老当即沉声道，“五长老，台上比武，拳脚无眼，且这是萧若狂自己挑战，怪得了谁？以逸儿完胜他的实力，仅是将他打伤，没有取他性命，已是仁义之举。”
<br/>　　“哼。”五长老怒哼道，“三长老，依你之言，萧逸将我儿打伤，我还要谢他不成？”
<br/>　　“你要谢我也不拦你。”三长老似笑非笑地说道。
<br/>　　三长老心里那个乐阿，以前，五长老总拿萧逸‘废物’的事来打击他，借此处处针对他，而他为了力保萧逸，没少受五长老的气。
<br/>　　现在，萧逸连败萧若寒和萧若狂，受家族所有人一致认可，他算是可以扬眉吐气了。
<br/>　　“混账。”五长老怒喝道，“三长老，老夫没兴趣与你口舌之争。如今，大长老、二长老，以及其他长老皆在此，老夫斗胆为我儿讨个说法。”
<br/>　　二长老沉声问道，“五长老你要什么说法？”
<br/>　　五长老道，“敢问二长老，依照族规，萧家子弟若打伤家族炼药师，该当处置？”
<br/>　　二长老皱起了眉头，直言道，“家族炼药师，日夜炼丹，劳苦功高，理当受家族格外礼遇。家族中，除长老执事，寻常子弟不可冒犯，若胆敢欺辱，乃至殴打，轻则杖责八十，重则逐出萧家。”
<br/>　　炼药师，本身就是这个世界最尊贵的职业，凌驾于同等级的武者之上。
<br/>　　比如一个一品炼药师，就比一个凡境武者尊贵得多。
<br/>　　而萧家的炼药师，就这么几个，还日夜为家族炼丹，劳苦功高，在萧家的权利和待遇自然非常高。
<br/>　　二长老的话刚说完，三长老心中便一突，不可置信地问道，“五长老，你的意思，难道是...”
<br/>　　就连一直没有说过话，假寐着观看比武的大长老，此刻也微微睁开了眼睛。
<br/>　　“不错。”五长老言语中带着不可一世的自豪。
<br/>　　被他搀扶着的萧若狂，更是自傲地说了一句，“不错，我乃是一品炼药师。虽然我算是玄火门的弟子，但，这并不妨碍我多回家族居住，为家族炼制丹药。”
<br/>　　边说着，萧若狂轻蔑而憎恨地看了萧逸一眼，暗道，“小废物，你打赢了我又怎样？我的炼药师身份，足以让你高攀不起，同样可以玩残你。”
<br/>　　比武台下方的族人中，顿时引起了轩然大波。
<br/>　　“什么？萧若狂竟然还是一品炼药师？”
<br/>　　“天阿，虽然只是一品，但他不过二十出头，以这样的天赋来看，日后就算成为三品炼药师都不是不可能吧？”
<br/>　　“啧啧，未来的三品炼药师阿，这乃是万中无一阿。”
<br/>　　“后天境武者，一品炼药师，萧若狂真不愧是整个紫云城最出色的天才之一。”
<br/>　　成为炼药师，有诸多的要求，天赋和资质更是重中之重。可以说，一千个武者中，都不一定出一个炼药师，甚至几率会更低。
<br/>　　也难怪族人们会如此惊讶。
<br/>　　万万没想到，萧若狂竟然还是个一品炼药师。
<br/>　　连一直沉默的大长老也微微惊讶，“不过二十出头，已经成为一品炼药师，资质比百炼更强。”
<br/>　　三长老顿时慌了，暗道，“杖责八十，不行，五长老那群阴险的家伙，必然会往死里打，该怎么办阿，逸儿肯定会受不住的。”
<br/>　　然而，在族人们的惊讶声中，五长老和萧若狂的冷笑声中，三长老的着急担忧眼光中...
<br/>　　萧逸却淡淡地‘哦’了一声，“哦，没事的话，我先下去了。”
<br/>　　“没事？你想得美。”五长老冷笑道，“来人，将萧逸捆起来，老夫要教教他怎样尊敬家族里的炼药师。”
<br/>　　“不行。”三长老已经顾不得什么了，脚步一顿，瞬间来到萧逸面前，将萧逸护在身后。
<br/>　　“我倒要看看，谁敢动逸儿？”三长老横眉冷对，一副阔出去的样子。
<br/>　　萧逸看在眼中，暗暗感动。
<br/>　　五长老心中一阵冷笑，暗道，“三长老，我就知道你会忍不住出手护萧逸的，哼，不知公允，包庇萧逸，我看你这个代理家族还怎么当下去。”
<br/>　　“大长老，二长老，三长老如此蛮横霸道，还请为我儿主持公道。”五长老高声说道。
<br/>　　“不好意思，打断一下。”
<br/>　　萧逸忽然从三长老身后站了出来。
<br/>　　“孽障，你还有什么话好说？”五长老质问道。
<br/>　　“没什么。”萧逸耸耸肩，道，“家族里应该没有说过，如果炼药师打伤了另外一个炼药师，该怎么惩罚的吧？”
<br/>　　“哼，是没有，那又如何？”五长老道，“你现在还有心思想这些？识相地就自己站出来接受惩罚。”
<br/>　　“没有就行。”萧逸不屑地看了眼五长老，随即拉住三长老，准备往台下走去。
<br/>　　“不许走，你到底什么意思？”五长老怒喝一声。
<br/>　　“没什么意思，我只是想告诉你，我也是一品炼药师。”萧逸嘴角咧过一道冷笑，淡淡地说道。
<br/>　　声音虽淡，却仿佛一枚重磅炸弹，轻松传入了所有人耳中。
<br/>　　“什么？你也是一品炼药师？”族人们有些不敢相信自己的耳朵。
<br/>　　大长老更是立马睁开了眼睛，紧紧地盯住萧逸，希望自己没有听错。
