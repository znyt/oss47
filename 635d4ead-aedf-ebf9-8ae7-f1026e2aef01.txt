　　在一些夜猫子在刷新微博的时候，一条突兀的消息却是让所有人都震惊失色。

<br/>　　“大魔王深夜住院，荣耀战队所有队员全部赶赴医院，现场图片曝光。”
<br/>　　“大魔王身患重病，可能无法继续比赛！”
<br/>　　“大魔王病危！！！”
<br/>　　无数的消息第一时间从微薄上流传出来，起初大家还不怎么相信，但是看到蒋念等人将叶帝背下车这一幕的时候，所有人却是又不得不信。
<br/>　　“卧槽，什么情况，昨天大魔王不还在场上比赛呢么，怎么突然间就住院了。”
<br/>　　“不要吓我啊，大魔王住院了，今天肯定是愚人节，肯定又是哪家无良媒体在那边瞎报道呢。”
<br/>　　“那个好像是上海二院，有没有附近的兄弟，赶紧帮忙去看看大魔王现在到底怎么个情况，如果有需要帮忙的请赶紧在微博上说一下。”
<br/>　　“我的天，这到底是什么情况，这肯定不是真的！”
<br/>　　......
<br/>　　惊闻这条消息，无数的粉丝这一刻也都懵了，不敢相信这个事实。
<br/>　　而此刻的叶帝已经是因为肩膀处的疼痛，此刻处于半昏迷的状态，看着这个状况，叶颖这一次是真的慌了神，他怎么也不会想到白天还和自己嬉笑的叶帝，晚上居然就出了这样的状况。
<br/>　　“爸，小叶子他住院了。”看到叶帝已经处于半昏迷的状态，叶颖连忙拿起了自己的手机朝着叶天江那里打了一个电话。
<br/>　　正在工作的叶天江听到这个消息，也是愣了半晌：“在哪里？我马上赶过去！”
<br/>　　火神总部是在上海，叶天江在这边工作，所以他离这边倒是比较近。
<br/>　　听着自己父亲的问话，叶颖连忙将医院的地址告诉了那边。
<br/>　　“先不要通知你大伯他们，我先去医院那边看看。”叶天江再次叮嘱道。
<br/>　　只不过这个时候网上到处都是关于叶帝住院的消息，叶天正这边就是想不知道也难，早就有警务员将这个消息告诉了叶天正和秦芳两人了。
<br/>　　当天夜里叶天正和秦芳两人便是乘飞机向着上海这边赶来，与此同时正在军队的叶天英听到这个消息后，也是立刻放下了手中的事情，向着上海这边赶来。
<br/>　　理论上来说他这种级别的想要离开军队去其他地方是需要向上面打报告的，不过这个时候担忧叶帝的安慰，叶天英也是顾不上了。
<br/>　　要知道叶帝可是叶家唯一的单传，他要是出了什么事情，这对于叶家的打击实在是太大了。
<br/>　　大概过了半个小时左右的时间，在医院外围无数记者的关注下，他们震惊的看到两排警车开道，快速的向着这边行驶过来。
<br/>　　“我的天，那不是火神总部负责人叶天江么，他怎么来医院了。”
<br/>　　“大魔王也姓叶，这人不会和大魔王有亲戚吧。”
<br/>　　“不是说京都叶家只有一个嫡系男性后代么，大魔王不可能是他们的亲戚吧。”
<br/>　　“如果那个后代就是大魔王呢！！！”
<br/>　　“......”
<br/>　　大概又过了一个半小时左右，当他们看到一号在重重保护下出现在他们的视野当中的时候，这些记者终于是知道了叶帝的身份了。
<br/>　　几乎瞬间网上便是流传出了叶帝是一号儿子的这个消息。
<br/>　　“我的天，大魔王居然是一号的儿子，这个身份也太恐怖了吧。”
<br/>　　“一号的儿子，这你妹的还打什么游戏啊，天天去泡明星多好啊。”
<br/>　　“这个消息我还真的是第一次听说，大魔王和其他的官二代真的不同，他根本没有靠家庭，他有着今天这一切都是自己打拼出来的。”
<br/>　　“我对这个世界再次充满了信心，连大魔王这样的人都这么努力，我还有什么借口不努力呢。”
<br/>　　“祈祷大魔王平平安安的，一定不要有事啊。”
<br/>　　“为大魔王祈福！！！”
<br/>　　深夜里无数的粉丝这一刻都已经睡不下去了，每隔几秒他们就要刷一下微博，看看有没有叶帝的消息。
<br/>　　叶帝住院到现在已经有2个小时了，一点消息都没有传出来，这些粉丝真的是非常的担心。
<br/>　　“什么情况？”叶天江来到医院的时候，院长就已经接到了消息，当得知那个已经半昏迷状态的人是一号的儿子的时候，院长当即便是将全员最好的医生全部叫出来为叶帝会诊。

<br/>　　在一号的面前，就算是一号没有说话，他也会感受到迎面而来的那种压力。
<br/>　　“静养？”听着院长的话，叶天正皱着眉问道。
<br/>　　叶帝的性格他最清楚不过了，马上就要世界赛了，这个时候让他静养，他肯定不会干的。

<br/>　　听到母亲的话，一旁的叶帝和叶颖两人眼睛都是一红，只不过是强忍着没有掉下泪水。
