　　后土部落，后羿之所。
<br/>　　这天，一道身穿素色月袍的绝色女子出现在后羿门前，看着门前那道同样绝丽而且正在发呆的女子，她的眼中不由的闪过一丝恍惚。
<br/>　　这绝世女神自然就是嫦羲的师傅，太阴之主望舒了。而她看的女子，也自然是嫦娥了。
<br/>　　“你还要玩到什么时候？”
<br/>　　良久，望舒开口对着呆呆的嫦娥问道。
<br/>　　“你是谁？”
<br/>　　嫦娥看到望舒，眼中闪过一丝惊艳，不知为何，她心中对于眼前这个比她还要漂亮几分的女子感到几分亲切，却是没有丝毫的戒备。
<br/>　　“徒儿，你还没有觉醒吗？”突然，望舒对着女子说道。
<br/>　　“徒儿？你是在说我吗？”嫦娥疑惑道。
<br/>　　望舒沉默良久，说道：“也罢，你也该觉醒了。”
<br/>　　说吧，她将一道灵光打入嫦娥脑海，顿时，仙雾缭绕，仙音袅袅，神光闪烁，月华降临，嫦娥缓缓闭上双眼，身体却慢慢脱离地面，缓缓升了起来，他的全身上下，笼罩着一层月华，看上去，就仿佛一轮明月，她就好像是月中仙子。
<br/>　　不知过了多久，突然，嫦娥睁开眼睛，眼中闪过丝丝精芒，更多明悟，出现在她心中。
<br/>　　“原来，我是嫦羲啊。”
<br/>　　嫦娥低声喃喃道，接着，她对着望舒缓缓拜道：“徒儿嫦羲，见过老师。”
<br/>　　“你也终究是觉醒了，为师早已算出，你有一次劫难，度过去了，便万事大吉，度不过去，便万事皆休。是劫，同时也是缘。如今，妖族已经失势，你也已归来，是时候该继承为师之责，太阴之主了。自今以后，你便是当代太阴之主。”
<br/>　　望舒看着自己的徒儿，言语之中，有几分欣喜，她开口缓缓说道。
<br/>　　“师傅，不可！”嫦羲急声说道。
<br/>　　“不，你必须做到，为师即将突破混元大罗金仙之境，太阴之主这个身份，是不该再继续担任下去了，否则因果太深，不利于我突破。以后保卫太阴星，处理太阴星上各种事物之事，便交由你了。你可不要让我失望啊。”
<br/>　　说着，望舒伸手向着自己额头一指，另一只手结出各种复杂玄妙的印法，将自己额头之上的太阴印记强行取了出来，之后打入嫦羲额头之上，顿时，她的气息一阵萎靡。而嫦羲的气息，却是在此刻缓缓变得强大起来。九天之上的太阴星上，月光大涨，无数月华仿佛受到某种冥冥中的接引，如光柱一般落了下来，落入嫦羲身体之内，将她衬托的更加高贵，完美了。
<br/>　　此刻的嫦羲，就如同真正的绝世月神，额头之上，一轮轻轻的月光印记，正在闪闪发光，将她衬托得更加完美出尘了。
<br/>　　嫦羲的身体缥缈如仙，气势如虹，她缓缓腾空而起，不由自主地向着高空升去。
<br/>　　“后羿！后羿！后羿！……”
<br/>　　突然，嫦羲对着远处大声叫了起来。而也就正在此时，后羿等祖巫也回来了。毕竟他们是夫妻，心中有所牵连，这一刻后羿仿佛听到有人在呼喊他的名字。顿时，他抬头向上循着心声望去，便看到了嫦羲踏月登天的一幕。
<br/>　　“嫦娥！”
<br/>　　后羿忍不住大声呼喊，本能的伸出双手，循着她的方向追了过去，想要将她抓到手中。他不明白嫦娥此刻为什么会飞了起来，但他已经不需要明白了。
<br/>　　“后羿！”
<br/>　　嫦羲看到自己的心上人，忍不住为之泪流满面，她不时回头，想要向下抓来，但却无论如何，也抓不到。
<br/>　　“嫦娥！”
<br/>　　后羿一声高呼，此刻，看着那嫦娥高升的身影，他使出神通，追着嫦娥向着天上飞去，只不过，他们之前好似总保持着一段距离，永远也抓不到。
<br/>　　就这样，后羿一直追嫦娥道天上，之后嫦娥再次向着太阴星而去，转瞬间，嫦娥便飞进了太阴星，同时，后羿也追进了太阴星。
<br/>　　“痴儿啊，痴儿！望你今后好自为之。”
<br/>　　地面上，望舒看着后羿追着嫦羲进入太阴星，她摇了摇头道。之后，她便不再多管，而是将目光投向昆仑山方向。看了良久，她又将目光看向混沌。之后低声呢喃道：
<br/>　　“想要突破混元大罗金仙，在洪荒之中，天道压制太严重了，不能轻易突破，否则定会遭遇许多阻碍，所以，我还是远离洪荒，到混沌深处突破吧。”
<br/>　　说着，她便起身，飞向混沌之外。
<br/>　　昆仑山方向，元始仿佛若有所感，他抬起头，看向远处，正好看到望舒飞出混沌这一幕，他顿时便知道了她的打算。
<br/>　　“也罢，既然那是你的选择，我也不便多管，只是希望你能成功。”
<br/>　　说完，他继续闭目，神游太虚，参悟天地大道去了。
<br/>　　太阴星上，后羿追嫦羲到太阴深处，便亲眼看着嫦羲消失不见，他心中大急，大声呼喊着嫦娥的名字。
<br/>　　“嫦娥！”“嫦娥！”“嫦娥！”……
<br/>　　直至良久，也听不到任何回应。突然，后羿仿佛看到了一座月华宫殿，宫殿的前方有一颗月桂树。他连忙追了进去，但很快，他便发现，自己的身体好似被这棵月桂树挡住了，无论如何，他也进不去。
<br/>　　一阵纠缠，后羿心中越来越急，担忧自己妻子的安危。突然，他内心深处一道苍茫之光闪过，紧接着他全身一震，仿佛有所明悟。
<br/>　　“无论什么阻挡在我前面，我都要将它毁灭！谁也不能阻挡我！”
<br/>　　后羿一声大叫，他二话不说，便将手中的长弓一阵变幻，化为一柄斧头，后羿举起斧头，便向着月桂树砍去。
<br/>　　月桂树乃是十大先天灵根之一，更是与太阴本源相连，若想要将其砍倒，便是要将这整颗太阴星一同毁灭，方能成功，如若不然，便不能损伤这月桂树分毫。
<br/>　　果然，斧头砍到月桂树上，一下子便看出一道缺口，可还没等后羿高兴，只见那缺口只在片刻的时间中，便已恢复完好，后羿大惊，拼命的砍向月桂树，可是，月桂树还是一样快速恢复过来。
<br/>　　以后羿那强大的祖巫之力，完全可以将太阴星连同月桂树一起毁灭，但，后羿此刻心中十分担忧嫦娥的安危，他怕自己一下子毁灭了太阴星，会伤害到嫦娥，这是一种直觉，他相信自己的直觉。
<br/>　　当然，更重要的是，太阴星为洪荒两大至尊星辰之一，关乎到洪荒万物发展万灵兴衰，可谓是重中之重，不可缺少，若是毁灭，必将遭受天大的因果，这些因果之力的爆发，完全可以将一个祖巫毁灭。他后羿现在还没想过死亡。
<br/>　　就这样，后羿用自己手中的斧头在不停的砍伐太阴星上的月桂树，月桂树也是在不停的恢复，他们陷入了毫无止境的循环之中。
<br/>　　渐渐地，后羿心中的那丝苍茫之光越来越盛了，但这一丝灵光隐藏在后羿内心深处，谁也发现不了。而与此同时，后羿心中的执念也越来越模糊，他仿佛像是变了一个人似的，完全只知道像一个机器一般不停地伐桂。
<br/>　　渐渐地，他的模样也大变了，变成了一个粗犷的壮汉，手持一柄斧头，身穿一袭简单的麻布衣，在不停的砍树，他的眼神也越来越空洞。但是，他内心深处的苍茫之光，却越来越明显了。
<br/>　　而此时，与后羿分别的嫦羲，却是已经到了太阴深处的一个神秘秘境之中，接受着太阴本源的传承，成为一个全新的太阴之主。
<br/>　　渐渐地，太阴星上伐桂的壮汉，被人们起了一个名字，叫吴刚。
<br/>　　嫦娥奔月，嫦娥奔月千年一曲，万载风月，斩不断的恩怨情仇，道不清的生死离别……
