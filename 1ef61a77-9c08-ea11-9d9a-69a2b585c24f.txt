<br/>　　显然，后者只是魂殿中垫底的存在。
<br/>　　也正是因此，慕清霄有能力将他击杀，而魂殿高层，最低恐怕都是斗尊级别，根本不是他现在能接触的。
<br/>　　战斗落下帷幕，朝附近聚拢过来的气息也越来越多，空间一阵扭曲，慕清霄就消失的无影无踪。
<br/>　　回到酒店，只见小医仙站在窗前，美眸担忧的望着枫城中央方向，如同望夫石一般，慕清霄坏笑一声，悄无声息的出现在她身后。
<br/>　　双手攀上纤腰，将她一把搂进怀中，俯下身子在她耳边轻轻一吹，道：“小妮子，在看什么呢？”
<br/>　　听到这熟悉的声音，小医仙则总算是松了口气，脸上涌上一抹嫣红，转过身来，语气温柔，道：“我刚刚听到鸟鸣声，你没事吧？”
<br/>　　闻言，慕清霄嘴角一抽，凤鸣和鸟鸣能一样么，他又不是鸟人，而是凤凰，双手在她玉臀上狠狠的捏了一把。
<br/>　　“......刚才的是凤鸣。”
<br/>　　小医仙娇呼一声，不明所以，却是点了点小脑袋，身体中的力量仿佛被抽空，软趴趴的躺在他的怀中。
<br/>　　低头一看，见她娇美柔弱，我见犹怜的模样，慕清霄食指大动，双手也变得不安分，渐渐在玉体上游走起来。
<br/>　　片刻，惹得佳人娇喘连连，直接将她抱起，向着床榻走去。
<br/>　　缠绵过后，小医仙趴在他的怀中，香甜的睡去，慕清霄并没有因为，一时邪火上涌而吃了她，替她捋了捋青丝，拿出韩枫的纳戒。
<br/>　　韩枫已经死亡，上面的灵魂印记也变得薄弱起来，挥手就能抹除。
<br/>　　抹除灵魂印记，打开纳戒，映入眼中的是玲琅满目的药材，金币堆积如山，最起码也有百万，各种极品宝甲，地阶斗技...
<br/>　　玉瓶整齐的摆放在货物架上，其中放着各种各样的丹药，其中的宝贝加起来，买下一座巨型城市，恐怕都绰绰有余。
<br/>　　然而，慕清霄并没有在意这些价值连城的宝物，而是被放在中央的一座药鼎所吸引，药鼎散发着古朴的气息，上面刻满了魔兽图案。
<br/>　　万兽鼎，上古时期由人类著名匠师借助魔兽灵魂而打造成功，位列天鼎排行第十三。
<br/>　　慕清霄脸上浮现一丝笑意，低喃道：“正愁没有好的药鼎，真是瞌睡就有人来送枕头，戒指中所有物品总合，都不及这药鼎十分之一，真是捡到宝了。”
<br/>　　悄悄起身拉上帘幕，直接取出万兽鼎，一股浓郁的药香味就席卷开来，慕清霄卷动灵气，香气就被隔绝起来。
<br/>　　万兽鼎上有一缕微弱的灵魂印记，一缕神火落下，瞬间将印记抹除，双手结印，在药鼎上成功设下了自己的灵魂印记。
<br/>　　对于药鼎，慕清霄并没有什么追求，万兽鼎虽只位列天鼎排行第十三位，但已经很不错，只要能炼丹就行。
<br/>　　手一挥，药鼎就装进储存空间中，随后，又将一些能派上用场的灵药全都装了进去。
<br/>　　至于斗技功法，极品宝甲，高级丹药，对于他来说并没有什么价值，不过总比没有好，说不定什么时候就能派上用场了。
<br/>　　将纳戒随手丢进储存空间，慕清霄就地盘坐而下，双手结印，设下结界后，微微闭上双眼。
<br/>　　此时，他的体内，正有一团深蓝色火焰悬浮在阴阳灵气湖泊上。
<br/>　　火焰升腾间，如同清澈海水般缓缓的扩散而开，淡淡的涟漪恍若水波在阴阳灵气湖上荡起，看上去极其玄妙。

<br/>　　海心焰曾经被收服过，狂暴的气息早已散的差不多，如此一来也不会在体内作乱，倒也能让慕清霄少吃一些苦头。
<br/>　　旋即，九霄琉璃焱就蔓延而上，开始吞噬海心焰...
<br/>　　半个小时候，悬浮在阴阳湖泊上的海心焰已经消失不见，九霄琉璃焱蔓延至身体每一个角落，锤炼强化体质。
<br/>　　慕清霄就如同老僧入定，在九霄琉璃焱的锻造下，痛苦并快乐着，一夜无话，时间就这样悄悄溜走。

<br/>　　但仔细想想也就释然了，海心焰毕竟只是异火榜排名第十五，能够让他提升一个小台阶已经非常不错了。
