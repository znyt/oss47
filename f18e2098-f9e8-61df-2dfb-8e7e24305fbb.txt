　　洛克希德和通用原子都是美国著名的军火巨头，双方彼此竞争，但是又彼此合作。
<br/>　　洛菲克和拉塞尔相识多年，去年的“源代码拍卖会”也是他们代表各自公司前来参加的，他们相互合作，打压其他的竞拍者，毕竟能跟洛克希德和通用原子相竞争的对手，太少了。
<br/>　　拉塞尔拉开洛菲克对面的座椅坐下，看了眼扭身离开的安妮的背影，脸上露出不怀好意的微笑，对洛菲克问道：“老伙计，还是没有能得手吗？”
<br/>　　“你知道，这个女人不简单，很不简单。”洛菲克拿起在杯口夹了一片柠檬片的玻璃杯，喝了一口杯中的马丁尼酒，说道：“这个女人就像是这杯子中的酒，充满着神秘，不去喝一口根本不知道这杯酒的味道，但是这杯酒不是谁都能喝到的，除非她自己愿意给你喝，否则根本不可能。”
<br/>　　看到自己的好友郁闷，拉塞尔显得非常高兴，哈哈大笑起来。安妮将一杯加了冰块的杜松子酒端到拉塞尔面前，微笑着问道：“什么事情让拉塞尔先生如此高兴？”
<br/>　　“我在因为洛菲克又一次在你身上吃瘪，而感到高兴。”拉塞尔端起酒杯，喝了口酒，笑着对安妮问道：“安妮小姐，难道你真的不打算给洛菲克一个机会吗？你跟着他，可是要比在这里开一个酒吧，成天跟各种各样臭男人打交道要舒服多了。”
<br/>　　安妮歪了歪头，看了眼洛菲克，随后露出魅惑的笑容，说道：“我担心洛菲克先生不能满足我的需求，毕竟他这样的年纪要想在床上有所表现的话，可能需要一些药物的帮助，而那种药一般都会对身体造成损伤，所以为了洛菲克先生身体着想，我想还是没必要了。”
<br/>　　安妮的话说得可谓是赤裸裸，不过，安妮的话，在开放的西方国家可以说是很“含蓄”了。
<br/>　　拉塞尔笑得快要喘不过气来，而洛菲克则是涨红了脸，咬着牙对安妮说道：“安妮小姐，如果你怀疑我能力的话，你大可以来尝试一下，我绝对会让你有一个难忘的夜晚。”
<br/>　　“可是我害怕因为相信了你的话，最后让我失望，所以还是算了。”安妮抱着盘子转身离开，对身后的拉塞尔和洛菲克说道：“这顿算我的，免费！对了，你们两个人情报交换中介费也免了，就当是老朋友给你们的特别优惠。”
<br/>　　“真是个会令人发疯的女人。”拉塞尔看着安妮的背影，对洛菲克说道：“对明天晚上的拍卖会，你有什么消息吗？”
<br/>　　“有那么一点消息。”说起正事，洛菲克立刻就变得严肃起来，说道：“NSA下属的EG黑客病毒实验室丢失了2个G容量的网络病毒源代码，而据说其中一部分会出现在这次拍卖会上。你那边有什么消息吗？”
<br/>　　“我这里有一条绝对劲爆的消息。”拉塞尔眼睛警惕的扫了眼四周，靠近洛菲克对他说道：“据说，这次拍卖会上会公开拍卖P病毒的制作配方，不过具体是谁拿出来拍卖目前还没有消息。”
<br/>　　“P病毒一直都掌握在‘暗月’手里，他们会拿出来拍卖？”洛菲克皱眉说道。
<br/>　　洛菲克显然是对拉塞尔的话表示不相信，P病毒是这个世界上最恐怖的致命病毒，不论是世界各地的反政府武装，恐怖组织，还是包括美国在内的各国军队，都想要着把P病毒掌握在自己手里，谁掌握了P病毒，谁就有了令这个世界颤抖的力量！
<br/>　　所以洛菲克根本不相信“暗月”会把P病毒的配方拿出来进行交易，因为这就等于是把自己的杀手锏交到了别人手中。
<br/>　　“虽然不知道这则消息的真假，但如果是真的，明天晚上的‘源代码拍卖会’将会是这十年来最疯狂的一次拍卖会。”拉塞尔又喝了一口杜松子酒，然后轻轻摇晃着手里的玻璃杯，冰块跟玻璃杯相撞，发出“叮叮当当”的清脆声响，随后放下杯子，看着洛菲克说道：“好了，我的朋友，你似乎还有什么事情隐瞒着我。”
<br/>　　“什么都瞒不过你的眼睛啊。”洛菲克把杯子中的马丁尼酒喝完，然后拿起那片夹在杯口作为装饰的柠檬片，放在嘴里，慢慢的咀嚼着，丝毫没有受到柠檬片那令人嘴里发苦的酸涩感，说道：“GHOST好像来到了这个小镇。”
<br/>　　“GHOST！？”
<br/>　　拉塞尔收起了脸上的微笑，变得严肃起来，说道：“那个跟SHADOW齐名，跟SHADOW突然消失，然后在最近几年又突然出现，当年令我们两家公司蒙受了巨大损失，拿走了多份尚未公布的武器设计源代码的GHOST？”
<br/>　　“没错，就是这个GHOST。”洛菲克摸索着他下巴下蓬松的胡子回答道，他脸上的表情也变得严肃起来。
<br/>　　“确定吗？”拉塞尔问道。
<br/>　　洛菲克轻轻摇了摇头，回答道：“目前还不确定，这只是我的猜测，但是我觉得这种可能性很大。今天在我进入小镇的时候，看到了六个疑似中国军方的人，尽管还不能明确他们的身份，可是能够确认的是，他们是军人。当年GHOST失踪后，就有传闻，他被军队招安了，成为了中国军方的人。他们来了。”
<br/>　　洛菲克指了指酒吧门口的方向，罗昊、子午、李然三个人推开酒吧门走了进来。
<br/>　　子午不是第一次来这个小镇，所以他自然是知道这个情报酒吧的存在，所以他带着罗昊和李然来到这个酒吧，打算在拍卖会开始前从这里得到一些有价值的情报，他们是红色卡片持有者，属于没有背景的自由者，所以他们无法像洛菲克这种黑色卡片持有者那样，在拍卖会开始前得到拍卖品的全部资料。
<br/>　　“三位先生，你们需要喝点什么？”安妮靠在吧台前，对罗昊、子午、李然三个人问道。
<br/>　　“三瓶可乐，谢谢。”罗昊对安妮说道。
<br/>　　“来到酒吧喝可乐？”安妮扫了眼坐在酒吧前的三个人，似笑非笑的问道。
<br/>(本章完)
