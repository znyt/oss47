　　开辟剑术。
<br/>　　这是无数剑道宗师，都不敢想象的事情，最多创造而出的，也只是剑术的一招半式，而自己的剑道，却是走的另外一道。
<br/>　　而楚风眠，所走的，正是他所开辟的九域剑术。
<br/>　　九域剑术越强，楚风眠也会变的越发强大，九域剑术若是失败，楚风眠也依然会陨落。
<br/>　　这九域剑术，现在虽然还稚嫩无比，但是一直走下去，未来可能超越世间的一切剑术。
<br/>　　“好！好！好！”
<br/>　　云老脸色大喜，一连说出了三个好字。
<br/>　　“万古无一！好，本来我还担心，你会辱了天帝威名，现在看来，我云羽天宫振兴有望！”
<br/>　　“少主，你不是想要剑术吗？好，我就将云羽天宫之中所有的剑术，全部传授与你！”
<br/>　　云老一边开口，一边眼睛一闭，突然在他的面前，化为了一方水幕。
<br/>　　在这水幕之上，皆是无数剑术，足足是有着上万道剑术，全部出现在了楚风眠的面前。
<br/>　　上万道剑术，这数量，比起剑道古籍之中所记载的数量都不少。
<br/>　　剑道古籍，可是当初剑道之中一声积累的无数剑术，这云羽天宫不愧是当初七万年前的第一宗门，单单剑术，便是记载了上万道。
<br/>　　而且这上万道剑术，许许多多都是远古时代，昙花一现的剑术，并没有流传到如今这个时代之中。
<br/>　　这些剑术，虽然有着无数缺点，却也并非是没有可取之处。
<br/>　　楚风眠一眼望过去，便是将这些剑术的真谛，全部看在眼中，其中的精华，都融入到了九域剑术之中。
<br/>　　融入了这万道剑术，足以是让九域剑术，再度蜕变。
<br/>　　楚风眠站在原地，一直是足足站了一个时辰的时间，都是一动不动，他的目光，始终是在看着水幕上的剑术。
<br/>　　从楚风眠的身上，一股剑意笼罩着，这一股剑意，不断的变化这，代表着九域剑术的一次次蜕变。
<br/>　　“轰！”
<br/>　　从楚风眠的身上，凌厉的剑意猛然爆发出来。
<br/>　　这剑意，似乎是拥有着一万剑术上的种种精华，但是却也有些不同，都被楚风眠融为一体，化为了真正属于他的剑术。
<br/>　　“剑！”
<br/>　　楚风眠目光一动，在他的面前，无数剑意却是直接化为了一尊灵剑的样子，锋利无比，猛然向着前方斩了过去。
<br/>　　单单这一剑的力量，就比起之前楚风眠的全力一剑，还要强大。
<br/>　　这还单单是现在楚风眠动用剑意的力量，若是真正全力出手之下，现在的楚风眠，实力比起进入这云羽天宫之前，要强上数倍之多。
<br/>　　在让楚风眠碰到那荒圣君，楚风眠便是有自信，足以是将荒圣君，轻易的斩杀在剑下。
<br/>　　他的九域剑术越是完善，也就代表着楚风眠的实力越发的强大。
<br/>　　“好！”
<br/>　　楚风眠大喜，他的力量，自然是越强越好，这整个云羽天宫的财富，都比不上这万道剑术，对楚风眠的作用大。
<br/>　　“一个时辰的时间，居然就这样过去了。”
<br/>　　楚风眠回过神来，才发现已经是过去了一个时辰的时间，楚风眠沉浸到剑道之中，他感觉仅仅是度过了一瞬间，但是实际上，却已经是过了一个时辰。
<br/>　　“很好，一个时辰的时间，足以参悟这万道剑术，你的剑道资质，万古无一。”
<br/>　　云老看着楚风眠，都不由的赞誉道。
<br/>　　这万道剑术，就算是给最为妖孽的剑道天才来，都至少要数日时间，才可以全部领悟贯通。
<br/>　　可现在楚风眠仅仅是一个时辰，便是做到了这一切，这让云老都是极为惊叹。
<br/>　　“看来，老夫也可以将这云羽天宫，安心的交给少主了。”
<br/>　　云老感叹一声，在他的手心中，便是一座阵图，突然凝聚出来，向着楚风眠递了过来。
<br/>　　这阵图的样子，正是云羽天宫缩小版的样子。
<br/>　　“这就是云羽天宫的阵图核心，你只要催动灵力，便是可以打开隧道，直接回到这云羽天宫之中来。”
<br/>　　云老看着楚风眠，出声说道。
<br/>　　“现在云羽天宫，被束缚于归墟天中，所以你无法将这云羽天宫带出去，不过你却是可以随时回到这云羽天宫来，若是遇到危险，便是可以进入其中来躲避，在这云羽天宫中，绝对无人可以伤及到少主。”
<br/>　　云老自信的开口道。
<br/>　　云羽天宫，天级灵器之中的佼佼者，论起品级，甚至都在祖龙至尊剑跟战龙宝车之上。
<br/>　　躲在其中，自然是没有任何危险，高枕无忧。
<br/>　　“这云羽天宫，居然无法带走？”
<br/>　　听到云老的话，楚风眠眉头一皱道。
<br/>　　“云老，您可知道这归墟天，是什么来历？”
<br/>　　归墟天，楚风眠一直以来都不清楚到底是什么来历，眼下他虽然是得到了这云羽天宫，但是实际上，在归墟天中。
<br/>　　云羽天宫也只能算是很小的一部分，整个云羽天宫，其中的洞府，宝藏，遗迹，不知道是有多少。
<br/>　　单单一个云羽天宫，只是其中的冰山一角罢了，谁也不知道在这归墟天中，到底是有着多少宝藏。
<br/>　　但是肯定会有，不亚于这云羽天宫遗迹的存在。
<br/>　　归墟天，处处极为神秘，令人无法看清其中的本质。
<br/>　　“这归墟天到底是什么来历，老夫也不清楚。”
<br/>　　云老摇了摇头道。
<br/>　　“曾经这云羽天宫的所在，也并非是在归墟天中，而且在中域的一处荒山之中，但是就在五万年前，突然这云羽天宫被一股巨大的力量收走，随后才落入到了这归墟天之中。”
<br/>　　“什么？这云羽天宫是被强行带到归墟天中的？”
<br/>　　楚风眠大吃一惊道。
<br/>　　这云羽天宫，便是整个天羽神域，简直是一方小世界的大小。
<br/>　　在强大的武者，在这一方小世界的面前，都会感觉到自己的力量渺小极致。
<br/>　　可是这云羽天宫，居然是被归墟天强行抓到其中的，这让楚风眠如何不惊讶。
<br/>　　这到底是什么力量，才可以将云羽天宫硬生生从中域，带到这归墟天中来。
<br/>　　这种力量，简直是超出了楚风眠的想象之外。
<br/>　　“老夫只是一介器灵，无法离开这云羽天宫，至于这归墟天，我也没什么了解，还请少主见谅。”
<br/>　　云老开口道。
