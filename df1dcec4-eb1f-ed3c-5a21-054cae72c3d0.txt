　　大伙吃了东西，休息了一会，聚在叶少阳的房间里讨论起来，觉得还是不要冒昧上山，因为他们都没身份，也没门派，哪个阵营都不属于，也只能先在山下转悠一下，看看形势再说。
<br/>　　“你们两个都是借尸还魂的，没事还是不要出去，这里法师太多，也不乏能够一眼看穿你们的宗师，万一暴露，还是很麻烦的，你们没事就在这呆着，等我们查到线索再确定怎么干。”叶少阳望着凤兮和崔颖说道。
<br/>　　“知道了，如果需要的时候，我们再出手！”
<br/>　　叶少阳拍了拍崔颖的肩膀，“这几天也委屈你一下，也是为了你们的安全。”
<br/>　　崔颖嘿嘿一笑，“全听风儿的。”
<br/>　　叶少阳想笑。这一路过来，他也算是看明白了，崔颖跟凤兮的情况，有点类似橙子跟萧逸云，都是女强男弱，凤兮一路使唤崔颖，跟小二似的，崔颖任劳任怨也不反抗，叶少阳看着觉得很好玩。又想到自己，如果真的跟芮冷玉在一起，恐怕也会是这个样子吧。如果是小九的话，或许是另一种情况。
<br/>　　想到芮冷玉和小九，叶少阳心中又是一阵苦涩。
<br/>　　丢下凤兮两口子，叶少阳一行三人从村子里出来，先在附近镇子上溜达了一下，看到了不少道佛打扮的人，自然都是各个门派的。
<br/>　　“……照这么说，我们这一趟是白来了？”
<br/>　　“自然也不是，虽然龙华会是不用参加了，但是我看啊，这次法术公会跟桃花山只怕是要开战……”
<br/>　　“不至于吧？公会虽然掌控一切，但是……也不至于直接对一个门派下手吧？”
<br/>　　三个法师在交头接耳地讨论着，虽然也有心将说话声放低，但之前因为道渊真人的提醒，三人都提早换了普通装束，看着就像三个普通的乡村二逼青年。这几个法师并没有注意他们，一边走，一边继续着讨论。
<br/>　　叶少阳三人假装赶路，远远跟着，距离保持着刚刚能听到他们的说话声。
<br/>　　“你不知道，桃花山上来的那位，据公会的人说，是一个甚为强大的邪物，并擅长伪装之术，用邪术蒙蔽了桃花山上下的一众道士，试图通过桃花山来颠覆整个法术界，公会自然不能让他得逞……”
<br/>　　另一个惊道：“你们是如何知道这消息的，何以我安平山不知道？”
<br/>　　之前说话那人一脸得意，说道：“日前法术公会召开会谈，我派有一位祖师有幸前去旁听，却是听到了这个消息，实际也算不上什么秘密，因为不日就会公布，我们也只是提前知道，也可先行布置一番。”
<br/>　　“是是，多谢道兄告知！”那人连忙拱手，又询问了一些具体的事情。提问那人又问道：“道兄，这桃花山上的妖孽，究竟是什么，值得法术公会如此大张旗鼓，连我们这些门派都叫来一起对付。”
<br/>　　“是什么妖孽，这个还真不知，不过……他之前连伤法术界数位高人，自然不是一般的邪物，莫说是我们这些中小门派，这次行动，也顶多是来凑个人数，真正的主力，必然是那几位。”
<br/>　　“道兄是说……那几大弟子？”
<br/>　　“自然是了，这对于他们来说，也算是一场考验和历练，毕竟他们将来是要执掌整个法术界的，这一战也可建功。”
<br/>　　“那……若是失败呢？”
<br/>　　那道士大笑起来，有点不屑地说道：“老弟你好没见识，星月老祖在当世就选了这么几个弟子，悉心栽培多年，实力就算跟几大门派的宗师相比，也未必不及，怎可能会败给那邪物？”
<br/>　　“是是，在下失言了。”
<br/>　　另外一个缓缓点头，说道：“长江后浪推前浪，以我之见，就算是茅山、龙虎山的几位宗师，也未必能胜过比这几个年轻人，尤其是张晓寒……”
<br/>　　“是了。茅山也好，龙虎山也罢，道门五大山门，佛门四大道场，也不过是曾经辉煌，到了今天，也只剩下虚名而已，面对这几位公会的天才少年，如何抵挡？”
<br/>　　身边两人都深表赞同。三人得意洋洋地朝远处走去，本来还想聊点什么，一回头看到跟在后面的叶少阳等人，三人早就看到他们，以为是路人，也没在意，结果走了这么久他们还在身后，这就有点警觉了，互相使了个眼色，闭口不再说话，匆匆离开。
<br/>　　“走，找个地方说。”叶少阳低声嘱咐，三人继续假装闲逛，一直走出村子，找了个树荫下视野开阔的地方，这才开始讨论之前偷听到的信息。
<br/>　　“高明啊……”道渊真人哼了一声，“法术公会来到人间，别的没学会，倒是把这种卑劣的手段学会了。我是不相信桃花山上那个神秘人是邪物。”
<br/>　　叶少阳笑了笑。想对付一个人，先给对方扣一顶帽子，然后不管自己怎么打压对方，就算是赶尽杀绝，也是名正言顺的了，这在古代就是所谓的出师有名，不管是不是真的有名，至少能找出一个搪塞大众的理由。何况，大部分被法术公会控制的门派，也会揣着明白装糊涂，用这个借口说法术公会的好话。
<br/>　　毛小方道：“不管桃花山上那位是不是邪物，只要法术公会说是，那就只能是了。”
<br/>　　叶少阳沉吟起来，想到龙虎山也有宗师过来，于是请他去找他的师兄弟们了解一下当前的形势，看法术公会是不是真的要对桃花山动手了。
<br/>　　道渊真人答应一声，龙虎山在这边也包了个客栈，道渊真人之前来过一次，知道地方，正好也是往桃花山去的方向，于是三人一起朝桃花山方向闲逛。
<br/>　　快到桃花山的地方，有一些人家，靠路边有一家客栈，走到跟前，道渊真人告诉叶少阳，这就是茅山包下的客栈。
<br/>　　叶少阳听说茅山弟子就住在这里，心中不免有些异样的感觉。偏巧他们走过去的时候，一行几人从客栈里走出来。
<br/>　　为首的是一个白胡子的老道士，身边走着看上去五十多岁的黑须男子。
<br/>(本章完)
