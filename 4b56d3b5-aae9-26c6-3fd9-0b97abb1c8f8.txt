    陈明看向老人，很是关心地道：“老人家，你的眼睛流血了......”<br/>    <br/>    但是老人如同是浑然不觉一般，笑道：“不该看的东西，确实不能看，原来是祂啊！可是，是祂中的哪一位呢？”<br/>    <br/>    “不必去想，凡是自有祂注定，即便是天道，又能如何？”<br/>    <br/>    “不能继续推测了，这是我不能窥探的东西。”<br/>    <br/>    陈明看着老人自言自语，心中震撼不已，这个神秘老人，是因为看了自己的前世一眼，所以瞎了吗？<br/>    <br/>    若真是如此，自己的前世，到底是什么来头，居然是让这个神秘老人看了一眼就瞎了！<br/>    <br/>    那么，能够将自己的前世打入轮回的，又是何种存在？<br/>    <br/>    陈明问道：“你到底看到了什么？”<br/>    <br/>    老人笑道：“我什么都没看到，只是看见了光，光太耀眼，所以我瞎了。”<br/>    <br/>    陈明继续问道：“难道你就不能推测出我的前世到底是谁？”<br/>    <br/>    老人继续道：“这对于你来说，没有好处，你必须斩尽你与你前世的因果。又或者说，你实际上是那种不该出现的存在，又或是，你是那种注定会出现，却不知道结局的存在。这些我也不是很明白，所以我不会接着说下去。”<br/>    <br/>    陈明很是疑惑，他完全不知道这个神秘的老人在说些什么。<br/>    <br/>    “你说的这些到底是什么意思？”<br/>    <br/>    老人飞空而起，只是淡淡道：“若你不半途夭折，我们会再次相见的，我只能告诉你一点，你并不是不死之身，你只是，三千个中的一个。至于这三清观，我送与你了，但是我却不能收你为徒，你也不能拜任何人为师，你的路，只能你自己去走。所以，做一场交易吧，有朝一日，替我击败真武老道。老道，去也！”<br/>    <br/>    神秘老人瞬间消失，如同是不在这天地间一般。<br/>    <br/>    陈明满头的疑惑，什么叫做我只是三千个之中的一个？<br/>    <br/>    这个老人，到底看到了什么？<br/>    <br/>    不过陈明算是明白了一点，那就是这个老家伙留下了条件，那就是让自己去击败真武老道，这两个家伙，是宿敌吗？<br/>    <br/>    不过陈明倒是知道了一点最重要的，那就是自己不是不死之身，还是不能随便作死啊。<br/>    <br/>    好好活着，比什么都好。<br/>    <br/>    如今日子过得这么舒坦，好好活着才是正途，没事去想什么打打杀杀啊，等着卓青瑶轮回之后娶回家才是正道啊！<br/>    <br/>    凌仙有些惊讶地看向陈明，“师父，这个神秘的老家伙到底在说什么？”<br/>    <br/>    陈明道：“就是说他这里留下的造化都是你的了，只是你要去击败真武大帝！”<br/>    <br/>    凌仙的眉头跳了跳，“师父，他不是让你去吗？”<br/>    <br/>    陈明摆手道：“尊老爱幼懂不懂，真武大帝一大把年纪了，为师去，也欺负人了，还是你这种辈分小的比较好。”<br/>    <br/>    凌仙心中升起万丈豪情，“放心吧师父，交给弟子了！”<br/>    <br/>    陈明懒得去想这些有的没的，直接走进道观大殿之中，可以看得到，在道观的正中央，挂着一幅画，上面画着一团神光，并没有画模样。<br/>    <br/>    在这幅画的上面，有着六个字：万圣天母娘娘。<br/>    <br/>    这个神秘老人，供奉的是万圣天母娘娘，但是这万圣天母娘娘，又是哪一位啊？<br/>    <br/>    不知道，这些东西，距离自己太远，就算是知道了，也没有任何用处。<br/>    <br/>    蚂蚁思考苍鹰吃什么，是没有任何意义的。<br/>    <br/>    陈明走到道观之后，有着三座阁楼，其中有一座的门上，便是有着五个字：一气化三清。<br/>    <br/>    这便是十大仙术之一，这么容易就到手了？<br/>    <br/>    不愧是系统小姐姐的，该直接送的东西，绝对不让自己废手脚！<br/>    <br/>    陈明走进其中，可以看得到在正中央有着一本经书，上面正是写着五个古字：一气化三清。<br/>    <br/>    陈明伸手拿起经书，翻开之后在心中道：“顿悟这本经文。”<br/>    <br/>    “叮，顿悟此经文需要十亿师德！目前师德不足！”<br/>    <br/>    陈明很是满意地点点头，嗯，很好，真货！陈明继续在心中道：“目前的师德可以参悟几卷？”<br/>    <br/>    “叮，一气化三清共有九卷，参悟三卷可以施展，前三卷共需师德1800w！”<br/>    <br/>    三卷就要一千八百万？<br/>    <br/>    不愧是十大仙术之一，这个价，够狠！<br/>    <br/>    目前师德肯定是不足的，想要得到这些师德，就必须从七宗罪上想办法了。<br/>    <br/>    陈明直接将这本经文收在储物戒中，至少也是十大仙术之一，不能留在这里，自己随身带走比较好。<br/>    <br/>    陈明看了一眼周围，还有着三种大仙术，分别是《撒豆成兵》，《方寸人间》，《五道掌》。<br/>    <br/>    《撒豆成兵》很容易理解，大概便是一种能够洒出豆子，便能战斗的傀儡。<br/>    <br/>    《方寸人间》则是一种空间法术，能够大小如意。<br/>    <br/>    《五道掌》则是一种能够糅合五行的掌法。<br/>    <br/>    这些大仙术，也带走，带回去给门下的记名弟子修行，他们功法是有了，但是这术法嘛，还是只有一些普通仙术，但是现在就不一样了，他们可以修行大仙术了。<br/>    <br/>    甚至于，陈明连一气化三清都准备教出去。<br/>    <br/>    都是自己的徒弟，有什么值得遮遮掩掩的，直接教出去，这才是燕山的风格。<br/>    <br/>    师父领进门，修行在个人，学是能学，但是他们能不能学得会，那就是另外一回事了。<br/>    <br/>    正在这时，凌仙从旁边的一座阁楼中走了出来，递上了四本经书，然后道：“师父，我发现了四种大仙术，分别是《诛仙剑》《绝仙剑》《陷仙剑》《杀仙剑》！”<br/>    <br/>    陈明仔细看了一眼，这不是就是布置诛仙剑阵最为重要的四种剑道吗？<br/>    <br/>    陈明问道：“可发现了诛仙剑阵？”<br/>    <br/>    陈明最为强大的手段，还是阵法，他对于这诛仙剑阵，可是垂涎已久！<br/>    <br/>    凌仙摇头道：“并没有发现诛仙剑阵，也没有发现和诛仙剑阵有关的经文。”<br/>    <br/>    陈明有些失望，但是还是想通了，这一行，已经得到了不少术法了，可以满足了。<br/>    <br/>    而且，不是还有着一座阁楼吗？不知道这一座阁楼之中，有着什么仙术。<br/>    <br/>    <br/>    <br/>
