        其实，诺司峻诈死离开监狱后，还被逼得供出了自己“上线”的几个领头人。
<br/>
<br/>        本来，他是打算视死如归的。
<br/>
<br/>        但是，直到有狱警暗中偷偷地告诉他，他的妹妹还有他的女朋友，以及他的父母，都在外头等着他平安回去。
<br/>
<br/>        如果想活着出去见到他们，他就必须老实交代他所有知道的事情。
<br/>
<br/>        所以，诺司峻出卖了自己道上“朋友”，同时，他也交代了自己在滨江城利用在婚介网站上当主编的职务便利做了当中的“牵线人”。
<br/>
<br/>        事如今，诺司峻不得不带着一家子，搬去其他地方居住，从此隐姓埋名。
<br/>
<br/>        童筱颖从村上的老家拖着行李箱回到西镇上的时候，傅夜沉已经将车停在了路边等候她已久。
<br/>
<br/>        倘若，这次陪她一起来的人，换做是傅昼景，其实压根就没办法替她解决梁家的事情。
<br/>
<br/>        就算傅昼景直接拿出两百万出来，还不一定能把哥哥诺司峻从监狱里救出来。
<br/>
<br/>        说到底，遇上这种事情，金钱比不过权利的重要。
<br/>
<br/>        傅夜沉的幕后身份，童筱颖不敢猜，也不想去猜。
<br/>
<br/>        其实，从她邂逅他的第一晚开始，她就知道他的身份一定不简单。
<br/>
<br/>        傅夜沉从后视镜里看到路边童筱颖正拖着行李箱朝他的车子这边走过来，连忙打开了驾驶座的车门下了车，去帮童筱颖提行李箱。
<br/>
<br/>        回到车上后，童筱颖沉默了许久，才很诚恳地说了一声谢谢。
<br/>
<br/>        因为，她知道，她这一句谢谢，其实是远远不够的。
<br/>
<br/>        傅夜沉对于她的这声谢谢，显得有些云淡风轻地应了声：“我只不过是帮阿景而已，不是帮你。”
<br/>
<br/>        他不想给她造成什么心理负担。
<br/>
<br/>        童筱颖嘴角微扬，看着傅夜沉露出了一个既释然又欣慰的笑容。
<br/>
<br/>        傅夜沉看着她甜美的微笑，仅仅只是看了几秒，便努力克制自己收回了目光，一边发动车子，一边看向后视镜。
<br/>
<br/>        童筱颖深吸了口气，抬起手来，将诺母传给她的金镯子给取了下来，拿出包包里的丝帕包着。
<br/>
<br/>        傅夜沉的余光里，看到童筱颖手上有一个金晃晃的东西，便问道：“戴你手上挺好看的，为什么要取下来？”
<br/>
<br/>        “想留着，将来传给我和昼景的女儿。”童筱颖微笑着说，漂亮的脸蛋上似乎对未来的向往满是憧憬。
<br/>
<br/>        可傅夜沉听到她说这句话的时候，心口又一次隐隐作痛。
<br/>
<br/>        如果这样的痛，会麻木该多好。
<br/>
<br/>        但是，每一次，都痛得那般真真切切，让他难以呼吸。
<br/>
<br/>        “你……喜欢女儿？”傅夜沉试图转开话题。
<br/>
<br/>        童筱颖下意识地摸了摸自己的肚子，微微偏头看向了车窗外，莞尔一笑：“嗯！”
<br/>
<br/>        她希望将来，能给昼景生个女宝宝。
<br/>
<br/>        如果，是个男宝宝的话，其实也不错啦！
<br/>
<br/>        只要是她和昼景生的，女宝宝和男宝宝都可以。
<br/>
<br/>        童筱颖心里就是这么想的。
<br/>
<br/>        以前，她和何明旭谈恋爱的时候，从未想过自己将来要给何明旭生孩子，那个时候的她，每天只想着要努力工作，要努力赚钱供何明旭读研，还有自己要挤出时间去学习，不能落后于人。
<br/>
<br/>        或许，是因为生活所迫，那个时候没有那份心思，让她去设想未来的事情。
<br/>
<br/>        而现在，今时不同往日。
<br/>
<br/>        傅昼景给了她物质上的需求，有了家，她就会想要有个孩子。
<br/>
<br/>        如今，她可以一门心思地放在学习上，空暇之际，还会想想她自己和傅昼景的将来。
<br/>
<br/>        在童筱颖的设想里，她和傅昼景的未来，是美好的。
<br/>
<br/>        坐在副驾驶座上，童筱颖还是忍不住想要提前打开诺母转交给她的，外婆的小木箱。
<br/>
<br/>        这是外婆特意留给她的，也不知道这个小木箱里会装着一些什么东西？
<br/>
<br/>        童筱颖带着这份好奇心，轻轻地打开了小木箱。
<br/>
<br/>        木箱里除了四大医术书的手抄本以外，还有外婆自己写的《千草集》和《行医日记》，里面记载着这片大山里所有的可入药的草本，和外婆所医治过的所有病例。
<br/>
<br/>        外婆要诺母转交给她，是希望她学医吗？
<br/>
<br/>        童筱颖若有所思地望着车窗外。
<br/>
<br/>        傅夜沉带着童筱颖回到了傅家大庄园里后，就被傅老太爷叫去训话了。
<br/>
<br/>        童筱颖这刚到家，前脚才踏进自己的屋内，冷夫人后脚就进来了，并当着跟随而来的苏漫雪的面，猝不及防地给了童筱颖一巴掌。
<br/>
<br/>        “怪不得小景冷落你，你这吃着碗里的，还看着锅里的了！就因为你仗着跟我们家小雪长的一模一样，你连你大哥都敢勾引！”冷夫人气急败坏地呵斥。
<br/>
<br/>        苏漫雪连忙搀扶着冷夫人，故作心疼地揉着冷夫人的手，安慰道：“妈，你这何必自己动手呢？打得手疼不疼？”
<br/>
<br/>        “妈不疼，妈这是为你在出气！”冷夫人气呼呼地说道。
<br/>
<br/>        童筱颖捂着被冷夫人扇肿了的脸颊，站直了身子，看着冷夫人和苏漫雪，为自己辩解道：“夫人，我想您一定是误会了。我没有仗着自己跟苏漫雪长的一模一样，去勾引大哥！”
<br/>
<br/>        “你没有？你没有，你带着小沉去你老家做什么？难道不是去见你父母？筱颖，我是看在小景的份上，才在这个家里好生待你的。但是，我没想到，你如此不要脸！”冷夫人指着童筱颖的鼻子大骂。
<br/>
<br/>        苏漫雪见状，还不忘在一旁添油加醋地说：“妈，你就不要这样说弟妹了。说不定，弟妹她是无心的呢？”
<br/>
<br/>        “还无心的？！哼！小雪，你去看看，我请来的妇科医生到了没？”冷夫人接着说道。
<br/>
<br/>        苏漫雪微微点头，应了声：“那我去给您问问。”
<br/>
<br/>        她说着，便转身出去了。
<br/>
<br/>        童筱颖一脸茫然，不知道她们两到底想干什么。
<br/>
<br/>        等到苏漫雪再次回来的时候，她身后跟着一个穿着白马褂的女医生，还有两名女护士。
<br/>
<br/>        “夫人，您这是要做什么？”童筱颖不解。
<br/>
<br/>        冷夫人白了童筱颖一眼，冷冷地回答道：“当然是给你做检查！”
<br/>
<br/>        “检查？！什么检查？”童筱颖吓得打了个冷噤，挪动着步子往后退了退。
<br/>
<br/>        冷夫人随即转身，在苏漫雪的搀扶下走了出去，并将房门给带关了。
<br/>
<br/>        童筱颖刚想反抗，却被迎面走上来的两个女护士给拦住。
<br/>
<br/>        “你们……你们要对我做什么？”童筱颖紧张得手心里冒出了冷汗。
<br/>
<br/>        冷夫人和苏漫雪离开主屋后没多久，身后就传来了屋内童筱颖发出的惨叫声。
<br/>
<br/>        苏漫雪听着童筱颖那痛苦的叫声，暗自得意地笑了笑。
<br/>
<br/>        冷夫人却嗤之以鼻：“只不过是做个检查而已，还如此反抗，十有八九是做了什么龌龊事。”
<br/>
<br/>        没过多久，屋内安静了。
<br/>
<br/>        女医生和两个女护士也从屋子里开门走了出来。
<br/>
<br/>        冷夫人就站在原地，等着她们过来汇报情况。
<br/>
<br/>        “夫人，二少奶奶还是如假包换的处子之身。”女医生颔首汇报道。
<br/>
<br/>        冷夫人一脸诧异地反问：“那会不会是重新补做的？”
<br/>
<br/>        “不是！是天生的！”女医生一脸淡定地说道。
<br/>
<br/>        苏漫雪顿时也诧异不已。
<br/>
<br/>        这童筱颖之前和何明旭在一起过，现在又和傅昼景在一起了，而且还和傅夜沉出去了三天三夜，如今居然仍旧是处子之身。
<br/>
<br/>        怎么可能！
<br/>
<br/>        冷夫人抿了抿唇，自知冤枉了童筱颖，心里虽然有些过意不去，但是又碍于面子，不想去跟童筱颖道歉，索性这事，她就当没发生过好了。
<br/>
<br/>        但是，隔墙有耳。
<br/>
<br/>        童筱颖被冷夫人请来的私家医生强行给做那种检查的事情，还是传到了傅夜沉的耳朵里。
