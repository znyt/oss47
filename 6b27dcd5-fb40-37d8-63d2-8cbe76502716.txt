　　火云蟒，是一种极为强大的妖兽，身躯庞大。像火狼、蛮火牛等妖兽，它一口就能轻松吞了。
<br/>　　而作为武魂，它更是位列黄阶，极为强大。
<br/>　　那扭动的蛇身，几乎占据了比武台三分之一的范围；蛇尾啪啪作响，带着火焰的威势，将地面砸出一个个大坑。
<br/>　　“好强！”台下观战的族人们吃了一惊，“单单蛇尾的拍打，甚至还未发动过攻击，其威力竟已不亚于凡境八重武者的全力一击。”
<br/>　　“萧逸的招式虽然古怪，拳若猛虎，但和火云蟒的差距太大了。火云蟒随便张嘴，都能一口将萧逸咬成两截吧。”
<br/>　　战斗才刚刚打响，但族人们已经认定萧逸必败。
<br/>　　在他们眼中，那威势十足十足的火焰猛虎，恐怕在火云蟒面前撑不了片刻吧。
<br/>　　“吼”一声虎啸声响起。
<br/>　　火焰猛虎率先朝火云蟒扑去，两者不同比例的身形，怎么看都想是火焰猛虎在自寻死路。
<br/>　　“哼，不自量力。”萧若寒冷笑一声，操控火云蟒张开了蛇口。
<br/>　　萧逸看着那狰狞的血盆大口，却没有丝毫退缩，仍旧直直冲去。
<br/>　　“找死。”萧若寒眼中透漏出浓烈的杀意，他感觉，自己下一秒就能让萧逸葬身蛇腹。
<br/>　　然而，真的是这样吗？
<br/>　　当火云蟒即将吞噬火焰猛虎时，火焰猛虎的速度竟陡然间加快，轻松躲开了蛇口，而后猛虎翻身，一把咬住了火云蟒巨大的脖子。
<br/>　　火焰猛虎锋利的牙齿直入蛇身，而后猛虎随着蛇身滑落。
<br/>　　猛虎的牙齿，实则是萧逸的双手。
<br/>　　嘶拉一声，如同利刃划破薄纸般。庞大的火云蟒身躯由上而下被切割而开。
<br/>　　嘭的一声，由真气凝聚而成的火云蟒顷刻间消散。
<br/>　　“怎么可能？”萧若寒瞪大了眼睛，“火云蟒乃是黄阶武魂，由我凡境九重的真气凝聚而成，岂会如此轻松地消散掉。”
<br/>　　下方观战的族人们，更是倒吸一口凉气。
<br/>　　“好厉害，萧逸只用了一击吧，竟将火云蟒武魂打散了！”
<br/>　　比武台上，面对萧若寒的不可置信，萧逸冷笑道，“没什么不可能的，萧若寒，你我的帐，也该是时候好好算一算了。”
<br/>　　话语刚落，萧逸的眼神变了，变得冷酷而杀意大增。
<br/>　　嗖，萧逸消失在原地，如同一只势若疾风的猎豹。
<br/>　　萧若寒瞳孔一缩，完全跟不上萧逸的速度。
<br/>　　嘭，当他反应过来时，已经后背一疼，被萧逸一拳打啪在地。
<br/>　　萧逸趁势而上，一把将萧若寒按在地上。
<br/>　　再次一拳，朝萧若寒脸上轰去。
<br/>　　萧若寒直接被打得满脸溢血，头脑发懵。
<br/>　　“这一拳，是替已经死去的萧逸打的。”萧逸心中暗道。
<br/>　　再次一拳，这次是朝萧若寒腹中轰去。
<br/>　　“噗”萧若寒立刻大口大口地喷着鲜血。
<br/>　　“这一拳，是你请杀手杨伏暗杀我，害依依受伤而打的。”萧逸心中发狠，这一拳，用出了十成的力度。
<br/>　　又是一拳，朝萧若寒的胸膛轰去。
<br/>　　“这一拳，是报答你多次算计老子打的。”
<br/>　　“这一拳，本应该打你父亲，现在一并算你头上了。”
<br/>　　“这一拳，单纯是我看你不爽。”
<br/>　　“这一拳，我暂时没有理由，但打得过瘾了，不想停手。”
<br/>　　“这一拳.....”
<br/>　　“.....”
<br/>　　萧逸按着萧若寒，一顿狂风暴雨般的狠揍。
<br/>　　萧若寒其实很强，但在萧逸手中，却连还手的机会都没有。
<br/>　　同为凡境九重，使用形意五绝的萧逸足以碾压一切同等级武者，甚至是越阶战斗也轻而易举。
<br/>　　即使萧若寒拥有黄阶武魂，却也无法拉平与形意五绝的差距。
<br/>　　没有直面与萧逸对战过的人，永远不知道形意五绝强悍到什么地步。
<br/>　　而如今萧逸将形意五绝与武魂力量结合，更是摧枯拉朽般瞬间击败了萧若寒，以致萧若寒连还手的机会都没有。
<br/>　　“噗、、噗、、噗、、、”
<br/>　　萧逸每轰出一拳，萧若寒就是一口鲜血吐出。
<br/>　　萧若寒已经被打蒙了，除了疼痛地哀嚎，不断地吐血，便只剩下微弱的呼吸。
<br/>　　曾经的萧家年轻一辈第一人，此刻的模样只能用凄惨去形容。
<br/>　　但，萧逸完全没有留手。
<br/>　　萧若寒这种小人，他杀之而后快。
<br/>　　同为萧家族人，却勾结慕容家的女子，暗中设计，私定奸情，置同族中人于死地，这是何等的恶毒。
<br/>　　之后更是多番陷害算计，罔顾同族之情，一心只想加害他人，这又是何等的无耻与卑鄙。
<br/>　　恶毒、无耻、卑鄙也就罢了，但他惹到萧逸，便只能自吞苦果。
<br/>　　萧逸很清楚，萧若寒是那种隐藏于黑暗中的毒蛇，随时可能噬人夺命。
<br/>　　“小废物，给我住手。”
<br/>　　这时，长老席上，一阵怒喝声响起。五长老见自己儿子如此凄惨，当即打算出手。
<br/>　　“五长老，还是给我安分地坐下吧。”三长老一跃而起，拦住了准备出手的五长老。
<br/>　　一旁的二长老则沉声道，“都坐下吧，比武还未结束，也还未到我们出手干预的时候。有我看着，不会出意外的。”
<br/>　　果然，二长老的话音才刚落，比武台上，萧若寒竟忽然双手一动，凝聚出火云蟒武魂。
<br/>　　凡境九重武者的肉体经过庞大的灵气淬炼，极为强悍，自然不会如此不堪。
<br/>　　萧若寒虽然刚才确实没有还手之力，如今也受了不轻的伤势，但还不至于致命。
<br/>　　只不过，阴险的他一直在等一个可以反扑萧逸的机会罢了。
<br/>　　而长老席上，五长老的忽然一声暴喝，微微吸引了萧逸一下目光，便让他捉住了机会。
<br/>　　“小废物，去死吧。”萧若寒狰狞地笑了起来。
<br/>　　火云蟒出现的时机太过突然了，他相信萧逸绝对反应不过来。
<br/>　　而此时，火云蟒已经张开了血盆大口，距离萧逸不过数米距离，下一秒就能将萧逸吞入腹中。
<br/>　　“萧逸，死吧。被我的火云蟒吞噬，你会在蛇腹中被火焰焚烧，最后化做一撮****，凄惨地死去，哈哈哈。”萧若寒仿佛已经看到自己胜利的那一刻。
<br/>　　“哼。”萧逸冷笑一声，“果然是一对阴险的父子。”
<br/>　　面对火云蟒的来袭，萧逸怡然不惧，快速地伸出了手臂，呈虎爪之势，一把扣住了蛇口。
<br/>　　在虎形加持的力量下，萧逸打算再次将火云蟒武魂打散。
<br/>　　然而，当他注视到火云蟒那阴冷的目光时，却忽然神色一阵恍惚。
<br/>　　“不好，那是火云蟒的凝视，逸儿小心。”
<br/>　　三长老大喝一声，尽是担忧之色。
<br/>　　下方族人们也发现了其中的异常，道，“那就是火云蟒武魂的天赋吗？蛇眼的阴冷，足以将凝视之人冻僵在原地，不能动弹。”
<br/>　　武魂是妙用无穷的东西，赋予了武者各种能力。像蛮火牛，赋予了武者控火和力量；火狼，赋予了武魂控火和速度。
<br/>　　而这火云蟒这更为厉害，除了控火能力外，最厉害的就是它的那双眼睛，无比阴冷，足以让敌人在他的注视下瑟瑟发抖，陷入无限恐慌；同时，让人如临冰窖，全身冰冷，无法动弹。
<br/>　　同境界比拼中，武者在这种蟒蛇的凝视下，将毫无抵抗之力。
<br/>　　“火云蟒，去，将那小废物吞掉。”萧若寒张狂地大笑了起来。
<br/>　　长老席上，二长老目光一凝，“该出手了。”
<br/>　　然而，他刚准备出手，却又忽然停了下来，原本沉稳的脸色变得目瞪口呆。
<br/>　　只见，比武台上，萧逸竟瞬间恢复了正常，并全力一握，将火云蟒捏散于手中。
<br/>　　“哼，如此卑鄙的技俩，也就你这种小人才会使用。”萧逸冷喝一声，随即再次对萧若寒展开了狂风暴雨般的攻击。
<br/>　　火云蟒的凝视，说白了，只是一种让敌人陷入恐惧，内心崩溃的手段。
<br/>　　而这种手段，在萧逸面前，不值一提。
<br/>　　再冰冷严酷的寒霜，再毛骨悚然的恐惧，也敌不过一颗坚如磐石的心。
<br/>　　所谓的冷意，所谓的恐惧，对于萧逸这个曾经纵横杀手界的传奇来说，不过如此。
<br/>　　长老席上，二长老不禁赞叹了一声，“好小子，竟有如此坚定沉稳的心智，以他现在表现出来的修炼天赋，日后必然能成为名动一方的强大武者。”
<br/>　　比武台上，萧逸涌现了更浓郁的杀意，仍旧在狠揍着。
<br/>　　他当然有实力要了萧若寒的命，只需一拳轰在脑袋上，萧若寒必死无疑。
<br/>　　但他更清楚，台上有裁判看着，长老席上还有九大长老在观战。
<br/>　　以先天长老的速度和力量，绝对能在他打出致命一拳时，轻松救下萧若寒。
<br/>　　所以，他无法下杀手，只能这般不断地攻击胸膛、腹部等部位。
<br/>　　当然，萧若寒就算死不了，但伤势会越来越重，先是身骨重创，之后损至经脉。
<br/>　　再这样下去，萧若寒不死也残。
<br/>　　对付萧若寒这种小人，萧逸自然不会心软。
<br/>　　五长老不是傻瓜，他已经看出了萧逸的打算，当即道，“二长老，比武该结束了吧，萧逸再这样打下去，将对若寒造成难以挽回的伤势。”
<br/>　　二长老点了点头，准备宣布比武结束。
<br/>　　萧逸目光一凝，要结束了吗？没办法了，能多打一拳是一拳。
<br/>　　萧逸打算再打出一拳，朝丹田轰去，起码要让萧若寒一年半载下不了床，免得这个阴险的家伙再算计自己。
<br/>　　恰在此时，一道火红色的人影快速跃上了比武台，速度极快。
<br/>　　人影接住了萧逸的拳头，并将萧若寒救了下来。
<br/>　　“轰。”而后人影趁萧逸没反应过来，一拳将萧逸轰飞。
<br/>　　“额。”萧逸嘴溢鲜血。
<br/>　　“好强。”萧逸站稳身子后，冷冷地盯着来人。
<br/>　　族人们也吃了一惊，待看清火红色人影时，纷纷脸色一变。
<br/>　　“是萧若狂，萧若寒的亲哥哥，五长老之子。”
<br/>　　“听说这萧若狂的天赋比萧若寒更强，年纪轻轻已经加入了玄火门，怎么忽然回来萧家了？”
<br/>　　长老席上，五长老阴笑一声，暗道，“若狂，终于等到你回来了。哼，小废物，打算一鸣惊人吗？老夫今天就让你知道，你只是老夫手中的一只蝼蚁罢了，随时都可以捏死你。”
