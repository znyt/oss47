　　第589章
<br/>　　张三斤想卖掉程大雷，以做进身之阶。最好的选择，自然是青州城的官兵，如此也能混个官身。
<br/>　　但出于对官家天然的不信任，张三斤并没有选择官府。他想要报告书院，却也没有这方面的关系。所以，他最后的选择是黑虎寨的刘黑虎。
<br/>　　刘黑虎点了五百余人，这已经是山寨的精锐。据张三斤所说，程大雷身染重病，那么刘黑虎还不趁你病要你命。一来可以拿住程大雷，天下扬名，二来，也可以趁机拿下白马山，扩张自己的势力。
<br/>　　由张三斤这个奸细带路，他们抄小路上山，一路上神不知鬼不觉。到得山寨门前，由张三斤诓开寨门，隐藏在他身后的刘黑虎才一窝蜂的杀进去。
<br/>　　等大家反应过来时，刘黑虎的人马已经杀了进来。一进来便杀人放火，以雷霆手段制住了白马山的土匪。
<br/>　　程大雷在房间里睡得踏实，外面的喊杀声惊动了他。他在床上一个鲤鱼打挺，抓起匹夫剑就跃了出去。
<br/>　　外面喊杀成一片，白马山的山贼大多已被制服。山贼么，互相打来打去，今天既然打不过，放下刀投降也算不得什么。
<br/>　　只有小白狼几个人拿着兵器战斗，他们已被挤在角落中。
<br/>　　“张三斤，枉我待你不薄，你竟勾结外人，出卖山寨弟兄。”小白狼咬着牙骂道。
<br/>　　“大当家，你不要怪我，真正要害山寨的是你呐。”张三斤道：“程大雷不杀，山寨如何安生，我看咱们还是跟着刘大当家更有前途些。”
<br/>　　这个时候，程大雷已经走出屋子，站在院子中央。他如黑猫般起身，窜上屋顶，几个起落便闯入重围，拦在小白狼身前。
<br/>　　张三斤和刘黑虎同时咬牙冷喝：“程大雷。”
<br/>　　“是我。”程大雷耸耸肩，手摁在剑柄上：“谁来？”
<br/>　　人的名树的影，程大雷的确是有几分名气的，尤其是在山贼中间。哪怕他现在只是持剑不发，同样能给人带来巨大压力。
<br/>　　小白狼见到程大雷也有了底气，握着刀冷冷笑道：“好贼，我怕你是自讨苦吃，那个不想活的，上来送死。”
<br/>　　刘黑虎心中暗暗打鼓，不是说他患了重病么，此刻模样看上去不像呐。那一日，程大雷一剑斩了韩通的威风犹在眼前，面对这魔头，他不得不谨慎。
<br/>　　“程大雷，你以为天下英雄都怕了你不成。”
<br/>　　在大队伍中，一个声音突然响起。一人分开众人，出现在程大雷面前。对方白面无须，手中握一柄短刀，双眼像毒蛇的眼睛一般，盯着令人毛骨悚然。
<br/>　　刘黑虎眼前一亮，他差点把对方忘了。青州一地并不是没有高手，如自己身边这位就是，他使一柄直刀，乃是青州一地的独行大盗，姓林，名懂刀。
<br/>　　绿林之中，藏龙卧虎，有人投奔到官家，想着裂土封侯，有人杀人越货，活个潇潇洒洒。而林懂刀当然是后者，他想要程大雷的人头，以成全自己在江湖上的名气。
<br/>　　程大雷冲起招招手，做了个『来』的手势。林懂刀冷笑一声，身子扑了上去，他砍出的第一刀是有虚有实，先佯攻转移程大雷的注意力，自己再趁机砍出必杀的一刀。
<br/>　　锵啷。
<br/>　　匹夫剑在那一刹那出鞘，所有人都看到，林懂刀的身体突然僵在那里，如一头全力奔驰的野马，被人拽住了缰绳。随之，他喉咙出现一道血线，大家恍惚听到咔嚓的声音，他的尸体跌倒在地上。
<br/>　　这就是程大雷的剑，刘黑虎并不明白，所谓的青州大盗林懂刀，根本没有与程大雷动手相格的资格。
<br/>　　剑刃上仍然在滴血，程大雷的目光冷冷扫过，在挑选下一个对手，或者说，在等下一个目标主动冒出来。
<br/>　　无人敢与他的目光对视，刘黑虎感受到一种前所未有的恐惧。
<br/>　　他忽然暴怒，或许是因为这种畏惧而愤怒：“程大雷，你以为你是谁，任你天大本事，能把我们全都杀光么。”
<br/>　　程大雷没有回答，只是用冰冷的目光注视着他，刘黑虎直视着他，最终败下阵来，心脏剧烈的跳动。
<br/>　　世上有百人敌，也有万人敌，程大雷的本事说万人敌的确差些，但面前这几个山贼，当真还不值得他忌惮。
<br/>　　他口中突然一声呼哨，众人就觉得耳边轰隆隆震动，那头黑牛从牛棚冲出来，像一架推土机一般冲了过来。
<br/>　　凭刘黑虎几个货，当然是不可能拦住。那黑牛冲过人群，撞得刘黑虎手下山贼人仰马翻，到程大雷面前时，才突然停下。
<br/>　　程大雷翻身跃到牛背上，一扯小白狼将她拽了起来。
<br/>　　小白狼还没反应过来，已经被程大雷抱在牛背上，耳边听得程大雷哈哈笑了一声。
<br/>　　“走喽，走喽。”
<br/>　　刘黑虎立刻大叫道：“拦住他，莫让他走了。”
<br/>　　可又哪里拦得住，黑牛撒开蹄子，直接撞开重围，向山下跑去。
<br/>　　刘黑虎的人着急上马，在身后紧追不舍。追不上还好，一旦追上后，也就被程大雷一剑斩了。
<br/>　　如此杀威，吓得一帮山贼早已没有追击了意志。程大雷携着小白狼，一溜烟便跑没了影。
<br/>　　直等到东方露出鱼肚白，程大雷才让黑牛停下，小白狼晃晃肩膀，口中埋怨道：“你放开我。”
<br/>　　程大雷此时才悟到自己一直将小白狼揽在怀里，他哈哈笑了一阵，尴尬的将对方放开。
<br/>　　小白狼立刻跃了下来，气哼哼盯着程大雷：“你究竟想做什么？”
<br/>　　程大雷摸了摸鼻子，叹口气道：“白当家，不是我说你，你瞧瞧你这队伍是怎么带的，说反就让人反了。我看你也回不去了，还是和我一起回蛤蟆城吧。”
<br/>　　小白狼恨恨瞪了程大雷一眼，却也是没什么好办法。她倒也是洒脱之人，事情既然已到这一步，自然也没什么看不开的。
<br/>　　“走吧。”小白狼无可奈何道：“我算是被你哄了，现在不走也没法子了。”
<br/>　　程大雷做了个请的姿势，请小白狼上牛。
<br/>　　“哼，你给我下来。”
<br/>(本章完)
