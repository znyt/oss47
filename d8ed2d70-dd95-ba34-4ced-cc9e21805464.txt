　　“怎么杀？”一人朝着徐锋打手势。
<br/>　　徐锋皱了下眉后，打出手势：“避让。”
<br/>　　第一小队不打算与这批丧尸队发生冲突，而是采取尽可能的躲开，作为杀羽战队的队长徐锋更能从大局上看待问题，这批超级丧尸队显然与他们的侦查队差不多，是埋骨中心特地派出来查看的一批丧尸。
<br/>　　在不清楚远近丧尸数量，以及对方是否留有联系丧尸的时候，徐锋不能擅自行动。
<br/>　　第一小队的全体成员都匍匐在地上不动，静静的看着这批丧尸队从他们面前走过，一只两只一路数过去，出动的超级丧尸竟然有上百头。
<br/>　　这样的规模若是放在华夏任何一个城池或是基地，都足以造成致命的毁灭危机，但在这个丧尸遍野的埋骨之地，却稀疏平常。
<br/>　　这批超级丧尸队的智力水平都很高，各方面的能力也无从探测，为了保险起见徐锋所在的第一小队一直按耐不动，直到对方走出到不见身影。
<br/>　　“已经远离。”一名成员打出手势，依然趴在地上不发出任何声音。
<br/>　　徐锋同样如此，这时候周围已经没有丧尸的身影，远处也听不到什么古怪的声音。
<br/>　　缓慢的，徐锋起身张望了一下四周，又等了几秒钟后，这才出声：“继续前进。”
<br/>　　嗖嗖！
<br/>　　五十人迅速爬起来，用极其轻微的声音朝前移动。
<br/>　　但就在此时——
<br/>　　刷！
<br/>　　忽然一道诡异的声音从旁边的树叶上响起，众人一扭头就看到一只黑色的鸟迅速朝右边飞离，只比正常鸟类大一点点。
<br/>　　有人松了一口气：“原来是只鸟。”
<br/>　　徐锋却忽然大惊，道：“打下来！”
<br/>　　话落，嘭！
<br/>　　一道声音在远处响起，一名成员手中的最新型枪口处冒出一缕白烟，已经飞远的那只鸟‘啪’的一声落在地上，抽·搐了几下便不再动弹。
<br/>　　“抓过来。”徐锋有些紧张的下令。
<br/>　　一人快速跑过去执行命令，只是在他抓着那只鸟回来的时候，其神色满脸惊恐。
<br/>　　“队长，是丧尸鸟。”那人回来后说的第一句话就是这个。
<br/>　　五十人全体倒吸一口凉气，不禁全部朝那鸟看了过去。
<br/>　　纯黑色的羽毛很茂密，样子就像是乌鸦，除了身上弹孔处流出来的黑血，根本看不出任何丧尸的迹象。
<br/>　　徐锋上前一把将其眼皮翻起，不出意外的众人都看到了其内森白的瞳孔。
<br/>　　真的是一只丧尸鸟！
<br/>　　“怎么会这么小？”有人不解问道。
<br/>　　常态中的飞行丧尸但凡是能够飞上天空的，都已经进化出部分羽毛，而能成长到有羽毛的飞行丧尸大多体型巨大，像眼前这样娇小体型的鸟类一般都属于动物的范畴，怎么会是丧尸？
<br/>　　这时候的徐锋却忽然想到了一个可能，顿时面色发白：“二代。”
<br/>　　二代丧尸！
<br/>　　这是只雏鸟，一出生就带有智力的丧尸雏鸟！
<br/>　　能有生育能力的丧尸一定是超级丧尸，因为那是属于五阶丧尸之后成为超级丧尸的能力之一，只是这样的丧尸数量及其稀少，少到整个世界恐怕都没有几头。
<br/>　　但狼牙在安罗城遇到过丧尸蜘蛛及其还未浮出的蜘蛛卵，楚涵更是与超级丧尸鸟及其后代单打独斗过，只有两个案例但足以证明某些东西。
<br/>　　丧尸之中有可以生育的丧尸！
<br/>　　在华夏或许因为狼牙针对丧尸种群的凶猛，导致还未有二代丧尸横行，但这里是埋骨之地，一切都不一样。
<br/>　　“三次，都是动物丧尸。”徐锋盯着那只死去的二代丧尸呐呐自语。
<br/>　　“队长，这个尸体怎么处理？”有成员发着懵问道。
<br/>　　“太危险了，应该立即烧掉。”有人提议。
<br/>　　徐锋有些难以下决定，只能叹口气道：“烧掉吧，时机不对，不然真想送回狼牙给研究部。”
<br/>　　二代丧尸有什么样的能力还无人知晓，其出生到成长的过程更没有任何案例存在，这是一个全新却又必然会遇到的事情，不得不让徐锋重视。
<br/>　　但就是因为全新未知和谨慎，让他不得不以保守方式行动，因为这不仅是他一个人的事，还关系着整个狼牙战团十万人的性命。
<br/>　　能让楚涵忌惮成那样的二代丧尸，徐锋没有把握……
<br/>　　雏鸟丧尸的尸体很快被燃烧起来，只是与其他丧尸不一样的是，其火焰竟然冒出一缕幽蓝色，整个燃烧场景都尤其诡异，连气味都带着一股不同寻常，让人下意识的退后好几步。
<br/>　　徐锋紧抿嘴角，幽幽道：“幸亏还是只雏鸟，看那样子估计刚出生没多久。”
<br/>　　刚出生不久的丧尸雏鸟，又拥有飞行能力，在这一带都属于无解的存在，他们可以依靠逆天的隐匿能力避开超级丧尸队，也能穿梭在地面远离高空的那些飞行丧尸侦查，但对于这种体型较小的东西，狼牙战团只能看运气。
<br/>　　徐锋一直看着那只丧尸雏鸟燃烧成灰烬，这才下令道：“去联系何峰汇报这件事，其他人继续出发。”
<br/>　　顿时两个人离队原路返回，其他人则跟着徐锋继续小心的朝前摸索，对比之前他们行动的更加小心，警惕心更十足。
<br/>　　——————
<br/>　　在距离第一小队不远处的第五小队，这是又黑芒战队队长江凌若率领的五十人队伍，位置在埋骨之地中心的东侧，与其他小队一样，他们的任务也是摸索前进占领前方的那处圆圈定点。
<br/>　　与简单的埋炸药引爆不同，占领的意义非凡，需要与大部队前来接应的后续任务，更需要这次行动的绝对成功。
<br/>　　和带有冲锋模式的第一小队类似，第五小队的行动也是快很准，他们速度相当快，安静又鬼魅的穿梭在这片地带，躲过了一群又一群的丧尸。
<br/>　　运气更好的是，他们并未遇到过需要尤其注意的超级丧尸队，所以在所有进发的小队中，第五小队的速度为最先一列。
<br/>　　只是在这样的快速进度之中，忽的走在最前方的江凌若猛然一顿，紧接着毫无预兆的从后背摸出那把巨大银色镰刀。
<br/>　　刷！
<br/>　　忽然的一个横斩扫出一个大半圆，在空中挥舞出‘呼呼’的破空声，以及一道显然不太对的飙血声。
