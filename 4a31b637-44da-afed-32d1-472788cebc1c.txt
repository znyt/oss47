<br/>　　不过在不到20分钟的时候，他们的经济便是已经被拉倒了将近6000，这已经是一个相当大的数据了，相当于白队这边每个位置上都要领先对方半个大件。

<br/>　　“皇族这边真的是难打了啊，这样打下去就等于是在慢性死亡，现在泰坦就是连塔下都不敢待啊，以鳄鱼现在的状态，是完全可以越塔强杀这个泰坦的，这一把鳄鱼真的是起飞的节奏了。”看着场上的局势，米粒开口分析着。

<br/>　　看到这个画面，无数的观众都有一种说不出的怪异的感觉。
<br/>　　“怎么感觉白队像是在打人机呢，不过这个人机好像是困难程度的。”
<br/>　　“233333333333，你这样说话就不怕皇族战队的粉丝过来打死你么。”

<br/>　　“心疼皇族，管怎么说我们也是一个老牌强队啊，如今在世界范围内都用有着很多的粉丝，可是今天真的太凄惨了。”
<br/>　　直播间里的无数观众纷纷开口议论着。

<br/>　　可以说皇族的防御塔就是在他们一步一步避让之下，被叶帝他们缓缓的敲击开的。

<br/>　　而他所说的那一波自然是上路破掉对方的高地了，事实上当时就是连他也没有想到香蕉的推塔速度回那么的快，如若不是这样的话，恐怕那一波就真的要被皇族破塔、拿龙了。
<br/>　　“这一波是不是蕉爷carry了，无敌的鳄鱼散发出来的气息，使得敌人都胆寒了。”香蕉在一旁自恋的开口说着。
<br/>　　“是是是，蕉爷牛逼、蕉爷威武。”看着香蕉那臭屁的样子，几人倒是也没好意思去打击他，一旁的茄子则是在那里附和着他。
