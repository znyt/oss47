　　林峰作为十阶强者，肺活量自然已经没得说了，菲尔普斯直接被他甩几条街。
<br/>　　此时此刻。
<br/>　　孙悟空在平民区内疯狂杀戮。
<br/>　　一个棚户之中，有两个貌美如花的女学生她们此时此刻都吓得哭都哭不出来了。
<br/>　　“要死了，我不想死啊。”
<br/>　　“我也不想死，我二十一岁了，还没有男朋友。”
<br/>　　“呜呜呜呜。”
<br/>　　两个人眼神之中充满了绝望。
<br/>　　砰！
<br/>　　这个时候，屋顶上忽然掉下来了一群东西，直接把脆弱的棚子给砸穿了。
<br/>　　几个女大学生一瞧，顿时尖叫起来。
<br/>　　这些掉下来的，不是什么好东西！而是丧尸！这群丧尸可能是追人爬到更高的地方然后一脚踩空掉下来了。
<br/>　　不过他们根本摔不死，其中一个丧尸的胳膊都已经断了！在那里甩来甩去，但他没有任何疼痛，从地上爬起来后就朝两个女大学生冲去。
<br/>　　“啊啊啊啊啊啊啊！”
<br/>　　两个女学生此时此刻，只能靠在墙角里抱着等死。
<br/>　　砰！
<br/>　　下一刻。
<br/>　　一抹抹热乎乎的液体落在了她们脸上。
<br/>　　“你们安全了，俺老孙奉师傅之命来了。”
<br/>　　一道声音响起。
<br/>　　两个女大学生睁开眼，这才看到了一个穿着帅气战甲的人站在了自己面前。
<br/>　　“孙悟空吗？”
<br/>　　“不对你肯定是带着孙悟空的面具吧。”
<br/>　　两个女大学生喃喃道。
<br/>　　“你能不能让我看看你的真面目？我的救命恩人。”
<br/>　　两个女学生都有些害羞，这么高大帅气的男人，若是能够嫁给他，该多好啊。不对，就算能够看一眼他的真面目，那也心满意足了吧。
<br/>　　“真面目？奇了怪了，这里的人说话真是让俺老孙糊涂。还是俺师傅，说话好听。”
<br/>　　孙悟空摇了摇头，就要走。
<br/>　　“恩人，能否告知我你的名字！！”
<br/>　　两个女学生立即道。
<br/>　　“叫俺老孙……斗战胜佛！！！”
<br/>　　孙悟空说完就是离开了。
<br/>　　“斗战胜佛！好奇特的名字！！”
<br/>　　两个女学生喃喃道。
<br/>　　“不过，很好听！很帅！”
<br/>　　……
<br/>　　另外一边。
<br/>　　穿着海军大将的千年僵尸一蹦一跳在街道之中穿梭。
<br/>　　此时此刻，在一个供水大水箱上，躲着七个学生，他们每个人都拿着钢棍。
<br/>　　在水箱附近，有十几个丧尸在伸着双手要去抓他们，虽然他们够不着，但是被困在这里，迟早也是要死的。
<br/>　　“怎么啊，毛子，你特么不是注意最多的吗？”
<br/>　　“你想个办法啊。”
<br/>　　“毛的办法，现在这种情况，除非会飞！不然死！”
<br/>　　几个人都满脸的绝望。
<br/>　　“不好！有个四阶丧尸跑过来了！！”
<br/>　　其中一个女学生忽然看到了一个极其灵活的丧尸爬上了屋顶然后朝他这边冲来。
<br/>　　“完蛋了！就这么要死了吗！！”
<br/>　　砰！
<br/>　　也就在下一刻，他们每个人瞪大了眼睛，他们只看到了那个四阶丧尸忽然被一个跳起来的东西给撕掉了脑袋。
<br/>　　然后这个东西一蹦一跳的朝他这边冲来，一手一个，就好像撕纸片一样，全部给他们活生生的撕成碎片。
<br/>　　然后调过头，这个东西就跑了，露出了背部的，正义两个大字。
<br/>　　“这，这是海贼王里面的海军大将吗？”
<br/>　　每个人都呆住了，然后狂喜。
<br/>　　“有海军大将来救我们了！！”
<br/>　　他们自然知道，这是有个强者，穿着海军大将的衣服罢了。
<br/>　　但是他们就是愿意称之为，为海军大将。这是他们对于动漫之中一些元素的憧憬。
<br/>　　……
<br/>　　还有一处。
<br/>　　三个宅男蹲在一起。
<br/>　　他们满脸的绝望。
<br/>　　“阿强，你不是王者荣耀荣耀王者几百颗星吗？还是大主播，你特么想想办法啊。”
<br/>　　“有毛的办法，老子虽然是第一李白，但那只是游戏啊，老子又不会青莲剑歌！！”
<br/>　　“那怎么办，我们等死吗？”
<br/>　　三人都瑟瑟发抖。
<br/>　　“女王崇拜！”
<br/>　　也就这个时候，一道娇媚的声音传来，然后她们就看到一个身材火辣的少女跑了过来，一连串粉色爱心把那些包围他们的丧尸全部击毙，极其的利落。
<br/>　　“这，这不是妲己吗？”
<br/>　　三人之中的那个王者荣耀游戏主播呆住了。
<br/>　　妲己看了他一眼，抛了个媚眼，然后就扭着个毛茸茸的大尾巴走了。
<br/>　　“真的是妲己，王者荣耀来拯救我了。”
<br/>　　他喃喃道。
<br/>　　“白痴，这肯定是哪个美女穿的妲己的服装！”
<br/>　　“不管她是谁，妈妈，我恋爱了。”
<br/>　　一时之间。
<br/>　　斗战胜佛，海军大将，外加魅惑之狐，这三个无名强者，开始在人群之中传颂，他们救了数百个幸存者。
<br/>　　有人说，他们都是COSPLAY角色扮演社团的，因为一个带着孙悟空的面具，一个穿着海贼王海军大将的衣服，走路姿势也非同寻常，简直就好像吃了炫迈一样，还有个少女，屁股后面有个狐狸一样的大尾巴。
<br/>　　也有人说，他们是天兵天将，但不管怎么说，英雄二字，无愧。
<br/>　　特等区，此时此刻，这里已经有变异怪物在攻打基地，不过数量不多，而且也都是些变异狗变异猫之类的，实力不强，四阶五阶上下。浑身湿漉漉的，显然也是穿过河流过来的。
<br/>　　在城墙之上已经有重火力在横扫，人群之中，林峰看到了周华和陆勇，周华在帮陆勇巩固城防。
<br/>　　内围墙壁不过十米的高度，这个高度，一些弹跳力很强的变异怪物可以跳上去，比如说变异猫甚至还有干脆爬着墙上去的变异壁虎。这些变异壁虎一个个都有强大的防御力不说，舌头就好像青蛙舌头一样，一吐一吸就可以把一个机枪位给卷走。
<br/>　　林峰见此二话不说拔出斩灵剑就是冲了过去。
<br/>　　刷。
<br/>　　也就在这个时候忽然一个巨大黑影从水面之中跳出来然后就是朝林峰咬来。
<br/>　　林峰顿时一惊，他立即闪躲，这个黑影扑了个空落在地上居然是一条巨大的娃娃鱼，这个娃娃鱼已经变异了，浑身都长出了石头一样的东西。
<br/>　　这头变异娃娃鱼实力居然还有九阶。
<br/>　　林峰记得，好像说这个娃娃鱼本来就是在地球上存活时间很久的东西，这就说明这玩意他够猥琐，只有猥琐才可以活得长，这个娃娃鱼有九阶实力那也不奇怪了。
<br/>　　狂龙出海！
