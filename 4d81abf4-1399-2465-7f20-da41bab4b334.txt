        &lt;h3&gt;第三卷第三百二十九章  相互试探&lt;/h3&gt;
<br/>
<br/>        对于张函来说，他并没有过多的和陈子昂说话，因为陆宇霆和吴润竹都看着，他心里纵有千分情万分爱他也得默默的忍受着。
<br/>
<br/>        陆宇霆只是对陈子昂点点头，算是打了招呼，因为他的身边坐着吴润竹，好的是，陈子昂很快的离开了，张函和陆宇霆心里一松，但是有很难过，他们相视一眼，便又和身边的人说话。
<br/>
<br/>        他们都记得他们曾经说过：“陈子昂是他们心里一道明亮的光，美好和纯洁。”如今这道光更加明亮和耀眼，只是在看不见的洪流中不知道这道光能存在多久？
<br/>
<br/>        李长卿发挥了他助理的特效，他已经是第三次见陆宇霆夫妇了，他对张函很陌生，但这不妨碍他演戏的天赋，只是他演戏的时候让张函心里起了疑问。
<br/>
<br/>        江俞轩和陈子昂一走，整个桌子上就剩下寒伯安、李长卿、张函和陆宇霆夫妇。
<br/>
<br/>        寒伯安拿起一只烟，李长卿将烟拿下来嗔怪的说：“这么多人抽什么烟？”
<br/>
<br/>        “我又不抽，我就看看。”寒伯安尴尬的笑了一下。
<br/>
<br/>        “看了你就想抽，我还不知道你。”李长卿现在像是对着一个深爱的人在说话。
<br/>
<br/>        这语气让张函一愣，张函看了一眼寒伯安，寒伯安冲张函一笑：“我的这个助理比较黏人，管的比较宽，吃喝拉撒睡他都包了。”
<br/>
<br/>        “他们是真爱！”吴润竹笑着对张函解释。
<br/>
<br/>        张函正在喝着茶，差一点就喷出去了，死死的憋着，憋得面红耳赤。
<br/>
<br/>        “不好意思，我有点咳嗽，我出去一下。”张函实在是尴尬，站起来准备出去透透气。
<br/>
<br/>        “吴姐姐，你讨厌，你吓到他了。”李长卿转过身对身边的吴润竹笑着说。
<br/>
<br/>        “他少见多怪。”吴润竹也笑了，他们的接受能力比张函还是要强一些的。
<br/>
<br/>        “你不是要抽烟吗，去吧，去那边抽。”李长卿将烟递给寒伯安。
<br/>
<br/>        “让我抽烟了？”寒伯安拿起烟朝着张函的方向走去。
<br/>
<br/>        “长卿，最近好吗？”吴润竹大姐姐般的问候充满了关切。
<br/>
<br/>        “挺好的，就是工作上有些忙，他如今是越发懒惰了。”
<br/>
<br/>        “懒惰了？你这样说寒家总裁，不怕外界说闲话啊？”吴润竹微微一笑。
<br/>
<br/>        “在你这个好闺蜜面前有什么不好说的，他就是懒惰嘛，什么事情都让我替他操心，太依赖我了也不好。”
<br/>
<br/>        “长卿，说实话，外界都被你们给骗了。谁能知道帅气的寒家总裁喜欢的人是你？”陆宇霆看了一眼远去的寒伯安。
<br/>
<br/>        “我们没有骗人啊，只是大家关注点不一样而已。”李长卿淡淡的说。
<br/>
<br/>        “好奇你们是怎么在一起的。”吴润竹很想知道李长卿和寒伯安的具体情况。
<br/>
<br/>        “日久生情！”李长卿四个字将吴润竹打发了。
<br/>
<br/>        “长卿，寒家和苓家的关系怎么样？你认识苓希吗？”吴润竹摇动着手中的酒杯。
<br/>
<br/>        “关系挺好的啊，他们是姻亲嘛，苓希以前爱过伯安，我不太喜欢她。不过伯安说苓希只是他的妹妹。”李长卿拿起酒杯和吴润竹碰了一下。
<br/>
<br/>        “你这醋味儿我都闻到了。”陆宇霆笑着说。
<br/>
<br/>        “你们最近有没有想多方面发展？”吴润竹漫不经心的问。
<br/>
<br/>        “有啊，甸城谷家已经和我们签了合约，多一个朋友多一条路嘛。”李长卿看了一眼吴润竹。
<br/>
<br/>        “你们和谷家都已经开始合作了啊。”吴润竹假装吃惊的问。
<br/>
<br/>        “是啊，以后肯定会多方面发展的。吴姐姐，你是不是应该照顾照顾我们？”
<br/>
<br/>        “我不行啊，你得要让你姐夫罩着你才行，只有你姐夫才能罩着你呢。”
<br/>
<br/>        “那，姐姐，是不是伯安的朋友你都可以罩着？他那人毕竟仗义。”
<br/>
<br/>        “那是当然啊，你们的朋友就是我们的朋友嘛，是不是宇霆？”吴润竹看着陆宇霆，陆宇霆正张着嘴巴看着她。
<br/>
<br/>        “啊，啊，是的是的，你们都是我的朋友。”
<br/>
<br/>        .......
<br/>
<br/>        寒伯安随着张函到了走廊，他拿出一支烟递给张函，张函接过去点着吸了一口，呛得他有咳嗽起来了。
<br/>
<br/>        寒伯安等张函咳嗽结束后问：“你今天怎么和他们一起来了？”
<br/>
<br/>        “陆家当家的让我们一起来的，希望我们能和你们将关系处好。”张函苦涩的一笑。
<br/>
<br/>        “陆家是盯上陈子昂了？”
<br/>
<br/>        “我和陈子昂的关系陆家已经知道了，今天落家和吴家发了祝贺无疑将陈子昂推到了风口浪尖上。而陆家今天的目的很明显，也是来证明他们和陈子昂的关系不错而且可能比落家的关系还好。”
<br/>
<br/>        “你打算怎么办？”
<br/>
<br/>        “走一步看一步吧，就看陈子昂下一步怎么走。”
<br/>
<br/>        “陈子昂去甸城的行程都定好了，不会在北京呆很久的。”
<br/>
<br/>        “呵，甸城也是陆家的地盘！”张函长长的叹口气，现在他感觉到自己是多么的无助和孤独。
<br/>
<br/>        “陈子昂很聪明，她肯定会知道怎么做的，你不用担心。”寒伯安想起了安林曦的话。
<br/>
<br/>        “我知道她很聪明，我不想她去涉险，我已经欠了她很多。”张函看了一眼灯火辉煌的大厅，那里人来人往热闹非凡，可是他觉得心很冷很痛。
<br/>
<br/>        “你要去甸城吗？”
<br/>
<br/>        “是的，我和张倩楠都会过去，以工作的名义，现在在等通知。”张函已经申请了调离书。
<br/>
<br/>        “倩楠还好吧？”
<br/>
<br/>        “挺好，想通了很多事情，估计在走之前会找江俞轩的。”
<br/>
<br/>        “他们出问题了吗？”
<br/>
<br/>        “我们接近江俞轩是有目的的，倩楠为了帮我追回陈子昂才接近江俞轩的。”
<br/>
<br/>        “你说什么？你们？你们？”寒伯安简直是吃惊不小，这都多少年了？
<br/>
<br/>        “陈子昂和江俞轩都不知道，我知道我很失败，让你看笑话了。”张函将烟掐灭，扔进了垃圾桶。
<br/>
<br/>        “张总，有些事情强求不来，在你还有家人的情况，这样做确实不好，他们原本应该很好的。”寒伯安叹息了一声，如果江俞轩知道情况会怎么样？
<br/>
<br/>        “我承认我很自私，我想要更多，可是我却没有能力，最后还是以失败告终，希望以后的他们会很幸福吧。”
<br/>
<br/>        张函说的是他们。
<br/>
<br/>        “过一段时间我去甸城，也许就在甸城安家了呢，陈子昂有你们这样的一帮朋友真的很幸运，希望你们以后能多多帮助她，她没有什么背景，很简单！”
<br/>
<br/>        “怎么听你口气好像在告别似的，放心，你也是我们的好朋友！江俞轩和陈子昂不会责怪你们的。”寒伯安叹了口气，事情已经这样有必要去纠结过去吗？
<br/>
<br/>        这会江俞轩和陈子昂已经双双在飞机上了吧！
