　　第137章
<br/>　　程大雷在苏家的时候，李如便已暗中派人去通知城中官兵。
<br/>　　官兵来得晚了一些。
<br/>　　自然无法奢望这时代的官府有多高的办事效率，在李大头颇费了一般力气，又洒出不少钱财后，方才见到城内有权调兵之人。
<br/>　　对方一听说是程大雷，顿时如临大敌，本来是想假装不知道的，也就是在知道程大雷只有一个人后，才泛起些蠢蠢欲动的野心。
<br/>　　如果能抓到这为祸的程大雷，可是不小的功劳。
<br/>　　但这程大雷绝不可轻视！
<br/>　　此人杀人不眨眼，老弱妇孺都是他斧下之鬼，更可怖的是，传说中此人还会神仙法术，要抓到他可不是一件容易的事情。
<br/>　　于是，这位名叫乔野的武将调动了他可以调动的所有官兵，十个小队，足足上千人。
<br/>　　大军杀到苏家，程大雷已经走了。于是乔野又带人来到苏樱居住的小院，将院落外三层里三层围住后，才大着胆子走进院子。
<br/>　　前面有盾牌兵挡着，还有人拎着一桶黑狗血，随时准备泼到程大雷身上。
<br/>　　“程大雷，你不要负隅顽抗，你已经……”
<br/>　　“我投降。”程大雷高举双手。
<br/>　　“咦！”乔野有些奇怪，这传说中的程大雷，好像不是很有骨气。
<br/>　　“姓程的，你不要耍花样，你敢动一动，我的人就可以将你万箭穿心。”
<br/>　　“我绝不会乱动，让你的人也不要乱放箭，大家有话好好说。”
<br/>　　乔野紧紧盯着程大雷，专注在他的肩膀和膝盖。
<br/>　　“去人，把他抓了。”
<br/>　　几个官兵畏畏缩缩，不敢靠近。
<br/>　　“怕什么，他只有一个人，你们也害怕！”乔野很为手下的不争气而恼火。
<br/>　　“老大，他可是程大雷，听说这人本事很大的！”
<br/>　　“哼！”乔野挥手：“让盾牌兵顶上去，给他戴上手铐脚镣。”
<br/>　　结果比想象中顺利，过程却被乔野搞得十分麻烦。最后程大雷手上脚上带上了沉重的镣铐。
<br/>　　如此，乔野才稍稍放心，他冲程大雷道：“好了，你过来吧，不要耍什么花样！”
<br/>　　程大雷站在原地一动不动。
<br/>　　“你为什么不过来，想搞什么？”乔野握紧刀柄。
<br/>　　“废话，我走得动么。”程大雷很无语，这副镣铐足有上百斤重，生生将程大雷往地上坠。
<br/>　　乔野大手一挥：“抬走！”
<br/>　　乔野倒是没有为难苏樱和小蝶两个，苏樱站在门口一脸担忧的看着程大雷。
<br/>　　“不用担心。”程大雷被几个官兵扛在肩头，冲苏樱和小蝶挥手：“我还会回来的！”
<br/>　　……
<br/>　　徐神机今天和程大雷分开后，兴冲冲去往青楼，他的穿着相貌自然不是青楼会欢迎的那种人。
<br/>　　刚进门就被老鸨拦住了，用一双职业专有的势利眼打量着徐神机。
<br/>　　“你有银子么？”
<br/>　　“你这狗眼看人低的东西，老子拿出银子……”徐神机一拍腰间，脸色瞬间变了。
<br/>　　老鸨嘎嘎笑道：“来人，给我赶出去，在这里给我招摇撞骗，你来错地方了！”
<br/>　　徐神机没有心想事成的喝成花酒，还被人奚落一通，差点挨了青楼的大棒。
<br/>　　被冷风一吹，他就想到是程大雷干的。这混蛋，自己去找姑娘，却偷走我的钱，不让我去喝花酒！
<br/>　　什么时候，大当家做贼的手段也这么高超了。
<br/>　　不行，他坏自己的好事，自己也去坏他的好事。
<br/>　　徐神机带着一股怨气，一路上骂骂咧咧，招呼着程大雷的十八代祖宗。
<br/>　　走到半路，他突然想起自己好像不知道苏樱住在哪里。而落叶城徐神机其实并不熟，自己好像是……迷路了。
<br/>　　正在此时，忽听得周围有动静，大队官兵从路上经过。徐神机多官兵是有经验的，同时也有些奇怪，大晚上，出动这么多官兵做什么。
<br/>　　他一直躲在官兵外围，心里琢磨着到底是那路贼这么了得，值得落叶城如此大张旗鼓，不过，不管是谁，对方也够倒霉的。
<br/>　　正胡思乱想的时候，就见几个官兵扛着程大雷从街道上经过。本来徐神机还担心天黑视线不好，有可能认错。但程大雷被人扛着也不老实，口中不停高喊着。
<br/>　　“我还会回来的！”
<br/>　　如此，就再也不会认错了。
<br/>　　“不好，大当家被官兵抓了！”
<br/>　　徐神机一激灵，心中的怨气顿时烟消云散。
<br/>　　他急急忙忙来到湖底捞，深夜砸开酒楼大门。
<br/>　　湖底捞内几乎都是蛤蟆寨的人，即便原先不是的，如今也因近墨者黑被蛤蟆吸收了。
<br/>　　廖甲和廖乙见到徐神机后还有些兴奋：“军师，你……”
<br/>　　“都别废话了，大当家被抓了！”
<br/>　　“什么！”
<br/>　　晚上，酒楼内灯火通明。
<br/>　　厨师，伙计，伙计……此刻都全副武装，菜刀，勺子，擀面杖都被握在怀里。
<br/>　　“军师，你说怎么办吧。”
<br/>　　“大当家待我们不薄，绝不能让他落在官兵手中。”徐神机重重一捶桌子：“为今之计，只要劫牢一条路！”
<br/>　　“好！”
<br/>　　众人齐声答应。
<br/>　　唯有廖甲和廖乙觉得事情有些不对，单单抓程大雷就动用了上千人，看押他的人还能少么。凭这十几号人劫牢，不是帮对方增加牢房人口么。
<br/>　　都说这徐军师足智多谋，大当家做什么事都要带着他，大当家的口头禅是：我没了你不行！
<br/>　　可单听军师这条计策，不像足智多谋的样子啊。莫非这是计中计，军师这一条计策只是伪装，后面还有更高深的计谋。
<br/>　　那军师后面的计谋是什么呢……廖甲忽然有些头疼。他觉得像自己这样的普通人，还没有资格踏入军师的思考领域。
<br/>　　“军师，我们是不是该先通知山寨，就算是劫牢，凭我们这几个人也做不成事啊。”
<br/>　　徐神机深吸一口气，道：“好，我在此地坐镇，派人回山寨传消息，我们发动所有人马，打下落叶城，救出大当家！”
<br/>　　廖甲觉得头又有些疼了，军师的计策，自己还真是看不破啊。
<br/>　　廖乙来做这件事，他连夜从城墙缺口出城，马不停蹄返回蛤蟆寨，将程大雷被捕的消息传告诉众人。
<br/>　　随之，这件事在青牛山掀起轩然大波。
<br/>
