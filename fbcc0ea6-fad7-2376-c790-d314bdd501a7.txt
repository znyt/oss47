　　随着夜色一点点的加深，迎宾楼里的人也越来越多，终于在某一时刻，一声清脆悦耳的铃声响过之后，迎宾楼的大堂之中数十宫娥彩女提着各式彩灯、熏香、花环款款而来，而在她们的身后，便是身着华丽盛装的长乐与小武。
<br/>　　大唐顶级贵族的出场不管是排场还是奢华程度，都不是一般人可以想像的。
<br/>　　单就彩灯来讲，那可不是普通意义上的几个玻璃片加上小电珠，而是采用上好的五色透明水晶为灯罩，里面燃烧的是参杂了‘龙延香’的鲸鱼油。
<br/>　　鲸鱼这种东西在古代几乎是无人可以捕捉的大鱼，因为其体型过于巨大，再加上经常在深海活动，想要捕捉一头绝对不是易事。
<br/>　　但其油脂经过特殊手段提炼之后却可以产生一种特殊的味道，历来为各国贵族所追捧。
<br/>　　不过如果仅仅是这样，还不足以称其为奢华，毕竟那些番邦异族的国内也不是没有这种东西。
<br/>　　真正让人吃惊的其实还是鲸油燃烧后产生的另外一种异香，那是‘龙延香’，是从活动于深海的抹香鲸肠子上提取的一种分泌物，数量极为稀少，甚至可以用有价无市来形容。
<br/>　　但就算是这样稀少的东西，一出场竟然点燃了八盏，那特殊的香味瞬间便充斥了整个迎宾楼。
<br/>　　而且这还不算，其他那些熏香也不是普通的物品，取材自雪域高原的熏香闻上去几乎没有任何味道，如果不注意很有可能就会忽略过去。
<br/>　　这同样也是对那些异族的一种考验，如果他们闻不出来这种熏香的特殊，那么他们就不配称为贵族，根本不配参加如此高档的洽谈会。
<br/>　　与这些东西相比，那些让人闻所未闻，见所未见的花环同样让异族吃惊不已，尺余高的各式珊瑚，让人眼花缭乱。
<br/>　　当然，最最让人震惊的还是那八颗如鸡蛋大小的夜明珠，这东西被侍女托于掌心，看的那些异族迷醉不已，有些贪心的已经有了动手去抢的念头。
<br/>　　有钱，不，太有钱了！大唐的富有已经远远超过了异族番邦的想像，在他们看来大唐几乎就是金山的代名词，而大唐皇帝就是这座金山的主人。
<br/>　　至于说长乐与小武，她们两人同样也是不容忽视的存在，雍容华贵的盛装衬托下，这两个女人几乎就成了神女的代名词，无数番邦异族在两女面前低下他们曾经高贵的头颅。
<br/>　　而那些异族的贵族女子虽然心有不甘，但在‘钱’的压迫下，也不得不俯首！
<br/>　　想要装犊子不是摆摆架子就可以的，没有钱再怎么装、、逼也有被踩的那天。
<br/>　　虽然所有人都在一次又一次告诉自己，钱不是万能的，但在说这句话的同时，那些人也知道，没有钱是万万不能的！
<br/>　　比钱多，两大唐的这两位殿下手中不知掌握着多少财富；比势力，一个长公主，一个大唐皇妃，谁手下的势力都不小，都是那种跺跺脚天下都要抖三拌的人物。
<br/>　　比钱比势都比不过的情况下，那些异族的贵妇除了低头还能怎么样。
<br/>　　……
<br/>　　就在下面一群番邦贵族自惭形秽，大唐商人得意洋洋的时候，长乐在一片祥和的气氛中缓缓开口：“诸位，本宫大唐长公主长乐，仅代表大唐皇帝陛下，欢迎诸位的来访！”
<br/>　　“吾等拜谢大唐皇帝陛下，大唐长公主殿下，大唐皇妃殿下！”就像事先安排好的一样，一群番邦贵族和那些大唐贵族、商人纷纷躬身行礼，声音出奇的一致。
<br/>　　立于长乐身边的小武与她对视一眼，交换了一个眼神，长袖一挥：“免礼！”
<br/>　　“谢殿下！”出于对皇权的尊重，一群或是嫉妒或是羡慕的家伙们齐齐谢了一声，然后纷纷站直了身体。
<br/>　　……
<br/>　　这一次的洽谈主要是以商业为目的，但是由于长乐和小武的到来，所以基本上所有人都集中在一楼的大厅里面。
<br/>　　这样既可以表现尊重，又可以近距离与两位大唐最尊贵的殿下接触，这才是那些异族和商人们喜欢的。
<br/>　　而长乐和小武似乎也知道这些人的主要目的不是听她们两个长篇大论，所以在简单的致意之后便进入了主题：“这次吾皇陛下邀请诸位到大唐来，其实目的一共有两个，这第一嘛是想加深一下各国之间的了解，举办一场马球赛；而这第二嘛，则是要加深一下民间的商业合作。”
<br/>　　下面众人鸦鹊无声，全都在聚精会神的听着，在这些人看来球不球赛的不重要，重要的还是商业合作，只要能够达成合作，能够赚到钱，那球就是让大唐年年都得第一也没啥。
<br/>　　异族的规矩与大唐不一样，他们更加看重利益，什么名声之类的东西那都是扯蛋，只要有钱，我跟你叫爹都没有问题。
<br/>　　但如果哪天你没有钱了，或者没有利用价值了，好吧，你给我当孙子我都不要，这就是那些异族番邦的理念，与大唐的三纲五常绝对是不相融的。
<br/>　　所以随着长乐的声音落下，一个大食国的贵族便站了出来，先是行了一礼，然后才大声问道：“尊敬的殿下，不知您能不能说一下，商业合作的具体方面都有哪些？我们彼此间可以交易的物品又包括什么，有哪些是禁运品！”
<br/>　　长乐与小武再次对视一眼，心道这些家伙果然是够直接，竟然上来就直奔主题，果然与大唐人婉转的方式有些不同。
<br/>　　不过下面那家伙既然问了，她们不答总是不好，所以在对视一眼之后，小武站了起来，用一种极缓慢的语速说道：“这次的商业合作并没有什么限制，也不存在什么禁运，如果你们有心，就是武器也不是不可以卖！”
<br/>　　“嗡……”的一声，随着小武的话落，下面那些异族番邦瞬间就炸了，一个个脸上全都闪过抹兴奋的光芒，其中数人顾不得礼仪，大声向小武求证，是不是真的可以向大唐够买武器。
<br/>(本章完)
