　　万毒鼎落入到楚风眠的手中，还在不断的挣扎这。
<br/>　　但是随着楚风眠一股股的战龙威压，直接镇压下来，将这万毒鼎镇压的死死的。
<br/>　　如今就算是毒手魔君在如何心神催动，都没有任何的意义，
<br/>　　“万毒鼎，这倒是一件不错的灵器，可以融入到黑玄剑阵之中，不过地级的品级，还是有些太低了，既然如此，我就将你的血肉，融入其中，将这万毒鼎，彻底晋升！”
<br/>　　楚风眠打量了一眼这万毒鼎，突然出手，只见毒手魔君的身躯，被楚风眠一捏，直接破碎开来。
<br/>　　“啊！”
<br/>　　这一声惨叫，还未完，毒手魔君的身躯已经是破碎爆炸开。
<br/>　　无数血气，瞬间爆发出来，还有毒手魔君曾经得到的奇遇，他的空戒都是炸开，从中飞出了无数灵器，都是地级灵器，足足是有着十几件之多。
<br/>　　这十几件灵器，都是蕴含了远古的气息，显然都是远古灵器，许多件的威力，都不如万毒鼎，都是只差一步，便是足以晋升到天级灵器地步的灵器。
<br/>　　随着数万年的时间，这些灵器的力量也开始生化，只差一步，就足以晋升。
<br/>　　“好，看来这毒手魔君在归墟天中，倒是得到了不少的奇遇，这些灵器，都足以晋升了，我就利用毒手魔君的血肉，将这些灵器晋升，在全部融入到黑玄剑阵之中！”
<br/>　　天级灵器跟地级灵器之间，还是有着极大的差距中，现在一件地级灵器融入到黑玄剑阵中，已经没有太大的提升，但是融入一件天级灵器，得到的好处至少要相当于百件地级灵器。
<br/>　　如今这毒手魔君空戒中的灵器，许多都可以晋升，只是欠缺一些，楚风眠正好以毒手魔君的血肉，来弥补这欠缺。
<br/>　　一尊生死境巅峰强者的血肉，带来的力量，足以是让这些灵器晋升。
<br/>　　“起！”
<br/>　　这一团血雾，随着楚风眠的手心一动，直接燃烧起来，同时那一件件灵器，也随着楚风眠的心神控制，全部投入到了这火焰之中。
<br/>　　只见毒手魔君血气的力量，被这一件件灵器，全部吞噬，其中那最强的万毒鼎，便是第一个开始发生着变化，上面的禁制阵法，都突然破碎，随后焕然一新。
<br/>　　这些禁制阵法，并非是以人力设下的，而是来自于这些灵器自己的力量，凝聚而出的，跟楚风眠身上远古龙王身上的阵法类似，代表着灵器本身的力量。
<br/>　　“碰！”
<br/>　　阵法彻底凝聚完成的一刻，这万毒鼎也是焕然一新，直接化为了一件天级灵器，虽然只是品级最低的天级灵器，但是已经踏入了天级之列，整体已经是跟地级灵器，天差地别。
<br/>　　“好！多晋升几件，晋升的越多，融入到黑玄剑阵中得到的好处也就越多！”
<br/>　　楚风眠继续观察在火焰之中，又是一件长刀，晋升到了天级地步，随后又是一根长矛晋升。
<br/>　　这无数灵器中，一共三件灵器，晋升到了天级的地步，虽然是被楚风眠以力量强行催动晋升，跟真正的天级灵器相比，还要逊色许多，但是已经足以了。
<br/>　　在晋升完这三件灵器之后，火焰中毒手魔君的力量，已经是百不足以了，以这毒手魔君的一身血气，也就最多晋升这三件。
<br/>　　这还是这三件灵器，早已经是积累到了极限，只差一个机会的原因。
<br/>　　真正想要炼制出一件天级灵器，还是要比起这要困难无数倍。
<br/>　　“剑！”
<br/>　　楚风眠心神呼唤一身，黑玄剑阵便是猛然出现，随后将那些灵器，全部围在其中。
<br/>　　随着楚风眠灵力的催动，那些灵器，纷纷破碎，都化为了一道道的微粒，开始全部融入到了这黑玄剑阵之中。
<br/>　　在融入了先天玉如意之后，这黑玄剑阵中的十万道灵剑，都已经是达到了玄级灵器的地步。
<br/>　　如今这些灵器全部融入其中，足以是让黑玄剑阵的力量，再度晋升。
<br/>　　“轰！轰！轰！”
<br/>　　无数灵剑，在沾染吞噬了这些灵器的微粒之后，纷纷晋升，达到了地级灵器的地步。
<br/>　　晋升的响声，连连不断，最终这所有微粒被吞噬的一刻，这十万灵剑之中的一万把，都已经是晋升到了地级灵器的地步。
<br/>　　虽然没有全部都晋升到地级灵器的地步，但是这结果也是让楚风眠无比欣喜了。
<br/>　　十万把灵剑全部都是玄级灵器的时候，组成剑阵，都足以令夜天君这样的强者，无法从中脱困而出，虽然无法斩杀，但是却足以缠的死死的。
<br/>　　现在楚风眠估计，这黑玄剑阵要是动用全力，起码那毒手魔君，肯定是无法从中脱困，在生死境之中，没有任何人可以从中脱困。
<br/>　　就算是那些超越了生死境的大人物，碎命之境的强者，楚风眠都有自信，可以拖延一段时间。
<br/>　　要是其中的十万把灵剑，全部达到地级灵器的地步，只怕倒时候，就算是超越生死的强者，落入到这黑玄剑阵之中，都只有陨落一条路。
<br/>　　远古时代之一的庞然大物上剑宗，其中护山大阵，便是十万把天级灵器，组成的剑阵，在当时的九域，无人可破。
<br/>　　就算是九域中的最强者，都不敢说足以破解，足以看出其中的恐怖。
<br/>　　要是楚风眠可以将这十万把灵剑，全部晋升到天级灵器的地步，便是足以就在这九域横着走了，甚至是七大宗门，都足以轻易毁灭。
<br/>　　只是这个想法，也就在楚风眠的脑海之中，一闪而过。
<br/>　　一件天级灵器晋升，都是一件极为困难的事情，十万把天级灵剑，可是当年上剑宗以合宗门之力，历时数万年的积累，楚风眠只怕是永远达不到这一点。
<br/>　　不过就算是这十万把灵剑，全部晋升到地级灵器，起码都已经足以斩杀超越生死境的强者了。
<br/>　　现在这黑玄剑阵，已经是足以称得上是楚风眠的一大助力了，足以帮助楚风眠对付不少强者所用。
<br/>　　站在楚风眠身边的寒月漓，一直是亲眼目睹这毒手魔君，被楚风眠斩杀，甚至是就连一身血肉，都是被融入到了灵器之中去，也是不由的叹息了一声。
