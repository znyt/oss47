　　叶晨的逝去永远是叶家一抹化不去的沉痛，要知道叶晨可是被叶家最给予厚望的一个后代，如果不是为了领土战，恐怕如今的叶晨早就已经光芒四射了。
<br/>　　在知道叶帝这边没有什么事情后，当天夜里叶天正他们便是赶回了各自的地方，叶天英也只不过是冲冲过来看了一眼，然后就回军区去了。
<br/>　　最后蒋念他们一行人也都被叶帝赶回了基地，病房内只留下了陈献忠陪着他。
<br/>　　“忠哥帮我发条微博吧，我这胳膊现在还有些痛。”知道外面很多人还在担心着自己，叶帝对着一旁的陈献忠说道。
<br/>　　“发什么？”陈献忠接过手机问道。
<br/>　　“就说伤病复发了，现在已经没什么大碍了，大家不要担心，下周的比赛咱们见。”叶帝思索了一下后，开口说道。
<br/>　　片刻后夜谈的微博下方便是发布了一条这样的动态，看到叶帝没事的消息，无数的粉丝这一刻也都是松了一口气。
<br/>　　“还好大魔王没有事，这一晚上担心死我了。”
<br/>　　“赶紧补个回笼觉，早上还要去上班呢。”
<br/>　　“感觉像是在做梦，没想到大魔王居然是京都叶家的人，这个身份真的是太牛了。”
<br/>　　“还在颓废么，还在抱怨么？像大魔王这样的人都这么努力，我们还有什么可去埋怨的呢。”
<br/>　　“大魔王一定要养好身体啊，我还等着下周去看你的比赛呢。”
<br/>　　“我去，叶哥什么时候受的伤啊，以前怎么一点消息都没听过，这个真的是太突然了。”
<br/>　　“期待下周叶哥带秀，千万不要变成瓜皮中单啊。”
<br/>　　.......
<br/>　　无数的粉丝在微博下方调侃着，知道叶帝没事后，大家都恢复了以前的状态。
<br/>　　而叶帝这边在医院住了两天院，恢复的差不多后便是回到了基地。
<br/>　　回到基地的时候，大家还特意为了弄了一顿接风宴，好不丰盛。
<br/>　　“弄的我还真像一个病号是的。”看着满桌子的菜，叶帝笑着说道。
<br/>　　“恢复的怎么样了？”秦霄问道。
<br/>　　“已经没事了，之前就是这几周秀的太频繁了，这才导致伤病复发的，不过我现在这状态打蒋念照样没问题。”叶帝自信的说着。
<br/>　　“我去，别以为你是病号我就不敢和你solo啊，我这躺着也中枪！”正吃着东西的蒋念一脸无语。
<br/>　　在叶帝回到基地内后，基地的气氛再次变的热闹了起来，这两天叶帝不再，蒋念他们还真感觉缺了些什么是的。
<br/>　　而值得一提的是，周方和李玲两人因为拿到了这个大新闻，在周一的例会上被领导好一顿表扬，直接转正了，这倒是让两人越发的感激起了叶帝。
<br/>　　当然了转正过后，两人也就不用来荣耀基地蹲点了，能有一次那样的大新闻已经是得之不易了，想要在看到荣耀战队发生什么事情，那肯定是非常的难了。
<br/>　　而且知道叶帝是一号的儿子后，这些媒体对于这里也都是讳莫如深，不敢过深的探查叶帝的生活，生怕一不小心便被请进去喝茶。
<br/>　　时间一天一天的走着，一转眼便是到了周五这一天，这一天有着白队和HY战队的比赛。
<br/>　　虽然HY战队属于联赛中水平靠后的队伍，对于白队这种屹立在世界之巅的队伍造不成威胁，不过毕竟这是叶帝康复过后的第一战，大家都想看看现在叶帝到底是什么情况，所以在开赛前，这场比赛的热度便是已经被炒了起来。
<br/>　　“终于等到大魔王的比赛了，也不知道他现在恢复的怎么样了。”
<br/>　　“那天住院真的吓死我了，我还以为大魔王真的得了什么重病呢，还好没事。”
<br/>　　“期待今天大魔王的表演，希望他能够再次震惊四座。”
<br/>　　“要我看还不如让大魔王继续休息一段时间呢，让小艺在打一阵，这样的话大魔王也可以更好的修养一下。”
<br/>　　“想的太天真了，马上就要季后赛了，然后就是世界赛，时间非常的短，现在大魔王如果不打比赛，那他到时候恐怕都找不到手感，你们没看到现在那些顶尖战队的核心每场比赛都首发么，现在已经没有战队再去练兵了。”
<br/>　　........
<br/>　　当叶帝一行人身穿黑色队服走进比赛场的一刹那，现场的观众顿时爆发出了轰鸣般的欢呼声，渐渐的这些欢呼声全部汇聚成了一句话。
<br/>　　“大魔王加油！！！”
<br/>　　“大魔王加油！！！”

<br/>　　进入游戏的一刹那，现场再次响起了一阵整齐的加油声，反观HY那边的加油声与之相比，倒是要弱上了很多个档次。
