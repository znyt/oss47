　　一股冰冷刺骨的寒意，突然之间便是从楚风眠身上传来。
<br/>　　楚风眠转过头看去，在寒月漓的身上，她的肌肤上，都是凝聚成了一层冰晶，这一股股的寒意，就算是楚风眠，都不由的要动用灵力来抵挡。
<br/>　　一直是足足持续了一个时辰的时间，寒月漓身上的冰晶，已经是足足有着一寸之厚。
<br/>　　“轰！”
<br/>　　突然一声响声。
<br/>　　寒月漓身边的冰晶，纷纷破碎，她身上的寒意，逐渐的收敛起来。
<br/>　　但是这寒意，虽然收敛起来，但是如今却是足以感觉得到，现在的寒月漓，比起刚刚还说要强大不少。
<br/>　　生死境九重。
<br/>　　她的境界，再度突破，如今已经是达到了生死境九重的地步。
<br/>　　看来在这被追杀之中，寒月漓一次次的显现环生，倒是她的境界有着进展。
<br/>　　只是一直在伤势之下，看不出来，现在吞噬这冰系圣药，寒月漓身上的伤势一扫而空，她便是破而后立，一举踏入到了生死境九重的地步。
<br/>　　仅仅是生死境九重的境界，楚风眠便是感觉得到，寒月漓身上的气势，比起曾经的剑祖荒，鹿战神等人，都毫不逊色。
<br/>　　甚至隐约之间，都要超越他们二人。
<br/>　　“我的玄寒之体，如今更进一步，领悟出了更多的先天本源之力，现在那些人，还敢追杀我，我便是要将他们，全部斩杀！”
<br/>　　寒月漓嘴角露出几分冷笑，眼神冰冷。
<br/>　　她自从来到这天羽神域中，便是被无数人追杀，现在，也该到了她报仇的时候了。
<br/>　　“走，这冰裂谷中，还有着不少武者的存在，这些人一个个都想要杀了我，既然敢来追杀我，今日都得死！”
<br/>　　寒月漓冷哼一声，她的身躯猛然一动，在这冰裂谷中，席卷无数风雪，如同一道暴风雪一般，直接向着外面飞去。
<br/>　　楚风眠看着楚风眠的举动，也并没有说什么，他的身躯一动，便是催动起来战龙宝车，跟随在后。
<br/>　　冰裂谷中，不少武者，都在其中穿行。
<br/>　　有着独自一人的人物，也有着一些势力武者，在其中结伴而行。
<br/>　　一眼望去，至少也有着数十人在其中。
<br/>　　“师兄，你说那女子真的是躲在这冰裂谷中吗？”
<br/>　　两尊武者，正在其中一路寻觅这寒月漓的踪影，一边交谈着，其中一人环视四周，开口道。
<br/>　　“冰裂谷中，倒是没有半个人影啊。”
<br/>　　“哼，当然是真的，不然的话剑祖荒，鹿战神他们怎么会带人过来，一个先天神体的女子，简直比起任何宝藏都要有价值，我们碰到她，便是将她直接带走，回去慢慢享受，在将她的先天本源，全部吞噬！”
<br/>　　令一名武者，冷哼一声道。
<br/>　　他们二人，看起来并非是大宗门的弟子，所以才二人联手，不过能够来到归墟天，在这天羽神域中生存下去的武者，都绝对没有弱者。
<br/>　　这二人的实力，也不必寻常七大宗门的弟子要弱上多少。
<br/>　　他们二人，一路寻觅，突然就在前方，一道白色身影，突然降临，相距他们百米距离。
<br/>　　“这是？”
<br/>　　“玄寒之体？”
<br/>　　那二人看向这白色身影，突然之间，眼神瞪得斗大，随后大喜道。
<br/>　　“果然是上天助我，这女子拥有玄寒之体，应该正是我们要找此人！”
<br/>　　“据说她已经是身受重伤，快，先将她擒下再说！”
<br/>　　那二人同时脸色大喜，身形一动，猛然向着这白色身影，冲了过去。
<br/>　　但是就在他们二人，即将出手的一刻，突然之间，无尽冰雪，却是突然落下，将周围，直接化为了一团冰域。
<br/>　　这寒月漓猛然出剑，一剑之下，冰封千里，那两名武者，看到这一幕，猛然色变，二话没说，直接转身打算要逃。
<br/>　　看到寒月漓出手的一刻，他们便是一下子明白过来，寒月漓的实力，远在他们之上。
<br/>　　能够在这天羽神域之中生存下来，实力并非是最为关键的，最关键的一点，便是眼力。
<br/>　　遇到危险，第一反应便是直接要逃。
<br/>　　但是现在寒月漓，如何会跟这二人机会，她剑锋一斩，这两名武者，顿时全部四分五裂，被直接斩杀，不留任何机会。
<br/>　　这二人的空戒，都被寒月漓拿走，随后身形消失在了原地。
<br/>　　“这些精血，不能浪费啊。”
<br/>　　等到寒月漓离开，楚风眠在驾驶宝车过来，看着地上那二人的尸体，他心神一动，一股灵力将其中的精血，全部抽走，这才继续离开。
<br/>　　这样的场面，在这冰裂谷中，持续了一日之久。
<br/>　　其中的六十三位武者，无一例外，全部都死在了寒月漓的剑下，整个冰裂谷中，除了楚风眠跟寒月漓外，再也没有任何人。
<br/>　　这也没什么残忍不残忍的。
<br/>　　杀人者人恒杀之。
<br/>　　这些武者，一个个既然都想要杀了寒月漓，甚至是不少人都想要将寒月漓掳走，当做奴隶。
<br/>　　寒月漓若是落入到他们的手中，只怕是比起死，都要痛苦千倍万倍，现在死了，也正是给他们一个便宜了。
<br/>　　“我的实力，已经全部恢复了，杀了这些人，也足以是让我的剑术，再进一步，再好的剑术，也必须要在鲜血的磨练之下，才能大成。”
<br/>　　冰裂谷之中，寒月漓擦拭这剑锋上的鲜血，平静开口道。
<br/>　　她的心中的怒火，已经平息不少，脸色也逐渐的融化了，化为了平常的样子。
<br/>　　“据说这里，是一处名为天羽神域的地方，一般武者无法离开，不过我们现在有着玉符，想要什么时候离开，都可以，要不要现在离开？”
<br/>　　寒月漓试探性的询问道。
<br/>　　她来到这天羽神域，也是机缘巧合之中，不过眼下，这天羽神域中，实际上真正的宝物，却是不多。
<br/>　　天羽神域真正蕴含宝藏的地方，便是天际之上的云羽天宫。
<br/>　　但是这云羽天宫，高高在上，足有万米之遥。
<br/>　　现在楚风眠虽然有着战龙宝车，足以御空而行，但是百米，已经是极限，想要攀登万米之高的云羽天宫，根本没有任何可能。
<br/>　　但是，就此放弃，也不是楚风眠的性格。
<br/>　　他来到这天羽神域中，正是为了云羽天宫而来，现在总不能眼看着云羽天宫，半途而废。
<br/>　　“云羽天宫，绝对有着办法，可以进入其中。”
<br/>　　楚风眠抬起头，看向这云羽天宫。
<br/>　　就在楚风眠看向这云羽天宫的时刻，在云羽天宫中，一道彩虹，突然出现。
