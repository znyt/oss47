　　花费数分钟后，众人再次跑回到五色祭坛边，可惜，此刻距离一个小时的任务，才过去四十分钟，还有二十分钟，任务才结束。
<br/>　　“进不去，我们该怎么办？”有人试了试，发现自己并不能进去混沌神殿中，有些绝望道。
<br/>　　“怎么办？”众人绝望了，看着周边的同学一个个的死去，他们此刻心中只剩下无尽的恐惧，如今连最后一根救命稻草混沌神殿都进不去了，他们更加绝望了。
<br/>　　“任务时间还有二十分钟，只要我们待足二十分钟，我们便安全了，大家不要灰心。”有人劝慰道。
<br/>　　“是啊。”有人顿时想起了混沌神殿布置的任务，心中好似燃烧起了希望。
<br/>　　“噗嗤！”
<br/>　　一道乌光闪过，又有人倒地不起。
<br/>　　“啊……”
<br/>　　阵阵惊恐声响起，祭台外乱做一团，乌光闪现，那便代表着死神来临了。没有神物的几人，顿时扑向叶梵、周毅等人，抓住他们不敢放松，共用佛器。倒是夜邪身边，却没有人敢上去与他共用佛器，而他自己也不强求。
<br/>　　“嗷吼……”突然风暴之中，传来阵阵令人毛骨悚然的吼叫声，惊天动地，雷鸣般的沙尘暴都被它覆盖了。
<br/>　　“那是什么？”这一刻，不少人脸色发白，惊恐不已。
<br/>　　“难道这火星上还有什么别的生物？”有不少人如此猜测。
<br/>　　“吧唧！”
<br/>　　有声音从地上的尸体内响起，让不少人心中一紧。那声音像是咀嚼什么东西，听起来让人难受，毕竟那可是一颗人类头颅，竟然有未知生物在里面进食，在场所有人身上的寒毛都倒立起来了。
<br/>　　“咯咯咯……”
<br/>　　骨头磨碎的声音响起，现场气氛一阵压抑与紧张，谁也不敢说话，场面一下子寂静下来。
<br/>　　时间一点一滴地过去了，这是一种煎熬，进食的声音，咬碎骨骼的声音，未知生物的吼叫声，寒风的呼啸声，风沙的咆哮声……许多声音混在一起，像是地狱惊魂曲，折磨着人的心神。
<br/>　　很多女生都无助自己的嘴巴想哭，但又不敢哭出来，这情景，让不少人的内心近乎崩溃。
<br/>　　随后就有东西从头颅血洞之中钻了出来，所有人都屏住了呼吸，看着那未知生物的黑影，这东西便是带给他们无尽死亡的死神。
<br/>　　那是一种形似鳄鱼的奇异生物，从血洞之中钻了出来，身上不仅沾染着鲜血，还有白色的脑浆，让人望之头皮发麻。
<br/>　　“唧唧……”
<br/>　　就在众人惊恐不已时，只听一阵尖锐的叫声响起，那声音极为刺耳，非常难听，在众人的耳边响起，给人一种极为难受的感觉。
<br/>　　紧接着，众人便看到不远处一阵密密麻麻的黑点飞过，众人脸色顿时一白。
<br/>　　嗖嗖嗖……
<br/>　　只见一阵阵破空声响起，天空上无数黑点宛若蝗虫过境一般，铺天盖地的飞了过来，将众人围住。
<br/>　　众人顿时脸色苍白无比，每人脸上都露出无比恐惧的神色，甚至有人心理奔溃，一下子哭了起来。
<br/>　　砰砰砰……砰砰砰……
<br/>　　紧接着，无数砰砰声响起，却是这些恶心的凶鳄朝着众人攻击而来，乌光闪过，在众人身上顿时亮起一片片黄色的光芒，将众人笼罩，光辉闪现，宛如天神降临，凶鳄们并不能突破进来。
<br/>　　“啊……”
<br/>　　当然，也有人被袭击了，一声尖叫，仰头倒地，却是没了生命。
<br/>　　虽然有神物的人一时之间被保护了起来，但随着时间的流逝，那些凶鳄不断的进攻，神物所散发出来的光辉已经越来越暗淡了，眼看就快要支撑不了多久了。
<br/>　　凶鳄的每一次攻击，都会让众人心中的恐惧增加一分。
<br/>　　怎么办？
<br/>　　“你娘个三寸丁，你还没有耗子长呢，竟敢杀死我这么多同学，老子砸死你！”
<br/>　　庞博拎起大雷音寺铜匾，猛力向着周边的黑点拍去。雷音阵阵，光芒冲天，铜匾光华万丈，电芒汹涌，交织成一片天罗地网。
<br/>　　顿时就有几道凄厉的惨叫声响起，声音之大，让人震惊，宛若厉鬼在嘶吼，让人听了脊背发凉。
<br/>　　见此，其他人也挥动着手中的神物，向着周边的“黑点”攻去。谁也不希望自己坐以待毙，就算死，也不能如此憋屈的死去。虽然他们不会武功什么的，但因为有着神物的光芒，他们还是能够抵挡一二。
<br/>　　夜邪手持金刚杵，向着周边挥舞着，金刚杵力量极大，一挥便是光芒一大片，在加上夜邪本就是杀手，眼光最独到，出手时机也最精准，倒是杀了好多凶鳄。
<br/>　　叶梵手中青灯上前，将灯芯梦猛地一吹，顿时一道炙热的火光冲天，将周边的凶鳄活活烧死。
<br/>　　顿时，天地间除了“黑点”的破空声，沙尘的呼啸声之外，还有“黑点”凄厉的惨叫声，这片地区就宛如鬼蜮一般。
<br/>　　随着时间的流逝，众人手中的神物光芒越来越暗淡了，而凶鳄的进攻却是越来越凶猛了。
<br/>　　“噗！”
<br/>　　有人倒地，却是他手中的神物已经失去了作用，光芒消散，神性弥散，抵挡不住凶鳄的进攻。
<br/>　　“啊……”
<br/>　　有了第一个，紧接着便有了第二个，第三个……
<br/>　　顿时，所有人心中一沉，无限恐惧笼罩其身，难道今日真的要死在这里？
<br/>　　“咔擦！”
<br/>　　随即，一道咔擦声响起，众人抬眼望去，顿时所有人瞳孔一缩，脸上充满了绝望。
<br/>　　因为，那笼罩着此地的光罩，此刻已经开始出现了裂缝，光罩之外，依旧是那狂暴的足以席卷天地的沙尘暴在咆哮着。
<br/>　　“怎么办？我还不想死！”
<br/>　　“呜呜！我要回家，我不要仙缘了！呜呜……”
<br/>　　“我们都要死了吗，老天，怎么会这样！”
<br/>　　……
<br/>　　“咔擦……咔擦……”
<br/>　　紧接着，喀嚓声不断响起，那光罩之上，已经布满了裂痕，眼看就要碎裂开来，众人的命运将无法掌控。
<br/>　　“轰隆隆！”
<br/>　　顿时，光罩破碎，无尽风暴冲了进来，天地震动，乾坤倒转，日月失色，无尽风沙卷起千重风浪，汹涌澎湃，狂涌而来。
<br/>　　生死一刻，所有人都心中绝望了。
<br/>　　就在此时，久违的声音在他们耳边响起，那淡漠的声音，在此刻，就如同仙乐一般，让人难以忘怀：
<br/>　　“叮！时间到，任务结束，欢迎回到混沌神殿！”
<br/>　　紧接着，众人只感觉一道青色光芒自自己的手上亮起，将每人包裹起来，一阵眩晕，众人便再次回到那个熟悉的神殿之中。
<br/>　　“呜呜……得救了！”有人激动地大哭起来。
<br/>　　“赞美你！混沌主神！我们刚刚以为自己要死了。”有林佳美女感慨道。
<br/>　　“混蛋，老子刚刚与死神擦肩而过了。”庞博爆粗口道。
<br/>　　叶梵与夜邪对视了一眼，两人心有余悸，刚刚他们都以为自己快要死了，这种感觉，谁也不想经历第二次。
<br/>　　“嗡！”
<br/>　　铜棺一阵晃动，光芒闪烁，照耀八方，求仙之路，再次起航！
<br/>　　……
