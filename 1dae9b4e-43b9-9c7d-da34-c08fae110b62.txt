　　第433章
<br/>　　“大当家，你自己去？”
<br/>　　徐神机、黄三元和高飞虎都惊得张大嘴巴，难以置信的看着程大雷。
<br/>　　千军万马丛中，想要救一个人出来，谈何容易，但确实又是眼下最好的选择。假若蛤蟆城倾巢而动，胜算的确更大些。但是，路上消耗的时间太大，宋伯康提重兵去长安，用了一个月时间。蛤蟆城虽然行军速度快些，时间上也来不及，去了估计林少羽就被杀死了。
<br/>　　而程大雷曾在草原上上救过林婉儿，对于这方面的事情，并不是完全没有经验。
<br/>　　徐神机睁大的嘴巴合不拢，竖起拇指，表达了感慨。
<br/>　　“单枪匹马，与整个帝国的军队做对，大当家真勇士也。”
<br/>　　程大雷点点头，又摇摇头，紧跟着一句话，立刻让徐神机把嘴巴闭上了。
<br/>　　“不是单枪匹马啦，是你和我一起去。”
<br/>　　“啊！”徐神机发出一声尖叫。
<br/>　　程大雷揽住了他的肩膀，冲着高飞虎和黄三元面带微笑：“看把他高兴的。”
<br/>　　“大当家，不是，我是说……”徐神机有些语无伦次：“我差点忘了告诉你，我最近得了咳病，你看，咳咳咳，哎呦……”
<br/>　　程大雷用力捶打徐神机的后背，徐神机剧烈咳嗽，当真是说不出话来。
<br/>　　“瞧瞧，我说你们都瞧瞧，就算身患重病，徐军师也义不容辞，这是什么精神，这种精神都是你们要学习的。”程大雷冲高飞虎和黄三元道。
<br/>　　“明白，明白。”二人以钦佩的眼神看着徐神机。
<br/>　　“这个……那个，老朽年老体衰，怕是不能出什么力，大当家不若多带几个人，事情也稳妥些。”徐神机。
<br/>　　“嗳……这件事宜快不宜迟，人带的多了，路上反而麻烦，容易拖慢时间，就我和军师二人就好了。”
<br/>　　徐神机无语，既然你知道人多了麻烦，那为何还要带着我。虽然我足智多谋，神机妙算，但动手打仗，很明显我是个废物么。
<br/>　　程大雷仿佛知道徐神机在想什么，用力拍打着他的肩膀：“徐军师是山寨第一军师，只要带着他，我心里就踏实。他就是定海神针，是我迷航时的灯塔，你是电，你是光，你是唯一的神话。”
<br/>　　这件事和攻打凉州城相比，在程大雷心中更重要些。而且时间上也更紧急，毕竟多拖一天，林少羽就多一分危险。
<br/>　　哪怕是现在，林少羽是生是死，程大雷都不知道。
<br/>　　根据高飞虎带来的消息，林少羽应该活动在并州一地，至于具体在什么位置，大家都在找呢。
<br/>　　也只有程大雷到了并州，再具体打听林少羽的下落，希望自己到的时候，林少羽还活在世上，自己有机会将他救回来。
<br/>　　对于林少羽，程大雷心里总是有一份愧疚。总是觉得，如果不是自己当初对他说那么一番话，他也不会落得天下人人皆欲杀的地步。
<br/>　　更何况杀父之仇不共戴天，从这个角度讲，程大雷不能说林少羽做得有什么不对。至于起义军在长安城烧杀劫掠的事情，程大雷相信，也并不是林少羽的主意。
<br/>　　所以，程大雷的目的是明确的，单独把林少羽救出来，至于其他人，程大雷不会去管。目前他的善良，只针对自家兄弟，至于其他人，程大雷也不想去管。
<br/>　　确定计划后，程大雷立刻开始着手准备工作。他携带武器，准备干粮，随行只带两个人，一个是徐神机，另外一个是力大无穷的背斧少女银眸。
<br/>　　程大雷骑着墨玉乌月兽，身披大氅，背后一头吞月蛤蟆，银眸和徐神机各自骑着一匹马，银眸背后是一柄与她身材极其不相配的斧头。
<br/>　　程大雷精神亢奋，跃跃欲试，银眸面无表情，唯独徐神机是一副生无可恋。
<br/>　　和珅、刘悲等人一直将程大雷送到城门外。
<br/>　　“回去吧，回去吧，事情解决了我就会回来的。”程大雷挥着手。
<br/>　　“大当家，真不再考虑了么，不若多带几个人去，我们兄弟都不怕死的。”张肥。
<br/>　　“好啦，好啦，回去吧，我一个去就行。”
<br/>　　张肥几人都跃跃欲试，想要和程大雷一同去执行这次任务。当然，他们不是对林少羽有多深的感情，大概原因是在山寨游戏闲不住。
<br/>　　想到这里，程大雷还是要嘱咐和珅两句：“我走之后，你们不要轻举妄动，一切计划，只等我从长安回来，再做打算。”
<br/>　　程大雷心里还是有些放心不下，现在山寨内部人人思战，自己一走，没人去辖制他们，万一他们二杆子精神大爆发，去打凉州城怎么办。
<br/>　　程大雷要看着珅亲口答应，这才算松口气，和徐神机、银眸出了城，离开蛤蟆城。
<br/>　　目前得到的信息，林少羽活动在并州一地，具体位置不详。从蛤蟆城出发，通过凉州、京州，进入并州是最短的一条路。
<br/>　　但程大雷没打算走这一条路。自己毕竟是挂在通缉令上的人物，林少羽现在是人人都欲杀，但自己的情况，其实不比林少羽好上多少。要是知道自己露面了，或官或匪，不知多少人准备杀自己。
<br/>　　所以程大雷准备走草原这条路，路程会远些，但好在不会有太多麻烦，可以一路直抵并州。
<br/>　　三人出现在草原上，徐神机很显然是个累赘，他没有什么本事，必定会拖慢行路的时间。但是，就算这样，程大雷也一定要带着他，原因为何，目前确实只有程大雷自己知道。
<br/>　　冬初之交，草原上一片荒芜，路上会有饿肚子的黄狼、野豹，或者某些更危险的生物。当然，无论什么时候，最危险的存在永远是人。
<br/>　　帝国是兵荒马乱，到处都在打仗。而草原上的情况，其实也没有好到哪里。戎族王被程大雷杀死后，戎族各部落就处于各自为战的的状态。
<br/>　　中间有许多小部落兴起，也有许多强大的部落陨落。当初程大雷答应李婉儿，用戎族王的死，给帝国五年和平。
<br/>　　结果，帝国没有和平，却迎来了内乱。现在五年时间快到了，帝国目前最该担心的事情，怕不是林少羽。
<br/>　　而是随时有可能南下的戎族。
<br/>(本章完)
