        连星儿循声抬头，一瞬间，她怔住了。
<br/>
<br/>        她的目光直勾勾地撞上一双冷冽如冰霜般的眸子，而那黑色的瞳仁里恍若隐藏着不可言喻的金色光芒。
<br/>
<br/>        郎安就站在面前这棵大树的粗大的树干上，眉宇间透着不羁与冷然，令人不敢轻易将他忽视。
<br/>
<br/>        “那个人，是你，对不对？”连星儿质问道，顿了一下，她又改了口，“还有攻击我的那只狼！也是你！”
<br/>
<br/>        郎安居高临下地俯视着连星儿，看着她微蹙的眉心，良久后，他的唇边忽然漾起一抹难得可见的邪肆冷笑。
<br/>
<br/>        “嗖”地一声。
<br/>
<br/>        周遭的落叶被这阵风卷起后又飘然落下。
<br/>
<br/>        连星儿下意识地抬起手来挡在眼睛前，遮挡被这阵风吹来的沙尘。
<br/>
<br/>        当她放下手来时，郎安已经伫立在她的跟前。
<br/>
<br/>        “连大小姐身上的血腥味很重，所以，脾气也很大吗？”
<br/>
<br/>        冷静到毫无波澜的声音，传入到连星儿的耳中。
<br/>
<br/>        连星儿再次对上郎安的眼睛，恍若又看到了那双向她扑过来的金色兽眸。
<br/>
<br/>        就在郎安上前一步时，连星儿下意识地后退了一步。
<br/>
<br/>        潜意识里，她对他还是有些心理上的畏惧。
<br/>
<br/>        “还有小时候，追逐我的那只流浪狗，也是你！”连星儿双手握拳，严肃地质问。
<br/>
<br/>        郎安此刻面无表情，比面瘫还要面瘫，波澜不惊地回答道：“连大小姐您这是拐着弯骂我吗？又是狼又是狗的。不知道，我跟连大小姐有什么仇什么怨？”
<br/>
<br/>        “你——”连星儿欲言又止。
<br/>
<br/>        郎安慢条斯理地说道：“连大小姐是要去找连爷爷吗？他今天不在实验室，裴奶奶也不在。只不过，还有其他几个工作人员在，你是要去见他们吗？”
<br/>
<br/>        “算了。”连星儿抿了抿唇，低头从手提包里翻出一包曲奇饼，递向郎安，“不管你承不承认，起初都是你救了我。这个就当是谢礼。”
<br/>
<br/>        郎安垂眸看着连星儿递过来的曲奇饼干，没伸手。
<br/>
<br/>        连星儿又说道：“这是我妈咪今早做给我吃的，我还没来得及吃。你不要，那我自己……”
<br/>
<br/>        “谢谢。”待她的话还未说完，郎安便从她手中夺走了那包曲奇饼干，并淡漠地道了个谢。
<br/>
<br/>        连星儿收了手，看了郎安一眼，抿了抿唇，索性转身离去。
<br/>
<br/>        他伤过她，但同时也救过她，这事，算不算扯平呢？
<br/>
<br/>        连星儿离开后，郎安重新爬到了大树上，一边眺望着远方那辆连星儿所乘坐的私家车渐驶渐远，一边打开包装袋，吃起了里面的曲奇饼干。
<br/>
<br/>        他刚把曲奇饼干全部吃完，连中凯和裴冉便回来了。
<br/>
<br/>        郎安立即从大树上跳了下来，直接狂奔回实验室。
<br/>
<br/>        没过多久后……
<br/>
<br/>        “为什么不把他给杀了？”连中凯气急败坏地质问，只差没拿那条铁鞭，鞭打郎安。
<br/>
<br/>        郎安此刻赤脚蹲在墙角，一言不发。
<br/>
<br/>        裴冉接到一通电话后，看向连中凯，替郎安解释道：“32号当时劫持的人是星儿，是郎安救了星儿。我想，郎安不杀32号，只是为了不让星儿看到血腥的场面。此外，32号已经被我们的人在暗中注射了神经毒素，他已经疯了，警察从他嘴里问不出什么的。对我们的事情，造成不了什么威胁。”
<br/>
<br/>        “星儿在场？！”连中凯惊愕地看向裴冉。
<br/>
<br/>        本来，他们是打算让郎安杀了32号，然后伪装成是他自己疯了才自杀的现场，谁知道，会遇上连星儿。
<br/>
<br/>        裴冉点了下头，接着说道：“据情报组的人说，星儿在警局里，没有提到郎安的事情。”
<br/>
<br/>        “还好是那丫头，不然，我们都得完蛋了！”连中凯顿时松了口气。
<br/>
<br/>        裴冉走到郎安的面前，蹲下身去，拍了拍郎安的肩膀，安慰道：“郎安，谢谢你照顾了星儿。”
<br/>
<br/>        郎安听着这话，依旧面无表情。
<br/>
<br/>        连星儿回到家后，见童筱颖一个人在厨房里准备午餐，于是走上前去，旁敲侧击地问道：“妈咪，外公他到底是好人还是坏人啊？”
<br/>
<br/>        “你这丫头，回来也不提前跟我说一声。”童筱颖被连星儿这突如其来的话给吓了一跳。
<br/>
<br/>        连星儿咧嘴一笑：“谁叫妈咪做个饭都这么认真呢？”
<br/>
<br/>        “你没事突然问这个做什么？”童筱颖一边煮着锅里的咖喱汤，一边回到了正题上问道。
<br/>
<br/>        连星儿索性胡诌了一个理由：“因为外公总是神神秘秘的啊！他表面上肆无忌惮地让我进出他的实验室，但心里其实有鬼的吧！”
<br/>
<br/>        “你外公的那个实验室，你能不去，就不要去。”童筱颖忽然间板着一张脸，冷冷地说道。
