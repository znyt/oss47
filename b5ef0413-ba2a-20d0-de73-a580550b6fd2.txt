        只为验证他连华生的猜测，同时也是为了验证现在待在他傅夜沉身边的童筱颖到底是不是他深爱的那个童筱颖。
<br/>
<br/>        连华生打开报告后，看到结果时，不禁瞪大了眼睛，勃然大怒地质问：“你有没有搞错？我要你做的DNA对比，你怎么给我搞了个DNA亲子鉴定！童筱颖才多大？居然能有一个跟她年龄一样大的女儿？难不成，她在娘胎里的时候就怀孕了？你是在侮辱我的智商吗？”
<br/>
<br/>        “我也觉得很奇怪，但是你看过这份DNA鉴定结果，结果，你就会更加难以置信了！”女助手接着向连华生递上另一份DNA检测报告。
<br/>
<br/>        连华生接过后，连忙翻看起来，顿时忍不住谩骂起来：“我靠！裴金澈跟她是父女关系？是不是我们的检测仪器出错了啊？”
<br/>
<br/>        “检测仪器没错，如果排除这三份血样是父、母、女的关系，那么还有一种可能就是，在某种特定的原因下发生了体内基因突变。”女助手给出了自己的观点。
<br/>
<br/>        而正是这个观点提醒了连华生。
<br/>
<br/>        连华生不禁打个响指，认同道：“阿澈给童筱颖输过血！阿澈说过，是他的血辅助童筱颖的自身修复，让童筱颖复活的。所以，童筱颖的体内有阿澈的DNA，他们两的基因融合重组，然后才出现了类似于父母与子女的基因那样。我去，我老爸真是个疯子！怪不得夏阳枭不肯认童筱颖这个女儿。阿澈和童筱颖，在他们眼里，就等于是怪物。”
<br/>
<br/>        “连少，我老师说，希望你能提供那个女人的血液，因为很有医学研究价值。”女助手试探性地问。
<br/>
<br/>        这个女助手的老师，现在是连华生手下最得力的生化学家，是他花高薪聘进来的，曾经获得过很多国际生化基因类的金奖，已经有六十岁多了，现在却仍旧对基因工程非常感兴趣，还愿意把自己的一生都奉献在这个上面。
<br/>
<br/>        “你跟老师说，要老师别打她的主意。免得引来杀身之祸！这事可不是威胁的话，那女人不简单。”连华生淡定地回绝了女助手的询问并好心警告。
<br/>
<br/>        女助手很识趣地点了点头：“好的。”
<br/>
<br/>        一个星期后，连华生把自己的研究报告发了一份电子档给傅夜沉看。
<br/>
<br/>        童筱颖给傅夜沉的那些“解药”，傅夜沉拿了两盒给连华生去做研究。
<br/>
<br/>        连华生悄悄地偷了自己父亲的科研成果报告和观察日记，再结合自己的研究，得出来的结果是……
<br/>
<br/>        “解药”于本体来说，只要是本体的死亡时间在四十八小时之内，“解药”便能让本体基因突变，促使心脏复苏。
<br/>
<br/>        不过本体的大脑会进入休眠状态，类似于植物人，直到“解药”将本体各个细胞、基因、组织、器官等修复到完整如初，并治好本体自身所有疾病，本体才会苏醒过来。
<br/>
<br/>        但本体自身已丢失的部分，将无法重新长出，只能进行伤口修复，例如童筱颖移植给傅昼景的那颗肾，是无法再长出来。
<br/>
<br/>        同时，这些“解药”于他人而言确实能治百病，但“解药”也有很多不明确的副作用，其中之一最大的副作用就是。
<br/>
<br/>        “解药”含有一种未知的神经毒素，会控制使用者的大脑，使其主人格会进入休眠状态，类似于心理疾病中产生第二人格，第二人格听命于第一眼见到的人，又类似于动物界刚出生的雏类的“印随行为”。
<br/>
<br/>        当“印随行为”消失后，“解药”所产生的第二人格会随着时间的推移完全吞噬主人格，成为新个体。
<br/>
<br/>        新个体将会出现何种副作用的发展，暂时无法知晓。
<br/>
<br/>        此解药只能通过母体血液提炼，无法自主培养，所以，没有童筱颖的血液作为来源主体，他们压根就不可能批量生产。
<br/>
<br/>        远在美国哥伦比亚郊区的某幢别墅里的傅夜沉，在刚把报告看完后，就接到了连华生打来的一通电话。
<br/>
<br/>        “你报告上的意思是，筱颖还是筱颖，只不过，筱颖的主人格进入了休眠状态，现在出现在我面前的是筱颖的第二人格？”傅夜沉难以置信地反问。
